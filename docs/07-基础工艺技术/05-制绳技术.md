- ✓ 立即有用
- ✓ 成功率100%

## 一、纤维材料

### 1.1 植物纤维（推荐）

#### 🌾 荨麻/大麻/亚麻

**特点:**
- 纤维最强
- 耐腐蚀
- 制绳首选

**识别:**
- 麻类植物
- 茎秆笔直
- 茎皮有韧性

**采集:**
- 秋季茎秆枯黄时
- 割下茎秆
- 剥取茎皮

**处理:**
```
1. 剥皮: 剥下外层茎皮
2. 浸泡: 清水浸泡7-14天（沤麻）
3. 去胶: 冲洗，去除胶质
4. 晒干: 阳光下晒至半干
5. 打软: 反复揉搓，分离纤维
6. 梳理: 梳成细纤维束
```

#### 🌾 芦苇/香蒲

**特点:**
- 随处可见（水边）
- 纤维较粗
- 适合粗绳

**处理:**
- 叶片晒干
- 撕成条状
- 直接搓绳

#### 🌳 树皮纤维

**适用树种:**
- 椴树（最佳）
- 柳树
- 桑树
- 榆树

**处理:**
```
1. 春季采集（水分多，易剥）
2. 剥取内层韧皮
3. 浸泡7天
4. 刮去外层
5. 撕成条状
6. 晒干备用
```

#### 🥥 其他纤维

- 椰子壳纤维: 极耐腐蚀（船用绳）
- 棕榈纤维: 粗糙但强韧
- 草类: 应急使用，强度低

### 1.2 动物纤维

#### 羊毛/兽毛

- 搓成细绳
- 强度中等
- 不耐水

#### 皮条

- 生皮切成细条
- 非常强韧
- 适合重物捆绑

#### 动物筋腱

- 强度极高
- 制作弓弦
- 需要鞣制

## 二、基本搓绳法

### 2.1 两股绳（最简单）

**原理:** 两束纤维反向搓，自动缠绕

**步骤:**

```
1. 准备:
   - 纤维束2根
   - 每根长度60-100cm
   - 粗细均匀

2. 固定一端:
   - 打结或系在固定物上
   - 或用脚踩住

3. 搓绳:
   ┌─────────────────┐
   │   手势示意:      │
   │                  │
   │  左手: ↶ 顺时针  │
   │  (Z搓)           │
   │                  │
   │  右手: ↷ 逆时针  │
   │  (S搓)           │
   │                  │
   │  两手同时向下移动│
   │  纤维自动缠绕    │
   └─────────────────┘

具体操作:
- 左手拿一束，顺时针搓紧
- 右手拿一束，逆时针搓紧
- 松开，两束自动反向缠绕
- 重复至末端

4. 接续:
   - 末端20cm处加入新纤维
   - 重叠搓入
   - 确保连接牢固

5. 收尾:
   - 末端打结
   - 或用细线缠绕固定
```

**时间:** 1米绳索约需10-15分钟

**强度:** 可承重5-10公斤

### 2.2 三股绳（标准绳）

**原理:** 三股编织，强度提升3倍

**步骤:**

```
1. 准备:
   - 纤维束3根
   - 一端打结固定

2. 编织法:

   起始:    A   B   C
           │   │   │

   步骤1:  A移到B和C之间
          B    A    C
          │    │    │

   步骤2:  C移到A和B之间
          B    C    A
          │    │    │

   步骤3:  B移到C和A之间
          C    B    A
          │    │    │

   重复: B→C之间, A→B之间, C→A之间...
   规律: 永远是最左边的跨到中间

3. 技巧:
   - 编织时拉紧
   - 保持均匀力度
   - 边编边整理
```

**时间:** 1米绳索约需20-30分钟

**强度:** 可承重15-30公斤

### 2.3 加粗绳

**方法:** 多股绳再编

```
步骤:
1. 先搓3根两股绳
2. 将3根两股绳当作3股
3. 再次编织成三股绳

结果: 粗绳，强度50-100公斤
```

## 三、实用绳结

### 3.1 基础绳结

#### 平结（方结）

**用途:** 连接两根同粗细的绳

```
[示意图]
绳A: ─────┐  ┌─────
           │  │
绳B: ──┐   │  │   ┌──
       └───┼──┼───┘
           └──┘

步骤:
1. A压在B上，绕过B从下穿出
2. A再压在B上，从另一侧穿出
3. 拉紧

口诀: 左压右，右压左
```

#### 称人结

**用途:** 救人、吊物、固定腰间

```
[示意图]
    ○  ← 活结环（套人）
   ╱│╲
  ╱ │ ╲
 ╱  │  ╲
───┘  └───

特点: 越拉越紧，但不会勒死
```

#### 双套结

**用途:** 固定在柱子、树上

```
[示意图]
   ╭──────╮
   │      │
───┴──┬───┴───
      │
     树干

绕两圈，打结，极牢固
```

### 3.2 高级绳结

#### 双半结

**用途:** 拉紧帐篷、捆绑物品

#### 8字结

**用途:** 防止绳子滑脱

#### 渔人结

**用途:** 连接钓鱼线、细绳

## 四、编织技术

### 4.1 渔网编织

**基本网结:**

```
[网格示意]
┌─┬─┬─┐
│ │ │ │
├─┼─┼─┤
│ │ │ │
└─┴─┴─┘

每个交叉点打一个结
```

**步骤:**

```
1. 第一行:
   - 绳子穿过横杆
   - 每隔5-10cm打一个圈

2. 第二行:
   - 新绳穿过上一行的圈
   - 打结形成网格

3. 重复:
   - 一行一行往下编
   - 保持网眼大小一致

4. 收尾:
   - 底部穿杆或绳加重
```

**网眼大小:**
- 捕大鱼: 5-10cm
- 捕小鱼: 2-3cm
- 捞虾: 1cm

**时间:** 1平方米渔网需10-20小时

### 4.2 篮筐编织

**材料:** 柳条、藤条、竹篾

**基础编法:**

```
[编织示意]
  ↓ ↑ ↓ ↑  ← 经线（竖）
→─┼─┼─┼─← 纬线（横）
  ↑ ↓ ↑ ↓

1. 经线: 放射状摆放（篮底）
2. 纬线: 穿梭编织
   - 上、下、上、下...
3. 边编边整形
```

**类型:**
- 圆篮: 底部呈圆形
- 方篮: 底部呈方形
- 背篓: 背部平整，前部深

**时间:** 小篮1-2小时，大篓3-6小时

### 4.3 草席/草帘

**材料:** 芦苇、稻草、香蒲

**方法:**

```
1. 准备:
   - 稻草晒干
   - 整理成束

2. 编织:
   - 横向排列稻草
   - 用细绳纵向穿梭
   - 每10cm打结固定

3. 压实:
   - 用木板压紧
   - 更加耐用
```

**用途:**
- 睡垫
- 遮阳
- 包装材料

## 五、绳索保养

### 5.1 防腐处理

**天然纤维绳易腐烂!**

**防腐方法:**

1. **油脂涂抹:**
   - 动物油、植物油
   - 延长寿命2-3倍

2. **松脂浸泡:**
   - 熔化松脂
   - 浸泡绳索
   - 防水、防腐

3. **烟熏:**
   - 悬挂在烟火上方
   - 烟熏3-7天
   - 杀菌、防虫

### 5.2 储存

**正确储存:**
- 阴凉、干燥处
- 卷成圈挂起
- 避免阳光直射
- 定期检查

**错误储存:**
- ✗ 潮湿环境（发霉）
- ✗ 长期阳光下（老化）
- ✗ 打结存放（难解开）

## 六、强度测试

### 6.1 测试方法

**负重测试:**

```
1. 将绳索固定
2. 悬挂重物
3. 逐渐增加重量
4. 记录断裂重量

安全系数 = 断裂重量 / 实际使用重量
建议: 安全系数 > 5
```

**目测检查:**
- 有无断股
- 有无霉变
- 有无过度磨损

### 6.2 强度等级

| 绳索类型 | 直径 | 承重 | 用途 |
|----------|------|------|------|
| 细绳 | 3-5mm | 5-10kg | 捆绑轻物、钓鱼线 |
| 中绳 | 8-12mm | 20-50kg | 帐篷、晾晒 |
| 粗绳 | 15-20mm | 50-150kg | 拖拽、吊物 |
| 缆绳 | >25mm | 200kg+ | 船用、建筑 |

## 八、常见问题

### Q1: 没有工具能搓绳吗?

**A: 完全可以！**
- 双手即可
- 或用大腿搓（坐姿）
- 更快更省力

### Q2: 绳子总是散开怎么办?

**A: 技巧问题**
- 搓得不够紧
- 方向错误（两股都同向了）
- 纤维太短（<30cm）

**解决:**
- 用力搓紧每一股
- 确保反向搓
- 使用长纤维

### Q3: 多粗的绳最实用?

**A: 8-12mm最常用**
- 适合大部分用途
- 不太重
- 强度足够

### Q4: 绳子断了能修吗?

**A: 可以**
- 两端各解开10cm
- 纤维交叉重叠
- 重新搓合
- 强度略降

## 九、实战应用

### 9.1 搭建庇护所

**用绳量:**
- 小帐篷: 20-30米
- 简易棚屋: 50-100米

**绑法:**
- 柱子连接: 十字绑法
- 屋顶固定: 双套结
- 拉绳: 称人结+调节扣

### 9.2 捕鱼陷阱

**简易鱼篓:**
- 编织锥形篓
- 入口内折（鱼进不出）
- 放入水中

**渔网:**
- 需要细绳
- 网眼根据鱼大小

### 9.3 搬运重物

**拖拽:**
- 粗绳套住物品
- 肩背拖拽
- 省力

**起重:**
- 滑轮组（如有）
- 绳索倍力系统

## 十、进阶技巧

### 10.1 染色绳索

**天然染料:**
- 红色: 茜草根
- 黄色: 姜黄、洋葱皮
- 蓝色: 蓝靛
- 绿色: 黄+蓝混合
- 黑色: 橡碗、核桃皮

**方法:**
- 染料煮沸
- 绳索浸泡30分钟
- 晾干

**用途:**
- 区分不同用途的绳
- 美观

### 10.2 防水绳索

**方法:**
- 绳索浸泡融化的蜂蜡
- 或涂抹松脂
- 晾干

**效果:**
- 防水
- 增加强度
- 延长寿命
