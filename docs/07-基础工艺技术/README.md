# 基础工艺技术

> **重建文明的基石 - 传统手工技艺全集**

## 📖 引言

**末日后，工业化生产停止，一切需要从头开始！**

本章节涵盖人类文明最基础、最重要的手工技艺：
- 👔 **纺织** - 从纤维到衣物
- 🏺 **制陶** - 从泥土到器皿
- 🎒 **制革** - 从兽皮到皮具
- 🪵 **木工** - 从树木到建筑
- 🪢 **制绳** - 从纤维到绳索

**这些技术决定了:**
- 能否有衣穿（保暖、防护）
- 能否储存食物和水（容器）
- 能否建造庇护所（建筑）
- 能否制作工具（皮革、绳索）

### [01-纺织技术](./01-纺织技术.md)

从植物纤维到布料的完整流程。

**核心内容:**
- 🌾 纤维来源：棉花、麻、羊毛、蚕丝
- 🧵 纺线技术：手摇纺车、简易纺锤
- 🪡 织布技术：简易织机、编织法

**关键要点:**

- 简易纺锤30分钟即可制作
- 基础织机1-2天可搭建

**时间投入:**
- 纺100米线：8-12小时
- 织1米布：6-10小时

### [02-制陶技术](./02-制陶技术.md)

从黏土到陶器的烧制技术。

**核心内容:**
- 🪨 黏土的识别和处理
- 👐 成型技术：手捏、泥条盘筑、陶轮拉坯
- 🔥 烧制技术：露天烧、简易窑炉
- ✨ 上釉技术：釉料制备

**关键要点:**
- 黏土需充分揉捏去气泡
- 烧制温度600-1000°C
- 露天烧成功率50%，窑炉80%+

**用途:**
- 储存容器（罐、缸）
- 烹饪器具（锅、碗）
- 建筑材料（砖、瓦）

### [03-制革技术](./03-制革技术.md)

皮革鞣制和加工技术。

**核心内容:**
- 🦌 皮革来源：牛、羊、猪、鹿等
- 🧂 鞣制方法：植物鞣、盐鞣、脑鞣
- 🔨 皮革加工：软化、整理
- 👞 皮具制作：鞋、袋、腰带等

**关键要点:**
- 鲜皮需立即处理（24小时内）
- 植物鞣最安全可靠
- 鞣制周期7-30天

**用途:**
- 衣物鞋靴（保暖、耐用）
- 容器（水袋、背包）
- 防护（盔甲、手套）

### [04-木工技术](./04-木工技术.md)

木材加工和建筑技术。

**核心内容:**
- 🌲 木材的选择和处理
- 🪚 基本木工工具：锯、斧、凿、刨
- 🏠 榫卯技术：不用钉子的连接
- 🏘️ 简易建筑：木屋、仓库、家具

**关键要点:**
- 木材需干燥（6个月-1年）
- 榫卯连接比钉子更牢固
- 基本家具（桌椅床）1-3天可制作

**用途:**
- 住所建筑
- 家具制作
- 工具制作

### [05-制绳技术](./05-制绳技术.md)

绳索和编织物制作。

**核心内容:**
- 🌾 纤维来源：植物纤维、动物毛发
- 🪢 搓绳技术：Z搓、S搓、三股绳
- 🎣 编织技术：渔网、篮筐、席子
- 🔗 绳结大全：常用实用绳结

**关键要点:**
- 最简单的绳：两股反向搓
- 三股绳强度提升3倍
- 渔网编织需掌握基本网结

**用途:**
- 捆绑、拖拽
- 渔猎（渔网、陷阱）
- 编织容器

## 🎯 学习路线

### 阶段一：必修技能（1-3个月）

**优先级排序:**

1. **制绳**（最简单，立即有用）
   - 1周掌握基本搓绳
   - 2周掌握三股绳和常用绳结
   - 1个月掌握编织技术

2. **制陶**（容器必需）
   - 1周学会手捏成型
   - 2周掌握露天烧制
   - 1个月掌握简易窑炉

3. **木工**（建筑必需）
   - 2周学会基本工具使用
   - 1个月掌握简易家具制作
   - 2-3个月掌握建筑技术

### 阶段二：进阶技能（3-6个月）

4. **纺织**（衣物必需，但较复杂）
   - 2周学会纺线
   - 1个月掌握简易织布
   - 2-3个月制作完整衣物

5. **制革**（高级技能）
   - 1个月掌握基础鞣制
   - 2个月掌握皮具制作

## 💡 新手建议

### ✅ 推荐起步

1. **从制绳开始**
   - 最简单
   - 无需工具
   - 立即有用（捆绑、拖拽）

2. **尝试简单制陶**
   - 手捏小碗
   - 露天烧制
   - 积累经验

3. **学习基础木工**
   - 从简单家具开始
   - 掌握基本工具

### ❌ 新手误区

1. ✗ **一开始就尝试织布**
   - 太复杂
   - 容易失败
   - 打击信心

2. ✗ **没有工具就做木工**
   - 效率极低
   - 先做简单工具

3. ✗ **鞣制不充分的皮革**
   - 会发臭、腐烂
   - 浪费资源

## 🌍 材料来源

### 纤维类

**植物纤维:**
- 麻：田野、荒地
- 棉花：需种植
- 树皮：内层韧皮
- 草类：芦苇、香蒲等

**动物纤维:**
- 羊毛：剪羊毛
- 兽毛：猎物
- 蚕丝：养蚕（高级）

### 黏土

**识别:**
- 黏性强
- 细腻
- 潮湿时可塑

**位置:**
- 河床
- 湖边
- 山坡黄土层

### 皮革

**来源:**
- 家畜：牛、羊、猪
- 野生动物：鹿、野猪等
- 鱼皮：大型鱼（特殊用途）

### 木材

**硬木（家具、工具）:**
- 橡木、榆木、槐木
- 坚硬、耐用

**软木（建筑）:**
- 松木、杉木、柳木
- 易加工、生长快

## ⚠️ 安全提示

### 工具使用

**刀具安全:**
- 向外切，不向身体
- 保持锋利（钝刀更危险）
- 使用时专注

**火的安全:**
- 烧窑远离易燃物
- 准备灭火工具
- 不在大风天烧窑

### 化学安全

**鞣制皮革:**
- 某些鞣剂有毒（如铬鞣）
- 植物鞣最安全
- 戴手套操作

**染色:**
- 某些天然染料有毒
- 通风良好
- 不用于食品容器
