# 水产养殖指南

## 水产养殖的价值

```
优点:
✓ 蛋白质丰富
✓ 充分利用水资源
✓ 某些品种生长快
✓ 经济效益好

需求:
- 水源(池塘、水库、稻田、水箱)
- 一定技术
- 水质管理
```

---

## 鱼类养殖

### 常见养殖鱼类

**四大家鱼**:
- 草鱼:吃草,生长快
- 鲢鱼:滤食浮游生物
- 鳙鱼(花鲢):滤食浮游生物
- 青鱼:吃螺蛳等

**其他**:
- 鲫鱼:适应性强
- 鲤鱼:杂食,易养
- 罗非鱼:热带鱼,生长快
- 黄颡鱼:经济价值高

### 池塘养鱼

#### 池塘条件

```
面积:
- 小型: 1-3亩
- 中型: 3-10亩
- 大型: >10亩
(1亩=666.7平米)

水深:
- 1.5-3米
- 深浅结合更好

水源:
✓ 充足
✓ 清洁
✓ 无污染
✓ 进排水方便

底质:
- 淤泥10-20cm(过厚需清淤)
- 或沙泥底

形状:
- 长方形为好
- 东西向(受光好)
```

#### 池塘准备

**清塘消毒**:
```
放鱼前15-20天:

1. 排干水
2. 清除杂鱼、野杂鱼、敌害(黑鱼、蛇等)
3. 修整池埂
4. 清除过多淤泥

消毒:
- 生石灰: 100-150kg/亩(干法)
  * 在池底挖坑放生石灰
  * 加水化开
  * 泼洒全池
- 或漂白粉: 20kg/亩(带水消毒)

5天后注水
```

**施基肥**:
```
放鱼前7-10天:
- 有机肥(发酵鸡粪、猪粪)200-300kg/亩
- 培养浮游生物(鱼苗开食)
- 水色呈黄绿色或茶褐色为好
```

#### 鱼种放养

**时间**: 春季(3-5月),水温10°C以上

**密度**(混养,每亩):
```
主养草鱼:
- 草鱼: 800-1000尾(规格100-200g/尾)
- 鲢鱼: 200-300尾
- 鳙鱼: 50-100尾
- 鲫鱼、鲤鱼: 50-100尾

主养鲢鳙:
- 鲢鱼: 500-800尾
- 鳙鱼: 200-300尾
- 草鱼: 200-300尾
- 其他: 100-200尾

注:规格、密度根据条件调整
```

**混养原理**:
- 充分利用水体各层次
- 鲢鳙:上层(浮游生物)
- 草鱼:中层(草食)
- 青鱼、鲤鱼:底层(底栖生物)
- 互利互补

**投放方法**:
```
✓ 选健康鱼种
✓ 规格整齐
✓ 同池鱼种一次放足
✓ 晴天上午投放
✓ 鱼种消毒(3-5%食盐水浸泡5-10分钟)
✓ 水温适应(温差<5°C)
```

#### 饲养管理

**投饲**:
```
草鱼:
- 青草、水草为主
- 或颗粒饲料
- 日投饲量:鱼体重的3-5%(夏季),1-2%(春秋)
- 设置食台

鲤鱼、鲫鱼:
- 颗粒饲料
- 或玉米、麸皮、豆饼等

鲢鳙:
- 主要靠浮游生物
- 施肥(有机肥)培养浮游生物
- 可补充豆浆、麸皮浆

投饲原则:
✓ 定时(每天同一时间)
✓ 定位(固定食台)
✓ 定质(新鲜,不腐败)
✓ 定量(根据季节、天气、鱼情)

口诀: "四定"投饲
```

**水质管理**(重要!):
```
"养鱼先养水"

水色:
✓ 黄绿色或茶褐色(好)
✓ 清澈见底:水瘦,需施肥
✓ 浓绿、蓝绿:水肥,易缺氧

透明度:
- 25-30cm为宜

溶氧:
- >3mg/L(正常)
- <2mg/L(缺氧,鱼浮头)

调节:
✓ 定期加注新水(每7-10天)
✓ 有条件使用增氧机
✓ 施肥(有机肥,培养浮游生物)
✓ 调节pH(生石灰)
✓ 清除水面浮草(不能太多)

巡塘:
✓ 每天早中晚各巡视一次
✓ 观察水色、鱼活动、摄食
✓ 检查设施
```

**施肥**:
```
目的:培养浮游生物(鲢鳙食物)

方法:
- 有机肥(腐熟):每次100-150kg/亩
- 间隔10-15天
- 少量多次
- 看水施肥(水瘦多施,水肥少施)

注意:
- 高温季节少施或不施(易缺氧)
- 阴雨天不施
- 有机肥必须发酵腐熟!
```

**越冬管理**:
```
- 深水越冬(水深1.5-2米以上)
- 保持冰下水位
- 冰面扫雪(增加光照)
- 冰下充氧(打冰眼)
- 减少投饲或停喂
```

#### 病害防治

```
预防为主:
✓ 清塘消毒
✓ 鱼种消毒
✓ 工具消毒
✓ 食台定期消毒
✓ 投喂新鲜饲料
✓ 水质管理良好
✓ 合理密养
✓ 发现病鱼及时隔离

常见病:
鱼瘟(烂鳃、赤皮、肠炎):
- 细菌性
- 药物防治
- 全池泼洒消毒药

寄生虫病(锚头鳋、鱼虱):
- 药物杀虫
- 全池泼洒

缺氧浮头:
- 立即加注新水
- 开增氧机
- 严重时减少投饲

⚠️ 病害严重咨询水产技术员!
```

#### 收获

```
时间:
- 秋冬季,水温下降
- 或达到商品规格随时

方法:
- 拉网捕捞
- 干塘捕捞(排水后)
- 可捕大留小(轮捕轮放)

产量:
- 一般管理:300-500kg/亩
- 精养高产:800-1500kg/亩
```

### 稻田养鱼

```
优点:
✓ 一水两用
✓ 鱼吃害虫、杂草
✓ 鱼粪肥田
✓ 稻谷+鱼双丰收

方法:
- 在稻田周围挖鱼沟(宽1米,深0.5米)
- 鱼坑(供鱼栖息)
- 进排水口设栅栏
- 放养鱼种:鲤鱼、鲫鱼、草鱼等
- 密度:50-150尾/亩

管理:
- 稻田正常管理
- 施农药时排水至鱼沟(保护鱼)
- 适当投饲

效益:
- 亩产鱼20-50kg
- 稻谷也增产!
```

### 水箱(桶)养鱼(小规模)

```
适合:
- 无大水面
- 家庭小规模
- 观赏+食用

容器:
- 大水箱、水族箱、大塑料桶
- 至少50-100升水

品种:
- 鲫鱼、鲤鱼(耐低氧)
- 罗非鱼(热带,需保温)

密度:
- 不宜过高
- 1-2条/10升水

管理:
✓ 增氧(气泵)
✓ 过滤
✓ 定期换水(1/3,每周)
✓ 投饲(颗粒料,少量)
✓ 防夏季高温、冬季低温

产出有限,主要体验和观赏
```

## 虾类养殖

### 小龙虾养殖

**池塘条件**:
```
- 面积:3-5亩
- 水深:1-1.5米
- 进排水方便
- 防逃设施(围网)
```

**放养**:
```
- 虾苗:30-50kg/亩
- 时间:春季或秋季
- 提供隐蔽物(水草、瓦片、网片)
```

**管理**:
```
饲料:
- 杂食性
- 小鱼虾、螺蛳、水草、人工饲料

水质:
- 保持清新
- 适当种植水草(净化水质,提供饵料和隐蔽)

注意:
- 小龙虾生长过程蜕壳
- 蜕壳时易受攻击,需隐蔽物
- 定期巡塘,防逃
```

**收获**:
```
- 地笼捕捞
- 可以边养边捕
- 4-10月为捕捞旺季
```

### 青虾(河虾)养殖

```
类似小龙虾,但:
- 对水质要求更高
- 密度可稍高
- 价值更高
```

## 螃蟹养殖

### 河蟹(大闸蟹)养殖

**池塘条件**:
```
- 水深:1-1.5米
- 底部硬质(防打洞)
- 防逃设施(光滑围板或网)
- 种植水草(伊乐藻等)
```

**放养**:
```
- 蟹苗(扣蟹):500-800只/亩
- 时间:春季
- 规格:100-150只/kg
```

**管理**:
```
饲料:
- 杂食性
- 小鱼、螺蛳、玉米、南瓜、河蟹专用料

水质:
- 清新,溶氧充足
- 水草覆盖50-60%

蜕壳:
- 生长需多次蜕壳
- 蜕壳期营养补充
- 提供隐蔽场所

注意:
- 防逃(特别是大雨、闷热天气)
- 防盗
```

**收获**:
```
- 9-11月(秋季)成熟
- 地笼、网捕
- 鲜活上市,价值高
```

## 泥鳅养殖

**特点**:
```
✓ 适应性强
✓ 耐低氧
✓ 杂食性
✓ 经济价值好
✓ 适合小水面、稻田
```

**养殖**:
```
池塘:
- 小型池塘、水泥池均可
- 水深30-50cm
- 底部淤泥10-15cm

放养:
- 50-100kg/亩
- 苗种3-5cm

饲喂:
- 杂食
- 米糠、麸皮、蚯蚓、小杂鱼、专用料

管理:
- 水质保持
- 防逃(泥鳅善钻)
- 夏季遮阳

收获:
- 养殖4-6个月
- 规格达10-15cm
- 地笼捕捞
```

## 📝 水产养殖要点

### 新手推荐

```
容易上手:
1. 池塘混养(四大家鱼)
2. 稻田养鱼(一水两用)
3. 泥鳅(适应性强)

需经验:
- 河蟹(技术要求高,投入大)
- 小龙虾(管理要求高)

观赏体验:
- 水箱养鱼(少量)
```

### 成功关键

```
✓ 水质管理!(最重要)
  - 溶氧充足
  - 水色适宜
  - 定期换水或加新水

✓ 合理密度(不贪多)

✓ 科学投饲
  - 定时定量
  - 新鲜饲料

✓ 病害预防
  - 消毒
  - 早发现早治疗

✓ 日常巡塘(勤观察)

✓ 学习技术
```

### 家庭水产配置(有1-2亩池塘)

```
池塘混养:
- 草鱼为主,搭配鲢鳙、鲤鲫
- 或小龙虾+鱼混养

稻田(如有):
- 稻田养鱼

目标:
- 自家食用为主
- 剩余出售

预期:
- 1亩池塘产鱼300-500kg
- 够一家人一年吃鱼
- 剩余可出售
```

### 注意事项

```
✓ 水源:
  - 确保水源充足、清洁
  - 了解水权

✓ 技术:
  - 学习基础知识
  - 请教有经验者
  - 小规模试验

✓ 投入:
  - 池塘改造
  - 鱼种
  - 饲料
  - 设备(增氧机等)

✓ 风险:
  - 病害
  - 自然灾害(洪水、干旱)
  - 市场价格波动
  - 盗窃、天敌(鸟类等)

✓ 环境:
  - 不过度投饲施肥(污染水质)
  - 合理用药
  - 废水处理
```

### 效益估算(1亩池塘混养)

```
投入:
- 清塘、池塘整理: 200元
- 鱼种: 1000-1500元
- 饲料: 2000-3000元
- 肥料、药物: 300-500元
- 电费、其他: 200-300元
总计: 3700-5500元

产出:
- 产量: 400kg
- 价格: 平均15元/kg
- 收入: 6000元

净利润: 500-2300元/亩

注:
- 新手第一年利润较低
- 经验增加后提高
- 自食用价值更高
- 配合其他种养效益更好
```
