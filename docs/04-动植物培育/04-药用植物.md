## 常见易种药用植物

### 薄荷

**功效**: 疏风散热,清利头目,利咽,透疹

**识别**:
```
- 多年生草本
- 茎四棱
- 叶对生,卵形,有锯齿
- 揉搓有强烈薄荷香
- 夏季开淡紫色小花
```

**种植**:
```
✓ 极易种植!
✓ 分株或扦插
✓ 喜湿润
✓ 半阴或全日照
✓ 生长快,需控制(侵占性强)

采收:
- 生长期随时采叶
- 开花前采收药效最好
- 鲜用或晒干
```

**用法**:
- 泡茶: 解暑、清咽
- 外用: 止痒(蚊虫叮咬)

### 金银花(忍冬)

**功效**: 清热解毒,疏散风热

**识别**:
```
- 多年生藤本
- 叶对生,卵形
- 花初开白色,后变黄色(金银相映)
- 花香浓郁
- 5-7月开花
```

**种植**:
```
✓ 适应性强
✓ 扦插或分株
✓ 喜阳光
✓ 耐旱耐瘠
✓ 需搭架或地栽让其攀援

采收:
- 花蕾期采收(未开放)
- 早晨采收最佳
- 及时晒干或烘干
```

**用法**:
- 泡茶: 清热解毒,预防感冒
- 煎服: 咽喉肿痛,热毒疮疡

### 菊花

**功效**: 疏风清热,平肝明目

**识别**:
```
- 多年生草本
- 叶互生,羽状分裂
- 秋季开花,颜色多样
- 气味芳香
```

**种植**:
```
✓ 容易种植
✓ 分株或扦插
✓ 喜阳光充足
✓ 排水良好
✓ 秋季开花

采收:
- 花盛开时采收
- 晒干或烘干
```

**用法**:
- 泡茶: 清肝明目,降火
- 药用多用杭白菊、贡菊

### 生姜

**功效**: 发汗解表,温中止呕,温肺止咳

**识别**:
```
- 多年生草本
- 地下根茎肥大,淡黄色
- 叶披针形,互生
- 独特辛辣味
```

**种植**:
```
见[蔬菜种植]章节

✓ 温暖湿润环境
✓ 种姜(根茎)种植
✓ 春季种植
✓ 充足水分
✓ 半阴环境

采收:
- 嫩姜: 夏季
- 老姜: 秋季霜前
```

**用法**:
- 煮水: 风寒感冒,恶心呕吐
- 泡茶: 姜枣茶(暖胃)
- 食用: 调味,温中

### 大蒜

**功效**: 解毒消肿,杀虫,止痢

**识别**: 见[蔬菜种植]

**种植**: 秋季种植,初夏收获

**用法**:
- 食用: 抗菌,预防感冒
- 外用: 捣烂敷患处(消肿)

### 蒲公英

**功效**: 清热解毒,消肿散结,利尿通淋

**识别**:
```
- 多年生草本
- 叶基生,倒披针形,羽状裂
- 花黄色,头状花序
- 种子有白色冠毛(蒲公英伞)
- 折断有白色乳汁
```

**种植**:
```
✓ 野生常见
✓ 极易种植
✓ 种子繁殖
✓ 耐寒耐旱
✓ 适应性极强

采收:
- 全草入药
- 春夏采收
- 鲜用或晒干
```

**用法**:
- 嫩叶可食(凉拌、炒食)
- 煎服: 热毒疮疡,咽喉肿痛
- 泡茶: 清热利尿

### 鱼腥草(折耳根)

**功效**: 清热解毒,排脓消痈,利尿通淋

**识别**:
```
- 多年生草本
- 茎紫红色
- 叶心形,全缘
- 揉搓有鱼腥味
- 白色花穗
- 地下茎蔓延
```

**种植**:
```
✓ 喜湿润半阴
✓ 地下茎繁殖
✓ 生长旺盛(需控制蔓延)
✓ 耐寒

采收:
- 全草或根茎
- 夏季采收
- 鲜用或晒干
```

**用法**:
- 食用: 凉拌(嫩茎叶,根茎)
- 煎服: 肺热咳嗽,尿路感染

### 艾草(艾蒿)

**功效**: 温经止血,散寒止痛

**识别**:
```
- 多年生草本
- 茎直立
- 叶互生,羽状深裂,背面有白色绒毛
- 强烈香气
- 夏季开小黄花
```

**种植**:
```
✓ 适应性强
✓ 种子或分株
✓ 耐旱
✓ 喜阳光

采收:
- 端午前后采收最佳
- 全草入药
- 晒干悬挂保存
```

**用法**:
- 艾灸: 温经通络
- 煮水泡脚: 散寒,助眠
- 驱蚊: 燃烧艾叶

### 枸杞

**功效**: 滋补肝肾,益精明目

**识别**:
```
- 落叶灌木
- 枝条细长,有刺
- 叶披针形或卵状披针形
- 花紫色
- 果实红色浆果(枸杞子)
```

**种植**:
```
✓ 耐旱耐瘠
✓ 种子、扦插或分株
✓ 喜阳光
✓ 适应性强

采收:
- 夏秋采收成熟果实
- 晒干或烘干
```

**用法**:
- 泡茶,煮粥,炖汤
- 滋补肝肾,明目

### 甘草

**功效**: 补脾益气,清热解毒,调和诸药

**识别**:
```
- 多年生草本
- 根和根茎粗壮,淡黄色
- 羽状复叶
- 紫色蝶形花
- 豆科植物
```

**种植**:
```
- 种子或根茎繁殖
- 耐旱耐盐碱
- 生长缓慢
- 3-4年后采收根部

注意:
- 种植有地区限制(保护植物)
- 购买合法种源
```

**用法**:
- 煎服: 补气,止咳,解毒
- 炙甘草: 炒制后功效增强

## 药用植物种植要点

### 选择原则

```
新手推荐(易种植):
1. 薄荷(极易)
2. 金银花(适应性强)
3. 菊花(普通)
4. 生姜、大蒜(蔬菜兼药用)
5. 蒲公英(野生,易采)

需经验:
- 甘草(生长慢)
- 黄芪、当归等(专业)
```

### 家庭药园配置

**小面积(10-20平米)**:
```
- 薄荷: 1平米
- 金银花: 攀援架
- 菊花: 2-3株
- 蒲公英: 散种
- 艾草: 2-3株
- 枸杞: 1-2株

+ 生姜、大蒜(蔬菜地)
```

### 采收与加工

**采收时间**:
```
- 全草类: 开花前或初花期
- 花类: 花蕾期或初开期
- 果实类: 成熟时
- 根茎类: 秋冬季(养分储存)

原则:
✓ 晴天采收
✓ 早晨露水干后
✓ 清洁无污染
```

**加工方法**:
```
晒干:
- 薄层摊开
- 通风阴凉处或阳光下
- 经常翻动
- 至干燥

烘干:
- 低温(<60°C)
- 保持药效

鲜用:
- 某些药材鲜用效果更好
- 如蒲公英、鱼腥草
```

**储存**:
```
✓ 充分干燥
✓ 密封容器
✓ 阴凉干燥处
✓ 避光
✓ 防潮防虫
✓ 标签注明品种和日期

保质期:
- 一般1-2年
- 每年更新
```

## 常见中草药识别(野生)

### 车前草

```
识别:
- 叶基生,卵形,有明显平行脉
- 花穗直立,穗状花序
- 路边、田野常见

功效:
- 利尿通淋,清热明目

采收:
- 全草,夏秋采收
```

### 马齿苋

```
识别:
- 肉质草本
- 茎平卧,紫红色
- 叶倒卵形,肥厚
- 黄色小花

功效:
- 清热解毒,凉血止血

用法:
- 可食用(凉拌,炒食)
- 煎服
```

### 夏枯草

```
识别:
- 直立草本
- 叶对生,卵形或披针形
- 穗状花序,紫色
- 夏季花穗变褐色(夏枯)

功效:
- 清肝火,散郁结

采收:
- 花穗,夏季采收
```

### 益母草

```
识别:
- 直立草本,茎方形
- 叶掌状深裂
- 粉红或淡紫色花,轮生
- 田野、路边

功效:
- 活血调经,利尿消肿

采收:
- 全草,夏季采收
```

## ⚠️ 安全警示

### 有毒植物(避免误用)

```
常见有毒植物:
❌ 曼陀罗: 全株有毒,剧毒
❌ 商陆: 根似人参,有毒
❌ 天南星: 块茎有毒,需炮制
❌ 半夏: 有毒,需炮制
❌ 附子: 剧毒,需严格炮制
❌ 马钱子: 剧毒
❌ 苍耳子: 有毒

原则:
✓ 不认识不采不用!
✓ 确认无误才使用
✓ 初学者避免有毒品种
```

### 用药安全

```
✓ 咨询专业中医师
✓ 了解禁忌症
✓ 不过量使用
✓ 某些药材需炮制后使用
✓ 孕妇、儿童、老人慎用
✓ 观察不良反应
✓ 出现不适立即停用就医
```

### 采集伦理

```
✓ 不过度采集野生药材
✓ 保护植物资源
✓ 某些是保护植物(禁采)
✓ 了解当地法规
✓ 提倡人工种植
```
