# 家禽养殖指南

## 鸡的养殖

### 为什么养鸡?

```
优点:
✓ 鸡蛋(蛋白质)
✓ 鸡肉
✓ 鸡粪(肥料)
✓ 控制害虫
✓ 相对容易
✓ 占地小

投入:
- 初期: 鸡舍、围栏、设备
- 日常: 饲料、水、时间
```

### 品种选择

**蛋鸡**:
- 来航鸡: 高产蛋(280-300个/年)
- 褐壳蛋鸡: 适应性强

**肉鸡**:
- 白羽肉鸡: 生长快(45天出栏)
- 黄羽肉鸡: 肉质好

**蛋肉兼用**:
- 芦花鸡、三黄鸡: 综合性能好
- 土鸡: 抗病力强,适合散养

**新手推荐**: 本地土鸡品种

### 鸡舍建设

**选址**:
```
✓ 地势高燥
✓ 通风向阳
✓ 排水良好
✓ 远离居住区(气味)
✓ 便于管理
```

**规格**:
```
面积:
- 平养: 6-8只/平米(鸡舍内)
- 散养: 每只需1-2平米(室外活动区)

高度:
- 2-2.5米

结构:
✓ 墙壁:砖、木板均可
✓ 地面:水泥或土(铺稻草)
✓ 屋顶:防雨
✓ 通风窗
✓ 门(防兽)

内部:
✓ 栖架(晚上栖息,30cm高)
✓ 产蛋箱(每4-5只一个,30x30x30cm)
✓ 饮水器
✓ 料槽
✓ 垫料(稻草、木屑)
```

**运动场**:
```
- 面积: 鸡舍2-3倍
- 围网(防逃、防兽)
- 遮阴(树、棚)
```

### 育雏(0-6周)

**设备**:
```
- 育雏箱/笼
- 保温灯(冬季)
- 温度计
```

**温度**:
```
关键!
- 第1周: 32-35°C
- 第2周: 29-32°C
- 第3周: 26-29°C
- 之后每周降2-3°C
- 至21°C(约6周)

判断:
- 温度合适:小鸡分散,活泼
- 太冷:扎堆,叫声
- 太热:远离热源,张嘴
```

**饲喂**:
```
0-6周:
- 雏鸡料(高蛋白)
- 随时供应
- 清洁饮水
- 每天清理

注意:
✓ 前3天温水(防应激)
✓ 逐渐过渡饲料
✓ 保持清洁
```

**管理**:
```
✓ 观察精神状态
✓ 定期清洁
✓ 通风(但保温)
✓ 防挤压
✓ 疫苗接种(咨询兽医)
```

### 成鸡饲养(6周后)

**饲料**:
```
商品饲料:
- 蛋鸡料(产蛋鸡)
- 肉鸡料(肉鸡)

自配饲料:
- 玉米(60-70%)
- 豆粕/饼(15-20%)
- 麸皮(5-10%)
- 贝壳粉/骨粉(3-5%,补钙)
- 盐(0.3%)
- 青饲料(菜叶、草,10-20%)

添加:
- 沙砾(助消化)
```

**饲喂**:
```
- 定时定量
- 蛋鸡: 每天100-120g
- 肉鸡: 自由采食
- 清洁饮水(随时)
```

**日常管理**:
```
每日:
✓ 观察鸡群(精神、食欲、粪便)
✓ 喂料、加水
✓ 捡蛋(2-3次)
✓ 清理粪便(或定期)

每周:
✓ 清洁饮水器、料槽
✓ 更换垫料(部分)
✓ 检查设备

定期:
✓ 全面清洁消毒
✓ 驱虫
✓ 疫苗
```

### 产蛋管理

**开产**:
- 5-6月龄开始产蛋
- 逐渐增加到高峰(8-10月龄)
- 高产期1年左右

**促进产蛋**:
```
✓ 营养充足(蛋白、钙)
✓ 光照: 14-16小时/天
  - 自然光不足,补充人工光
  - 早晚开灯
✓ 安静环境
✓ 适宜温度(13-23°C)
✓ 充足饮水
```

**捡蛋**:
```
- 每天2-3次
- 及时捡(防啄破、弄脏)
- 保存: 钝端向上,阴凉处,10-15°C
```

### 疾病预防

**预防为主**:
```
✓ 清洁卫生
✓ 通风良好
✓ 密度适当
✓ 营养均衡
✓ 疫苗接种
✓ 定期消毒
✓ 隔离病鸡
✓ 观察异常
```

**常见病**:
```
鸡瘟(新城疫):
- 疫苗预防!
- 症状: 精神差,呼吸困难,绿便,死亡率高

球虫病:
- 症状: 血便,消瘦
- 预防: 保持干燥,药物预防

白痢:
- 雏鸡常见
- 症状: 白色粪便
- 保温,抗菌药

⚠️ 疾病严重时咨询兽医!
```

### 淘汰与更新

```
淘汰:
- 产蛋1-2年后产量下降
- 淘汰老鸡,补充新鸡
- 淘汰病弱鸡

补充:
- 每年补充20-30%(保持产蛋)
- 或全进全出(肉鸡)
```

## 鸭的养殖

### 特点

```
优点:
✓ 适应性强
✓ 抗病力强
✓ 产蛋多(270-300个/年,某些品种)
✓ 肉质好
✓ 喜水,可水旱结合

需求:
- 水源(pool或溪流,更好)
- 较大活动space
```

### 品种

- 蛋鸭: 绍兴鸭、金定鸭
- 肉鸭: 北京鸭、樱桃谷鸭
- 蛋肉兼用: 高邮鸭

### 鸭舍

```
类似鸡舍,但:
✓ 地面防水(鸭便湿)
✓ 略倾斜(排水)
✓ 运动场大些
✓ 水池(如可能)
```

### 饲养

**育雏**:
```
- 保温(类似小鸡)
- 第1周: 29-32°C
- 逐周降温
- 饮水+雏鸭料
```

**成鸭**:
```
饲料:
- 鸭料或自配(类似鸡料)
- 喜青饲料(菜叶、水草)
- 可放养觅食(虫、草、螺)

放牧:
- 早晚放牧(鸭喜凉爽)
- 水面放养(吃水草、螺、鱼虾)
- 减少饲料成本

产蛋:
- 多在夜间或清晨
- 产蛋箱
- 定时捡蛋
```

## 鹅的养殖

### 特点

```
优点:
✓ 主要食草(成本低)
✓ 生长快
✓ 抗病力强
✓ 肉质好
✓ 羽绒valuable

需求:
- 草地
- 水源
- 空间大
```

### 品种

- 狮头鹅、太湖鹅、四川白鹅等

### 饲养

**育雏**(0-4周):
```
- 保温: 第1周28-30°C
- 雏鹅料+切碎青菜
- 比例: 料:青 = 1:2逐渐到1:4
```

**育成**(4周后):
```
- 以放牧为主
- 食草(新鲜嫩草)
- 补充少量精料
- 水源(游水)

放牧:
- 上午、下午各一次
- 草地、水边
- 可快速育肥(70-90天,3-4kg)
```

**产蛋**:
```
- 8-10月龄开产
- 季节性产蛋(春季)
- 年产蛋40-80个
- 蛋大(150-200g)
```

## 📝 家禽养殖要点

```
选择:
- 新手: 鸡(容易)
- 有水源: 鸭、鹅
- 有草地: 鹅(成本低)

数量(家庭):
- 10-20只鸡(够吃蛋+偶尔吃肉)
- 或混养: 鸡+鸭+鹅

成功关键:
✓ 清洁卫生
✓ 适当密度
✓ 营养充足
✓ 防病疫苗
✓ 细心观察
✓ 及时处理问题

效益:
- 投入: 设施+饲料+时间
- 产出: 蛋+肉+肥料
- 约半年回本(养得好)
- 长期稳定收益

注意:
✓ 了解当地法规
✓ 防止扰民(气味、噪音)
✓ 人道养殖
✓ 食品安全
```
