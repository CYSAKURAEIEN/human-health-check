# 蔬菜种植指南

## 快速生长蔬菜(30-60天)

### 生菜

**周期**: 30-45天

```
种植:
- 时间: 春秋(避开盛夏)
- 土壤: 疏松肥沃
- 播种: 撒播或条播,浅覆土
- 间距: 15-20cm

管理:
✓ 保持湿润
✓ 薄肥勤施
✓ 部分遮阴(夏季)

收获:
- 外叶采收(持续生长)
- 或整株收获
```

### 小白菜/青菜

**周期**: 30-40天

```
种植:
- 适应性强,四季可种
- 撒播或条播
- 间距: 10-15cm

管理:
✓ 充足水分
✓ 速效氮肥
✓ 防虫(黄板、网罩)

病虫害:
- 蚜虫: 肥皂水喷洒
- 菜青虫: 手工捉、BT杀虫剂
```

### 萝卜

**周期**: 30-60天(品种而异)

```
种植:
- 秋季最佳
- 直播(不移栽)
- 间距: 10-20cm
- 土壤: 深厚疏松

管理:
✓ 苗期保湿
✓ 间苗(保证间距)
✓ 膨大期充足水分
✓ 磷钾肥

收获:
- 根据品种,30-60天
- 及时收获(防空心)
```

### 菠菜

**周期**: 40-50天

```
种植:
- 秋冬春(夏季抽薹)
- 撒播
- 浅覆土

管理:
✓ 保持湿润
✓ 氮肥为主
✓ 间苗

收获:
- 10cm高开始间拔
- 或整株收获
```

## 常见茄果类

### 番茄

**周期**: 育苗+100-120天

```
育苗:
- 早春室内育苗
- 播种后7-10天出芽
- 4-5片真叶移栽

种植:
- 间距: 40-50cm
- 深栽(促发根)
- 插支架

管理:
✓ 充足阳光
✓ 打侧枝(单干整枝)
✓ 开花期减少氮肥,增加磷钾肥
✓ 均衡浇水(防裂果)

病虫害:
- 晚疫病: 避免叶面积水
- 蚜虫: 肥皂水
```

### 辣椒

**周期**: 育苗+90-120天

```
育苗:
- 早春育苗
- 温度: 25-30°C发芽

种植:
- 间距: 30-40cm
- 阳光充足

管理:
✓ 不耐涝(排水好)
✓ 开花结果期增加肥水
✓ 采摘促进持续结果

品种:
- 甜椒、辣椒、朝天椒
```

### 茄子

**周期**: 育苗+100-120天

```
育苗:
- 春季育苗

种植:
- 喜温暖
- 间距: 50cm
- 深厚肥沃土壤

管理:
✓ 大肥大水
✓ 整枝(去老叶)
✓ 支撑(防倒伏)
```

## 瓜类

### 黄瓜

**周期**: 50-70天

```
种植:
- 春夏
- 直播或育苗移栽
- 间距: 30-40cm
- 搭架或地爬

管理:
✓ 保持土壤湿润
✓ 开花结果期大肥水
✓ 及时采收(促进持续结果)
✓ 整枝打杈

病虫害:
- 白粉病: 通风,喷小苏打水
- 蚜虫: 肥皂水
```

### 南瓜/西葫芦

**周期**: 60-100天

```
种植:
- 春夏播种
- 直播
- 间距: 50-100cm(占地大)

管理:
✓ 耐旱,但开花结果期需水
✓ 粗放管理
✓ 人工授粉(提高坐果率)

储存:
- 南瓜可长期储存(数月)
- 是很好的储备食物
```

## 根茎类

### 土豆

**周期**: 90-120天

```
种植:
- 春秋两季
- 种薯切块(带芽眼)
- 沟栽,间距: 25-30cm
- 覆土10-15cm

管理:
✓ 出苗后培土(2-3次)
✓ 块茎膨大期充足水分
✓ 花前追肥

收获:
- 茎叶枯黄时
- 挖出晾晒
- 阴凉干燥处储存

产量高,耐储存!
```

### 红薯

**周期**: 120-150天

```
种植:
- 育苗:春季,薯块催芽
- 剪藤扦插
- 间距: 30-40cm

管理:
✓ 前期少水(促扎根)
✓ 中期膨大充足水分
✓ 翻藤(防止藤蔓生根)
✓ 少肥(氮肥少,防徒长)

收获:
- 霜前收获
- 可长期储存
```

### 胡萝卜

**周期**: 70-120天

```
种植:
- 秋季最佳
- 直播(不移栽)
- 撒播或条播
- 间距: 5-10cm
- 土壤: 深厚疏松

管理:
✓ 保持湿润(出苗前)
✓ 间苗
✓ 后期控水(防裂根)

收获:
- 根据品种,70-120天
```

## 豆类

### 豇豆(豆角)

**周期**: 60-80天

```
种植:
- 春夏
- 直播
- 搭架或矮生品种

管理:
✓ 开花结荚期充足水肥
✓ 及时采收(嫩豆荚)
✓ 持续结荚长

优点:
- 产量高
- 采收期长
- 豆科植物固氮(改良土壤)
```

### 豌豆

**周期**: 90-120天

```
种植:
- 秋冬(耐寒)
- 直播
- 搭架

管理:
✓ 苗期少肥水
✓ 开花结荚期增加
✓ 根瘤菌固氮

收获:
- 嫩豆荚或成熟豆粒
```

## 葱蒜类

### 大葱/香葱

**周期**: 60-120天

```
种植:
- 四季可种
- 育苗或分株
- 间距: 5-10cm

管理:
✓ 喜肥水
✓ 培土(大葱)
✓ 多次采收

优点:
- 易种
- 多次收获
- 驱虫
```

### 大蒜

**周期**: 180-210天

```
种植:
- 秋季(9-10月)
- 蒜瓣分开种植
- 间距: 10-15cm
- 尖端向上

管理:
✓ 越冬前浇水
✓ 春季返青追肥
✓ 蒜薹采收后增肥(膨大)

收获:
- 蒜薹:春季
- 蒜头:初夏
```

## 📝 蔬菜种植要点

```
新手推荐(易种高产):
1. 生菜、小白菜(快速)
2. 萝卜(简单)
3. 豆角(高产)
4. 葱(多次收获)
5. 番茄、辣椒(产量稳定)

轮作原则:
- 豆科→叶菜类→茄果类→根茎类→豆科

四季安排:
- 春: 叶菜、茄果类育苗
- 夏: 瓜类、豆类
- 秋: 萝卜、白菜、秋豌豆、蒜
- 冬: 耐寒叶菜

关键技巧:
✓ 土壤肥沃疏松
✓ 薄肥勤施
✓ 适时浇水
✓ 病虫害预防
✓ 轮作倒茬
✓ 及时采收
```
