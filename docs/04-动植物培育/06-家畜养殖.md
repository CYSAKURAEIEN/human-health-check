# 家畜养殖指南

## 猪的养殖

### 为什么养猪?

```
优点:
✓ 生长快(5-6月出栏)
✓ 繁殖力强
✓ 肉产量高
✓ 杂食性(饲料来源广)
✓ 猪粪是优质肥料

考虑:
- 投入较大(饲料、设施)
- 需要一定技术
- 气味较大
- 疫病风险
```

### 品种选择

**本地猪**:
- 适应性强,抗病力好
- 肉质好
- 生长较慢

**外来猪** (如杜洛克、大白猪、长白猪):
- 生长快
- 瘦肉率高
- 饲料转化率高

**杂交猪**:
- 综合优势
- 新手推荐

### 猪舍建设

**选址**:
```
✓ 地势高燥
✓ 向阳背风
✓ 排水良好
✓ 远离居住区(200米以上)
✓ 水源方便
```

**猪舍结构**:
```
面积:
- 育肥猪: 1.5-2平米/头
- 母猪: 3-4平米/头

设计:
✓ 保温通风(冬暖夏凉)
✓ 干燥清洁
✓ 光线充足
✓ 坚固耐用

设施:
✓ 食槽(水泥或铁制)
✓ 水槽或自动饮水器
✓ 漏缝地板(利于清洁)
✓ 或水泥地面(有坡度排水)

分区:
- 睡卧区(干燥)
- 排粪区(易清理)
- 喂食区
```

### 仔猪选购

**选择健康仔猪**:
```
✓ 精神活泼
✓ 眼睛明亮
✓ 被毛光亮
✓ 食欲旺盛
✓ 粪便正常(不拉稀)
✓ 呼吸正常(无咳嗽)
✓ 体重适中(断奶仔猪7-10kg)
✓ 有免疫记录

购买:
- 15-30kg仔猪(断奶后)
- 本地信誉好的养殖场
- 查看防疫记录
```

### 饲养管理

#### 仔猪期(断奶-30kg)

```
饲料:
- 仔猪料(高蛋白,易消化)
- 自由采食
- 清洁饮水

管理:
✓ 保温(18-22°C)
✓ 清洁干燥
✓ 防腹泻
✓ 分群饲养(10-15头/群)
✓ 去势(公猪,1月龄)
✓ 疫苗接种
```

#### 育肥期(30kg-出栏)

```
饲料:
- 育肥猪料
- 或自配:
  * 玉米60-70%
  * 豆粕/豆饼15-20%
  * 麸皮5-10%
  * 预混料3-5%
  * 青饲料(菜叶、草)适量

饲喂:
- 每天2-3次
- 定时定量
- 先粗后精,先干后湿
- 充足清洁饮水

管理:
✓ 温度适宜(15-25°C)
✓ 通风良好(不要贼风)
✓ 干燥清洁(每天清理粪便)
✓ 适当运动(圈养注意)
✓ 保持安静
✓ 观察健康状况

饲料用量:
- 育肥期总耗料约300-350kg
- 料肉比: 约3:1(3kg料长1kg肉)
```

#### 出栏

```
时间:
- 5-6月龄
- 体重90-120kg

判断:
- 体型丰满
- 背部平直
- 食欲开始下降(达到顶峰)

及时出栏(再养饲料转化率下降)
```

### 疾病防治

**预防为主**:
```
✓ 清洁卫生(每天清理)
✓ 消毒(定期,空栏消毒)
✓ 通风干燥
✓ 营养均衡
✓ 免疫接种(猪瘟、口蹄疫等)
✓ 驱虫(定期)
✓ 隔离病猪
✓ 观察异常(食欲、粪便、精神)
```

**常见疾病**:
```
猪瘟:
- 疫苗预防!
- 高热、不食、便秘后腹泻、皮肤紫斑
- 发病立即隔离,报告兽医

腹泻:
- 仔猪常见
- 保暖,清洁饮水
- 抗菌药治疗
- 补液防脱水

呼吸道病:
- 咳嗽、气喘
- 通风,保温
- 及时治疗

⚠️ 严重疾病咨询兽医!
```

### 成本与效益(参考)

```
投入(1头育肥猪):
- 仔猪: 300-400元
- 饲料: 900-1200元(300kg×3-4元)
- 疫苗、药物: 50元
- 其他: 50元
总计: 1300-1700元

产出:
- 出栏重: 110kg
- 价格: 15元/kg(市场波动大)
收入: 1650元

净利润: -50至+350元/头

注:
- 价格波动大,有风险
- 规模化降低成本
- 自配饲料可降低成本
- 家庭养殖主要自用,剩余出售
```

## 羊的养殖

### 为什么养羊?

```
优点:
✓ 以草为主(成本低)
✓ 适应性强
✓ 抗病力强
✓ 繁殖力好
✓ 羊肉、羊毛、羊奶

适宜:
- 有草地、山地资源
- 粗放管理
- 小规模养殖
```

### 品种

**肉羊**:
- 波尔山羊、杜泊羊(引进)
- 小尾寒羊(本地,繁殖力强)

**奶羊**:
- 萨能奶山羊

**毛用羊**:
- 美利奴羊

**本地羊**:
- 适应性强,推荐

### 羊舍建设

```
选址:
✓ 地势高燥
✓ 向阳避风
✓ 靠近草场

羊舍:
面积: 1.5-2平米/只(舍内)
高度: 2.5-3米
地面: 干燥,略有坡度
通风: 良好
分区: 成羊、羔羊、产房

运动场:
- 舍外,面积2-3倍
- 设草架、饮水槽、遮阳棚
```

### 饲养管理

**饲料**:
```
以草为主:
✓ 放牧(最佳,成本低)
✓ 干草(冬季)
✓ 青贮饲料
✓ 农作物秸秆

补充精料:
- 玉米、麸皮、豆粕
- 妊娠后期、哺乳期、羔羊补饲

矿物质:
- 食盐、骨粉
- 自由舔食盐砖
```

**饲喂**:
```
放牧:
- 早出晚归
- 夏季避开中午高温
- 冬季中午放牧
- 慢赶慢走
- 不食毒草、露水草(易胀气)

舍饲:
- 早晚各喂一次草
- 夜间加喂
- 先粗后精
- 少喂勤添
- 充足饮水
```

**繁殖**:
```
配种:
- 母羊8-10月龄
- 公羊1-1.5岁
- 自然交配或人工授精

妊娠:
- 150天(5个月)

产羔:
- 大多单羔,某些品种双羔、多羔
- 做好接产准备(产房、保温)

哺乳:
- 及早吃初乳(出生后2小时内)
- 母乳或代乳
- 2-3月龄断奶
```

**日常管理**:
```
✓ 定期清理羊舍
✓ 通风干燥
✓ 修蹄(定期)
✓ 剪毛(春季)
✓ 驱虫(春秋各一次)
✓ 疫苗(口蹄疫、羊痘等)
✓ 观察健康
```

### 疾病防治

```
预防:
✓ 清洁卫生
✓ 通风干燥
✓ 免疫接种
✓ 定期驱虫
✓ 营养均衡

常见病:
- 寄生虫病:定期驱虫
- 腹泻:清洁饮水,卫生管理
- 口蹄疫:疫苗预防
- 羊痘:疫苗,隔离病羊
```

## 兔的养殖

### 为什么养兔?

```
优点:
✓ 繁殖力极强(1年数窝)
✓ 生长快(3-4月出栏)
✓ 占地小
✓ 投资少
✓ 饲料转化率高
✓ 以草食为主(成本低)
✓ 兔肉营养丰富
✓ 兔粪是优质肥料

非常适合小规模家庭养殖!
```

### 品种

**肉兔**:
- 新西兰兔、加利福尼亚兔、比利时兔
- 本地肉兔

**毛兔**:
- 安哥拉长毛兔

**皮兔**:
- 獭兔

### 兔舍建设

**笼养**(推荐):
```
兔笼规格:
- 成兔: 60×50×40cm(长×宽×高)
- 母兔: 70×60×40cm(需产仔箱)

材料:
- 木制或铁丝网笼
- 底网(防粪便积累)

放置:
- 室内或有遮蔽处
- 通风良好
- 避免阳光直射
- 离地30-50cm
```

**舍内散养或围栏**:
- 需要更大空间
- 清洁管理较难

### 饲养管理

**饲料**:
```
主料:
✓ 青草、青菜(主食)
✓ 干草(常备,磨牙)
✓ 胡萝卜、白菜等

精料:
- 兔饲料(颗粒料)
- 或自配:玉米、麸皮、豆粕

禁忌:
❌ 带露水的草(易胀气)
❌ 腐烂变质饲料
❌ 突然换料(逐渐过渡)

注意:
- 宁干勿湿(干草为主,少喂水分大的)
- 但保证饮水

口诀: "兔吃百样草,看你找不找"
```

**饲喂**:
```
- 早晚各喂一次
- 夜间加喂(兔有夜食性)
- 定时定量
- 少喂勤添
- 清洁饮水(自动饮水器)

用量:
- 成兔: 每天150-200g青料+50-100g精料
```

**繁殖**:
```
配种:
- 母兔5-6月龄
- 公兔6-7月龄

妊娠: 31天

产仔:
- 每窝6-10只
- 做好产仔箱(干草、兔毛保暖)
- 检查母兔是否哺乳

哺乳:
- 母乳30-40天
- 逐渐添加饲料
- 断奶后分笼

繁殖力:
- 年产5-6窝!
- 增殖非常快
```

**日常管理**:
```
✓ 每天清理粪便
✓ 保持干燥清洁
✓ 通风(兔怕潮湿、闷热)
✓ 夏季防暑(遮阳、通风)
✓ 冬季保暖(防风,不需过高温度,兔耐寒)
✓ 观察粪便(正常为圆球状)
✓ 观察食欲、精神
```

### 疾病防治

```
预防:
✓ 清洁干燥!
✓ 通风
✓ 勤清粪
✓ 食具消毒
✓ 免疫接种
✓ 隔离病兔

常见病:
腹泻:
- 食物不洁、受凉、换料过快
- 停喂青料,喂干草,保温
- 口服抗菌药

球虫病:
- 幼兔常见,严重!
- 预防为主(药物,卫生)
- 症状:腹泻、消瘦、死亡

疥癣:
- 皮肤病,脱毛
- 隔离治疗,涂药

⚠️ 兔病来得快,重在预防!
```

### 成本与效益

```
家庭养殖(10只母兔+2只公兔):

投入:
- 种兔: 600-1000元
- 兔笼: 500-800元
- 饲料: 约2000-3000元/年(部分青草自采)
总计: 3000-5000元(首年)

产出:
- 每只母兔年产30-40只(平均6窝×6只)
- 10只母兔年产300-400只
- 肉兔出栏: 2-2.5kg
- 总产肉: 600-1000kg
- 价格: 20元/kg(活兔)
- 收入: 12000-20000元

净利润: 第一年5000-12000元,之后更高(无种兔成本)

注:
- 很高的效益!
- 自食+出售
- 需要一定劳动投入(喂料、清洁)
```

## 📝 家畜养殖要点

### 新手推荐

```
最适合:
1. 兔(繁殖快,投入小,管理相对简单)
2. 羊(有草地,粗放,耐病)

需经验:
- 猪(投入大,技术要求高,疫病风险)
```

### 选择建议

```
考虑因素:
✓ 资金(兔<羊<猪)
✓ 场地(兔占地小,猪羊需较大)
✓ 饲料来源(羊适合有草地,猪需购买饲料)
✓ 技术(兔羊粗放,猪精细)
✓ 目的(自食、出售、综合)
✓ 劳动力

家庭配置:
- 鸡10-20只(蛋肉)
- 兔10-20只(肉,高繁殖)
- 羊3-5只(有草地)

或:
- 鸡+猪1-2头(每年)
```

### 成功要素

```
✓ 清洁卫生(第一重要!)
✓ 科学饲喂(营养均衡,定时定量)
✓ 防疫驱虫(预防为主)
✓ 细心观察(早发现早治疗)
✓ 学习技术(书籍、有经验者)
✓ 记录管理(饲料、疫病、效益)
✓ 适度规模(别贪大,稳步发展)
```

### 注意事项

```
✓ 了解当地法规(养殖许可、环保)
✓ 防疫要求(动物防疫条件合格证)
✓ 环境保护(粪污处理)
✓ 邻里关系(气味、噪音控制)
✓ 人畜共患病防范(布病、狂犬病等)
✓ 人道养殖
✓ 市场风险(价格波动)
```
