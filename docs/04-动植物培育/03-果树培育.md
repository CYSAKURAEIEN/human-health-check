# 果树培育指南

## 果树种植的价值

```
优点:
✓ 维生素和营养
✓ 多年生(种一次收多年)
✓ 相对省工
✓ 经济价值高
✓ 美化环境

考虑:
- 需要耐心(2-5年结果)
- 占地较大
- 前期投入
- 需要一定技术
```

---

## 果树种植基础

### 选址

```
✓ 阳光充足(每天6小时以上)
✓ 排水良好(不积水)
✓ 土壤深厚肥沃
✓ 避风(或设风障)
✓ 水源方便

❌ 避免:
- 低洼易涝地
- 重粘土
- 地下水位过高
- 风口
```

### 定植

**时间**:
- 落叶果树:秋季落叶后至春季发芽前(休眠期)
- 常绿果树:春季(3-4月)

**株行距**:
根据树种,一般:
- 大型果树(苹果、梨):4x5米
- 中型(桃、李、杏):3x4米
- 小型(樱桃、枣):2x3米

**定植坑**:
```
规格: 60-80cm宽深

步骤:
1. 挖定植坑
2. 底部填有机肥+表土混合
3. 放入树苗(根系舒展)
4. 填土踩实
5. 浇透定根水
6. 树盘覆盖(保湿)

深度:
- 嫁接口露出地面5-10cm
- "深栽不过嫁接口,浅栽不露根"
```

## 常见果树

### 苹果

**品种**: 红富士、嘎拉、金帅等

**气候**: 温带,需要一定冬季冷量

**结果**: 3-4年开始,盛果期15-40年

**种植要点**:
```
授粉:
- 大多需异花授粉
- 建议种2-3个品种
- 或嫁接不同品种

修剪:
- 冬季修剪(休眠期)
- 疏除过密枝、病虫枝
- 开心形或纺锤形

施肥:
- 基肥:秋季(有机肥)
- 追肥:春季萌芽、花后、果实膨大期

病虫害:
- 腐烂病:刮治病斑
- 蚜虫:早春喷药
- 红蜘蛛:生长季防治
```

**收获**: 9-11月(品种而异)

### 梨

**品种**: 雪花梨、鸭梨、砀山梨等

**气候**: 适应性强,较耐寒

**结果**: 3-5年,盛果期20-30年

**种植要点**:
```
特点:
- 需异花授粉(配置授粉树)
- 喜湿润

管理:
- 整形修剪(疏散分层形)
- 充足灌溉
- 疏花疏果(提高品质)

病虫害:
- 梨黑星病:清除落叶,喷药保护
- 梨木虱:早春防治
```

### 桃

**品种**: 水蜜桃、油桃、黄桃等

**气候**: 喜温暖,较耐旱

**结果**: 2-3年(早),盛果期10-15年

**种植要点**:
```
特点:
- 结果早
- 寿命相对短
- 生长快

修剪:
- 重要!
- 夏季摘心、疏枝
- 冬季短截、回缩
- 开心形

施肥:
- 喜肥
- 生长季多次追肥

病虫害:
- 桃缩叶病:春季发芽前喷药
- 蚜虫、红蜘蛛:及时防治
- 桃蛀螟:果实套袋
```

**特别**:
- 需疏果(早期摘除部分幼果)
- 提高单果品质

### 葡萄

**品种**: 巨峰、夏黑、阳光玫瑰等

**气候**: 适应性强

**结果**: 2-3年,盛果期20-30年

**种植要点**:
```
架式:
- 篱架、棚架
- 需搭架支撑

繁殖:
- 扦插(易成活)
- 压条

修剪:
- 冬季重剪
- 夏季摘心、疏梢

管理:
✓ 疏花疏果
✓ 果穗套袋(防病虫、提高品质)
✓ 充足水肥(果实膨大期)
✓ 冬季埋土防寒(北方)

病虫害:
- 霜霉病:雨季注意防治
- 白粉病:通风,喷药
```

**收获**: 7-10月(品种而异)

### 柑橘类(橘、橙、柚、柠檬)

**气候**: 亚热带,不耐寒

**结果**: 3-5年

**种植要点**:
```
特点:
- 常绿果树
- 喜温暖湿润
- 怕霜冻

管理:
✓ 保持水分
✓ 施有机肥
✓ 修剪(通风透光)
✓ 防寒(北方盆栽,冬季移入室内)

病虫害:
- 溃疡病:剪除病枝叶
- 红蜘蛛、蚜虫:及时防治
- 潜叶蛾:夏秋梢防治
```

### 樱桃

**品种**: 甜樱桃、酸樱桃、中国樱桃

**气候**: 喜冷凉,怕热怕涝

**结果**: 3-4年(甜樱桃),2-3年(中国樱桃)

**种植要点**:
```
特点:
- "春果第一枝"(最早成熟)
- 对环境要求高
- 经济价值高

要求:
✓ 排水极好(忌涝!)
✓ 配置授粉树
✓ 防鸟害(网罩或驱鸟)

管理:
- 轻剪缓放(促早结果)
- 夏季摘心
```

### 枣

**品种**: 冬枣、金丝小枣、大枣等

**气候**: 耐干旱,适应性强

**结果**: 2-3年(嫁接苗)

**种植要点**:
```
特点:
✓ 极耐旱
✓ 耐瘠薄
✓ 管理粗放
✓ 适合干旱地区

管理:
- 开甲(环剥,提高坐果率)
- 花期喷水(提高坐果)
- 枣头摘心

病虫害:
- 枣步曲:早春防治
- 枣锈病:雨季注意
```

**收获**: 鲜食或晒干

### 石榴

**气候**: 喜温暖干燥

**结果**: 3-4年

**种植要点**:
```
特点:
- 耐旱
- 观赏和食用兼具

管理:
✓ 充足阳光
✓ 花期、幼果期充足水分
✓ 疏除部分花果
✓ 修剪(通风透光)
```

### 无花果

**气候**: 喜温暖,较耐寒

**结果**: 当年或次年

**种植要点**:
```
特点:
✓ 结果极早!
✓ 管理简单
✓ 病虫害少
✓ 扦插易成活

管理:
- 粗放管理
- 修剪(控制树形)
- 冬季防寒(寒冷地区)

新手友好!
```

## 果树管理技术

### 整形修剪

**目的**:
- 控制树形和大小
- 通风透光
- 促进结果
- 更新复壮

**时期**:
- 冬季修剪:休眠期(落叶果树)
- 夏季修剪:生长季

**基本方法**:
```
短截:
- 剪短枝条
- 促发新枝

疏枝:
- 从基部剪除
- 减少枝量

回缩:
- 剪去多年生枝的一部分
- 更新复壮

缓放:
- 不剪
- 促进结果
```

**常见树形**:
- 开心形:中心空虚,通风好
- 疏散分层形:有中心干
- 纺锤形:密植园

### 施肥

**原则**: "有机肥为主,化肥为辅"

**基肥**(秋季):
```
- 有机肥(腐熟)
- 在树冠投影外缘挖沟施入
- 量:每株20-50kg(成年树)
```

**追肥**(生长季):
```
关键期:
1. 萌芽前:氮肥为主
2. 花后:氮磷配合
3. 果实膨大期:氮磷钾
4. 采收后:恢复树势

方法:
- 沟施或穴施
- 或叶面喷肥
```

### 浇水

```
关键期:
✓ 萌芽前
✓ 花后
✓ 果实膨大期
✓ 采收后
✓ 封冻前

方法:
- 沟灌、穴灌
- 滴灌(节水)

排水:
- 雨季及时排水(防涝)
```

### 疏花疏果

**目的**:
- 提高果实品质
- 防止大小年(隔年结果)
- 保持树势

**时期**:
- 疏花:花期
- 疏果:坐果后2-4周

**方法**:
```
- 疏除畸形、病虫果
- 疏除过密、弱小果
- 保留位置好的果

留果量:
- 根据树势、品种
- 叶果比:30-50:1(苹果)
```

### 病虫害防治

**预防为主**:
```
✓ 选抗病品种
✓ 合理修剪(通风透光)
✓ 清理落叶、病果(减少病源)
✓ 翻耕(消灭越冬虫源)
✓ 树干涂白(防冻、防虫)
```

**物理防治**:
- 糖醋液诱杀
- 黑光灯诱杀
- 套袋(果实)
- 粘虫板

**生物防治**:
- 保护利用天敌
- 生物农药

**化学防治**:
- 关键期喷药
- 轮换用药(防抗性)
- 注意安全间隔期

## 📝 果树种植要点

### 新手推荐

```
容易种植(适合新手):
1. 无花果(极易,结果快)
2. 枣(耐旱粗放)
3. 桃(结果早)
4. 葡萄(易管理)
5. 石榴(观赏食用)

需经验:
- 苹果、梨(需授粉、修剪技术)
- 樱桃(环境要求高)
- 柑橘(北方难越冬)
```

### 家庭果园配置(1亩)

```
混合种植:
- 苹果/梨: 10-15株(主体)
- 桃/李: 5-8株
- 葡萄: 20-30株(篱架)
- 石榴/枣: 3-5株
- 草莓/覆盆子: 边缘空地

优点:
✓ 品种多样
✓ 延长采收期
✓ 互相授粉(某些)
✓ 降低风险
```

### 省力管理要点

```
✓ 选择适应当地的品种
✓ 抗病虫品种
✓ 适当密度(不过密)
✓ 地面覆盖(保湿、抑草)
✓ 滴灌系统
✓ 矮化砧木(方便管理)
```

### 采收与储存

**适时采收**:
- 过早:品质差
- 过晚:不耐储运

**储存**:
```
短期:
- 阴凉通风处
- 分层摆放

长期:
- 冷藏(适合的品种)
- 加工(果酱、果干、罐头)
```

### 繁殖方法

**嫁接**(常用):
```
优点:
✓ 保持优良性状
✓ 矮化控制
✓ 提早结果
✓ 适应性强(砧木)

方法:
- 枝接(春季)
- 芽接(夏季)

需学习技术!
```

**扦插**:
- 葡萄、无花果:易生根
- 其他需生根剂

**压条**:
- 葡萄等蔓性

**实生**(种子)**:
- 不能保持母本特性
- 结果晚
- 一般不推荐(除非培育砧木)
