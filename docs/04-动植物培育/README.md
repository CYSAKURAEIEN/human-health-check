## 🌱 种植基础知识

### 土壤

#### 土壤类型

**沙土**:
- 特点:透气好,排水快,贫瘠
- 改良:增加有机质(堆肥、腐叶)
- 适合:根茎类、豆类

**粘土**:
- 特点:保水好,排水差,粘重
- 改良:加沙、有机质,深耕
- 适合:水稻、白菜

**壤土**(最佳):
- 特点:沙粘适中,肥沃
- 适合:大多数作物

#### 土壤改良

```
堆肥制作:
1. 材料:
   - 绿色(氮):菜叶、草、咖啡渣
   - 棕色(碳):干叶、木屑、纸
   - 比例:绿1:棕2-3
2. 方法:
   - 分层堆放
   - 保持湿润
   - 定期翻堆
   - 2-6个月成熟
3. 使用:
   - 混入土壤
   - 覆盖表面
```

### 水分管理

**浇水原则**:
```
✓ 时间:早晨或傍晚
✓ 方式:浇根部,避免叶面(防病)
✓ 频率:见干见湿
✓ 深度:浇透(鼓励深根)
✓ 排水:防止积水
```

**判断缺水**:
- 叶片萎蔫
- 土壤干燥
- 颜色暗淡

### 阳光需求

**全日照**(6-8小时):
- 茄果类:番茄、辣椒
- 瓜类:黄瓜、西瓜
- 根茎类:胡萝卜、土豆

**半日照**(3-6小时):
- 叶菜类:生菜、菠菜
- 十字花科:白菜、西兰花

**耐阴**(2-3小时):
- 姜、芋头
- 某些香草

### 施肥

**三要素**:
- **N**(氮):叶子生长
- **P**(磷):根、花、果
- **K**(钾):整体健康、抗病

**有机肥**:
- 堆肥
- 动物粪便(腐熟)
- 骨粉(磷)
- 草木灰(钾)
- 绿肥(豆科植物)

**施肥原则**:
```
✓ 薄肥勤施
✓ 避开根颈
✓ 浇水后施肥
✓ 不同生长期不同需求:
  - 生长期:多氮
  - 开花期:多磷
  - 结果期:多钾
```

## 🌾 作物轮作

### 原则

不要连续在同一地块种植相同科作物

### 轮作计划(4年)

```
第1年:豆科(固氮)- 豌豆、大豆
第2年:叶菜类 - 白菜、生菜
第3年:茄果类 - 番茄、辣椒
第4年:根茎类 - 萝卜、土豆
```

### 好处

```
✓ 减少病虫害
✓ 平衡土壤养分
✓ 提高产量
✓ 减少化肥使用
```

## 🐛 病虫害管理

### 预防

```
✓ 选择抗病品种
✓ 合理密植(通风)
✓ 清除病株、落叶
✓ 轮作
✓ 伴生种植(驱虫植物)
```

### 有机防治

**物理方法**:
- 手工捉虫
- 黄板诱杀
- 防虫网
- 清水冲洗

**生物方法**:
- 引入天敌(瓢虫吃蚜虫)
- 种植驱虫植物:
  * 万寿菊:驱线虫
  * 薄荷:驱蚂蚁、蚜虫
  * 大蒜:驱多种害虫

**自制杀虫剂**:
```
辣椒水:
- 干辣椒+水,煮沸
- 冷却,喷洒
- 驱虫效果

肥皂水:
- 少量洗洁精+水
- 喷在虫体上
- 杀蚜虫、粉虱

大蒜水:
- 大蒜捣烂+水浸泡
- 过滤,喷洒
- 广谱驱虫
```

## 🌟 自给自足指南

### 基本配置(1家4口)

**种植面积**(约100-200平米):
- 蔬菜:50-100平米
- 粮食:50-100平米(可选)
- 果树:几棵

**养殖**(可选):
- 鸡:5-10只(蛋、肉)
- 兔:几只(肉)

**预期**:
- 蔬菜基本自给
- 鸡蛋自给有余
- 部分肉类
- 需要时间和劳动投入

### 循环系统

```
厨余→鸡→鸡粪→堆肥→蔬菜→厨余
```

### 时间投入

```
日常:1-2小时
高峰期(播种、收获):更多时间
随经验增加,效率提高
```
