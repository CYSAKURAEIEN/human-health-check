# 粮食作物种植指南

## 主食作物重要性

```
粮食作物 = 生存基础
- 提供碳水化合物(能量)
- 可长期储存
- 单位面积产量高
- 自给自足核心
```

---

## 水稻

### 基本信息

```
类型: 主粮作物
周期: 120-150天
产量: 约500kg/亩(精米)
适宜: 水源充足地区
```

### 品种选择

**按生育期**:
- 早稻: 90-120天
- 中稻: 120-150天
- 晚稻: 150-170天

**按种植方式**:
- 水稻(淹水种植)
- 旱稻(旱地种植,产量较低)

**新手推荐**: 本地适应品种

### 种植方法

#### 育苗移栽法(常用)

**1. 育秧(约30天)**:
```
时间: 清明前后(南方)

秧田准备:
- 平整、肥沃
- 浅水层(2-3cm)

播种:
- 浸种催芽(2-3天)
- 播种密度适中
- 保持湿润

管理:
✓ 保持浅水
✓ 薄施肥料
✓ 防病虫
```

**2. 整田**:
```
- 犁田(翻耕)
- 耙平
- 做好田埂(保水)
- 灌水
```

**3. 移栽**:
```
时间: 秧苗4-5叶时

密度:
- 行距: 20-25cm
- 株距: 15-20cm
- 每穴2-3株

深度:
- 浅插(2-3cm)
- "寸水插秧"

要点:
✓ 边起秧边栽
✓ 当天栽完
✓ 保持水层
```

#### 直播法(简单但产量略低)

```
- 整田后直接播种
- 省工省时
- 适合小面积
- 杂草控制难
```

### 田间管理

**水层管理**:
```
关键!

分蘖期(移栽后):
- 浅水(3-5cm)
- 促进分蘖

孕穗期:
- 深水(5-8cm)

抽穗开花:
- 浅水
- 保持湿润

成熟期:
- 排水晒田
- 收获前7-10天断水
```

**施肥**:
```
基肥(整田时):
- 有机肥为主
- 腐熟农家肥

追肥:
- 分蘖肥(移栽后7-10天)
- 穗肥(孕穗期)
- 以氮肥为主,配合磷钾

原则:
- 前促(分蘖)
- 中控(防徒长)
- 后补(壮籽)
```

**病虫害防治**:
```
常见病害:
- 稻瘟病: 选抗病品种,避免偏施氮肥
- 纹枯病: 控制水层,不过密

常见虫害:
- 稻飞虱: 田间灯光诱杀
- 螟虫: 及时排除枯心苗

预防为主!
```

### 收获与储存

**收获**:
```
时间:
- 90%谷粒黄熟
- 早收损失大
- 迟收易落粒

方法:
- 割稻
- 脱粒
- 晾晒(水分<13%)
```

**储存**:
```
✓ 充分干燥
✓ 密封防潮
✓ 防鼠防虫
✓ 通风阴凉处
✓ 定期检查

糙米/稻谷可储存1-2年
```

## 小麦

### 基本信息

```
类型: 主粮作物
周期: 180-240天
产量: 约300-400kg/亩
适宜: 北方、温带
```

### 品种

**按种植季节**:
- 冬小麦: 秋播春收(主要,产量高)
- 春小麦: 春播秋收(高寒地区)

### 种植方法

**整地**:
```
- 深耕(20-25cm)
- 耙细整平
- 施足基肥
```

**播种**:
```
时间:
- 冬小麦: 寒露至霜降(10月)
- 春小麦: 清明前后

方法:
- 条播(行距20-25cm)
- 或撒播

播量:
- 10-15kg/亩

深度:
- 3-5cm
- "深种不过寸"
```

### 田间管理

**冬前管理**(冬小麦):
```
✓ 及时浇灌(出苗水、越冬水)
✓ 中耕除草
✓ 镇压(防冻拔)
✓ 追施腊肥
```

**春季管理**:
```
✓ 返青水肥(早春)
✓ 拔节期追肥
✓ 防倒伏
✓ 病虫害防治
```

**灌溉**:
```
关键期:
- 出苗期
- 越冬前
- 返青期
- 拔节期
- 孕穗期
- 灌浆期

原则:
- 干旱时及时浇
- 雨季注意排水
```

**病虫害**:
```
- 锈病: 选抗病品种
- 赤霉病: 扬花期喷药预防
- 蚜虫: 早防早治
```

### 收获储存

**收获**:
```
时间:
- 蜡熟末期至完熟初期
- 90%籽粒变硬

方法:
- 割晒(割后晾晒再脱粒)
- 或联合收割

晾晒:
- 水分<12%
```

**储存**:
```
✓ 干燥通风
✓ 防虫防鼠
✓ 避光
✓ 可储存1-2年
```

## 玉米

### 基本信息

```
类型: 主粮/饲料
周期: 100-150天
产量: 约500-600kg/亩
适宜: 广泛,适应性强
```

### 品种

- 普通玉米
- 甜玉米(鲜食)
- 糯玉米
- 饲料玉米

### 种植方法

**整地施肥**:
```
- 深耕
- 施足有机肥
- 做垄或平作
```

**播种**:
```
时间:
- 春播: 4-5月(地温稳定12°C以上)
- 夏播: 6月(麦后复种)

方法:
- 穴播或条播
- 行距: 60cm
- 株距: 25-30cm
- 密度: 3000-4000株/亩

深度:
- 5-7cm
- "深种不过寸,浅种不露籽"

每穴:
- 2-3粒
- 出苗后间苗留1株
```

### 田间管理

**苗期**:
```
✓ 间苗、定苗(3-4叶)
✓ 中耕除草
✓ 防治地下害虫(蛴螬、蝼蛄)
```

**拔节期**:
```
✓ 追肥(大喇叭口期)
✓ 培土(防倒伏)
✓ 除蘖(去分蘖)
```

**抽雄开花期**:
```
✓ 充足水分
✓ 防治玉米螟
✓ 人工辅助授粉(小面积,提高结实率)
```

**灌浆成熟期**:
```
✓ 保持水分
✓ 防早衰
✓ 防倒伏
```

**施肥**:
```
基肥: 有机肥为主
追肥:
- 苗肥: 3-4叶期
- 穗肥: 大喇叭口期(重要!)
- 粒肥: 抽雄后

重施穗肥!
```

### 收获储存

**收获**:
```
时间:
- 苞叶枯黄
- 籽粒变硬
- 乳线消失

方法:
- 掰穗
- 剥皮
- 晾晒
```

**储存**:
```
玉米棒:
- 挂起通风晾干
- 水分<14%

玉米粒:
- 脱粒后晾晒
- 密封防潮防虫
- 可储存1-2年
```

## 大豆

### 基本信息

```
类型: 粮食/油料
周期: 100-120天
产量: 150-200kg/亩
特点:
- 蛋白质丰富
- 可榨油
- 豆科固氮(改良土壤)
```

### 种植方法

**播种**:
```
时间: 5-6月(夏大豆)

方法:
- 条播或穴播
- 行距: 40-50cm
- 株距: 15-20cm

深度: 3-5cm
```

**管理**:
```
✓ 苗期中耕除草
✓ 开花结荚期追肥、浇水
✓ 根瘤菌固氮(少施氮肥)
✓ 防治病虫害
```

**收获**:
```
- 叶片脱落
- 豆荚变黄干燥
- 摇动有响声

晾晒后脱粒储存
```

## 高粱

### 基本信息

```
周期: 90-130天
产量: 300-400kg/亩
特点:
✓ 耐旱耐瘠
✓ 适应性强
✓ 可食用、可酿酒、可饲料
```

### 种植

**播种**:
```
- 春夏均可
- 条播或穴播
- 行距50-60cm
- 株距20-25cm
- 深度3-5cm
```

**管理**:
```
✓ 间苗定苗
✓ 中耕除草
✓ 追肥1-2次
✓ 防倒伏
✓ 防治蚜虫
```

## 📝 粮食作物种植要点

### 新手推荐种植顺序

```
1. 土豆/红薯(简单高产,必种!)
2. 玉米(容易,产量稳定)
3. 大豆(改良土壤,蛋白质)
4. 水稻(有水源地区)
5. 小麦(需要经验)
```

### 自给自足配置(1家4口)

**面积约1-2亩**:
```
- 水稻/小麦: 0.5亩(主粮)
- 玉米: 0.3亩
- 土豆/红薯: 0.2亩(高产补充)
- 大豆: 0.1亩(蛋白)
- 其余: 蔬菜

预期产出:
- 稻米/小麦: 250kg
- 玉米: 150kg
- 薯类: 400-800kg
- 大豆: 30kg

可基本满足1年粮食需求(节约使用)
```

### 轮作方案

```
3年轮作:
年1: 水稻/小麦
年2: 玉米/高粱
年3: 大豆/花生

或:
年1: 薯类
年2: 玉米
年3: 豆类

好处:
✓ 平衡土壤养分
✓ 减少病虫害
✓ 提高产量
```

### 种子保存

```
重要!必须留种!

选种:
✓ 选健壮植株
✓ 饱满籽粒
✓ 无病虫害

处理:
✓ 充分干燥
✓ 密封保存
✓ 低温阴凉
✓ 防虫防鼠

注意:
- 杂交种(F1)后代会分离,不适合留种
- 传统品种(传家宝)可留种
- 建议储备传统品种种子!
```

### 储存要点

```
粮食长期储存:
✓ 充分干燥(水分<13%)
✓ 密封容器(防潮防虫)
✓ 阴凉通风处
✓ 避光
✓ 定期检查翻晒

延长储存:
- 真空或充氮包装
- 冷藏
- 混入干燥剂

注意:
- 糙米比白米更营养,但储存期短
- 全谷物营养更好,但需防虫
- 可分批储存,轮换使用
```

### 工具设备

**最小化工具**:
```
- 锄头(翻地、除草)
- 铁锹(挖沟)
- 镰刀(收割)
- 耙子(整地)
- 桶(浇水、运输)
- 筐篮(采收、晾晒)

进阶:
- 打谷机/脱粒机
- 磨米/磨面机
- 扬场簸箕(去糠)
```

### 时间投入

```
1亩粮田(手工):
- 整地: 5-10天
- 播种/移栽: 2-5天
- 日常管理: 每周1-2小时
- 收获: 5-10天
- 加工: 3-7天

总计约20-30工作日/年/亩
```
