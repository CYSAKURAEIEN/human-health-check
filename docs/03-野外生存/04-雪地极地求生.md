# 雪地/极地求生指南

## 核心挑战

```
- 极寒
- 失温风险
- 冻伤
- 暴风雪
- 白茫茫易迷路
- 食物水源困难
```

## 保暖 (最关键!)

### 穿着系统

**多层穿衣法**:
```
1. 内层(排汗):
   - 合成纤维或羊毛
   - NOT棉!(湿后失温)

2. 中层(保暖):
   - 羊毛、抓绒
   - 多层薄胜于一层厚

3. 外层(防风防水):
   - 防水外套
   - 防风

原则:
✓ 可调节(热了脱,冷了穿)
✓ 保持干燥!(湿=death)
```

**重点部位**:
```
✓ 头部(30%热量从头散失):
  - 帽子、头巾

✓ 手脚:
  - 手套(连指更暖)
  - 厚袜
  - 靴子(防水保暖)

✓ 面部:
  - 围巾、面罩
  - 防冻伤

✓ 眼睛:
  - 雪盲防护
  - 太阳镜/护目镜
```

### 保持干燥

```
⚠️ 湿冷=deadly组合!

✓ 脱掉湿衣服
✓ 拧干
✓ 体温烘干
✓ 不出大汗(及时脱层)
✓ 刷掉雪(不要让化)
✓ 防水层
```

## 庇护所

### 雪洞

**最佳选择** (保暖,隐蔽风):

```
步骤:
1. 找深雪堆或积雪坡
2. 挖入口(小,保温)
3. 向上挖:
   - 入口低于睡眠平台
   - 冷空气下沉,暖空气上升
4. 内部:
   - 圆顶状(结构stable)
   - 睡眠平台高于入口
   - 足够空间(但越小越暖)
5. 通气孔!(戳洞,防CO2)
6. 标记入口(防埋)
7. 铺垫隔离雪地

注意:
✓ 必须通气!
✓ 入口可用雪块或背包部分堵
✓ 内温可达0°C(外零下,相对温暖!)
```

### 雪墙

```
挡风:
- 堆雪墙
- 围成半圆或U形
- 背风侧开口
- 配合火堆
```

### 雪沟

```
紧急:
- 挖雪沟
- 铺松枝、草
- 覆盖树枝、雪块
- 简易快速
```

### Quinzhee雪屋

```
方法:
1. 堆大雪堆
2. 等1-2小时(雪固化)
3. 从一侧挖空
4. 向上挖sleeping platform
5. 壁厚20-30cm
6. 通气孔!
```

## 水源

### 融雪

**⚠️ 不要直接吃雪!**
```
原因:
- 降低体温(危险)
- 消耗能量

正确方法:
✓ 融化后饮用
✓ 煮沸(更暖和)
✓ 或体温融化(放水壶贴身,慢)
```

**融雪技巧**:
```
✓ 先放少量水,再加雪(防烧焦容器)
✓ 选干净白雪
✓ 压实雪(比蓬松雪水多)
✓ 不要用黄、粉、脏雪
```

### 冰

```
✓ 冰比雪更好(水多)
✓ 选干净冰
✓ 融化饮用
```

## 火

### 生火挑战

```
困难:
- 湿木
- 大雪覆盖
- 风大
- 火绒难找

解决:
✓ 找枯死树上枝条(干燥)
✓ 劈开湿木用内芯
✓ 松树脂(极好fire starter)
✓ 桦树皮(防水易燃)
✓ 挡风
✓ 雪地:先铺木头平台(防雪融化灭火)
```

### 保持火

```
✓ 准备足够燃料
✓ 防风雪遮蔽
✓ 持续维护
✓ 夜间:灰覆盖保留火种
```

## 食物

### 动物

**可猎**:
```
- 兔、松鼠
- 鸟类
- 鱼(冰钓)
- 大型动物(需技能)

方法:
- 陷阱、snares
- 冰钓(凿冰洞)
- 追踪雪上足迹
```

**所有肉煮熟!**(防寄生虫)

### 注意

```
⚠️ 食物需水消化
- 吃太多脱水
- 平衡水和食
```

## 导航

### 白茫茫易迷路

```
预防:
✓ 指南针/GPS
✓ 地图
✓ 做标记(旗帜、堆雪)
✓ 视线差时不乱走
✓ 暴风雪时停下
✓ 留在原地(如迷路)
```

### 暴风雪

```
如遇:
1. 立即寻找/建造庇护所
2. 雪洞、雪墙
3. 等待过去
4. 不要继续前进(迷路风险)
5. 保持温暖
```

## 危险

### 失温症

**最大杀手!**

见[急性症状-失温](../../01-日常疾病和伤害/05-急性症状.md#失温症)

**预防**:
```
✓ 保暖多层
✓ 保持干燥
✓ 避免出汗
✓ 充足食物(燃料)
✓ 庇护所
✓ 保持活动(但不过度)
```

### 冻伤

**症状**:
```
浅表:
- 皮肤发白、麻木
- 刺痛

深度:
- 皮肤发黑
- 无感觉
- 组织坏死
```

**预防**:
```
✓ 覆盖暴露部位
✓ 保持循环(活动手指脚趾)
✓ 不要太紧(手套、鞋)
✓ 保持干燥
✓ 感觉麻木立即处理
```

**处理**:
```
✓ 转移温暖处
✓ 温水复温(37-40°C)
✓ 不要:
  ❌ 揉搓
  ❌ 用热水、火烤
  ❌ 复温后再冻(更危险)
✓ 就医
```

### 雪盲

**原因**:雪反射UV

**症状**:
```
- 眼痛
- 流泪
- 视力模糊
- 怕光
```

**预防**:
```
✓ 太阳镜/护目镜
✓ 或自制:
  - 布条留缝
  - 木片刻细缝
✓ 阴天也需要!
```

**治疗**:
```
- 避光rest
- 冷敷
- 通常24-48小时恢复
```

### 雪崩

**识别危险**:
```
- 坡度30-45度
- 新降雪后
- 气温骤升
- 坡面无植被
- "嗡嗡"声
```

**避免**:
```
✓ 避开危险坡
✓ 了解路线
✓ 早晨(雪冻结)更安全
✓ 逐一通过(团队)
```

**如被埋**:
```
1. 全力游向上
2. 雪崩停前:
   - 一手护脸(造空气空间)
   - 一手向上(标记位置)
3. 停止后:
   - 扩大空间
   - 向上挖
   - 节省氧气
   - 等待救援
```

危险:
