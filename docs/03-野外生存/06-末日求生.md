# 末日求生指南

⚠️ **本章节提供长期生存策略,适用于社会秩序严重崩溃的极端情况**

## 核心原则

```
1. 安全第一
2. 水、食物、庇护
3. 社区合作
4. 技能>物资
5. 适应和灵活
6. 保持希望
```

## 初期(0-30天)

### 立即行动

**撤离 vs 固守**:
```
撤离,如果:
- 当前位置不安全
- 有安全目的地
- 有交通工具和燃料
- 有足够物资

固守,如果:
- 相对安全
- 有储备
- 有防御
- 熟悉环境
```

### 基本需求

**水**:
```
✓ 储备(每人每天4升)
✓ 净化方法(煮沸、过滤、化学)
✓ 备用水源(雨水收集、附近水源)
✓ 至少2周储备
```

**食物**:
```
✓ 长期储备:
  - 罐头(数年)
  - 干粮(大米、面、豆类)
  - 能量棒
  - 盐、糖、油
✓ 至少1个月储备
✓ 轮换使用(先进先出)
```

**庇护所**:
```
✓ 加固住所
✓ 备用避难所
✓ 防御准备
✓ 保持低调
```

### 安全

**防御**:
```
✓ 加固门窗
✓ 应急逃生路线
✓ 观察哨(了解周围)
✓ 警报系统(improvise)
✓ 保持警惕
```

**人员**:
```
✓ 建立信任的小组
✓ 分工合作
✓ 守望相助
✓ 谨慎对待陌生人
```

## 中期(1-6个月)

### 自给自足

**水源**:
```
✓ 雨水收集系统
✓ 水井(如可能)
✓ 附近溪流
✓ 净化设备
✓ 循环利用(灰水)
```

**食物生产**:
```
✓ 种植:
  - 快速作物(萝卜、生菜-30天)
  - 高产作物(土豆、玉米)
  - 储藏作物(南瓜、红薯)

✓ 养殖:
  - 鸡(蛋、肉)
  - 兔(繁殖快)
  - 鱼(如有水源)

✓ 觅食:
  - 野生植物
  - 狩猎、捕鱼
  - 昆虫
```

**能源**:
```
✓ 柴火(烹饪、取暖)
✓ 太阳能(如有设备)
✓ 人力(自行车发电机等)
✓ 保存热量
```

## 特殊情境

### 核灾难

```
初期:
✓ 躲避(地下室、内部房间)
✓ 密封门窗
✓ 收听信息
✓ 储备的食物和水
✓ 不外出

中期:
✓ 听从官方指导
✓ 避免污染食物和水
✓ 除污(脱衣、淋浴)
✓ 碘片(官方指导下)

长期:
✓ 监测辐射
✓ 远离高污染区
✓ 定期健康检查
```

### 瘟疫

```
✓ 隔离sick
✓ 个人防护(口罩、手套)
✓ 消毒
✓ 避免接触
✓ 增强免疫(营养、rest)
✓ 照顾病患(protective措施)
```

### 社会崩溃

```
✓ 低调(不显富)
✓ 守望相助
✓ 避免冲突
✓ 保持移动性(能快速撤离)
✓ 多个藏身处
✓ 建立信任网络
```
