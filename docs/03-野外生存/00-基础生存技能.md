## 生存优先级

### 3-3-3法则

```
3分钟 - 没有空气
3小时 - 极端环境无庇护所
3天   - 没有水
3周   - 没有食物
```

### 生存任务优先顺序

```
1. 急救(处理伤病)
2. 庇护所(体温调节)
3. 水(生存必需)
4. 火(取暖、煮食、信号)
5. 食物(能量补充)
6. 信号/导航(获救/回家)
```

**记住**:人可以数周不吃饭,但只能数天不喝水!

## 导航与定向

### 使用指南针

```
基础使用:
1. 水平放置
2. 让磁针自由转动
3. 红针指向北(通常)
4. 转动表盘对准方向
5. 沿方向前进
```

### 无指南针辨方向

#### 太阳法

**手表法**(北半球):
```
1. 手表平放
2. 时针指向太阳
3. 时针与12点刻度夹角平分线指向南
4. 简化:时针指向太阳,12点方向大致是北
```

**影子法**:
```
1. 直立木棍
2. 标记影子顶端(A点)
3. 等待15-20分钟
4. 再次标记影子顶端(B点)
5. 连线A-B,指向东
6. 垂直线指向北

原理:太阳东升西落
```

#### 星星法(夜晚)

**北极星**(北半球):
```
找法:
1. 找到北斗七星(勺子形)
2. 勺口两颗星连线
3. 延长约5倍
4. 就是北极星
5. 北极星方向=正北
```

**南十字星**(南半球):
```
1. 找到南十字星
2. 十字长轴延长约4.5倍
3. 垂直到地平线
4. 就是正南方向
```

#### 自然标志

**树木**:
```
北半球:
- 树冠: 南侧茂密
- 树桩年轮: 南侧宽
- 苔藓: 北侧多
```

⚠️ 这些方法不绝对准确,仅作参考

### 避免迷路

```
✓ 经常回头看(记住来时的样子)
✓ 做标记(布条、石堆)
✓ 保持直线(选定远处目标前进)
✓ 记录路线(地标、方向、时间)
✓ 不要离开水源太远
```

### 如果迷路

```
STOP原则:
S - Stop:停下,不要慌
T - Think:思考你的位置
O - Observe:观察周围环境
P - Plan:制定计划

原则:
✓ 留在原地(更易被找到)
✓ 发出信号
✓ 找水源
✓ 建庇护所
✓ 等待救援
```

## 水源获取与净化

### 寻找水源

**地形线索**:
```
✓ 山谷、低洼处
✓ 植被茂盛区
✓ 动物踪迹汇聚处
✓ 昆虫飞行方向(傍晚)
✓ 鸟类飞行(早晚飞向水源)
```

### 收集雨水

```
方法:
✓ 干净容器接水
✓ 塑料布收集(中间低洼)
✓ 大叶子引流
✓ 挖坑铺塑料布

最安全的水源!
```

### 露水收集

```
清晨:
1. 用吸水布料擦草
2. 拧到容器里
3. 重复

可收集1-2升/小时
```

### 植物取水

**蒸腾水收集**:
```
1. 选择绿叶茂盛植物
2. 套透明塑料袋在枝叶上
3. 袋口扎紧
4. 袋内角落放小石子(让水聚集)
5. 数小时后收集

注意:
- 选择无毒植物
- 不要选枯叶
```

**竹子取水**:
```
1. 绿竹节间有水
2. 摇晃听水声
3. 顶部开口
4. 倒出饮用
```

### 太阳能蒸馏器

```
制作:
1. 挖坑(直径约1米,深50厘米)
2. 坑底中央放容器
3. 坑内可放植物(增加水分)
4. 坑上铺塑料布
5. 边缘用土压实
6. 中央放小石子(使塑料布下凹)
7. 凝结水滴入容器

产水:0.5-1升/天
```

### 水质净化

#### 煮沸(最可靠)

```
方法:
- 沸腾至少1分钟
- 高海拔(>2000米):3分钟
- 杀死所有病原体
```

#### 过滤

```
自制过滤器:
材料:瓶子/竹筒

层次(从下到上):
1. 小石子
2. 粗沙
3. 细沙
4. 木炭(烧过的,碾碎)
5. 布料

使用:
- 水从上倒入
- 下方收集
- 去除颗粒和部分杂质
- 仍需煮沸!
```

#### 化学消毒

```
碘片/氯片:
- 按说明使用
- 等待30分钟
- 杀死大部分病原体

漂白剂(次氯酸钠):
- 2滴/升清水
- 4滴/升浑水
- 摇匀,等30分钟
```

#### 紫外线消毒

```
条件:
- 清澈的水
- 透明塑料瓶
- 强烈阳光

方法:
1. 装入透明瓶
2. 阳光直射6小时
3. 或2天阴天
4. 杀死部分病原体
```

### 不能喝的水

```
❌ 避免:
- 死水、臭水
- 颜色异常(黄、红、黑)
- 有泡沫
- 有毒植物附近
- 海水(高盐)
- 尿液(浓缩毒素)
- 血液
```

## 生火技术

### 火的三要素

```
燃料 + 氧气 + 热量 = 火

缺一不可!
```

### 准备材料

**火绒**(最易燃):
```
- 干燥细纤维
- 蒲公英绒、芦苇绒
- 干草、树叶
- 桦树皮
- 棉絮、纸张
- 钢丝绒
```

**引火材**(细燃料):
```
- 细树枝(铅笔粗细)
- 松针
- 干树枝
- 木片
```

**燃料**(主燃料):
```
- 手指粗-手腕粗树枝
- 木柴
- 粗树枝
```

**选择木材**:
```
✓ 干燥木材
✓ 硬木烧得久
✓ 软木易点燃
✓ 松树含树脂(助燃)

❌ 避免:
- 湿木(冒烟不燃烧)
- 绿木(难燃)
- 腐烂木(不好)
```

### 生火方法

#### 现代方法

**打火机/火柴**:
- 最简单
- 保持干燥
- 防风

**打火石**(镁棒):
```
1. 刮镁棒屑到火绒上
2. 用刮铁快速刮打火石
3. 产生火花
4. 点燃镁屑和火绒
5. 添加引火材
```

#### 原始方法

**弓钻法**:
```
工具:
- 弓(弯曲树枝+绳)
- 钻轴(硬木,筷子粗细,削尖一头)
- 钻板(软木,平板)
- 手座(木块,手握)
- 火绒

步骤:
1. 钻板上刻V型缺口
2. 缺口下放火绒
3. 钻轴绕弓弦一圈
4. 钻轴尖端放钻板凹槽
5. 手座压住钻轴顶部
6. 快速拉弓(钻轴旋转)
7. 摩擦生热
8. 产生烟和炭粉
9. 炭粉掉入火绒
10. 轻轻吹,产生火苗
```

**手钻法**:
```
类似弓钻,但用手搓:
- 更费力
- 需要练习
```

**火犁法**:
```
1. 一块木板刻槽
2. 另一根木棍
3. 快速在槽里推拉摩擦
4. 产生炭粉
5. 引燃火绒
```

**透镜聚焦法**:
```
条件:
- 强烈阳光
- 透镜(眼镜、放大镜、冰透镜、水瓶)

方法:
1. 聚焦阳光到火绒上
2. 形成最小亮点
3. 保持不动
4. 开始冒烟
5. 轻吹成火
```

### 火堆结构

**锥形**(快速生火):
```
- 火绒中央
- 引火材围成锥形
- 顶部留通气口
- 点燃火绒
- 逐渐添加粗燃料
```

**木屋式**(持久燃烧):
```
- 井字交叉堆叠
- 中心放火绒和引火材
- 通风好,燃烧旺
```

**平台式**(湿地/雪地):
```
- 底部铺平台(粗木头)
- 隔离湿气
- 上面正常生火
```

**反射墙**(取暖):
```
- 火后方堆石头或木头墙
- 反射热量
- 增强取暖效果
```

### 生火技巧

```
✓ 准备足够材料(湿了不能停下找材料)
✓ 从小到大逐渐添加
✓ 保持氧气供应(不要压实)
✓ 防风(但不能完全挡风)
✓ 耐心(不要急于添粗燃料)
```

### 保存火种

```
方法:
- 用灰覆盖炭火(减少氧气)
- 可保持一夜
- 早上拨开灰,添引火材

或:
- 制作火绳(绳子浸树脂)
- 阴燃携带
```

## 庇护所建造

### 选址

```
✓ 地势较高(防洪水)
✓ 排水良好
✓ 靠近水源(但不太近,防蚊虫)
✓ 靠近材料来源
✓ 避风处
✓ 阳光充足(寒冷环境)
✓ 阴凉处(炎热环境)

❌ 避免:
- 山谷底(冷空气下沉)
- 干河床(山洪)
- 孤立高地(雷电)
- 悬崖下(滚石)
- 枯树下(倒塌)
- 动物巢穴附近
```

### 简易庇护所类型

#### A字庇护所

```
结构:
1. 主梁:长木杆,一端架在树叉或支撑物上
2. 肋骨:斜靠在主梁两侧
3. 覆盖:树枝、树叶、草
4. 底部:铺草、树叶(隔离地面湿气)

优点:
- 简单快速
- 防雨
- 两侧挡风
```

#### 单坡庇护所

```
结构:
1. 两根支柱
2. 横梁连接
3. 一侧斜靠木杆
4. 覆盖防雨材料

优点:
- 简单
- 可加反射墙(火堆前取暖)
```

#### 树叶小屋

```
结构:
1. A字框架
2. 厚厚覆盖树叶(30厘米以上)
3. 从下往上铺(瓦片状)
4. 内部铺干树叶

优点:
- 保暖
- 防水
- 隐蔽
```

#### 雪洞(雪地)

```
步骤:
1. 找雪堆或积雪
2. 挖入口(小,保温)
3. 向上挖(冷空气下沉,热空气上升)
4. 内部:睡眠平台高于入口
5. 顶部戳气孔(通风)
6. 用雪块或背包堵部分入口

注意:
✓ 保持通风!
✓ 入口低于sleeping平台
✓ 内部不要太湿(体温融化)
```

### 改进庇护所

```
✓ 防水:
  - 塑料布、雨衣
  - 厚层树叶(瓦片状)
  - 树皮

✓ 保温:
  - 厚覆盖物
  - 减小空间(保存体温)
  - 反射层(在热带用树叶反射热,寒带用emergency blanket反射体温)
  - 底部隔离层(厚树叶、草)

✓ 防虫:
  - 烟熏
  - 网状遮挡
  - 选址(通风处)

✓ 舒适:
  - 平整地面
  - 软垫
  - 排水沟(周围挖沟防雨水倒灌)
```

## 食物获取

详见各环境特定章节。基础原则:

### 可食植物测试

⚠️ 仅紧急情况使用!很多植物有毒!

```
通用可食性测试:
1. 确认只有一个人测试一种植物
2. 分离植物部分(叶、茎、根、果)
3. 闻:刺鼻气味?避免
4. 皮肤测试:内手腕,等15分钟,红肿?
5. 唇部:放嘴唇,等3分钟,刺激?
6. 舌部:舌尖,等15分钟,刺激?
7. 咀嚼:不吞,等15分钟,不适?
8. 吞咽:小量,等5小时,观察反应
9. 如无不适,可少量食用
10. 等24小时确认安全

每次只测一种!
```

### 昆虫

```
可食(煮熟):
✓ 蚱蜢、蟋蟀
✓ 蚂蚁(去头)
✓ 白蚁
✓ 甲虫幼虫
✓ 蠕虫

避免:
❌ 鲜艳颜色
❌ 有毛
❌ 散发臭味
❌ 蜘蛛、蜱虫

准备:
- 去头、翅、腿
- 煮熟或烤
- 高蛋白!
```

###捕鱼

```
方法:
- 钩和线(制作鱼钩:弯针、骨头、木头削尖)
- 鱼叉(快速刺)
- 陷阱(围困in浅水)
- 手抓(在石头下)

优点:
- 容易获取
- 营养丰富
- 相对安全
```

## 基础工具制作

### 刀具

```
石刀:
1. 找坚硬石头(燧石、黑曜石)
2. 用另一石头敲击边缘
3. 形成锋利边
4. 绑在木棍上

竹刀:
1. 斜切竹子
2. 边缘锋利
```

### 容器

```
竹筒:
- 节间作容器
- 煮水(不要太靠近火)

树皮:
- 桦树皮做容器
- 缝合边缘

叶子:
- 大叶子包裹
- 煮食(石头煮:烧热石头放水里)
```

### 绳索

```
材料:
- 植物纤维
- 动物筋腱
- 藤蔓

制作:
1. 取长纤维
2. 搓成小股
3. 两股交叉拧
4. 形成绳
```
