# 森林求生指南

## 方向辨别

### 不要迷路

```
✓ 沿已有小路/河流
✓ 做标记(布条、刻痕、石堆)
✓ 记住地标
✓ 定期回头看
✓ 河流一般通向人烟
```

### 已迷路

```
STOP:
S - Stop:停下
T - Think:思考
O - Observe:观察
P - Plan:计划

原则:
- 留在原地(搜救易找到)
- 沿河流下走(通向文明)
- 往高处(观察、信号)
```

## 水源

### 寻找

```
线索:
✓ 植被茂盛
✓ 动物踪迹汇集
✓ 昆虫多
✓ 鸟类飞向(傍晚)
✓ 山谷低洼处
✓ 跟随动物踪迹
```

### 水源

**优先级**:
```
1. 流动溪流(上游)
2. 泉水
3. 雨水
4. 露水
5. 植物蒸腾水
6. 池塘/湖水(必须净化)
```

**净化**:
```
⚠️ 所有野外水源都应净化!
- 煮沸(最佳)
- 过滤+煮沸
- 化学消毒
```

## 庇护所

### 自然庇护所

```
利用:
✓ 洞穴(检查无动物)
✓ 大树下(临时)
✓ 倒下的树干旁
✓ 岩石突出部
```

### 建造庇护所

**A字棚**:
```
1. 找主梁(粗长木棍)
2. 一端架高(叉树枝)
3. 另一端触地
4. 两侧斜靠木棍
5. 密铺树枝、树叶(瓦片状)
6. 越厚越好(30cm+)
7. 内铺干树叶
```

**树叶小屋**:
```
- 框架:A字
- 厚覆盖树叶
- 入口小
- 保暖效果好
```

**防雨tips**:
```
✓ 瓦片状铺设(下层在外)
✓ 树皮作顶层
✓ 坡度足够(水流走)
✓ 周围挖排水沟
```

## 火

### 选址

```
✓ 平坦地面
✓ 避风但通风
✓ 近庇护所(但安全距离)
✓ 近木材来源
✓ 清除周围可燃物
✓ 挖坑或围石
```

### 湿木生火

**找干材料**:
```
✓ 枯死树上的小树枝(未落地)
✓ 树皮下干燥层
✓ 松树脂(易燃)
✓ 桦树皮(防水易燃)
✓ 劈开湿木(inside干燥)
```

**方法**:
```
1. 准备大量火绒
2. 劈开湿木露出干芯
3. 削薄片(feather stick)
4. 生小火
5. 烤干湿木
6. 逐渐添加
```

## 食物

### 可食植物

**常见可食** (确认后):
```
✓ 橡子(去涩,煮熟)
✓ 蒲公英(全株)
✓ 车前草
✓ 野蒜野葱
✓ 某些野果(确认)
✓ 蕨菜嫩芽
✓ 竹笋
✓ 木耳、蘑菇(⚠️专业识别!)
```

**避免**:
```
❌ 白色浆果(大多毒)
❌ 聚合浆果的白色、黄色种类
❌ 乳白汁液(大多毒)
❌ 苦杏仁味
❌ 伞形花序(很多毒)
❌ 三片叶(很多毒)

⚠️ 蘑菇:不确定绝不吃!(致命)
```

### 动物蛋白

**昆虫**:
```
可食(煮熟):
✓ 蚱蜢、蟋蟀
✓ 白蚁
✓ 甲虫幼虫(朽木中)
✓ 蚯蚓(挤出内容物,冲洗)
✓ 蚂蚁(去头)

避免:
❌ 鲜艳颜色
❌ 有毛
❌ 臭味强烈
```

**小动物**:
```
陷阱捕猎:
- 简易绳索陷阱
- 挖陷坑
- 落石陷阱

⚠️ 需技能和时间
- 能量消耗可能>获得
- 优先昆虫和植物
```

**钓鱼**:
```
溪流中:
✓ 制作鱼钩(弯针、骨头)
✓ 鱼线(伞绳内芯)
✓ 诱饵(虫子)
✓ 或徒手(石头下)
✓ 鱼叉
```

## 危险

### 动物

**大型动物**:
```
遇到熊:
- 保持冷静
- 缓慢后退
- 不要奔跑
- 不要直视
- 发出声音示警
- 使自己显大(举高手臂)
- 如攻击:
  * 棕熊:装死
  * 黑熊:反击

遇到野猪:
- 不要靠近
- 爬树
- 背对大树

遇到蛇:
- 远离
- 不要惊动
- 检查睡袋、鞋子
- 用棍子探路
```

**昆虫**:
```
✓ 检查身体(蜱虫)
✓ 烟熏驱蚊
✓ 泥巴涂皮肤(防虫咬)
✓ 检查衣物、睡袋
```

### 植物

**有毒植物**:
```
❌ 毒藤、毒橡树(三片叶,触碰过敏)
❌ 毒芹(致命)
❌ 夹竹桃(全株有毒)

原则:
- 不认识不吃
- 不确定不碰
```

### 疾病

**预防**:
```
✓ 净化水
✓ 煮熟食物
✓ 个人卫生
✓ 避免积水(疟疾)
✓ 伤口清洁
```

## 导航

### 沿河下行

```
✓ 河流通向人烟
✓ 水源保证
✓ 食物(鱼)
✓ 相对平坦
```

### 找高处观察

```
✓ 爬到高处/树上
✓ 观察:
  - 烟(人烟)
  - 灯光(夜晚)
  - 道路、建筑
  - 开阔地
  - 河流方向
```

### 留记号

```
✓ 布条(鲜艳)
✓ 树干刻记号
✓ 石堆
✓ 折断树枝指向

作用:
- 搜救队找到
- 自己不走回头路
```

## 📝 森林求生要点

```
优先级:
1. 水(溪流、雨水、植物)
2. 庇护所(防雨保暖)
3. 火(驱兽、取暖、信号)
4. 食物(昆虫、植物、鱼)
5. 导航/信号

技巧:
✓ 沿河走
✓ 做标记
✓ 留在原地(如等救援)
✓ 利用天然庇护所
✓ 木材充足(火)

危险:
⚠️ 迷路
⚠️ 野兽
⚠️ 有毒动植物
⚠️ 失温(夜晚)
```
