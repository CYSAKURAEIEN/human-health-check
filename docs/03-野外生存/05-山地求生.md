# 山地求生指南

## 核心挑战

- 高原反应
- 气温低、风大
- 地形险峻
- 易迷路、坠落
- 天气多变
- 紫外线强

## 高原反应

### 症状

**轻度**:
- 头痛、头晕
- 乏力
- 失眠
- 食欲差

**中度**:
- 剧烈头痛
- 恶心呕吐
- 呼吸困难

**重度**(危险):
- 肺水肿:咳嗽、粉红痰、呼吸极度困难
- 脑水肿:意识障碍、共济失调、昏迷

### 预防和处理

```
预防:
✓ 逐渐上升(每天<500米海拔)
✓ 充分适应(每1000米休息1-2天)
✓ 保持水分
✓ 避免剧烈运动
✓ 高碳水饮食

轻中度:
✓ 休息、不再上升
✓ 补水
✓ 止痛药(头痛)
✓ 如恶化,下降!

重度:
⚠️ 立即下降!(生死攸关)
- 下降500-1000米
- 吸氧(如有)
- 就医
```

## 水源

```
山地水源丰富:
✓ 溪流
✓ 泉水
✓ 融雪
✓ 雨水
✓ 露水

优先:
- 高海拔源头
- 流动水

净化:
- 煮沸(高海拔需更久)
- 过滤+消毒
```

## 庇护所

### 选址

```
✓ 避风(山坳、岩石后)
✓ 避开山顶(风大、雷击)
✓ 避开谷底(冷空气聚集)
✓ 避开滑坡、滚石区
✓ 靠近水源(但不在河边-山洪)
✓ 平坦处

❌ 避免:
- 孤立高处(雷电)
- 悬崖边
- 干河床(山洪)
- 雪檐下(avalanche)
```

### 利用地形

```
天然庇护:
✓ 洞穴(检查safe)
✓ 岩石突出部
✓ 大石块后(挡风)
✓ 茂密树丛

改进:
- 石墙挡风
- 覆盖遮雨
- 底部隔离
```

## 导航

### 地形辨识

```
✓ 地图+指南针
✓ 明显地标(山峰、河流)
✓ 观察地形
✓ 跟随河流(下山)
```

### 下山原则

```
通常:
✓ 河流往下→山谷→人烟
✓ 沿山脊(视野好,不迷路)
✓ 避开悬崖、峡谷
```

### 迷路

```
✓ 往高处观察
✓ 寻找河流(跟随下山)
✓ 留在原地(如等救援)
✓ 发信号
```

## 危险

### 坠落

```
预防:
✓ 小心行走
✓ 试探石头(可能松动)
✓ 三点支撑(always保持3点接触)
✓ 避开悬崖边
✓ 雨雾天减少移动
```

### 滚石/山体滑坡

```
识别:
- 松动岩石
- 裂缝
- 新落石痕迹

避免:
✓ 不在陡坡下
✓ 快速通过危险区
✓ 雨后extra小心
```

### 雷电

```
山地雷电危险!

如遇雷暴:
1. 下到低处
2. 远离山顶、山脊
3. 不在孤立树下
4. 双脚并拢蹲姿
5. 远离金属物
6. 进入低洼处(但not水流处)
```

### 天气骤变

```
山地天气多变:
✓ 关注天气
✓ 早出发(午后易thunderstorm)
✓ 准备防雨防寒
✓ 天气差时不冒险
```

## 食物

```
高山:
- 植被稀少
- 动物少

可能:
✓ 鱼(溪流)
✓ 昆虫
✓ 小型动物
✓ 某些高山植物(需识别)
✓ 浆果(确认可食)
```

## 信号

```
山地:
✓ 镜子反光(远距可见)
✓ 烟雾(峡谷中上升明显)
✓ 哨声(山谷回声)
✓ 鲜艳衣物(对比背景)
✓ 往开阔高地(更visible)
```

## 📝 山地求生要点

```
优先级:
1. 高原适应(防高反)
2. 避免坠落(careful行走)
3. 庇护所(防风寒)
4. 水(相对容易)
5. 火、食物

关键:
✓ 逐渐上升
✓ 小心行走
✓ 避雷电
✓ 跟随河流下山
✓ 注意天气

危险:
⚠️ 高原反应(下降!)
⚠️ 坠落
⚠️ 失温
⚠️ 雷电
⚠️ 滑坡
```
