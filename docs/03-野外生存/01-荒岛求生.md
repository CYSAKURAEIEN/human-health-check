# 荒岛求生指南

## 优先任务

### 第一24小时

```
1. 急救处理伤病
2. 寻找/建造临时庇护所
3. 寻找淡水
4. 发出求救信号
5. 探索环境
```

## 水源

### 海岛取水

**雨水**(最佳):
```
✓ 收集雨水(容器、塑料布)
✓ 树叶引流
✓ 挖坑收集
✓ 最安全的水源
```

**露水**:
```
- 清晨用布擦草
- 拧到容器
```

**太阳能蒸馏器**:
```
1. 挖坑,中央放容器
2. 可放海水或湿沙(不直接接触容器)
3. 铺塑料布,边缘压实
4. 中央放石子
5. 蒸发凝结成淡水
6. 产量:0.5-1升/天
```

**植物取水**:
```
- 绿椰子(大量淡水)
- 竹子节间
- 藤蔓(切断,水滴出)
- 蒸腾水(塑料袋套枝叶)
```

**地下水**:
```
在沙滩:
1. 高潮线以上挖坑
2. 深1-2米
3. 可能渗出淡水(比海水轻,浮在上层)
4. 必须净化!
```

**⚠️ 绝不**:
```
❌ 喝海水!(脱水faster)
❌ 喝尿液!(浓缩毒素)
```

## 庇护所

### 选址

```
✓ 靠近水源
✓ 高于高潮线!
✓ 避风
✓ 接近信号区(海滩可见)
✓ 靠近资源(木材、食物)

❌ 避免:
- 潮水可达处
- 椰子树正下方(掉椰子dangerous)
- 岩石下(涨潮淹没)
```

### 简易庇护所

**单坡棚**:
```
1. 找两棵树或竖柱子
2. 架横梁
3. 斜靠木棍
4. 覆盖:
   - 棕榈叶(瓦片状,从下往上)
   - 草
   - 树皮
5. 地面铺干草、树叶
```

**改进**:
```
✓ 高离地面(防虫、潮湿、涨潮)
✓ 厚覆盖(防雨)
✓ 烟熏防蚊虫
```

## 火

### 生火

**火绒**:
```
- 干椰子壳纤维(excellent)
- 干树皮
- 干草
```

**燃料**:
```
✓ 漂流木(充足)
✓ 先晒干
✓ 硬木烧得久
```

**保护**:
```
✓ 防风
✓ 防雨(遮蔽)
✓ 保持火种(灰覆盖)
```

## 食物

### 海洋食物

**贝类**(最容易):
```
✓ 岩石上的牡蛎、藤壶
✓ 沙滩上的蛤蜊(看气孔挖)
✓ 低潮时收集

安全:
- 煮熟!
- 避免红潮季节
- 新鲜食用
```

**鱼**:
```
方法:
- 鱼叉(削尖木棍,多叉)
- 鱼钩(骨头、木头弯曲)
- 潮汐池(退潮后困住)
- 简易鱼网(衣服)

技巧:
- 夜晚用火把吸引
- 涨潮时鱼进浅滩
```

**螃蟹**:
```
- 夜晚活跃
- 用灯/火把引诱
- 小心钳子
- 煮熟
```

### 陆地食物

**椰子**:
```
获取:
- 地上捡(检查是否新鲜)
- 爬树摘(dangerous)
- 用长棍钩

开椰子:
1. 找椰子的"眼睛"(三个点)
2. 用尖石/棍戳破
3. 倒出椰汁
4. 砸开取椰肉
```

**可食植物**:
```
常见:
- 棕榈心(嫩芽)
- 香蕉、芭蕉
- 木瓜
- 海草(某些种类)

⚠️ 不确定不要吃!
```

**昆虫**:
```
✓ 甲虫幼虫(腐木中)
✓ 白蚁
✓ 蚱蜢
- 煮熟食用
```

## 信号与救援

### 信号方法

**烟雾**:
```
白天最有效:
- 三堆火(三角形)
- 添湿草、树叶产生浓烟
- 白烟(湿草)或黑烟(橡胶、油)
- 保持燃烧
```

**火光**:
```
夜晚:
- 三堆火
- 火焰高
- 可见数公里
```

**反光**:
```
材料:
- 镜子、金属片
- CD、手机屏幕
- 磨光的罐头

使用:
- 向天空、地平线扫射
- 看到飞机/船立即信号
- V字手势瞄准
```

**地面标记**:
```
沙滩/空地:
- SOS(大字,至少3米高)
- HELP
- X(需要援助)
- 用石头、木头、海草摆放
- 与周围对比明显(深色/浅色)
```

**声音**:
```
- 哨子(三声短)
- 敲击金属
- 呼喊
```

**旗帜**:
```
- 鲜艳衣物
- 绑在高处
- 挥舞
```

### 保持可见

```
✓ 保持海滩信号火准备好
✓ 定期到海滩观察
✓ 制作高处观察台
✓ 保持信号标记清晰
```

## 危险

### 动物

**鲨鱼**:
```
避免:
- 黄昏/夜晚游泳
- 出血时下水
- 浑水
- 单独游泳

如遇到:
- 保持冷静
- 面对它
- 缓慢移动
- 如攻击:打鼻子、眼睛、鳃
```

**有毒生物**:
```
- 水母:远离,vinegar治疗
- 海蛇:剧毒,不要触碰
- 石鱼:伪装,看清再踩
- 海胆:刺入皮肤,尿液或醋软化后取出
```

**陆地危险**:
```
- 蛇:小心行走
- 蝎子:检查鞋子、衣物
- 蜘蛛:检查庇护所
```

### 环境

**太阳**:
```
✓ 遮阳(帽子、庇护所)
✓ 避免10-16点户外
✓ 保持水分
```

**潮汐**:
```
✓ 观察高潮线
✓ 不要在潮汐区建庇护所
✓ 了解潮汐时间(约每12小时一次)
```

## 长期生存

### 资源管理

```
✓ 不过度捕捞一处
✓ 轮流利用资源点
✓ 保存食物(烟熏、晒干)
✓ 储备淡水(容器)
✓ 保持火种
```

### 保持健康

```
✓ 个人卫生(防感染)
✓ 营养均衡
✓ 充足饮水
✓ 伤口护理
✓ 避免过度劳累
✓ 保持希望和积极
```

### 日常routine

```
建立规律:
- 早:检查信号、收集露水
- 日间:改进庇护所、觅食、制作工具
- 傍晚:准备食物、水、燃料
- 夜间:保持火、休息
```

## 📝 荒岛求生要点

```
优先级:
1. 淡水(生存3天)
2. 庇护所(防sun、雨、风)
3. 火(煮食、信号、驱虫)
4. 食物(海洋资源丰富)
5. 信号(持续努力)

心态:
✓ 保持希望
✓ 保持忙碌
✓ 设定目标
✓ 记录日期
✓ 相信会获救
```
