# 沙漠求生指南

## 核心挑战

```
- 极度缺水
- 高温(日)
- 低温(夜)
- 暴晒
- 迷失方向
- 食物匮乏
```

## 水源 (最关键!)

### 保存体液

```
⚠️ 水比食物重要100倍!

减少水分流失:
✓ 避免日间活动(阴凉处休息)
✓ 夜间/清晨/傍晚行动
✓ 保持衣物(防晒保湿)
✓ 用嘴呼吸(鼻呼吸减少蒸发)
✓ 小口喝水(不要牛饮)
✓ 不要进食(消化消耗水)
✓ 保持冷静(焦虑增加water loss)

❌ 不要:
- 白天行走
- 脱光衣服(增加蒸发)
- 饮酒
- 吸烟
- 过度劳动
- 说话过多
```

### 寻找水源

**地形线索**:
```
✓ 干河床(挖底部,可能有地下水)
✓ 岩石裂缝(清晨有露水)
✓ 植被聚集处
✓ 动物踪迹汇聚
✓ 鸟类飞行方向(傍晚)
✓ 蜜蜂活动(距水<5km)
✓ 苍蝇(距水<100m)
```

**挖掘取水**:
```
干河床:
1. 选择外弯、低处
2. 挖坑至湿沙
3. 继续挖
4. 等待渗水
5. 必须净化!

特点:
- 费力
- 可能无水
- 最后手段
```

**露水收集**:
```
清晨:
- 用布擦岩石、植物
- 拧到容器
- 可收集少量
```

**植物取水**:
```
仙人掌:
⚠️ 谨慎!很多有毒!
- 桶状仙人掌(某些种类)
- 切开,捣碎,挤汁
- 苦味?有毒,不喝
```

**太阳能蒸馏器**:
```
效率低但可靠:
1. 挖坑(1m直径,50cm深)
2. 底部中央放容器
3. 坑内放植物、湿沙(增加水分)
4. 铺透明塑料布
5. 边缘压实密封
6. 中央放小石子(向下凹)
7. 蒸发→凝结→滴入容器

产量:0.5-1升/天
值得!
```

### 配给水

```
原则:
✓ 不要配给!该喝就喝!
✓ 脱水比缺水更危险
✓ 小口慢饮
✓ 湿润嘴巴

注意:
- 不喝水不会"适应"
- 只会脱水more严重
```

## 庇护所

### 防晒(白天)

**优先级最高!**

```
选址:
✓ 阴凉处(岩石、峡谷)
✓ 避免沙地(反射、辐射热)
✓ 通风

建造:
1. 利用自然阴影
2. 或搭建遮阳:
   - 任何布料、塑料
   - 离地面30-45cm(空气流通)
   - 不要触碰地面(热传导)
3. 两层更好(隔热)
```

**白天活动**:
```
❌ 不要在10:00-18:00暴晒下行动
✓ 待在阴凉处
✓ 节省体力和水分
```

### 保暖(夜晚)

**沙漠夜晚寒冷!**

```
方法:
✓ 挖浅坑(沙堆保温)
✓ 堆沙围挡风
✓ 火堆(如有燃料)
✓ 多层衣物
✓ 保暖blanket
✓ 活动身体
```

## 导航

### 白天

**避免迷失**:
```
✓ 沿车辙、小路
✓ 做标记
✓ 记住来时路
✓ 观察地标
✓ 使用指南针/GPS

⚠️ 极易迷失方向!
- 地形单一
- 缺乏地标
- 海市蜃楼
```

**沙尘暴**:
```
如遇到:
1. 立即寻找遮蔽
2. 背风方向
3. 用布遮住口鼻
4. 躺下(避免吸入沙尘)
5. 保护眼睛
6. 等待过去
```

### 夜间

```
✓ 星星导航(参见基础技能)
✓ 温度较低
✓ 可行动(如必须)
✓ 小心视线
```

### 是否留在原地?

**留在车旁**(如有):
```
✓ 更易被发现(大目标)
✓ 有遮蔽
✓ 可能有水、食物
✓ 镜子、轮胎(信号)

除非:
- 确知方向和距离
- 有足够水
- otherwise留下!
```

## 行动

### 何时行动

```
✓ 夜间、清晨、傍晚
✓ 温度较低
✓ 保存体力
✓ 减少水分流失
```

### 如何行走

```
✓ 稳定pace
✓ 不要赶路
✓ 定期休息
✓ 阴凉处休息
✓ 少量喝水(频繁)
✓ 避免出汗过多
```

### 穿着

```
✓ 保持衣物穿着:
  - 防晒
  - 减少蒸发
  - 防风沙
✓ 浅色宽松
✓ 长袖长裤
✓ 头部遮盖(improvise头巾)
✓ 太阳镜/眼部保护
✓ 坚固鞋子(沙地烫)
```

## 食物

### 原则

```
⚠️ 不吃比乱吃好!

- 食物需水消化
- 缺水时少吃或不吃
- 水优先!
```

### 可能食源

**植物**(极少):
```
- 仙人掌果(某些可食)
- 沙漠植物种子
- ⚠️ 需确认,大多有毒或不可食
```

**动物**:
```
- 蜥蜴、蛇(煮熟)
- 昆虫(蚂蚁、甲虫)
- 鸟蛋
- 小型哺乳动物

获取:
- 夜间活动多
- 岩石下
- 简易陷阱
```

## 信号

### 视觉

**镜子反光**:
```
⚠️ 沙漠最有效信号!
- 可见数十公里
- 向天空、地平线扫射
- 看到飞机/车立即信号
```

**烟雾**:
```
- 白天有效
- 轮胎、油(黑烟)
- 保持准备好
```

**地面标记**:
```
- SOS(大石头、深色材料)
- 至少3米高字母
- 与沙漠对比
```

**火光**:
```
- 夜晚
- 三堆火
```

### 声音

```
- 哨子
- 喊叫(省点体力)
```

## 危险

### 中暑

**症状**:
```
- 体温高
- 皮肤干热
- 意识模糊
- 停止出汗

⚠️ 致命!
```

**预防**:
```
✓ 遮阳
✓ 补水
✓ 避免日间活动
✓ 保持通风
```

### 脱水

**征兆**:
```
轻度:
- 口渴
- 尿色深

中度:
- 极度口渴
- 乏力
- 头晕
- 少尿

重度:
- 不再口渴(危险!)
- 无尿
- 意识模糊
- 皮肤干燥失弹性
```

### 动物

**蝎子**:
```
- 岩石下、鞋子里
- 检查睡袋、衣物
- 晚上活动
- 蜇伤:冰敷,就医
```

**蛇**:
```
- 日间藏身阴凉处
- 晚上活动
- 用棍子探路
- 检查岩石缝隙
```

**蜘蛛**:
```
- 检查鞋子、衣物
- 小心岩石下
```

## 📝 沙漠求生要点

```
记住:
1. 水是生命!(找水、省水、存水)
2. 遮阳防晒(白天不活动)
3. 保暖(夜晚温差大)
4. 不要迷路(标记、地标)
5. 留在车旁(如可能)
6. 信号(镜子反光最有效)

优先级:
水 >> 庇护所 >>> 其他

不吃不喝:
- 可活3周
不喝水in沙漠:
- 可能<3天

⚠️ 水是一切!
```
