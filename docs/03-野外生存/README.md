## 🔑 生存优先级:3-3-3法则

记住生存的关键时限:

```
3分钟 - 没有空气(窒息)
3小时 - 恶劣环境下无庇护所(失温/中暑)
3天   - 没有水
3周   - 没有食物
```

**生存优先顺序**:
1. 空气/呼吸
2. 庇护所/体温调节
3. 水
4. 火
5. 食物
6. 信号/救援

## 🔥 生火方法

### 火的三要素

```
燃料 + 氧气 + 热量 = 火
```

### 火种层次

1. **火绒**(最易燃):
   - 干草、树叶
   - 树皮纤维
   - 棉绒
   - 桦树皮

2. **引火材**(细燃料):
   - 细树枝、小树棍
   - 干树枝
   - 松针

3. **燃料**(主要燃料):
   - 粗树枝
   - 木柴
   - 干木头

### 生火结构

**锥形火堆**:
```
用途: 快速生火、照明
结构: 引火材围成锥形,中心放火绒
```

**木屋式**:
```
用途: 持久燃烧、烹饪
结构: 木材交叉堆叠成井字形
```

**平台式**:
```
用途: 湿地、雪地
结构: 底部铺平台,上面生火
```

### 生火方法

**现代方法**:
- 打火机
- 防水火柴
- 打火石(镁棒)

**原始方法**:
- 弓钻法
- 手钻法
- 火犁法
- 透镜聚焦(晴天)

### 火的维护

```
✓ 准备充足燃料
✓ 防风防雨
✓ 控制火势大小
✓ 永不离开无人看管的火
✓ 完全熄灭再离开
```

### 安全用火

```
选址:
✓ 远离帐篷、树木
✓ 清除周围可燃物
✓ 挖坑或围石

防火:
✓ 控制火势
✓ 准备灭火水/土
✓ 不要在大风天生火

环保:
✓ 使用枯木(不砍活树)
✓ 用完全熄灭
✓ 恢复原状
```

## 💧 寻找和净化水源

### 寻找水源

**地形线索**:
- 山谷、低洼处
- 植被茂盛处
- 动物踪迹聚集
- 昆虫活动区域
- 跟随鸟类(傍晚飞向水源)

**收集方法**:
- 雨水收集
- 露水收集(清晨用布擦草,然后拧出水)
- 雪/冰融化
- 植物蒸腾水(塑料袋套植物枝叶)
- 太阳能蒸馏器(挖坑+塑料布+容器)

### 水质判断

```
避免:
❌ 死水、臭水
❌ 颜色异常
❌ 泡沫过多
❌ 有毒植物附近

优先:
✓ 流动水
✓ 泉水
✓ 清澈水
✓ 高海拔水源
```

### 净化方法

**煮沸**(最有效):
```
- 沸腾至少1分钟
- 高海拔(>2000米):3分钟
- 杀死所有病原体
```

**过滤**:
```
自制过滤器:
1. 容器底部打孔
2. 从下到上分层:
   - 小石子
   - 砂砾
   - 细沙
   - 木炭(烧过的)
   - 布料
3. 水从上倒入,下方收集
4. 仅去除大颗粒,还需煮沸
```

**化学处理**:
- 净水片(碘片、氯片)
- 按说明使用
- 等待时间(通常30分钟)

**紫外线**(晴天):
- 清澈的水装入透明塑料瓶
- 阳光直射6小时
- 杀死部分病原体

**紧急情况**:
```
如必须喝未净化水:
✓ 选择流动清澈的
✓ 尽量煮沸
✓ 少量多次
```

## 🏠 庇护所原则

### 选址

```
✓ 考虑因素:
  - 地势平坦、排水良好
  - 靠近水源(但不太近,防洪水/蚊虫)
  - 有材料来源
  - 避风处
  - 避免危险(滚石、枯树、野兽巢穴)

❌ 避免:
  - 山谷底部(冷空气下沉)
  - 孤立高地(闪电风险)
  - 干河床(山洪)
  - 悬崖下(滚石)
```

### 类型

**天然庇护所**:
- 洞穴(检查无野兽)
- 大树下
- 岩石突出部

**简易庇护所**:
- A字棚
- 单坡棚
- 树枝小屋
- 雪洞(雪地)
- 沙堆遮蔽(沙漠)

### 保暖/散热

**保暖**(寒冷环境):
```
✓ 绝缘层:树叶、草、雪
✓ 防风防湿
✓ 减小空间(保存体温)
✓ 火堆反射墙
```

**散热**(炎热环境):
```
✓ 通风良好
✓ 遮阳
✓ 离地(空气流通)
✓ 浅色材料反射
```

## 🍎 野外食物

### 觅食原则

```
✓ 能量消耗 < 获得能量
✓ 水比食物重要
✓ 优先容易获得的
✓ 多样化饮食
```

### 可食植物

**通用可食性测试**(仅用于紧急情况):
```
⚠️ 谨慎!很多植物有毒!

1. 分离部位(叶、茎、根、果)
2. 闻:刺激性气味?避免
3. 皮肤测试:揉搓内手腕,等15分钟,是否红肿?
4. 嘴唇测试:放嘴唇上,等3分钟
5. 舌头测试:舌尖,等15分钟
6. 咀嚼测试:咀嚼不吞,等15分钟
7. 吞咽测试:小量,等5小时
8. 如无不适,可食用

每次只测试一种植物!
```

**常见可食植物**:
- 橡子(煮熟去涩)
- 蒲公英(全株)
- 车前草
- 芦苇根
- 野葱野蒜
- 野生浆果(确认可食)
- 松树(内皮、松针茶)

**绝对避免**:
- 白色浆果
- 伞形花序(很多有毒)
- 三片叶植物(很多有毒)
- 乳白汁液植物(大多有毒)

### 动物蛋白

**昆虫**(高蛋白):
```
可食:
✓ 蚱蜢、蟋蟀
✓ 白蚁、蚂蚁
✓ 蠕虫
✓ 甲虫幼虫

避免:
❌ 鲜艳颜色
❌ 有毛的
❌ 有刺激性气味

烹饪:
- 煮熟或烤熟
- 去掉翅膀和腿
```

**钓鱼**:
- 简易鱼钩(弯曲的针、骨头)
- 鱼线(伞绳内芯)
- 诱饵(虫子、小鱼)
- 鱼叉

**捕猎**(需技能和陷阱):
- 小型哺乳动物
- 鸟类
- 爬行动物

**安全**:
```
✓ 所有肉类煮熟
✓ 避免病态动物
✓ 谨慎处理(防咬伤)
```

## 🆘 求救与信号

### 国际求救信号

**SOS**: ... --- ... (短短短-长长长-短短短)

**三规则**:
- 三声哨声
- 三堆火
- 三个反光
- 间隔相等,重复

### 信号方法

**视觉信号**:
```
- 镜子/反光物(白天,可见数公里)
- 鲜艳衣物挥舞
- 火堆(夜晚)
- 烟雾(白天):
  * 白烟:湿草、树叶
  * 黑烟:油、轮胎、塑料
- 地面标记:
  * SOS
  * HELP
  * X(需要援助)
  * 箭头(指示方向)
  * 尽可能大(至少3米)
  * 与周围对比明显
```

**声音信号**:
- 哨子(省力)
- 敲击金属
- 大声呼救

**电子信号**:
- 手机(尽量登高,节省电量)
- GPS设备
- 个人定位信标(PLB)

### 提高可见性

```
✓ 到开阔地、高处
✓ 穿鲜艳衣物
✓ 移动(更易被发现)
✓ 制造对比(雪地用深色,草地用浅色)
```

### 飞机求救

```
如看到飞机:
1. 挥舞鲜艳衣物
2. 镜子反光
3. 点燃信号火
4. 烟雾信号

地面信号(国际通用):
✓ I - 需要医疗
✓ II - 需要物资
✓ X - 无法前进
✓ F - 需要食物和水
✓ ▲- 可能在这方向
✓ Y - 是/同意
✓ N - 否/拒绝
```
