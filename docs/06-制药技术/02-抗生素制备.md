# 抗生素制备

> **⚠️ 极高风险 - 仅限绝境使用**

## 🚨 严重警告

**本章节内容极度危险!**

**风险清单:**
- ☠️ 过敏反应: 青霉素过敏可致死(过敏性休克)
- ☠️ 无法保证纯度: 杂质导致严重副作用
- ☠️ 剂量无法精确: 过量中毒,不足无效
- ☠️ 无菌条件极难达到: 污染导致感染加重
- ☠️ 耐药性: 滥用导致细菌耐药
- ☠️ 二重感染: 杀灭正常菌群,致病菌趁虚而入

**死亡风险: 真实且极高!**

**使用前提:**
- ✓ 细菌感染确诊(不是病毒!)
- ✓ 生命垂危,别无选择
- ✓ 具备医学知识
- ✓ 已测试无过敏
- ✓ 接受所有风险

**⚠️ 在有任何其他选择的情况下,绝不使用自制抗生素!**

## 一、青霉素粗提取

### 1.1 青霉素简史

**发现:** 1928年,弗莱明发现青霉菌能抑制细菌
**意义:** 拯救了数亿生命
**问题:** 工业生产需要专业设备

**末日现实:**
- 无法制备纯品青霉素
- 只能粗提取青霉素粗品
- 效果不稳定,风险极高

### 1.2 青霉菌培养

#### 菌种获取

**来源:**
```
1. 发霉的柑橘类水果(橙子、柠檬)
   - 青绿色霉菌
   - 可能是青霉菌

2. 发霉的面包
   - 蓝绿色霉菌

3. 腐败食物上的霉菌
```

**识别特征:**
- 颜色: 蓝绿色、青绿色
- 质地: 绒毛状
- 气味: 霉味

**⚠️ 警告: 无法确定菌种纯度!可能混有有毒霉菌!**

#### 培养基配制

**简易培养基:**
```
配方:
- 面粉: 20克
- 糖: 10克
- 盐: 0.5克
- 水: 200ml

或:
- 玉米浆: 50ml
- 乳糖: 30克
- 硫酸铵: 5克
- 磷酸二氢钾: 1克
- 水: 1000ml

制法:
1. 混合所有成分
2. 煮沸溶解
3. 倒入浅盘(玻璃或陶瓷)
4. 冷却至室温
5. 高压灭菌(压力锅120°C,30分钟)
   或煮沸灭菌(不彻底但可用)
```

#### 接种与培养

```
1. 接种:
   - 无菌操作!
   - 取青霉菌少许
   - 涂抹在培养基表面

2. 培养:
   - 温度: 25-28°C
   - 通气: 盖纱布,不完全密封
   - 时间: 7-14天
   - 现象: 表面长满蓝绿色菌丝

3. 判断成功:
   - 大量青绿色菌落
   - 培养液变黄(产生青霉素)
```

### 1.3 青霉素粗提取

**⚠️ 以下方法得到的是极粗糙的青霉素,纯度<10%!**

#### 方法A: 简易水提法

```
1. 过滤:
   - 培养液过滤去菌丝
   - 得到黄色清液

2. 酸化:
   - 加入醋酸或醋
   - 调pH至2-3(用pH试纸)
   - 青霉素析出

3. 萃取:
   - 加入乙醚或乙酸乙酯(易燃!)
   - 振荡萃取
   - 有机层含青霉素

4. 反萃取:
   - 有机层加缓冲液(pH 6-7)
   - 青霉素转入水层

5. 冷冻干燥或真空蒸发:
   - 得到青霉素粗品(黄色粉末)
```

#### 方法B: 直接使用培养液(最简单但最危险)

```
1. 培养液过滤
2. 灭菌(煮沸5分钟会破坏部分青霉素,但必须!)
3. 直接使用

剂量: 完全无法确定!
风险: 极高!
```

### 1.4 青霉素使用

**⚠️ 极度危险操作!**

#### 过敏测试(必须!)

```
皮试:
1. 稀释青霉素100倍
2. 在前臂内侧皮内注射0.1ml
3. 20分钟后观察:
   - 红肿直径>1cm → 阳性,禁用!
   - 出现胸闷、呼吸困难 → 立即抢救!
   - 无反应 → 阴性,可用

⚠️ 即使皮试阴性,仍可能过敏!
⚠️ 准备好肾上腺素救命!(见下文)
```

#### 给药途径

**口服(相对安全):**
- 胃酸会破坏大部分青霉素
- 效果差
- 剂量: 无法确定

**注射(危险!):**
- ✗ 绝对不推荦!
- 过敏风险极高
- 无菌无法保证

**外用:**
- 创面感染
- 涂抹或浸泡
- 相对安全

#### 过敏抢救

**症状:**
- 皮疹、瘙痒
- 胸闷、气短
- 血压下降
- 意识丧失

**抢救(争分夺秒!):**
```
1. 立即停药!

2. 肾上腺素:
   - 1:1000肾上腺素0.5-1ml
   - 肌肉注射(大腿外侧)
   - 可重复,间隔5-15分钟

3. 体位:
   - 平卧
   - 抬高双腿

4. 保持气道通畅

5. 如呼吸心跳停止 → CPR

⚠️ 无肾上腺素 = 几乎必死!
   必须提前准备!
```

## 二、磺胺类药物

### 2.1 简介

**磺胺药:**
- 最早的人工合成抗菌药
- 二战期间广泛使用
- 现代已较少用(副作用大,有更好的药)

**优点:**
- 化学合成(不依赖发酵)
- 广谱抗菌

**缺点:**
- 副作用大(肾损害、血液系统)
- 容易耐药

### 2.2 合成(几乎不可能)

**⚠️ 需要专业化学实验室!**

**原料:**
- 苯胺
- 氯磺酸
- 氨水

**步骤(仅作了解):**
```
苯胺 + 氯磺酸 → 对氨基苯磺酰氯
        ↓ + 氨
      磺胺
```

**现实:**
- 原料难获得
- 反应危险(强酸、有毒气体)
- 纯化困难
- **末日情况下几乎不可能制备!**

### 2.3 替代: 回收药品

**现实方案:**
- 末日前储备成品磺胺药
- 搜索废弃药房、医院
- 检查有效期
- 妥善保存

## 三、天然抗菌物质

### 3.1 大蒜素

**优点:**
- 天然、安全
- 广谱抗菌、抗病毒
- 容易制备

**制备:**

```
1. 新鲜大蒜: 500克
2. 去皮、洗净
3. 捣烂成泥
4. 加水500ml
5. 浸泡2小时
6. 过滤取汁

或:
1. 大蒜切片晒干
2. 研粉
3. 75%酒精浸泡7天
4. 过滤得大蒜酊
```

**使用:**
- 外用: 稀释2-5倍,清洗伤口
- 内服: 蒜汁10-20ml,一日3次
- 生吃: 3-5瓣/天

**适应症:**
- 轻度感染
- 肠道感染
- 呼吸道感染

**效果:** 弱于抗生素,但安全!

### 3.2 蜂胶

**成分:** 多种抗菌物质

**制备:**
```
1. 蜂胶: 100克(从蜂箱刮取)
2. 冷冻后粉碎
3. 加95%酒精500ml
4. 密封浸泡7天,每天摇晃
5. 过滤得蜂胶酊
```

**使用:**
- 外用: 涂抹感染创面
- 内服: 10-20滴,一日3次(稀释后)

### 3.3 金银花、连翘等(详见中药制剂章节)

**配方: 五味消毒饮**
- 金银花、野菊花、蒲公英、紫花地丁、紫背天葵
- 煎汤服用
- 清热解毒,治疗感染

**效果:**
- 对早期感染有效
- 安全性高
- **首选!**

### 3.4 茶树油(如有)

**来源:** 茶树叶蒸馏

**用途:**
- 外用抗菌
- 治疗皮肤感染、真菌感染

## 四、抗生素使用原则

### 4.1 何时使用

**明确细菌感染:**
- 伤口化脓
- 高热不退(>3天)
- 白细胞升高(如能检测)
- 局部红肿热痛

**不要用于:**
- ✗ 病毒感染(感冒、流感)
- ✗ 预防性使用(浪费且致耐药)
- ✗ 发热原因不明

### 4.2 用药原则

**1. 能口服不注射**
- 注射风险大

**2. 能窄谱不广谱**
- 减少耐药
- 保护正常菌群

**3. 足量足疗程**
- 不要一有好转就停药
- 至少用药5-7天
- 症状消失后再用2-3天

**4. 联合用药(重症)**
- 两种抗生素联用
- 增强效果
- 延缓耐药

### 4.3 副作用监测

**常见副作用:**

**消化道:**
- 恶心、呕吐、腹泻
- 停药或减量

**过敏:**
- 皮疹→停药
- 呼吸困难→抢救!

**肝肾损害:**
- 尿少、水肿
- 黄疸
- 立即停药

**二重感染:**
- 用药后出现新感染(如真菌)
- 停用抗生素
- 治疗新感染

## 五、感染的预防

**预防远胜于治疗!**

### 5.1 伤口处理

**正确步骤:**
```
1. 止血
2. 清洗:
   - 大量清水或生理盐水冲洗
   - 去除异物、泥土
   - 冲洗5-10分钟
3. 消毒:
   - 75%酒精或碘酒
   - 从里向外擦拭
4. 包扎:
   - 清洁纱布
   - 不要包太紧
5. 观察:
   - 每日换药
   - 注意红肿、化脓
```

**预防胜于抢救!**

### 5.2 个人卫生

- 勤洗手(最重要!)
- 饮水煮沸
- 食物煮熟
- 环境清洁
- 人群隔离(传染病)

### 5.3 营养与免疫

**增强抵抗力:**
- 充足睡眠
- 营养均衡(蛋白质、维生素)
- 适度运动
- 阳光照射

**免疫力 = 最好的抗生素!**

**优点:**
- 保质期长
- 副作用小

## 八、常见问题

### Q1: 青霉素过期还能用吗?

**A: 非常危险!**
- 青霉素分解产物致敏性更强
- 过敏风险大增
- 绝对不用过期青霉素!

### Q2: 发霉的面包可以治疗感染吗?

**A: 民间传说,极度危险!**
- 面包上的霉菌种类复杂
- 可能是有毒霉菌(如黄曲霉→剧毒!)
- 即使是青霉菌,含量极低
- **绝不可用!**

### Q3: 抗生素是万能药吗?

**A: 不是!**
- 只对细菌感染有效
- 对病毒、真菌、寄生虫无效
- 滥用导致耐药
- 破坏正常菌群

### Q4: 如何判断是细菌还是病毒感染?

**A: 粗略判断**

**细菌:**
- 局部化脓
- 高热持续
- 白细胞↑(如能化验)
- 抗生素有效

**病毒:**
- 全身症状(乏力、肌肉酸痛)
- 发热3天内多自愈
- 白细胞正常或↓
- 抗生素无效

**不确定 → 观察,支持治疗,不滥用抗生素**
