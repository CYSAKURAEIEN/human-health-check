# 制药技术

> **末日情况下的药物制备 - 挽救生命的最后防线**

## ⚠️ 极度重要警告

### 🚨 使用条件限制

**本章节内容仅限于以下极端情况使用:**

- ✓ 文明完全崩溃,医疗体系瓦解
- ✓ 无法获得商业药品
- ✓ 生命垂危,无其他选择
- ✓ 具备基本化学知识

**在有现代医疗可及的情况下:**
- ✗ 绝对不要自制药物!
- ✗ 立即就医,使用正规药品!
- ✗ 自制药物风险极高!

### ☠️ 风险警示

**自制药物的致命风险:**

1. **剂量无法准确控制** → 中毒或无效
2. **纯度无法保证** → 毒副作用
3. **无菌条件困难** → 感染风险
4. **过敏反应难以预测** → 过敏性休克
5. **原料识别错误** → 致命中毒

**死亡风险: 真实存在且很高!**

## 🎯 学习路线

### 阶段一: 基础准备(必修)

1. **学习基础化学**(1-2周)
   - 酸碱盐

   - 剂量计算

2. **掌握消毒剂制备**(2-4周)

   - 植物识别
   - 炮制方法

   - 风险极高
   - 需要严格无菌条件

2. **化学合成药物**(3-6个月)
   - 需要化学专业知识
   - 设备要求高
   - 危险性极大

## 💡 新手建议

### ✅ 推荐起步

1. **从消毒剂开始**
   - 酒精蒸馏最简单
   - 立即有用
   - 风险最低

2. **学习中药**
   - 大量免费原料(野生)
   - 相对安全
   - 效果经过千年验证

3. **建立药材库**
   - 采集、晾干常用药材
   - 制作简单中成药
   - 储备应急

### ❌ 新手禁忌

1. ✗ **不要一开始就尝试抗生素**
   - 极度危险
   - 成功率极低
   - 容易致命中毒

2. ✗ **不要无知识就动手**
   - 先学习基础化学
   - 理解原理
   - 了解风险

3. ✗ **不要使用未确认的植物**
   - 很多有毒植物和药用植物相似
   - 误食致命!
   - 100%确认才可使用

4. ✗ **不要忽视剂量**
   - "是药三分毒"
   - 剂量过大=毒药
   - 宁少勿多

## ⚠️ 关键安全提示

### 毒性物质警示

**剧毒,严禁使用/接触:**
- ☠️ 砒霜(三氧化二砷)
- ☠️ 氰化物
- ☠️ 汞及其化合物
- ☠️ 铅化合物
- ☠️ 某些植物碱(马钱子碱、乌头碱等)

**即使"古方"中有,也绝对不用!**

### 过敏反应处理

**常见药物过敏症状:**
- 皮疹、红肿
- 呼吸困难
- 恶心、呕吐
- 晕厥

**急救:**
1. 立即停药!
2. 大量饮水稀释
3. 如有抗过敏药(扑尔敏),立即服用
4. 严重者:CPR,人工呼吸

**预防:**
- 首次使用新药,小剂量测试
- 观察30分钟
- 无反应再正常使用

### 感染控制

**所有制药操作必须:**
- 手部清洗消毒
- 器具煮沸消毒(15分钟以上)
- 工作区域清洁
- 尽可能无菌操作

**特别是注射用药:**
- 必须严格无菌!
- 否则宁可不用!

## 📖 术语表

- **剂量**: 单次用药量(mg、g、ml)
- **疗程**: 持续用药时间
- **半致死量(LD50)**: 导致50%实验动物死亡的剂量
- **治疗窗**: 有效剂量与中毒剂量之间的范围
- **配伍禁忌**: 不能同时使用的药物组合
- **炮制**: 中药的加工处理
- **提取**: 从原料中分离有效成分
- **纯化**: 去除杂质
- **结晶**: 固体从溶液中析出

## 🌍 替代方案

**在完全无法制药的情况下:**

### 预防为主
- 清洁饮水(煮沸)
- 个人卫生
- 营养充足
- 伤口及时消毒

### 物理疗法
- 发烧: 温水擦浴、冷敷
- 疼痛: 冰敷、热敷、按摩
- 炎症: 休息、抬高患处

### 免疫增强
- 充足睡眠
- 营养均衡
- 适度锻炼
- 阳光照射(维生素D)

### 天然食疗
- 大蒜、洋葱(抗菌)
- 蜂蜜(抗菌、止咳)
- 生姜(暖胃、发汗)
- 柠檬(维生素C)

**预防>>>治疗!**
