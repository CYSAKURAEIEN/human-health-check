# 基础制药知识

> **制药入门 - 安全第一,生命至上**

## ⚠️ 再次警告

**制药不是游戏!**

- 一个错误 = 中毒或死亡
- 无法保证纯度 = 严重副作用
- 剂量计算错误 = 无效或致死

**仅在文明崩溃、医疗体系瓦解、生死攸关时使用!**

## 一、基础化学概念

### 1.1 物质分类

**元素:**
- 氢(H)、氧(O)、碳(C)、氮(N)等
- 不能再分解的基本物质

**化合物:**
- 水(H₂O)、乙醇(C₂H₅OH)、食盐(NaCl)
- 由元素组成

**混合物:**
- 空气、溶液、药液
- 多种物质混合

### 1.2 酸碱盐

**酸(pH<7):**
- 例: 盐酸(HCl)、醋酸(CH₃COOH)
- 特征: 酸味,腐蚀性
- 用途: 溶解、中和

**碱(pH>7):**
- 例: 烧碱(NaOH)、石灰水(Ca(OH)₂)
- 特征: 苦涩,滑腻感,腐蚀性
- 用途: 中和、皂化

**盐:**
- 例: 食盐(NaCl)、苏打(Na₂CO₃)
- 酸+碱 → 盐+水

### 1.3 浓度计算

**质量百分比浓度:**
```
浓度(%) = (溶质质量/溶液总质量) × 100%

例: 75%酒精
  = 75克乙醇 + 25克水
  = 100克溶液
```

**体积百分比浓度:**
```
浓度(%) = (溶质体积/溶液总体积) × 100%

例: 2%碘酒
  = 2ml碘 + 98ml酒精
  = 100ml溶液
```

**稀释计算(重要!):**
```
公式: C₁V₁ = C₂V₂
C₁ = 原浓度
V₁ = 原体积
C₂ = 目标浓度
V₂ = 目标体积

例: 95%酒精稀释成75%
已知: C₁=95%, C₂=75%, V₂=100ml
求V₁: V₁ = C₂V₂/C₁ = 75×100/95 ≈ 79ml
即: 79ml 95%酒精 + 21ml水 = 100ml 75%酒精
```

## 二、剂量与用量

### 2.1 剂量单位

| 单位 | 符号 | 换算 |
|------|------|------|
| 克 | g | 1g |
| 毫克 | mg | 1g = 1000mg |
| 微克 | μg | 1mg = 1000μg |
| 毫升 | ml | 液体体积 |

**常用估算:**
- 1克 ≈ 一个回形针重量
- 1毫升 ≈ 20滴液体
- 1茶匙 ≈ 5ml
- 1汤匙 ≈ 15ml

### 2.2 治疗窗

```
[剂量-效果图]

致死量 --------☠️ (危险!)
         ↑
中毒量 --------⚠️
         ↑
有效量上限 ----✓ ← 治疗窗(安全范围)
         ↑
有效量下限 ----✓
         ↑
无效量 --------✗
```

**关键原则:**
- 剂量过小: 无效
- 剂量适中: 治疗
- 剂量过大: 中毒/死亡

**安全用药:**
- 从最小有效量开始
- 逐步增加
- 密切观察反应

### 2.3 成人vs儿童剂量

**儿童剂量计算(粗略):**
```
方法1: 按体重
  儿童剂量 = 成人剂量 × (儿童体重kg/70)

方法2: 按年龄
  儿童剂量 = 成人剂量 × (年龄/(年龄+12))

例: 成人剂量100mg,5岁儿童
  儿童剂量 = 100 × (5/(5+12)) ≈ 29mg
```

**特殊人群:**
- 老人: 剂量减少1/4-1/3
- 孕妇: 多数药物禁用!
- 肾/肝功能不全: 减量

## 三、提取与纯化技术

### 3.1 水提取法(煎煮法)

**原理:** 水溶性成分溶入水中

**步骤:**
```
1. 药材切碎
2. 加水(没过药材3-5cm)
3. 浸泡30分钟
4. 煮沸后小火煎煮30-60分钟
5. 过滤取液
6. 药渣可再煎一次

用途: 中药煎剂
```

### 3.2 醇提取法

**原理:** 醇(乙醇)可溶解更多有机成分

**步骤:**
```
1. 药材粉碎
2. 浸入50-70%酒精
3. 密封
4. 室温浸泡7-14天,每天摇晃
5. 过滤取液

用途: 药酒、酊剂
```

### 3.3 蒸馏法

**原理:** 沸点不同,分离纯化

**简易蒸馏装置:**
```
[示意图]

加热 ↓
  ╔═══════╗
  ║ 蒸馏瓶 ║ ← 原料
  ╚═╦═══╦═╝
    ║ 蒸汽
    ║   ╔═════════╗
    ╚═══╣ 冷凝管  ║ ← 冷水流过
        ╚═════╦═══╝
              ↓液滴
        ╔═══════╗
        ║接收瓶 ║
        ╚═══════╝
```

**用途:**
- 提纯酒精
- 提取精油
- 制备蒸馏水

### 3.4 过滤与沉淀

**过滤:**
```
粗过滤: 纱布、棉布
中过滤: 滤纸、细布
精过滤: 多层滤纸
```

**沉淀法:**
```
1. 加入沉淀剂
2. 静置
3. 杂质沉底
4. 上清液倒出
```

### 3.5 结晶法

**原理:** 降温或蒸发,溶质析出晶体

**步骤:**
```
1. 热溶液饱和
2. 缓慢冷却
3. 晶体析出
4. 过滤收集
5. 洗涤、干燥

用途: 提纯药物(阿司匹林等)
```

## 四、无菌操作

### 4.1 为什么需要无菌?

**污染后果:**
- 口服药: 腹泻、食物中毒
- 外用药: 伤口感染
- 注射药: 败血症、死亡!

**无菌要求(从低到高):**
1. 口服药: 清洁即可
2. 外用药: 较高清洁度
3. 眼药、创面药: 无菌
4. 注射药: **绝对无菌!**

### 4.2 灭菌方法

**煮沸灭菌(最简单):**
```
步骤:
1. 器具浸入水中
2. 煮沸15-30分钟
3. 保持沸腾状态
4. 取出后不接触污染物

杀灭: 大部分细菌和病毒
不能杀灭: 芽孢(但已足够一般使用)
```

**高压蒸汽灭菌(最可靠):**
```
设备: 压力锅
条件: 121°C,15分钟,1.1个大气压
杀灭: 所有微生物,包括芽孢

用于: 注射用器具、培养基
```

**干热灭菌:**
```
方法: 烤箱/炉子
温度: 160-170°C
时间: 2小时

用于: 玻璃器皿、金属器具
```

**化学灭菌:**
```
75%酒精浸泡: 30分钟
碘伏擦拭: 2次
高锰酸钾溶液: 浸泡

用于: 不耐热物品、皮肤消毒
```

### 4.3 无菌操作要点

**操作前:**
- [ ] 手洗净,消毒
- [ ] 器具灭菌
- [ ] 工作台清洁、消毒
- [ ] 穿清洁衣物

**操作中:**
- [ ] 减少暴露时间
- [ ] 不说话、不咳嗽
- [ ] 器具口部不接触外物
- [ ] 盖子内面朝上放

**操作后:**
- [ ] 及时密封
- [ ] 标记日期、名称
- [ ] 低温避光保存

## 五、药物保存

### 5.1 常见药物储存条件

| 药物类型 | 储存温度 | 其他要求 | 保质期 |
|----------|----------|----------|--------|
| 固体药片 | 阴凉(<25°C) | 干燥、避光 | 1-3年 |
| 液体药剂 | 冷藏(2-8°C) | 避光 | 1-6个月 |
| 酒精/碘酒 | 室温 | 密封 | 2年 |
| 中药饮片 | 阴凉干燥 | 通风、防虫蛀 | 1年 |
| 药膏 | 阴凉 | 密封 | 1-2年 |

### 5.2 变质判断

**外观变化:**
- ✗ 变色(褐色、黑色)
- ✗ 发霉(白毛、绿斑)
- ✗ 结块、潮解
- ✗ 液体浑浊、沉淀
- ✗ 异味

**任何变质,立即丢弃!**

### 5.3 容器选择

**玻璃瓶(最佳):**
- ✓ 不与药物反应
- ✓ 可高温消毒
- ✓ 密封性好
- 深色瓶更佳(避光)

**陶罐:**
- ✓ 存储干燥药材
- ✓ 透气性好

**塑料瓶(慎用):**
- ⚠️ 某些药物会溶解塑料
- ⚠️ 不耐高温
- 短期可用

## 六、安全防护

### 6.1 个人防护装备

**基本防护:**
- [ ] 口罩(防吸入粉尘、蒸汽)
- [ ] 手套(防接触腐蚀物)
- [ ] 护目镜(防飞溅)
- [ ] 围裙/工作服

**高危操作(强酸、强碱):**
- [ ] 防化手套
- [ ] 面罩
- [ ] 长袖长裤

### 6.2 通风要求

**必须通风操作:**
- 蒸馏酒精(易燃蒸汽)
- 使用强酸强碱(腐蚀性蒸汽)
- 处理硫化物(有毒气体)
- 培养微生物(避免污染)

**通风措施:**
- 室外操作
- 开窗对流
- 简易通风罩

### 6.3 急救措施

**酸碱烧伤:**
```
1. 立即大量清水冲洗(15分钟以上)
2. 酸灼伤: 用稀碱(苏打水)中和
3. 碱灼伤: 用稀酸(醋)中和
4. 包扎就医
```

**吸入有毒气体:**
```
1. 立即转移到新鲜空气处
2. 松开衣领
3. 保持呼吸道通畅
4. 如呼吸停止,人工呼吸
5. 就医
```

**误食:**
```
1. 立即大量饮水稀释
2. 催吐(非腐蚀性物质)
3. 喝牛奶或蛋清(结合毒物)
4. 活性炭吸附(如有)
5. 就医
```

## 七、简易实验器材

### 7.1 可自制的器材

**蒸馏装置:**
```
材料:
- 大锅(蒸馏瓶)
- 盖子上开孔,插竹筒/铜管
- 管道通过冷水桶(冷凝)
- 管道另一端滴入瓶(接收)

原理: 蒸汽→冷凝→液滴
用途: 蒸馏酒精、精油
```

**过滤装置:**
```
材料:
- 漏斗(或竹筒、陶罐底部开孔)
- 滤布/纱布/滤纸
- 接收容器

多次过滤可提高清洁度
```

**简易天平:**
```
材料:
- 直杆
- 中心支点(刀刃)
- 两端挂盘/吊钩
- 标准砝码(硬币、石块提前称重)

虽粗糙,但能估算重量
```

**温度计替代:**
```
无温度计情况:
- 水沸腾 = 100°C (海平面)
- 手能接触的热水 = 40-50°C
- 手不能接触的热水 = 60-80°C
- 冒蒸汽 = >80°C
- 木材开始燃烧 = 约300°C
```

## 八、毒性物质识别

### 8.1 剧毒植物(绝对避免)

| 植物 | 毒性成分 | 致死量 | 症状 |
|------|----------|--------|------|
| 断肠草 | 钩吻碱 | 2-3克 | 呼吸麻痹 |
| 曼陀罗 | 莨菪碱 | 15-100mg | 幻觉、昏迷 |
| 乌头 | 乌头碱 | 2-4mg | 心脏骤停 |
| 夹竹桃 | 强心苷 | 少量 | 心律失常 |
| 毒芹 | 毒芹碱 | 少量 | 抽搐、呼吸衰竭 |

**原则: 不认识的植物绝对不用!**

### 8.2 重金属(禁用)

- ☠️ 汞(水银): 神经毒
- ☠️ 铅: 慢性中毒
- ☠️ 砷(砒霜): 剧毒
- ☠️ 镉: 肾损伤

**古代"仙丹"常含这些,导致很多人中毒!**

### 8.3 安全性测试(谨慎)

**如果必须测试未知植物:**
```
第1步: 皮肤测试
  - 挤汁液涂抹在手臂内侧
  - 等待4小时
  - 如红肿、灼痛 → 放弃

第2步: 唇部测试
  - 少量触碰嘴唇
  - 等待15分钟
  - 如麻木、灼痛 → 放弃

第3步: 舌尖测试
  - 微量放舌尖
  - 立即吐出
  - 等待15分钟
  - 如不适 → 放弃

第4步: 小量吞服
  - 极少量(米粒大小)
  - 等待8小时
  - 如无不良反应,逐渐增加

整个过程需2-3天!
```

**⚠️ 风险极高!除非绝境,否则不要尝试!**

## 九、常见问题

### Q1: 没有精确称量工具怎么办?

**A: 使用容积估算**
```
1茶匙(约5ml) ≈ 5g(粉末)
1汤匙(约15ml) ≈ 15g
1杯(约250ml) ≈ 250g

液体: 可用容量估算
固体: 用水测体积后估算重量
```

**原则: 宁少勿多!**

### Q2: 如何判断提取是否完全?

**A: 多次提取对比**
- 第一次提取: 颜色深
- 第二次提取: 颜色浅
- 第三次提取: 几乎无色 → 完全

### Q3: 自制药物能保存多久?

**A: 保守估计**
- 液体药剂: 1-4周(冷藏)
- 干燥药粉: 3-6个月
- 药酒: 6-12个月
- 膏药: 1-3个月

**有任何可疑变质,立即丢弃!**

### Q4: 一定要无菌吗?

**A: 视用途而定**
- 口服: 清洁即可(非无菌)
- 外用创面: 尽可能无菌
- 注射: **绝对无菌!**(无条件不做!)
