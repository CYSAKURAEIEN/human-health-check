# 中药制剂

> **传统智慧 - 最安全可行的末日制药方向**

## ✅ 为什么推荐中药?

**相比化学制药的巨大优势:**

| 对比项 | 中药 | 化学药 |
|--------|------|--------|
| 原料获取 | ⭐⭐⭐⭐⭐野生采集 | ⭐化工原料难得 |
| 制备难度 | ⭐⭐简单 | ⭐⭐⭐⭐⭐极难 |
| 安全性 | ⭐⭐⭐⭐较高 | ⭐⭐风险高 |
| 经验积累 | ⭐⭐⭐⭐⭐数千年 | ⭐⭐现代技术 |
| 副作用 | ⭐⭐⭐⭐较小 | ⭐⭐较大 |
| 成功率 | ⭐⭐⭐⭐60-80% | ⭐⭐<20% |

**推荐作为末日制药的主要方向!**

## 一、常见症状对应中药

### 1.1 感冒发烧

#### 感冒初期(风寒感冒)

**症状:** 怕冷、流清涕、打喷嚏、无汗

**方剂:** 葱姜红糖汤
```
材料:
  - 葱白: 3-5根(带须)
  - 生姜: 5-8片
  - 红糖: 30克
  - 水: 500ml

制法:
  1. 水煮沸
  2. 加入葱姜
  3. 小火煎10分钟
  4. 加红糖溶化
  5. 趁热喝下
  6. 盖被发汗

用法: 每日2-3次
```

**方剂:** 麻黄汤(经典)
```
材料:
  - 麻黄: 9克
  - 桂枝: 9克
  - 杏仁: 9克
  - 甘草: 3克

功效: 发汗解表
```

#### 感冒中期(风热感冒)

**症状:** 发热、口渴、咽痛、流黄涕

**方剂:** 银翘散
```
材料:
  - 金银花: 15克
  - 连翘: 15克
  - 薄荷: 6克(后下)
  - 牛蒡子: 9克
  - 桔梗: 6克
  - 甘草: 3克

制法:
  1. 除薄荷外,其余药加水煎煮20分钟
  2. 最后5分钟加入薄荷
  3. 过滤取汁

用法: 每日2次,温服
```

**简易版:**
```
金银花30克 + 菊花15克 + 薄荷6克
泡茶频饮
```

### 1.2 消化问题

#### 食积腹胀

**方剂:** 保和丸
```
材料:
  - 山楂: 15克(消食)
  - 神曲: 15克(助消化)
  - 陈皮: 9克(理气)
  - 茯苓: 12克
  - 莱菔子: 9克

制法: 煎汤服用
功效: 消食化积
```

**简易版:**
- 山楂片30克泡水喝
- 生萝卜榨汁饮用

#### 腹泻

**方剂:** 藿香正气散
```
材料:
  - 藿香: 12克
  - 紫苏叶: 9克
  - 白芷: 9克
  - 陈皮: 9克
  - 厚朴: 9克
  - 茯苓: 12克
  - 甘草: 3克

功效: 解表化湿,理气和中
适用: 胃肠型感冒、水土不服
```

**简易方:**
```
大蒜: 3-5瓣,捣烂
加红糖水送服
杀菌止泻
```

### 1.3 外伤

#### 止血

**方剂:** 云南白药替代方
```
材料:
  - 三七: 30克(研粉)
  - 白及: 20克(研粉)
  - 仙鹤草: 15克(研粉)
  - 龙骨: 15克(煅,研粉)

制法:
  1. 各药分别炮制
  2. 研成极细粉
  3. 混匀
  4. 密封保存

用法:
  - 外用: 直接撒敷伤口
  - 内服: 每次3克,一日2次(内伤出血)
```

**野外应急:**
- 艾叶: 揉碎敷伤口止血
- 蒲公英: 捣烂敷伤口消炎
- 侧柏叶: 止血(鼻出血塞鼻)

#### 消炎

**方剂:** 五味消毒饮
```
材料:
  - 金银花: 15克
  - 野菊花: 15克
  - 蒲公英: 15克
  - 紫花地丁: 15克
  - 紫背天葵: 15克

制法: 煎汤服用
功效: 清热解毒,消痈散结
适用: 疮疖、皮肤感染
```

**外用:**
- 煎汤洗患处
- 或捣烂敷患处

## 二、中药炮制方法

### 2.1 净制

**目的:** 去除杂质、非药用部分

**方法:**

**拣选:**
- 挑出杂质、霉变、虫蛀部分

**筛选:**
- 用筛子去沙土、碎屑

**风选:**
- 扬去轻质杂物(毛、叶)

**水洗:**
- 快速漂洗,勿浸泡(损失成分)

### 2.2 切制

**目的:** 便于煎煮、服用、调剂

**方法:**

**切片:**
- 厚度: 2-3mm
- 药材: 生姜、白术、当归等

**切段:**
- 长度: 2-3cm
- 药材: 麻黄、桂枝等茎枝类

**切丝:**
- 药材: 陈皮等

**捣碎:**
- 药材: 种子类(莱菔子、杏仁等)

### 2.3 炮制(改变药性)

#### 炒制

**清炒(干炒):**
```
方法:
  1. 热锅
  2. 药材倒入
  3. 不断翻动
  4. 炒至微黄或焦黄
  5. 取出摊凉

目的:
  - 降低寒性(生地→熟地)
  - 增强补益(黄芪炒后补气更强)
  - 便于粉碎(炒后酥脆)

药材: 白术、黄芪、山药等
```

**麸炒:**
```
方法:
  1. 热锅,撒麦麸
  2. 冒烟时加药材
  3. 翻炒至黄色
  4. 筛去麦麸

目的: 缓和药性,健脾
药材: 苍术、枳壳等
```

**土炒:**
```
方法:
  1. 灶心土(或黄土)研细
  2. 热锅中炒热
  3. 加药材拌炒
  4. 筛去土

目的: 止泻,止血
药材: 白术等
```

#### 炙制

**蜜炙:**
```
方法:
  1. 蜂蜜加水稀释(蜜:水=1:1)
  2. 药材拌匀
  3. 闷润吸尽
  4. 文火炒至不粘手
  5. 取出摊凉

目的: 增强补益,润肺止咳
药材: 甘草、麻黄、百部等
```

**姜炙:**
```
方法:
  1. 生姜榨汁
  2. 药材拌姜汁
  3. 闷润
  4. 炒干

目的: 增强止呕,温中
药材: 半夏、厚朴等
```

#### 蒸煮

**蒸:**
```
方法:
  1. 药材放蒸笼
  2. 大火蒸透(筷子能扎透)
  3. 取出切片晒干

目的: 改变药性,增强补益
药材:
  - 何首乌: 生用通便,蒸熟补肝肾
  - 地黄: 蒸熟成"熟地"补血
```

**煮:**
```
目的: 降低毒性
药材: 附子、川乌等(剧毒,需久煮)
```

#### 煅制

**明煅:**
```
方法:
  1. 药材直接放火上
  2. 烧至红透
  3. 取出放凉

目的: 增强收敛、止血
药材: 龙骨、牡蛎、石膏等
```

**煅淬:**
```
方法:
  1. 药材烧红
  2. 立即投入液体(醋/酒/水)
  3. 取出干燥
  4. 重复7次(七煅七淬)

目的: 质地酥脆,易粉碎
药材: 自然铜等
```

## 三、中成药的简易制作

### 3.1 丸剂

**水丸:**
```
方法:
  1. 药材研成细粉
  2. 加水或药汁做黏合剂
  3. 制成小丸(绿豆大小)
  4. 阴干

优点: 简单
缺点: 易霉变,不易保存
```

**蜜丸:**
```
方法:
  1. 药粉过筛(极细)
  2. 蜂蜜加热炼制(去泡沫)
  3. 药粉:蜜 = 1:1-1.5(重量)
  4. 混合均匀
  5. 搓成条,切段
  6. 搓圆成丸

大小:
  - 大蜜丸: 9-15克/丸
  - 小蜜丸: 绿豆大小

保存: 蜡纸包裹,防潮
保质期: 1-2年

优点: 保存久,补益药多用
```

### 3.2 散剂

```
方法:
  1. 药材分别炮制
  2. 干燥
  3. 分别研粉
  4. 过筛(100目以上)
  5. 按配方混合均匀

用法:
  - 内服: 温水送服,3-6克/次
  - 外用: 撒敷患处

保存: 密封,防潮
保质期: 6个月-1年

优点: 简单,快速,吸收快
缺点: 不易保存
```

### 3.3 膏药(外用)

**黑膏药(传统):**
```
材料:
  - 药材: 500克
  - 麻油/植物油: 1000ml
  - 黄丹(四氧化三铅): 500克
  - 松香: 100克

方法:
  1. 药材浸油3-7天
  2. 熬煮至药枯(焦黑)
  3. 去渣,油温降至120°C
  4. 缓慢加入黄丹,不断搅拌
  5. 煎熬至"滴水成珠"(测试: 油滴入水中成珠,不散)
  6. 离火,加松香
  7. 倒入水中,反复揉搓去火毒
  8. 搓成条,备用

使用:
  1. 取一段加热软化
  2. 摊在布/纸上
  3. 贴患处

⚠️警告: 黄丹含铅,有毒!孕妇禁用!
```

**简易膏药:**
```
材料:
  - 药材粉: 100克
  - 蜂蜜: 200克
  - 凡士林/猪油: 100克

方法:
  1. 蜂蜜加热
  2. 加入药粉搅匀
  3. 加入油脂
  4. 小火熬成膏状
  5. 冷却装瓶

用法: 涂抹患处
保存: 冷藏,3-6个月
```

### 3.4 酊剂(药酒)

```
方法:
  1. 药材粗碎
  2. 装入容器(70%满)
  3. 加入50-70度白酒/酒精
  4. 密封
  5. 室温浸泡14-30天,每日摇晃
  6. 过滤装瓶

用法:
  - 内服: 10-30ml/次,一日2-3次
  - 外用: 擦涂患处

常用配方:
  - 舒筋活血酒: 当归、川芎、红花、乳香等
  - 补肾酒: 枸杞、淫羊藿、肉苁蓉等
  - 跌打酒: 三七、骨碎补、土鳖虫等

保存: 密封,阴凉处,1-2年
```

## 四、常用中药材识别与采集

### 4.1 全年可采

#### 蒲公英

**识别:**
- 叶: 羽状深裂,锯齿缘
- 花: 黄色,头状花序
- 果: 白色绒球(蒲公英降落伞)
- 根: 圆锥形,黄褐色

**采集:** 全草,春秋采挖

**功效:** 清热解毒,消痈散结
- 乳腺炎、扁桃体炎
- 尿路感染
- 外用治疮疖

**用法:**
- 鲜品: 捣烂敷患处
- 干品: 煎汤,15-30克

#### 艾叶

**识别:**
- 叶: 羽状深裂,背面灰白色绒毛
- 气味: 特异香气
- 高度: 50-120cm

**采集:** 端午前后最佳

**功效:**
- 温经止血
- 散寒止痛
- 外用止血

**用法:**
- 煎汤: 9-15克
- 外用: 捣烂敷,或烟熏
- 艾灸: 制成艾条

#### 车前草

**识别:**
- 叶: 基生,卵形,有平行脉
- 花: 穗状花序
- 生境: 路边、田埂

**采集:** 全草

**功效:** 利尿通淋,清热解毒

**用法:** 煎汤,15-30克

### 4.2 春季采集

#### 金银花

**识别:**
- 藤本植物
- 花: 初开白色,后变黄(金银相映)
- 气味: 清香

**采集:** 花蕾,5-7月

**功效:** 清热解毒,疏散风热
- 感冒发热
- 咽喉肿痛
- 疮疖

**用法:** 煎汤,10-15克,或泡茶

#### 桑叶

**识别:**
- 叶: 卵形,边缘锯齿
- 来源: 桑树

**采集:** 初霜后(霜桑叶更佳)

**功效:** 疏散风热,清肝明目

**用法:** 煎汤,5-10克

### 4.3 秋季采集

#### 菊花

**识别:**
- 花: 头状花序,黄色或白色
- 气味: 清香

**采集:** 花盛开时

**功效:** 疏散风热,清肝明目

**用法:** 泡茶,3-10克

#### 山楂

**识别:**
- 果实: 红色,球形
- 树: 落叶乔木,有刺

**采集:** 秋季果熟

**功效:** 消食化积,活血化瘀

**用法:** 煎汤,10-15克,或生食

### 4.4 根茎类(秋冬采)

#### 生姜

**识别:** 根茎,黄褐色,断面淡黄

**功效:** 解表散寒,温中止呕

**用法:**
- 感冒: 5-10片煎汤
- 止呕: 3-5片

#### 当归

**识别:**
- 根: 圆柱形,有浓郁香气
- 生境: 高山地区

**功效:** 补血活血,调经止痛

**用法:** 煎汤,5-15克

## 五、中药煎煮方法

### 5.1 器具选择

**最佳:**
- 砂锅、陶罐(不与药物反应)

**可用:**
- 不锈钢锅
- 搪瓷锅

**禁用:**
- 铁锅、铝锅(与药物反应)
- 铜锅

### 5.2 煎煮步骤

**标准煎法:**

```
1. 浸泡:
   - 药材放锅中
   - 加冷水没过药材3-5cm
   - 浸泡30分钟(种子类1-2小时)

2. 煮沸:
   - 大火煮沸

3. 文火慢煎:
   - 沸后转小火
   - 解表药: 15-20分钟
   - 一般药: 25-30分钟
   - 滋补药: 40-60分钟

4. 二煎:
   - 药渣加水再煎20-30分钟

5. 合并药液:
   - 两次药液混合
   - 总量约400-600ml
   - 分2-3次服用
```

**特殊煎法:**

**先煎:**
- 矿物、贝壳类(石膏、龙骨等)
- 提前30分钟下锅

**后下:**
- 芳香药(薄荷、藿香等)
- 最后5-10分钟下锅

**包煎:**
- 细小种子、花粉类
- 用纱布包裹煎煮

**另煎:**
- 贵重药(人参等)
- 单独煎2-3小时,取汁兑入

### 5.3 服用方法

**温服:**
- 一般药,温热服用

**热服:**
- 风寒感冒药
- 发汗解表

**冷服:**
- 清热药(夏季)
- 止吐药

**时间:**
- 饭前: 补益药
- 饭后: 消食药、刺激性药
- 睡前: 安神药

## 六、安全注意事项

### 6.1 十八反十九畏(配伍禁忌)

**十八反(不能同用,简化版):**
- 甘草 反 甘遂、大戟、芫花、海藻
- 乌头 反 半夏、瓜蒌、贝母、白蔹、白及
- 藜芦 反 人参、沙参、丹参、玄参、细辛、芍药

**十九畏(避免同用):**
- 硫磺 畏 朴硝
- 水银 畏 砒霜
- 狼毒 畏 密陀僧
- 巴豆 畏 牵牛子
- 丁香 畏 郁金
- 川乌、草乌 畏 犀角
- 牙硝 畏 三棱
- 官桂 畏 石脂
- 人参 畏 五灵脂

**原则: 不清楚不要混用!**

### 6.2 孕妇禁忌

**绝对禁用(剧毒或堕胎):**
- 巴豆、牵牛子、大戟
- 斑蝥、商陆、麝香
- 水蛭、虻虫、三棱、莪术
- 乌头、附子

**慎用:**
- 活血类: 红花、桃仁、牛膝
- 通便类: 大黄、芒硝、番泻叶
- 理气类: 枳实、枳壳

**安全药:**
- 党参、黄芪、白术
- 当归、熟地(小量)
- 陈皮、砂仁

### 6.3 儿童用药

**剂量:** 见基础制药知识篇

**禁用:**
- 含重金属药(朱砂、雄黄)
- 有毒药(附子、半夏等)

**首选:**
- 温和药物
- 食疗为主

## 七、常见问题

### Q1: 中药一定要按方抓药吗?

**A: 不一定**
- 常见病: 单味药也很有效
  - 感冒: 生姜、葱白即可
  - 消化不良: 山楂、陈皮
  - 外伤: 三七、白及
- 复杂病: 需配伍增效

### Q2: 中药有副作用吗?

**A: 有,但相对较小**
- 毒性药: 严格控制剂量
- 一般药: 按常规剂量安全
- 药食同源: 几乎无副作用

**原则: 是药三分毒!**

### Q3: 煎煮时间越长越好?

**A: 不是**
- 解表药: 久煎损失挥发油
- 滋补药: 需久煎充分提取
- 一般: 25-30分钟

### Q4: 可以用电热杯煎药吗?

**A: 不理想,但应急可用**
- 最好用砂锅文火
- 电热杯温度难控制
- 容易煎焦

### Q5: 中药可以替代西药吗?

**A: 视情况而定**
- 常见病: 多数可以
- 急危重症: 西药更快更强
- 慢性病: 中药调理更好
- 感染: 抗生素更可靠,中药辅助

**末日建议: 中西结合!**
