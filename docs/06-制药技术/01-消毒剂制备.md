# 消毒剂制备

> **防感染第一防线 - 最实用最重要的末日技能**

## ✅ 为什么消毒剂最重要?

**末日情况下,第一死因不是饥饿,而是感染!**

- 小伤口感染 → 败血症 → 死亡
- 饮水不洁 → 霍乱、伤寒 → 死亡
- 手术无消毒 → 感染 → 死亡

**有效消毒可以:**
- 阻断90%的感染途径
- 提高伤口愈合率
- 延长食物保存时间
- 净化饮用水

**成功率高、风险低、立即有用!优先掌握!**

## 一、酒精(乙醇)制备

### 1.1 酒精的重要性

**用途:**
- 皮肤消毒(75%最佳)
- 器械消毒
- 伤口消毒(轻度)
- 溶剂(制药)
- 燃料(酒精灯)

**有效浓度:**
- 75%酒精:最佳杀菌浓度
- 95%酒精:太高,反而效果差(蛋白质凝固阻碍渗透)
- <60%酒精:效果不足

### 1.2 发酵法制取酒精

#### 原理:
```
糖 + 酵母 → 酒精 + 二氧化碳
C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂↑
```

#### 材料:

**糖源(选一种):**
- 粮食(大米、小麦、玉米): 淀粉→糖→酒精
- 水果(葡萄、苹果): 直接含糖
- 蜂蜜: 高糖
- 甘蔗汁、甜菜: 高糖

**酵母:**
- 酒曲(传统)
- 面包酵母
- 野生酵母(水果皮上,葡萄皮最多)

#### 步骤(以大米为例):

**第1步: 糖化(淀粉→糖)**
```
1. 大米2公斤,洗净
2. 蒸煮至熟烂
3. 摊凉至30°C
4. 拌入酒曲100克(或甜酒曲)
5. 保温发酵24-48小时
6. 出现甜味 → 糖化成功
```

**第2步: 发酵(糖→酒精)**
```
1. 糖化好的饭加水4-6升
2. 混匀,倒入发酵桶
3. 加盖(不密封,留气孔)
4. 温度保持20-30°C
5. 发酵7-14天
   - 初期: 冒泡旺盛
   - 中期: 逐渐平息
   - 末期: 基本不冒泡
6. 有酒香味 → 发酵完成
```

**产物: 米酒(酒精度5-12%)**

#### 步骤(水果酒-更简单):

```
1. 葡萄/苹果10公斤,洗净晾干
2. 压碎(保留皮,皮上有野生酵母)
3. 加糖2公斤(提高酒精度)
4. 装入容器,70%满(留发酵空间)
5. 封口(不完全密封)
6. 室温发酵10-20天
7. 过滤得果酒(酒精度8-15%)
```

### 1.3 蒸馏提纯

**低度酒(5-15%) → 高度酒(75-95%)**

#### 简易蒸馏装置:

```
[示意图]

   木炭火
      ↓
  ╔═══════╗
  ║ 大锅  ║ ← 米酒/果酒
  ║(蒸馏瓶)║
  ╚═══╦═══╝
      ║
    蒸汽↑
      ║  ╱冷水进
  ╔═══╩═══════╗
  ║  冷凝管    ║ ← 铜管/竹筒盘绕在冷水中
  ╚═════╦═════╝
      水出╲
    液滴↓
  ╔═══════╗
  ║接收瓶 ║ ← 高度酒精
  ╚═══════╝
```

#### 操作步骤:

**1. 装填:**
- 大锅装入米酒/果酒(70%满)

**2. 加热:**
- 小火慢慢加热
- 控制温度78-85°C(酒精沸点78°C)

**3. 蒸馏:**
```
温度控制关键:
- 开始: 流出的是挥发性杂质(前段,丢弃50-100ml)
- 中段: 纯酒精(收集!)
- 末段: 水蒸出(>85°C,停止收集)

判断:
  - 滴在手上,很快挥发,凉爽感 → 高度酒✓
  - 不易挥发,黏手 → 低度酒或水✗
```

**4. 测浓度:**
```
方法1: 酒精计(理想)
  - 读刻度

方法2: 燃烧法(粗略)
  - 点燃一小杯酒精
  - 完全烧干无残留 → 约95%以上
  - 有少量残留 → 70-90%
  - 烧一半熄灭 → <60%

方法3: 重量法
  - 纯酒精比重: 0.79g/ml
  - 水比重: 1.0g/ml
  - 称量100ml计算
```

**5. 多次蒸馏:**
- 第一次: 30% → 60%
- 第二次: 60% → 85%
- 第三次: 85% → 95%

**产量:**
- 10升米酒(10%) → 约1升纯酒精(95%)
- 损耗约10-20%

### 1.4 酒精使用

**75%酒精配制:**
```
公式: 95%酒精稀释成75%
取95%酒精: 79ml
加蒸馏水(或凉开水): 21ml
混匀 = 100ml 75%酒精

或简化:
约4份酒精 : 1份水
```

**使用方法:**
- 皮肤消毒: 擦拭2次,间隔1分钟
- 器械消毒: 浸泡30分钟
- 手消毒: 揉搓至干(20秒以上)

**禁忌:**
- ✗ 深度伤口(疼痛、组织损伤)
- ✗ 黏膜(眼睛、口腔内)
- ✗ 大面积烧伤(吸收中毒)

## 二、碘酒制备

### 2.1 碘酒的优势

**vs酒精:**
- ✓ 杀菌谱更广
- ✓ 对细菌、病毒、真菌、芽孢都有效
- ✓ 用量少
- ✗ 有刺激性
- ✗ 染色(棕色)

### 2.2 碘的来源

**方法A: 海藻提碘(沿海地区)**

**材料:**
- 海带、紫菜等海藻: 10公斤
- 草木灰: 5公斤

**步骤:**
```
1. 海藻晒干
2. 燃烧成灰(注意:火焰呈紫色=碘蒸汽!)
3. 灰烬加水煮沸
4. 过滤取清液
5. 蒸发浓缩
6. 加入双氧水(H₂O₂)或漂白粉
   反应: 2I⁻ + H₂O₂ → I₂ + 2OH⁻
7. 加入酒精振荡
8. 碘溶于酒精(呈棕色)
9. 分离、蒸馏得固体碘

产量: 10公斤海带 → 3-5克碘
```

**方法B: 地下卤水(内陆地区)**

- 某些盐湖、盐井含碘
- 提取方法类似海藻法

**方法C: 购买/回收(现实)**

- 药店碘酒
- 碘伏
- 碘盐(含量极低,不实用)

### 2.3 碘酒配制

**标准碘酒(2%碘酒):**

**配方:**
```
碘(I₂): 2克
碘化钾(KI): 1.5克
75%酒精: 加至100ml
```

**步骤:**
```
1. 碘化钾1.5克溶于5ml水
2. 加入碘2克,搅拌溶解
   (碘化钾帮助碘溶解)
3. 加75%酒精至总体积100ml
4. 充分混匀
5. 深色瓶保存(碘见光分解)
```

**简化版(无碘化钾):**
```
碘2克 + 95%酒精100ml
  - 溶解度低,部分不溶
  - 效果略差,但能用
```

### 2.4 碘酒使用

**消毒:**
- 涂抹患处
- 等待2-3分钟
- 用75%酒精脱碘(去色)

**碘过敏测试:**
- 小范围涂抹
- 观察15分钟
- 如红肿、瘙痒 → 过敏,不用

## 三、碘伏(碘胺酮)

### 3.1 碘伏vs碘酒

| 特性 | 碘伏 | 碘酒 |
|------|------|------|
| 刺激性 | 小 | 大 |
| 疼痛感 | 无 | 明显 |
| 染色 | 轻 | 重 |
| 稳定性 | 好 | 较差(见光分解) |
| 制备难度 | 复杂 | 简单 |

### 3.2 简易碘伏制备

**原理:** 碘与表面活性剂络合

**配方:**
```
碘: 1克
表面活性剂(吐温80或洗洁精): 5克
水: 加至100ml
```

**步骤:**
```
1. 碘溶于少量酒精
2. 加入表面活性剂混匀
3. 缓慢加水稀释
4. 搅拌至均匀透明棕色
```

**使用:** 直接涂抹,无需脱碘

## 四、双氧水(过氧化氢H₂O₂)

### 4.1 双氧水的用途

**优点:**
- 杀菌广谱
- 分解成水和氧(无残留毒性)
- 清洁伤口(冒泡带出脏物)

**缺点:**
- 不稳定,易分解
- 对组织有轻微损伤

### 4.2 双氧水制备(难度高)

**⚠️ 警告: 制备危险,仅作了解!**

**方法A: 电解法**
```
电解稀硫酸:
  - 阳极: 生成过硫酸
  - 水解: 得过氧化氢
  - 蒸馏纯化

危险: 需要电力、硫酸,爆炸风险
```

**方法B: 过氧化钡法**
```
BaO₂ + H₂SO₄ → H₂O₂ + BaSO₄↓
  - 过氧化钡加稀硫酸
  - 生成双氧水和硫酸钡沉淀
  - 过滤分离

危险: 钡化合物剧毒!
```

**结论: 末日情况下,双氧水极难制备。优先使用酒精、碘酒!**

### 4.3 双氧水使用

**医用浓度: 3%**

**使用:**
- 清洗污染伤口
- 冒泡带出脏物和坏死组织
- 冲洗后用生理盐水再次冲洗
- 不用于深度伤口(气泡阻碍愈合)

## 五、高锰酸钾溶液

### 5.1 高锰酸钾的用途

**特点:**
- 强氧化剂
- 杀菌、消毒、除臭
- 深紫色,易识别

### 5.2 高锰酸钾来源

**工业级高锰酸钾:**
- 化工商店可能有售
- 深紫色晶体

**末日情况下难以自制,但可大量储备(稳定)**

### 5.3 高锰酸钾溶液配制

**不同用途不同浓度:**

| 用途 | 浓度 | 颜色 | 配法 |
|------|------|------|------|
| 伤口消毒 | 0.1% | 淡紫红 | 1克+1升水 |
| 皮肤消毒 | 0.05% | 很淡紫 | 0.5克+1升水 |
| 水果消毒 | 0.02% | 微紫 | 0.2克+1升水 |
| 饮水消毒 | 0.01% | 几乎无色 | 0.1克+1升水 |

**使用:**
- 浸泡10-15分钟
- 用清水冲洗

**禁忌:**
- ✗ 勿与甘油、酒精等还原性物质混合(爆炸!)
- ✗ 浓度过高伤皮肤

## 六、漂白粉(含氯消毒剂)

### 6.1 漂白粉的用途

**优点:**
- 便宜、高效
- 适合大面积消毒
- 饮水消毒

**缺点:**
- 刺激性强
- 腐蚀性
- 不稳定(受潮失效)

### 6.2 漂白粉使用

**配制消毒液:**
```
漂白粉10克 + 水1升 = 1%消毒液

用途:
  - 地面、墙壁消毒
  - 厕所消毒
  - 疫区消毒
```

**饮水消毒:**
```
1. 每升水加0.3-0.5ml漂白粉1%溶液
2. 搅拌
3. 静置30分钟
4. 有氯味即可饮用(余氯保护)
```

## 七、生理盐水

### 7.1 用途

**不是消毒剂,但极重要:**
- 冲洗伤口(等渗,不损伤组织)
- 眼部冲洗
- 补液(静脉、口服)

### 7.2 配制

**标准生理盐水(0.9%):**
```
食盐: 9克(约2茶匙)
蒸馏水或凉开水: 1000ml
混匀
```

**使用前煮沸消毒!**

## 十、常见问题

### Q1: 白酒可以消毒吗?

**A: 视度数而定**
- 65度以上白酒: 勉强可以应急
- 56度白酒: 效果很差
- <50度: 基本无效

**最好重新蒸馏提纯到75%!**

### Q2: 没有酒曲怎么办?

**A: 野生发酵**
- 葡萄皮表面有天然酵母
- 直接压碎发酵即可
- 或从上一批酒糟中提取酵母

### Q3: 蒸馏装置里的管子用什么材料?

**A: 优先级排序**
1. 铜管(最好,导热好,无毒)
2. 不锈钢管
3. 竹筒(凿通竹节)
4. 陶管
5. ✗ 塑料管(有毒物溶出!)
6. ✗ 铅管(剧毒!)

### Q4: 酒精度数怎么看?

**A: 简易判断**
- 点燃,蓝色火焰持续燃烧 → 高度(>60%)
- 火焰跳跃,很快熄灭 → 低度(<40%)
- 滴在手上,快速挥发凉爽 → 高度
- 黏手 → 低度

## 十一、应急替代方案

**完全无消毒剂情况下:**

### 物理消毒:

**煮沸:**
- 器械: 煮沸15分钟
- 纱布: 煮沸30分钟
- 杀灭大部分病原体

**干烤:**
- 160°C,2小时
- 金属、玻璃器具

**日晒:**
- 紫外线杀菌
- 晒6-8小时
- 效果较弱

### 天然消毒:

**盐水(高浓度):**
- 20%盐水有抑菌作用
- 清洗伤口

**醋:**
- 稀释醋(5%)
- 有一定抑菌作用
- 不能替代正规消毒剂

**茶水:**
- 浓茶水
- 茶多酚有抑菌作用
- 效果很弱

**大蒜汁:**
- 压碎大蒜取汁
- 涂抹伤口
- 有刺激性

**蜂蜜:**
- 涂抹伤口
- 抗菌、促愈合
- 不能用于深度伤口
