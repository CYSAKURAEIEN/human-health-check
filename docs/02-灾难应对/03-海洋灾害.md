## 海啸

### 什么是海啸

海啸是由海底地震、火山喷发、海底滑坡等引起的巨大海浪,具有极强的破坏力。

**特点**:
- 速度快(深海可达700-800公里/小时)
- 波长长(数百公里)
- 近岸突然增高(可达数十米)
- 连续多波(第一波不一定最大)
- 破坏力巨大

### 海啸预警信号

#### 自然预警(极重要!)

```
⚠️ 看到以下任一现象,立即撤离!

1. 强烈地震(沿海地区):
   - 持续超过20秒
   - 剧烈摇晃
   - 难以站立

2. 海水异常:
   - 突然大幅退潮(露出海床)
   - 海面快速上涨
   - 海水翻腾、旋转
   - 发出怪声

3. 其他征兆:
   - 远处轰鸣声
   - 海平面异常升降
   - 动物异常(鸟类惊飞)
```

**记住**:
- 地震后有10-30分钟逃生时间(近源海啸可能更短)
- 海水退去是海啸来临最明显信号
- 不要去看"奇观",立即逃命!

#### 官方预警

```
✓ 手机预警短信
✓ 电视、广播
✓ 警报系统
✓ 社区广播
```

### 海啸来临前

**立即撤离**(分秒必争!):

```
1. 往高处跑!
   目标:
   - 海拔至少30米(约10层楼)
   - 或距海岸3公里以上
   - 越高越远越好

2. 往内陆跑!
   - 垂直于海岸线
   - 远离海滩、港口
   - 远离河流(海啸会沿河逆流)

3. 携带:
   - 仅必需品
   - 不要浪费时间收拾
   - 生命>一切财物

4. 路线:
   - 提前规划逃生路线
   - 熟悉高地位置
   - 知道最近的坚固高层建筑
```

**如无法到达高地**:
```
最后选择:
- 爬上坚固高层建筑(至少3层)
- 钢筋混凝土结构最佳
- 避免木结构、临时建筑
- 爬到最高层
```

### 海啸期间

**在建筑内**:
```
✓ 最高楼层
✓ 远离窗户
✓ 抓牢固定物
✓ 保护头部
```

**被海水卷走**:
```
极度危险!尽全力:

1. 抓住:
   - 漂浮物(木头、门板)
   - 固定物(树、柱子)

2. 保护:
   - 头部(防撞击)
   - 保持口鼻在水面上

3. 等待:
   - 水流减弱
   - 救援到来

4. 注意:
   - 回流危险
   - 漂浮碎片
```

### 海啸后

**不要立即返回!**

```
⚠️ 关键:海啸通常不是一波!

- 第一波后会有多波
- 间隔几分钟到几小时
- 后续波可能更大
- 至少等待8-12小时
- 听从官方通知
```

**返回时**:
```
✓ 官方宣布安全
✓ 注意建筑结构
✓ 远离垂落电线
✓ 不要进入受损建筑
✓ 小心污染水源
✓ 检查燃气泄漏
✓ 拍照记录损失(保险)
```

**健康安全**:
```
✓ 煮沸饮用水
✓ 避免接触污水
✓ 清洁伤口
✓ 预防传染病
✓ 心理创伤辅导
```

### 海啸预防准备

**沿海居民**:
```
必备知识:
✓ 了解海啸风险
✓ 规划撤离路线(至少2条)
✓ 知道高地/避难所位置
✓ 家庭演练计划
✓ 外地联系人

必备物资:
✓ 应急包
✓ 重要文件防水保存
✓ 应急药物
```

**游客**:
```
✓ 询问酒店海啸撤离路线
✓ 了解预警系统
✓ 认识撤离标志
✓ 感受到地震立即撤离
✓ 不要低估危险
```

### 特殊提醒

**儿童教育**:
```
教导儿童:
- 地震后立即往高处跑
- 不要等大人
- 跟随人群往高处
- 记住集合地点
```

**夜间海啸**:
```
更危险:
- 难以观察海水变化
- 听到异响立即撤离
- 不要迟疑
```

## 风暴潮

### 什么是风暴潮

强烈大气扰动(台风、寒潮等)引起的海面异常升高。

**危害**:
- 海水倒灌
- 淹没沿海地区
- 破坏海堤
- 加剧台风/洪水影响

### 预警

```
关注:
✓ 台风预警(常伴随风暴潮)
✓ 天文大潮时间(农历初一、十五)
✓ 海洋预报
✓ 当地预警信息
```

### 应对措施

**与台风应对类似**:

```
提前准备:
✓ 加固房屋
✓ 准备应急物资
✓ 规划撤离路线

危险区撤离:
✓ 低洼沿海地区
✓ 海堤外
✓ 听从指挥
✓ 往高处转移

期间:
✓ 远离海边
✓ 不要观潮
✓ 不要在海堤上
```

### 特殊注意

**天文大潮 + 台风 + 暴雨 = 极度危险!**

```
三碰头时:
- 风暴潮最严重
- 提高警惕
- 提前防范
- 及时撤离
```

**沿海养殖户**:
```
✓ 关注预警
✓ 提前转移物资
✓ 固定设施
✓ 生命>财产
```

## 📝 总结要点

### 海洋灾害核心原则

**海啸**:
```
1. 感觉地震,立即撤离
2. 海水异常,立即撤离
3. 往高处跑(30米以上)
4. 往内陆跑(3公里以上)
5. 不要返回(多波海啸)
6. 生命>一切财物
```

**风暴潮**:
```
1. 关注预警
2. 低洼区撤离
3. 远离海边
4. 加固防护
```

### 记住

```
时间就是生命:
- 海啸逃生时间有限
- 不要犹豫
- 不要收拾东西
- 不要围观
- 立即逃命

准备:
- 沿海居民必须有撤离计划
- 定期演练
- 熟悉路线
- 教育家人
```

### 海啸生存链

```
1. 识别预警(自然/官方)
   ↓
2. 立即决定撤离
   ↓
3. 快速往高处/内陆跑
   ↓
4. 到达安全地点
   ↓
5. 等待多波过去
   ↓
6. 官方通知后才返回
```
