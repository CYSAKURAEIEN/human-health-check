## 台风/飓风

### 台风等级(中国标准)

| 级别 | 风速 | 影响 |
|------|------|------|
| 热带低压 | <17.2 m/s | 轻微 |
| 热带风暴 | 17.2-24.4 m/s | 中等 |
| 强热带风暴 | 24.5-32.6 m/s | 较大 |
| 台风 | 32.7-41.4 m/s | 严重 |
| 强台风 | 41.5-50.9 m/s | 极严重 |
| 超强台风 | ≥51.0 m/s | 毁灭性 |

### 台风来临前

**24-48小时前**:
```
关注预报:
✓ 气象局预警
✓ 台风路径
✓ 预计登陆时间和地点
✓ 影响范围

准备物资:
✓ 食物和水(至少3天)
✓ 应急包
✓ 手电筒、电池
✓ 收音机
✓ 急救包
✓ 药品
✓ 重要文件防水保存
✓ 充满电(手机、充电宝)
```

**12-24小时前**:
```
加固房屋:
✓ 关闭门窗
✓ 贴胶带防玻璃飞溅(米字形)
✓ 收起阳台、院子物品
✓ 清理排水沟
✓ 固定室外易飞物品

撤离准备:
✓ 如在危险区,听从指挥撤离
✓ 低洼地带
✓ 危房
✓ 沿海地区
✓ 携带应急包
```

### 台风期间

**在室内**:
```
✓ 远离门窗
✓ 躲在内墙角、卫生间
✓ 关闭电源总闸
✓ 不要使用电器
✓ 储水(可能停水)
✓ 保持通讯畅通

❌ 不要:
- 外出
- 靠近窗户
- 使用明火
- 乘电梯
```

**台风眼**:
```
⚠️ 警惕台风眼!
- 台风眼经过时风平浪静
- 但很快第二次风暴来袭
- 不要以为台风过去而外出
- 继续躲避
```

**在室外**(被困):
```
1. 寻找坚固建筑躲避
2. 避开:
   ❌ 大树下
   ❌ 广告牌下
   ❌ 临时建筑
   ❌ 电线杆附近
3. 趴在地势低的地方
4. 抓紧固定物
5. 保护头部
```

**在车内**:
```
- 立即停车
- 避开树木、电线、广告牌
- 留在车内
- 系好安全带
- 趴下保护头部
```

### 台风后

```
检查:
✓ 房屋结构
✓ 燃气泄漏
✓ 电线损坏
✓ 水质安全

清理:
✓ 积水(防蚊虫)
✓ 倒塌物
✓ 清洁消毒

注意:
❌ 不要触碰垂落电线
❌ 不要饮用未经处理的水
❌ 不要急于返回危险区
✓ 注意次生灾害(滑坡、洪水)
```

## 洪水

### 洪水类型

**暴雨洪水**: 短时强降雨
**河流泛滥**: 持续降雨、上游水库泄洪
**山洪**: 山区突发洪水,极度危险
**城市内涝**: 排水系统超负荷

### 洪水前

**关注预警**:
```
✓ 天气预报
✓ 河流水位
✓ 官方预警
✓ 上游降雨情况
```

**准备**:
```
✓ 应急包放在易取处
✓ 规划撤离路线
✓ 了解高地位置
✓ 准备救生衣/漂浮物
✓ 重要物品移至高处
```

### 洪水时

**撤离**:
```
立即撤离如果:
- 收到撤离命令
- 水位快速上涨
- 房屋进水
- 周围已被淹

撤离要点:
✓ 往高处走
✓ 避开河道、山谷
✓ 携带应急包
✓ 关闭电源、燃气
✓ 不要单独行动
```

**涉水安全**:
```
⚠️ 涉水极度危险!

如必须涉水:
- 水深>膝盖,不要冒险
- 水流>站稳,不要涉水
- 使用棍子探路
- 抓紧固定物
- 集体手拉手(增加稳定)

记住:
- 15厘米快速流水可冲倒人
- 60厘米水深可冲走汽车
```

**驾车遇洪水**:
```
❌ 绝对不要:
- 开车进入淹水区
- "我能开过去"的想法非常危险!

如被困车内:
1. 立即弃车
2. 往高处走
3. 如车辆漂浮:
   - 打开车窗/打破车窗
   - 爬到车顶
   - 等待救援
```

**被困建筑**:
```
1. 往高处移动(楼顶)
2. 携带:
   - 手机
   - 哨子
   - 鲜艳衣物
3. 发出求救信号
4. 不要进入电梯
5. 不要轻易游泳逃生(水流湍急)
```

**急流自救**:
```
如被冲入急流:
1. 面朝上,脚朝下
2. 仰泳姿势
3. 脚在前防撞击
4. 顺流而下
5. 斜向岸边移动
6. 抓住漂浮物
7. 抓住岸边可及物
```

### 洪水后

```
返回前:
✓ 等待官方通知
✓ 确认安全

检查房屋:
✓ 结构损坏
✓ 电气安全(请专业人员)
✓ 燃气泄漏

清理:
✓ 排水(慢慢排,防结构损坏)
✓ 清洁消毒
✓ 丢弃污染食物
✓ 不要饮用井水(污染)

健康:
✓ 预防疾病(霍乱、伤寒)
✓ 避免接触污水
✓ 清洁伤口
✓ 防蚊虫
```

## 暴风雪

### 危险

- 能见度低,迷路
- 失温
- 交通中断
- 停电断水
- 积雪压塌建筑
- 雪崩(山区)

### 暴风雪前

```
准备:
✓ 储备食物、水(至少1周)
✓ 备用取暖设备(安全的)
✓ 应急照明
✓ 电池、收音机
✓ 药品
✓ 充足燃料(汽车、发电机)
✓ 铲雪工具

房屋:
✓ 检查供暖系统
✓ 保温材料
✓ 清理排水沟
✓ 修剪树枝(防压断电线)
```

### 暴风雪中

**在家中**:
```
✓ 留在室内
✓ 保暖:
  - 多层衣物
  - 密闭一个房间取暖
  - 用毛巾堵门缝
✓ 节省燃料
✓ 防一氧化碳中毒:
  - 不要在室内烧炭取暖
  - 使用燃料取暖时保持通风
✓ 储水(管道可能冻裂)
✓ 定期清理屋顶积雪(防压塌)
```

**在车内被困**:
```
1. 留在车内!
   - 车是庇护所
   - 外出易迷失

2. 发信号:
   - 鲜艳布条系在天线
   - 定时打开危险信号灯
   - 偶尔鸣笛

3. 保暖:
   - 发动机每小时运转10分钟取暖
   - 确保排气管未被雪堵塞(防一氧化碳中毒)
   - 多活动身体
   - 和他人挤在一起

4. 保持清醒:
   - 轮流睡觉
   - 防止睡着冻伤

5. 节省:
   - 燃料
   - 食物
   - 电池
```

**在户外**:
```
⚠️ 暴风雪中外出极度危险!

如被困户外:
1. 寻找/建造庇护所:
   - 天然庇护(洞穴、岩石)
   - 雪洞
   - 挡风墙

2. 保持干燥:
   - 湿衣服致命
   - 脱掉湿衣
   - 用干雪擦干

3. 保暖:
   - 多层衣物
   - 保护头部(30%热量从头部散失)
   - 覆盖口鼻(温暖吸入空气)
   - 活动身体

4. 不要:
   ❌ 吃雪(降低体温)
   ❌ 饮酒(虽感觉温暖,实际降温)
   ❌ 睡觉(易失温)
```

### 暴风雪后

```
✓ 小心外出(路滑、能见度低)
✓ 避免过度劳累(铲雪是重体力活)
✓ 注意雪崩风险(山区)
✓ 检查冻伤
✓ 补充水分和能量
```

## 龙卷风

### 识别

**云层特征**:
- 乌云翻滚
- 漏斗云
- 低垂云层

**其他征兆**:
- 天空呈绿色
- 雷电频繁
- 大冰雹
- 响亮持续的轰鸣声(像火车)
- 突然平静(龙卷风前)

### 龙卷风警报

**Watch(关注)**: 条件适合,可能形成
**Warning(警告)**: 已发现龙卷风,立即行动!

### 应对措施

**在家中**:
```
立即行动!

最佳位置:
1. 地下室:
   - 最安全
   - 躲在坚固工作台下
   - 远离窗户

2. 无地下室:
   - 最低楼层
   - 内部房间(卫生间、壁橱)
   - 无窗户
   - 远离外墙

姿势:
✓ 蹲下或趴下
✓ 双手抱头
✓ 保护头颈
✓ 用厚垫子、床垫覆盖

❌ 不要:
- 待在移动房屋
- 开窗"平衡气压"(错误!)
- 躲在汽车里
```

**在学校/公共建筑**:
```
✓ 内部走廊(最低楼层)
✓ 小房间
✓ 远离窗户、大厅
✓ 蹲下,保护头部
✓ 听从指挥
```

**在室外**:
```
1. 寻找坚固建筑

2. 无建筑:
   - 趴在低洼处
   - 沟渠、水坑
   - 双手抱头

3. 不要:
   ❌ 躲在桥下(狭管效应,风更大)
   ❌ 躲树下
   ❌ 试图跑过龙卷风

4. 远离:
   - 电线杆
   - 树木
   - 汽车
```

**在车内**:
```
⚠️ 汽车不安全!

龙卷风接近:
1. 弃车!
2. 趴在附近低洼处
3. 距离车远一些(防被砸)

如无处可去:
- 系好安全带
- 趴下低于窗户
- 用手、毯子保护头
```

### 龙卷风后

```
✓ 检查伤情
✓ 远离倒塌建筑
✓ 远离垂落电线
✓ 使用手电筒(不要明火)
✓ 帮助他人
✓ 听从救援指挥
```

## 干旱

### 影响

- 缺水
- 农作物歉收
- 粮食短缺
- 火灾风险增加
- 健康问题

### 应对

**节水措施**:
```
日常:
✓ 缩短洗浴时间
✓ 修理漏水
✓ 重复用水(洗菜水浇花)
✓ 收集雨水
✓ 减少洗车
✓ 使用节水器具

农业:
✓ 滴灌
✓ 选择耐旱作物
✓ 覆盖保湿
✓ 改进灌溉时间(早晚)
```

**寻找水源**:
```
✓ 深挖水井
✓ 雨水收集系统
✓ 水资源共享
✓ 远距离运水
```

**健康防护**:
```
✓ 保证饮水
✓ 防中暑
✓ 注意卫生(水少易传染病)
✓ 营养补充
```

**火灾预防**:
```
✓ 清理可燃物
✓ 禁止野外用火
✓ 准备灭火工具
```

## 雷电

### 雷电危险

- 直接雷击(致命)
- 侧面闪络
- 接触电压
- 跨步电压

### 室外避雷

**寻找安全处**:
```
✓ 坚固建筑内
✓ 封闭汽车内
✓ 远离高大物体
```

**紧急姿势**(无处躲避):
```
双脚并拢蹲姿:
1. 双脚完全并拢
2. 蹲下
3. 双手抱膝
4. 头低下
5. 尽量小的接触地面面积

作用:
- 减小被击中概率
- 减少跨步电压伤害
```

**避开**:
```
❌ 开阔地、山顶、水面
❌ 大树下(至少远离5米)
❌ 金属物附近
❌ 电线杆、旗杆
❌ 孤立棚屋、凉亭
❌ 水中(游泳、钓鱼)
```

**集体避雷**:
```
- 分散开(至少3米)
- 不要手拉手
- 各自蹲姿
```

### 室内避雷

```
✓ 关闭门窗
✓ 拔掉电器插头
✓ 不要打电话(有线)
✓ 不要洗澡
✓ 远离水管、金属管道
✓ 远离窗户
✓ 不要靠墙
```

### 车内避雷

```
✓ 相对安全(金属车厢形成法拉第笼)
✓ 关闭车窗
✓ 不要触摸金属部件
✓ 停车,不要下车
✓ 等雷暴过去
```

### 雷击急救

```
⚠️ 可以触摸被雷击者(不带电)!

1. 立即检查呼吸心跳
2. 如无,立即CPR
3. 雷击者有很高复苏率
4. 持续CPR直到急救到达
5. 治疗烧伤
6. 所有雷击者都应就医
```

### 判断雷暴距离

```
闪电-雷声法:
1. 看到闪电开始计数
2. 听到雷声停止
3. 秒数÷3 = 约公里数

例:9秒 = 约3公里

如<30秒(10公里),立即躲避!
```

## 📝 总结要点

### 气象灾害共同原则

```
1. 提前关注预警
2. 准备应急物资
3. 及时撤离
4. 听从官方指挥
5. 不要冒险
```

### 各类灾害优先级

**台风**: 加固房屋,危险区撤离
**洪水**: 往高处走,不要涉水
**暴风雪**: 留在庇护所,保暖
**龙卷风**: 地下室/内部房间,保护头部
**雷电**: 室内或车内,双脚并拢蹲姿

### 记住

```
✓ 生命>财产
✓ 预防>应对
✓ 准备充分
✓ 保持冷静
✓ 帮助他人
```
