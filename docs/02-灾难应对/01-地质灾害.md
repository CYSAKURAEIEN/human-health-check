## 地震

### 地震前兆(不一定准确)

- 动物异常(不安、逃离)
- 地下水异常(变色、变味、涌出)
- 地光、地声
- 小震频繁

⚠️ **不要依赖前兆!准备最重要!**

### 地震时应对

#### 室内

**"伏地、掩护、抓牢"(Drop, Cover, Hold on)**

```
1. 伏地(Drop):
   - 立即趴下,双手双膝着地

2. 掩护(Cover):
   - 钻到坚固的桌子或家具下
   - 用手保护头颈

3. 抓牢(Hold on):
   - 抓牢家具
   - 随着震动移动
   - 保持姿势直到震动停止

如果没有桌子:
- 靠内墙角蹲下
- 双手护头
- 远离窗户、镜子、吊灯
```

**不同地点**:

**在床上**:
- 留在床上
- 用枕头保护头部

**在厨房**:
- 远离炉灶
- 远离吊柜
- 蹲在坚固家具旁

**在卫生间**:
- 相对安全(空间小、墙多)
- 躲在承重墙角
- 保护头部

**在高楼**:
- 不要跳楼!
- 不要乘电梯!
- 就地躲避
- 震后走楼梯疏散

#### 室外

```
✓ 跑到开阔地
✓ 远离建筑物
✓ 远离电线杆、路灯
✓ 远离立交桥、高架桥
✓ 蹲下,保护头部

避开:
❌ 建筑物旁(坠物)
❌ 狭窄街道
❌ 围墙、栅栏旁
❌ 悬崖边、河岸边
```

#### 特殊场所

**在车内**:
```
1. 靠边停车(避开桥梁、立交桥、电线)
2. 留在车内
3. 系好安全带
4. 趴在座位下
5. 震后小心驾驶(检查路况)
```

**在剧场、商场等公共场所**:
```
1. 不要拥挤冲向出口
2. 蹲在座椅旁、柱子边
3. 保护头部
4. 听从指挥
5. 震后有序疏散
```

**在海边**:
```
⚠️ 地震后可能引发海啸!
1. 震后立即向高处转移
2. 远离海岸至少3公里
3. 爬到高处(至少30米)
```

### ❌ 地震时不要做

```
- 不要跳楼
- 不要乘电梯
- 不要靠近窗户
- 不要拥挤推搡
- 不要在狭窄过道逗留
- 不要使用明火
```

### 地震后应对

#### 立即行动

**1. 检查伤情**:
```
- 先自己,再他人
- 提供急救
- 不要移动重伤者(除非有危险)
```

**2. 检查隐患**:
```
- 闻:燃气泄漏
  * 如闻到,开窗通风
  * 关闭总阀
  * 不要开关电器(防火花)
  * 撤离并报告

- 看:电线是否损坏
  * 关闭电源总闸
  * 远离垂落电线

- 查:水管破裂
  * 关闭进水阀
```

**3. 准备余震**:
```
✓ 余震可能持续数天甚至数月
✓ 可能和主震一样强
✓ 持续警惕
✓ 保持"伏地、掩护、抓牢"姿势
```

**4. 疏散**:
```
- 走楼梯(不要乘电梯)
- 携带应急包
- 关闭水电气
- 锁门(防盗)
- 前往安全地点
```

#### 被困时

```
如果被困在废墟下:

✓ 保持冷静
✓ 保存体力
✓ 寻找水和食物
✓ 创造生存空间
✓ 定期呼救:
  - 敲击管道、墙壁
  - 吹哨子
  - 不要大声喊叫(浪费体力、吸入粉尘)
✓ 用手机:
  - 发送短信(更省电)
  - 定时开机
✓ 掩住口鼻(防尘)
✓ 等待救援

避免:
❌ 不要点火(消耗氧气、可能引起爆炸)
❌ 不要乱动(防坍塌)
```

### 地震安全准备

#### 家庭准备

**固定家具**:
```
✓ 书柜、衣柜固定在墙上
✓ 重物放低处
✓ 柜门安装插销
✓ 固定热水器、冰箱
✓ 床头不要挂重物
```

**应急物资**:
- 参见[家庭应急包](README.md#家庭应急包)
- 准备在门口、车里、工作地点

**制定计划**:
- 避震位置
- 疏散路线
- 集合地点
- 外地联系人

**定期演练**:
- 家庭地震演练
- 每年至少2次

#### 建筑安全

**抗震加固**:
- 老旧房屋抗震鉴定
- 必要时加固
- 符合抗震标准

**了解房屋结构**:
- 承重墙位置
- 相对安全区域

### 地震强度参考

| 震级 | 烈度 | 感觉和破坏 |
|------|------|------------|
| <3级 | I-II | 基本无感 |
| 3-4级 | III-IV | 室内有感,悬挂物摇晃 |
| 5-6级 | VI-VIII | 破坏性地震,建筑受损 |
| 7-8级 | IX-XI | 大地震,严重破坏 |
| >8级 | XII | 特大地震,毁灭性 |

## 泥石流

### 特征

- 暴雨后或连续降雨
- 山谷或沟谷
- 水流浑浊、夹带泥沙
- 伴有轰鸣声
- 流速快(可达数十公里/小时)

### 前兆

- 河水突然断流或水位突降
- 听到轰鸣声(像火车)
- 地面震动
- 山体裂缝
- 动物异常惊恐

### 应对措施

#### 提前预防

**了解风险**:
```
高风险区域:
- 山区沟谷
- 暴雨后
- 植被破坏区
- 地震灾区
```

**避免**:
- 暴雨时不要在沟谷逗留
- 不要在沟口扎营
- 注意天气预报

#### 发生时

```
⚠️ 立即行动!

1. 向两侧山坡跑!
   ✓ 垂直于泥石流方向
   ✓ 往高处跑
   ✓ 向两侧跑

   ❌ 不要顺沟向下游跑
   ❌ 不要往沟里跑

2. 丢弃物品
   - 抛弃沉重物品
   - 保留必需品

3. 不要上树
   - 可能被冲倒

4. 不要躲在凹坡
   - 可能被掩埋
```

### 逃生原则

```
✓ 向高处跑
✓ 向两侧跑
✓ 跑得越快越好
✓ 跑得越高越好
✓ 跑得越远越好
```

## 山体滑坡

### 前兆

- 山体出现裂缝
- 泉水、井水突然干涸或浑浊
- 山坡渗水增加
- 听到异常声响
- 树木倾斜
- 动物异常
- 地面下沉

### 应对措施

#### 预防

```
避免:
❌ 在滑坡体上建房
❌ 破坏山体植被
❌ 在坡脚挖土
❌ 暴雨时登山

识别危险山体:
- 坡度陡(>40度)
- 土质松散
- 裂缝明显
- 植被破坏
```

#### 发生时

```
1. 向两侧撤离!
   ✓ 垂直于滑坡方向
   ✓ 往稳定区跑

   ❌ 不要顺滑坡往下跑

2. 跑不掉时:
   - 抱紧大树(根深的)
   - 但不是最佳选择

3. 在家中:
   - 听到异响立即撤离
   - 不要犹豫收拾财物
```

#### 撤离后

```
✓ 不要立即返回
✓ 注意次生滑坡
✓ 远离危险区
✓ 报告有关部门
```

## 火山喷发

### 危害

- 熔岩流
- 火山灰
- 火山碎屑流(极度危险,时速可达数百公里)
- 有毒气体
- 泥流
- 地震

### 预警信号

- 地震频繁
- 地热增加
- 水质改变
- 气体释放
- 动物异常
- 官方预警

### 应对措施

#### 喷发前

```
如果在火山区:

✓ 关注官方预警
✓ 准备疏散
✓ 准备应急包
✓ 计划撤离路线
✓ 准备防护用品:
  - 护目镜
  - 口罩/湿布
  - 长袖衣物
```

#### 喷发时

**疏散**:
```
1. 立即撤离火山区
2. 远离河谷(泥流风险)
3. 向上风方向撤离
4. 驾车撤离:
   - 关闭窗户
   - 打开空调循环模式
   - 缓慢行驶
```

**防护**:
```
✓ 戴护目镜(防火山灰)
✓ 用湿布捂住口鼻
✓ 穿长袖长裤
✓ 进入室内:
  - 关闭门窗
  - 密封通风口
  - 呆在室内
```

#### 火山灰防护

```
火山灰危害:
- 呼吸系统
- 眼睛刺激
- 损坏机械
- 压塌建筑

防护:
✓ 不要外出
✓ 戴口罩/呼吸器
✓ 护目镜
✓ 清除屋顶积灰(防压塌)
✓ 不要驾车(损坏发动机)
```

#### 喷发后

```
✓ 等待官方通知
✓ 不要立即返回
✓ 清理火山灰:
  - 戴防护装备
  - 避免扬尘
  - 用水冲洗
✓ 检查饮用水(可能污染)
✓ 检查建筑结构
```

## 📝 总结要点

### 地质灾害共同特点

- 突发性强
- 破坏力大
- 难以准确预测
- 可能引发次生灾害

### 应对核心原则

```
1. 事前准备
   - 了解风险
   - 准备物资
   - 制定计划
   - 定期演练

2. 事中应对
   - 保持冷静
   - 快速反应
   - 正确躲避
   - 及时撤离

3. 事后处理
   - 检查伤情
   - 警惕余震/次生灾害
   - 等待救援
   - 有序恢复
```

### 记住

```
✓ 生命第一
✓ 掌握正确逃生方向
✓ 不要贪恋财物
✓ 服从专业指挥
✓ 互助合作
```
