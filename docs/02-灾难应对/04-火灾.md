## 建筑火灾

### 火灾预防

**家庭防火检查**:
```
每月检查:
✓ 烟雾报警器(测试、换电池)
✓ 电线老化
✓ 插座过载
✓ 燃气管道
✓ 灭火器(有效期、压力)
✓ 消防通道畅通
✓ 阳台杂物清理
```

**消防设备**:
```
必备:
✓ 烟雾报警器(每层至少1个)
✓ 灭火器(厨房、卧室)
✓ 逃生绳(高层)
✓ 防烟面罩
✓ 应急手电

位置:
- 易取处
- 家人都知道
- 定期检查
```

**逃生计划**:
```
制定:
✓ 至少2条逃生路线
✓ 集合地点(楼外)
✓ 每个房间的逃生路线
✓ 特殊人员帮助计划

演练:
✓ 每半年一次
✓ 夜间演练(睡着时起火最危险)
✓ 摸黑演练(模拟浓烟)
✓ 计时(2分钟内逃出)
```

### 发现火灾

**立即**:
```
1. 大喊"着火了!"
2. 启动火警报警器
3. 拨打119
4. 准确报告:
   - 地址
   - 着火物
   - 火势大小
   - 有无人员被困
   - 姓名电话
5. 派人在路口接消防车
```

**小火扑救**:
```
仅当:
✓ 火势很小(如垃圾桶)
✓ 有灭火器
✓ 有逃生退路
✓ 已报警

方法:
1. 使用灭火器
2. 站在上风处
3. 对准火源根部
4. 左右扫射
5. 如无效,立即撤离
```

**大火逃生**:
```
立即逃生!不要:
❌ 试图扑救
❌ 收拾财物
❌ 寻找宠物
❌ 贪恋财物

优先:
✓ 生命
✓ 逃生
✓ 逃生
```

### 火场逃生

#### 逃生原则

**1. 判断逃生路线**:
```
开门前:
1. 用手背触摸门把手
   - 烫?不要开门!
   - 不烫?小心打开一条缝观察

2. 看门缝:
   - 有烟?不要开门
   - 无烟?可以开门

3. 选择:
   - 如能逃生,立即走
   - 如不能,固守待援
```

**2. 低姿前进**:
```
浓烟中:
✓ 弯腰或匍匐
✓ 用湿毛巾捂住口鼻
✓ 沿墙壁前进
✓ 关闭身后的门(防烟扩散)
```

**3. 逃生路径**:
```
✓ 首选楼梯
✓ 往下跑(除非下方是火源)
✓ 关闭经过的防火门

❌ 绝不:
- 乘电梯(断电会困住,烟囱效应)
- 往上跑(除非顶楼有出口)
- 返回取物
```

#### 被困时

**无法逃离房间**:
```
1. 关门(防火门可挡2小时)
2. 用湿布条堵门缝
3. 泼水降温(门、墙)
4. 往窗户移动
5. 发出求救信号:
   - 挥舞鲜艳衣物
   - 打手电
   - 敲击金属
   - 大声呼救
6. 等待救援
7. 湿毛巾捂口鼻
8. 低姿减少吸入烟雾
```

**被困楼梯间**:
```
- 楼梯间相对安全
- 如无法下楼:
  * 等待救援
  * 关闭楼梯门
  * 发信号
```

**高层被困**:
```
⚠️ 不要跳楼!(3层以上致命)

选择:
1. 固守房间(如上述)
2. 逃生绳(提前准备,练习使用)
3. 床单打结:
   - 固定牢靠
   - 只作为最后手段
   - 极度危险
4. 等待救援
```

### 身上着火

```
⚠️ 不要跑!(氧气助燃)

停-躺-滚:
1. 停下
2. 躺倒
3. 双手护脸
4. 滚动灭火
5. 或用厚衣物、毯子裹住灭火
```

### 特殊场所火灾

#### 高层建筑

```
特点:
- 疏散困难
- 烟雾扩散快
- 消防车高度有限

应对:
✓ 熟悉逃生路线
✓ 不要乘电梯
✓ 避难层(如有)
✓ 固守待援(无法下楼时)
```

#### 地下空间

```
特点:
- 出口少
- 烟雾聚集
- 方向难辨

应对:
✓ 记住入口位置
✓ 沿墙壁前进
✓ 往上走
✓ 跟随人群
```

#### 公共场所

```
电影院、商场、KTV等:

✓ 进入时观察安全出口
✓ 不要慌乱拥挤
✓ 听从指挥
✓ 寻找多个出口
✓ 不要返回拿物品
```

### 灭火器使用

**类型**:
- **ABC干粉**: 通用(最常见)
- **CO₂**: 电气火灾
- **水基**: 固体火灾
- **泡沫**: 液体火灾

**使用方法**:
```
提-拔-握-压:
1. 提起灭火器
2. 拔掉保险销
3. 握住喷管
4. 压下手柄
5. 对准火焰根部
6. 左右扫射
7. 距离3-5米
```

**禁忌**:
```
❌ 不要用水扑救:
- 电气火灾(触电)
- 油类火灾(飞溅扩大)
- 金属火灾(爆炸)
```

## 森林火灾

### 预防

```
野外用火安全:
✓ 遵守禁火令
✓ 指定区域用火
✓ 挖防火坑
✓ 完全熄灭:
  - 用水浇透
  - 土埋严实
  - 检查无明火和暗火
✓ 不要:
  - 乱扔烟头
  - 焚烧垃圾
  - 燃放烟花
```

### 发现森林火灾

```
立即:
✓ 拨打森林火警:95119
✓ 报告位置、火势
✓ 如火小,可扑救
✓ 如火大,立即撤离
```

### 逃生

**判断风向**:
```
⚠️ 关键!

逃生方向:
✓ 逆风方向
✓ 向河流、湖泊、空旷地
✓ 向植被稀疏处
✓ 向山下(火向上蔓延快)

❌ 避免:
- 顺风跑(追不上火)
- 往山上跑
- 往浓密植被区
```

**快速逃离**:
```
1. 丢弃易燃物品(背包等)
2. 用湿布捂口鼻
3. 低姿前进
4. 如被包围:
   - 点燃身前草地(制造隔离带)
   - 进入燃烧后的区域
   - 等待火过去
5. 如实在无路:
   - 卧倒在空地、沟渠
   - 用土掩埋身体
   - 头朝火来方向
```

### 扑救安全

```
⚠️ 仅专业人员!

普通人:
✓ 小火可扑救
✓ 拍打火焰(湿布、树枝)
✓ 扬土压火
✓ 组织隔离带
✓ 保持逃生路线
❌ 大火不要冒险
```

## 电气火灾

### 原因

- 线路老化、短路
- 插座过载
- 电器故障
- 私拉乱接

### 预防

```
✓ 定期检查线路
✓ 不超负荷用电
✓ 不私拉乱接
✓ 及时更换老化线路
✓ 使用合格电器
✓ 电器不用时断电
✓ 不要:
  - 电线压在重物下
  - 湿手触碰电器
  - 电器长期待机
```

### 扑救

```
⚠️ 首先断电!

1. 切断电源:
   - 关闭总闸
   - 或拔掉插头

2. 未断电前:
   ❌ 不要用水
   ❌ 不要用泡沫灭火器
   ✓ 使用干粉或CO₂灭火器

3. 断电后:
   - 可用水扑救
   - 注意复燃
```

## 燃气火灾

### 预防

```
✓ 定期检查燃气管道
✓ 使用燃气时有人看管
✓ 保持通风
✓ 及时更换老化管道
✓ 使用合格燃气器具
```

### 燃气泄漏

**发现泄漏**:
```
闻到燃气味:

立即:
✓ 开窗通风
✓ 关闭燃气总阀
✓ 疏散人员
✓ 到户外拨打报警

❌ 禁止:
- 开关任何电器(产生火花)
- 打电话(在室内)
- 使用明火
- 按门铃
- 开抽油烟机
```

**已起火**:
```
1. 关闭燃气阀门(如能接近)
2. 用湿布、灭火器扑救
3. 如无法靠近,立即撤离
4. 拨打119
5. 疏散周围人员
```
