## ⚠️ 灾难应对基本原则

### 灾前准备(Prevention & Preparedness)

```
1. 了解风险
   - 所在地区易发灾害
   - 预警系统
   - 疏散路线

2. 制定计划
   - 家庭应急计划
   - 集合地点
   - 外地联系人

3. 准备物资
   - 应急包
   - 食物和水
   - 重要文件备份

4. 学习技能
   - 急救知识
   - 自救技能
   - 应急演练
```

### 灾中应对(Response)

```
1. 保持冷静
2. 评估情况
3. 确保安全
4. 遵循指示
5. 及时撤离
6. 互助合作
```

### 灾后恢复(Recovery)

```
1. 检查伤情
2. 确认安全
3. 联系家人
4. 评估损失
5. 清理重建
6. 心理恢复
```

## 🎒 家庭应急包

### 基础装备

**水和食物**(至少3天份):
- 瓶装水:每人每天3-4升
- 不易腐食品:
  - 罐头食品
  - 压缩饼干
  - 能量棒
  - 干果、坚果
  - 婴儿食品(如需要)

**工具和用品**:
- 电池式或手摇收音机
- 手电筒(多个)
- 备用电池
- 充电宝
- 瑞士军刀/多功能工具
- 哨子
- 防水火柴/打火机
- 蜡烛
- 塑料布、胶带
- 绳索
- 纸笔

**个人用品**:
- 急救包
- 处方药(至少一周)
- 眼镜/隐形眼镜
- 个人卫生用品
- 湿巾、消毒液
- 垃圾袋
- 卫生纸

**衣物和寝具**:
- 换洗衣物
- 雨衣/防水服
- 结实的鞋
- 帽子、手套
- 毯子/睡袋
- 保温毯

**重要文件**(防水袋密封):
- 身份证、护照复印件
- 银行账户信息
- 保险文件
- 医疗记录
- 紧急联系人名单
- 家庭照片(用于寻人)

**现金**:
- 小额纸币和硬币
- ATM可能无法使用

**特殊需求**:
- 婴儿:奶粉、尿布、奶瓶
- 老人:药物、助听器电池
- 宠物:食物、水、牵引绳、笼子

### 车载应急包

- 应急包基础物品
- 跳线电缆
- 破窗器
- 反光三角牌
- 灭火器

## 🏠 家庭安全检查

### 日常检查

**结构安全**:
```
✓ 固定高大家具(书柜、衣柜)
✓ 检查房屋结构(裂缝、倾斜)
✓ 安装烟雾报警器
✓ 准备灭火器
```

**燃气水电**:
```
✓ 知道如何关闭总闸
✓ 检查燃气管道
✓ 电线老化更换
✓ 准备应急照明
```

**逃生路线**:
```
✓ 规划至少2条逃生路线
✓ 确保通道畅通
✓ 准备逃生绳(高层)
✓ 定期演练
```

## 👨‍👩‍👧‍👦 家庭应急计划

### 制定计划

1. **集合地点**:
   - 家附近:房子外的安全地点
   - 社区内:学校、公园等
   - 社区外:亲友家、指定地点

2. **外地联系人**:
   - 选择外地亲友
   - 全家都记住电话
   - 灾时向其报平安

3. **重要信息卡**:
   - 每人携带
   - 包含:
     * 紧急联系人
     * 医疗信息
     * 血型、过敏
     * 用药情况

4. **特殊成员**:
   - 老人
   - 儿童
   - 残疾人
   - 宠物

### 演练

- 每半年演练一次
- 包括:
  - 紧急疏散
  - 关闭水电气
  - 使用应急包
  - 联系外地亲友

## 🆘 求救信号

### 国际通用

**声音信号**:
- 三声短哨,停顿,重复
- 间隔1分钟重复

**视觉信号**:
- SOS(... --- ...)
- 三堆火(三角形排列)
- 反光镜反射
- 挥舞鲜艳衣物
- 地面大字(HELP、SOS)

**手机**:
- 发送位置信息
- 留言报告情况
- 节省电量

## 🔔 重要提醒

### 灾难来临时

```
✓ 生命第一,财产第二
✓ 听从专业指挥
✓ 不要盲目返回
✓ 帮助他人(在安全前提下)
✓ 保持通信联络
✓ 记录保存证据
```

### 避免谣言

```
✓ 关注官方信息
✓ 不信谣、不传谣
✓ 核实信息来源
✓ 保持理性判断
```

### 心理准备

```
✓ 灾后心理创伤是正常的
✓ 寻求专业心理支持
✓ 家人相互支持
✓ 参与社区重建
✓ 保持希望
```
