## 战乱与武装冲突

### 冲突前准备

**信息收集**:
```
✓ 关注时事新闻
✓ 注册使领馆
✓ 了解撤离路线
✓ 准备应急包
✓ 备份重要文件
✓ 储备现金
```

**应急计划**:
```
✓ 安全集合地点
✓ 多条撤离路线
✓ 外地联系人
✓ 多种通讯方式
✓ 安全屋(如可能)
```

### 冲突期间

**保护自身**:
```
✓ 留在室内(安全时)
✓ 远离窗户
✓ 地下室或内部房间
✓ 储备食物和水
✓ 减少外出
✓ 保持低调
✓ 避开军事目标
```

**遇到炮击/空袭**:
```
立即:
1. 寻找掩护:
   - 地下室
   - 坚固建筑底层
   - 远离窗户
2. 趴下
3. 保护头部
4. 远离外墙
5. 等待停止
```

**遇到枪战**:
```
立即:
1. 卧倒!
2. 寻找掩体:
   - 墙壁、柱子
   - 汽车发动机舱侧面
   - 坚固家具
3. 爬行到安全处
4. 不要站立或奔跑
5. 保护头部和vital organs
```

**检查站**:
```
✓ 保持冷静
✓ 双手可见
✓ 服从指令
✓ 缓慢动作
✓ 不要突然移动
✓ 携带身份证件
```

### 撤离

**时机**:
```
考虑撤离当:
- 官方建议撤离
- 冲突升级
- 生活物资短缺
- 有安全撤离通道
```

**撤离准备**:
```
✓ 监听官方信息
✓ 联系使领馆
✓ 选择安全路线
✓ 避开交战区
✓ 携带应急包
✓ 轻装上阵
✓ 标识中立(如白旗、红十字)
```

**集体撤离**:
```
✓ 跟随人群
✓ 互相帮助
✓ 保护弱者
✓ 保持联络
```

## 化学品泄漏

### 识别化学品泄漏

**征兆**:
- 异常气味
- 烟雾或蒸汽
- 死亡的动植物
- 刺激眼睛、喉咙
- 呼吸困难
- 头晕恶心
- 皮肤刺激

### 立即行动

**室外遇到**:
```
1. 判断风向:
   - 观察烟雾、旗帜
   - 感觉风向

2. 往上风、侧风方向撤离:
   ✓ 垂直于风向
   ✓ 远离泄漏区
   ✓ 往高处(有些气体比空气重)

3. 如在泄漏区:
   - 用湿布捂住口鼻
   - 快速撤离
   - 避免接触液体或烟雾

4. 脱离后:
   - 脱掉污染衣物(不要从头部脱)
   - 冲洗身体(清水,至少15分钟)
   - 就医
```

**室内应对**:
```
如泄漏在室外,你在室内:

1. 判断:
   - 室内更安全?进入
   - 还是撤离更安全?

2. 如留在室内:
   ✓ 关闭门窗
   ✓ 关闭通风系统、空调
   ✓ 用湿毛巾堵门缝
   ✓ 往上层移动(有些气体重)
   ✓ 备水(湿毛巾捂鼻)
   ✓ 听从官方指示

3. 如需撤离:
   - 往上风方向
   - 用湿布保护呼吸道
```

### 不同化学品

**氯气**(黄绿色、刺鼻):
```
- 比空气重,下沉
- 往上风、高处撤离
- 用碱性溶液中和(小苏打水)
```

**氨气**(刺激性臭味):
```
- 比空气轻,上升
- 往低处撤离
- 用酸性溶液中和(醋、柠檬汁)
```

**硫酸雾**:
```
- 不要用水冲洗(放热)
- 先用干布擦除
- 再用大量水冲洗
```

**未知化学品**:
```
- 不要接触
- 不要闻
- 远离
- 报警
```

### 除污

**皮肤接触**:
```
1. 脱去污染衣物
2. 大量清水冲洗15-20分钟
3. 不要使用中和剂(除非明确)
4. 就医
5. 带上化学品信息
```

**眼睛接触**:
```
1. 立即冲洗(清水)
2. 翻开眼皮冲洗
3. 至少15分钟
4. 不要揉眼
5. 就医
```

**吸入**:
```
1. 转移到新鲜空气处
2. 保持呼吸道通畅
3. 如呼吸停止,CPR
4. 就医
```

## 核辐射

### 辐射来源

- 核电站事故
- 核武器
- 放射性物质泄漏
- 脏弹

### 防护原则

**三要素**:
```
1. 时间:减少暴露时间
2. 距离:远离辐射源
3. 屏蔽:利用材料阻挡
```

### 核事故应对

**官方警报**:
```
✓ 听从指示
✓ 撤离或就地掩蔽
✓ 准备撤离
```

**就地掩蔽**:
```
如官方建议留在室内:

1. 进入室内:
   - 最好是地下室
   - 中心房间
   - 远离外墙

2. 关闭门窗:
   - 密封
   - 关闭通风系统

3. 脱去外衣:
   - 可能沾染放射性微粒
   - 密封在塑料袋中
   - 远离生活区

4. 淋浴:
   - 温和清洗
   - 不要擦伤皮肤

5. 储备:
   - 食物和水
   - 收音机
   - 保持通信

6. 等待:
   - 听从官方通知
   - 可能数天
```

**撤离**:
```
1. 按指定路线撤离
2. 携带:
   - 应急包
   - 药物
   - 身份文件
3. 关闭门窗
4. 开车时关闭车窗,空调内循环
```

**碘片**:
```
⚠️ 仅在官方指示下服用!

作用:
- 保护甲状腺
- 阻止吸收放射性碘

注意:
- 不防护其他辐射
- 不治疗辐射
- 有禁忌症
- 听从专业指导
```

### 辐射防护

**减少暴露**:
```
✓ 远离辐射区
✓ 留在室内
✓ 地下室最佳
✓ 减少外出时间
```

**避免摄入**:
```
✓ 不要饮用未经处理的水
✓ 不要食用当地食物(可能污染)
✓ 使用密封包装食品和瓶装水
✓ 清洗食物
```

**长期防护**:
```
✓ 监测辐射水平
✓ 避免高污染区
✓ 定期健康检查
✓ 心理辅导
```

## 恐怖袭击

### 类型

- 爆炸
- 枪击
- 车辆冲撞
- 化学/生物/核袭击

### 爆炸袭击

**现场应对**:
```
1. 卧倒:
   - 脸朝下
   - 双手护头
   - 张嘴(减少气压伤)

2. 寻找掩护:
   - 坚固物体后
   - 远离窗户、镜子

3. 二次爆炸:
   - 可能有
   - 不要立即返回
   - 快速撤离

4. 撤离:
   - 远离现场
   - 往上风方向
   - 帮助他人
   - 不要触碰可疑物品
```

### 枪击袭击

**"跑-躲-打"原则**:

**1. 跑(Run)**:
```
如有逃生路线:
✓ 立即跑
✓ 丢下物品
✓ 帮助他人(如可能)
✓ 阻止他人进入危险区
✓ 拨打报警
```

**2. 躲(Hide)**:
```
如无法逃跑:
1. 躲在坚固掩体后
2. 锁门
3. 堵门
4. 关灯
5. 静音手机
6. 趴下,远离门窗
7. 保持安静
8. 如可能,报警(文字信息)
```

**3. 打(Fight)**:
```
⚠️ 最后手段!

如袭击者进入,生命受威胁:
- 全力反击
- 使用任何武器:
  * 椅子、灭火器
  * 热水、化学品
  * 集体反击
- 目标:制服或逃脱
```

### 车辆冲撞

```
警惕:
✓ 观察周围
✓ 听到异常声音
✓ 看到车辆加速冲向人群

反应:
1. 往侧面跑
2. 躲到障碍物后:
   - 柱子、树木
   - 车辆、建筑
3. 如无法躲避,跳起或侧滚
```

### 人质劫持

```
如被劫持:

✓ 保持冷静
✓ 服从指令
✓ 不要反抗(初期)
✓ 观察周围(如可能)
✓ 保持低调
✓ 不要与劫持者对视
✓ 不要激怒
✓ 如救援到来:
  - 立即卧倒
  - 双手抱头
  - 等待指令
```

### 可疑物品

```
识别:
- 无人看管包裹
- 形状异常
- 电线、天线外露
- 异常声音、气味
- 标签奇怪

应对:
✓ 不要触碰!
✓ 远离
✓ 报警
✓ 疏散周围人
✓ 等待专业人员
```

## 📝 总结要点

### 人为灾难共同原则

```
1. 保持警惕
2. 关注预警
3. 快速反应
4. 保护自己
5. 帮助他人
6. 听从专业指挥
```

### 各类灾难应对

**战乱**:
- 低调行事
- 远离冲突区
- 及时撤离

**化学品**:
- 判断风向
- 往上风撤离
- 密闭室内

**核辐射**:
- 时间、距离、屏蔽
- 就地掩蔽或撤离
- 避免摄入

**恐怖袭击**:
- 跑-躲-打
- 远离现场
- 快速撤离

### 记住

```
✓ 提前准备
✓ 保持冷静
✓ 快速决策
✓ 不要恐慌
✓ 互相帮助
✓ 生命第一
```

### 心理准备

```
人为灾难的特点:
- 突发性
- 不确定性
- 恐惧性

应对:
✓ 心理准备
✓ 学习知识
✓ 模拟演练
✓ 保持希望
✓ 寻求支持
```
