# 基础冶金知识

> **冶金入门 - 从零开始掌握金属冶炼的基础**

## ⚠️ 安全第一

**冶金是高危作业!在开始之前必须了解:**

- 🔥 工作温度可达1500°C以上
- ☠️ 一氧化碳中毒风险(无色无味剧毒)
- 💥 熔融金属遇水会爆炸性飞溅
- 🌡️ 高温引起的严重烧伤
- 🏭 重金属污染

**必须在室外、通风良好、远离居住区的地方进行!**

---

## 一、冶金基本原理

### 1.1 什么是冶金?

**简单说:** 从矿石中提取金属的过程。

**化学原理:**
```
金属氧化物 + 碳(还原剂) → 金属 + 二氧化碳
例: Fe₂O₃ + 3C → 2Fe + 3CO↑
```

### 1.2 关键要素

| 要素 | 作用 | 说明 |
|------|------|------|
| **矿石** | 原料 | 含金属的天然矿物 |
| **燃料** | 提供热量 | 木炭、焦炭 |
| **还原剂** | 夺取氧气 | 通常是碳(木炭) |
| **助熔剂** | 降低熔点 | 石灰石、石英 |
| **鼓风** | 提高温度 | 增加氧气供应 |

### 1.3 温度需求

| 金属 | 熔点 | 所需炉温 | 难度 |
|------|------|----------|------|
| 铜 | 1085°C | 1200°C | ⭐⭐ 简单 |
| 金 | 1064°C | 1200°C | ⭐⭐ 简单 |
| 银 | 962°C | 1100°C | ⭐⭐ 简单 |
| 铁 | 1538°C | 1600°C+ | ⭐⭐⭐⭐ 困难 |
| 锡 | 232°C | 300°C | ⭐ 非常简单 |

**关键:**
- 木炭明火: ~700-800°C (不够)
- 木炭+鼓风: 1200-1400°C (可炼铜)
- 木炭+强鼓风+耐火材料: 1500-1600°C (可炼铁)

## 二、冶炼炉的建造

### 2.1 简易黏土炉(初级)

**适用于:** 铜、锡、铅等低熔点金属

#### 材料清单:
- 黏土: 50-100公斤
- 沙子: 20公斤(混入黏土增强)
- 稻草/干草: 1-2公斤(防开裂)
- 石块: 若干(砌炉底)

#### 建造步骤:

**第1步: 选址**
- 平整地面,清除可燃物
- 上风向10米内无建筑
- 靠近水源(安全)

**第2步: 打地基**
```
[俯视图]
○ = 通风口
═ = 石块

   正面
   ║
 ╔═══╗
 ║ ○ ║← 通风口(直径10cm)
 ╚═══╝
 ┌───┐
 │地基│← 石块垫高30cm
 └───┘
```

- 用石块垫高30-50cm(防潮)
- 留一个通风口

**第3步: 配制耐火黏土**
```
配方:
- 黏土: 70%
- 细沙: 25%
- 切碎的稻草: 5%
- 水: 适量,调至可塑状
```

**搅拌:** 反复踩踏、揉捏30分钟

**第4步: 砌炉墙**
- 墙厚: 10-15cm
- 内径: 30-40cm
- 高度: 50-60cm
- 炉底: 倾斜便于倒渣

**第5步: 晾干(关键!)**
- 阴干3-5天(避免阳光直晒开裂)
- 小火慢慢烘烤2-3天
- 第一天: 小火30分钟,停火,重复3次
- 第二天: 小火2小时
- 第三天: 中火4小时

**第6步: 高温定型**
- 大火烧烤6-8小时
- 黏土会变硬、略微收缩
- 出现裂缝用黏土糊补

#### 鼓风装置

**简易风箱(必需!):**

```
[侧视图]

活动板 ↓
┌─────┐
│     │  皮革/布料
│  □  │  ←空气
└──║──┘
   ║← 竹筒/木管
   ↓
  炉子
```

**制作:**
1. 木框: 40×30×15cm
2. 顶部蒙皮革或厚帆布
3. 底部开口连接竹筒
4. 竹筒插入炉子通风口
5. 手动或脚踩压缩空气

**高级版:**
- 双腔风箱(一压一松都送风)
- 连续鼓风,效率高

### 2.2 高温炉(炼铁用)

**材料:**
- 耐火砖或耐火黏土(普通黏土不够)
- 耐火黏土配方:
  - 铝矾土(或高岭土): 60%
  - 石英砂: 30%
  - 黏土: 10%

**结构改进:**
- 更厚的炉壁(20-30cm)
- 更高的炉身(100-150cm)
- 预热进风(利用废气加热鼓风)
- 多层装料

**炉型:**
```
[剖面图]
   ╔═══╗← 投料口
   ║   ║
   ║   ║← 炉腹(最宽处)
   ║   ║
   ╚═══╝
    ║║← 炉腰(收窄)
    ║║
   ╔═╗
   ║○║← 风口
   ╚═╝
   ╔═╗← 出铁口
```

## 三、燃料制作

### 3.1 木炭的重要性

**为什么必须用木炭?**
- ✗ 木柴: 含水分、杂质,温度低(仅700°C)
- ✓ 木炭: 纯碳,燃烧温度高(1200-1400°C)
- ✓ 木炭无烟,一氧化碳浓度更稳定

### 3.2 木炭制作(闷烧法)

**原理:** 隔绝空气加热木材,挥发物逸出,剩余碳。

#### 土窑法(大量生产)

**步骤:**

**1. 挖坑:**
- 直径2-3米
- 深度1-1.5米
- 坑底中心挖烟道

**2. 装柴:**
- 木材竖立摆放(直径5-15cm)
- 中心留空隙作烟囱
- 装满后顶部覆盖小树枝

**3. 封土:**
- 覆盖10-20cm厚的湿土
- 留顶部和底部通气孔
- 不能完全封死

**4. 点火:**
- 从顶部点燃
- 火势稳定后逐渐缩小顶部通气孔

**5. 闷烧:**
- 持续3-7天
- 观察烟色:
  - 白烟(水蒸气) → 初期
  - 黄烟(木焦油) → 中期
  - 青烟(接近完成)
  - 透明热气(完成)

**6. 熄火:**
- 完全封死所有通气孔
- 等待冷却2-3天
- 扒开取炭

**产量:**
- 1吨木材 → 200-300公斤木炭
- 优质硬木(如栎木)出炭率更高

#### 简易铁桶法(小量生产)

**步骤:**
1. 准备大铁桶,盖上开小孔(排气)
2. 装满木材块
3. 倒扣在火堆上加热
4. 小孔冒烟时点燃(燃烧可燃气体)
5. 烟变清后撤火,等待冷却
6. 取出木炭

**时间:** 4-8小时

### 3.3 木炭质量检验

**优质木炭:**
- ✓ 黑色有金属光泽
- ✓ 敲击声音清脆
- ✓ 断面纹路清晰
- ✓ 燃烧时无烟或少烟

**劣质木炭:**
- ✗ 褐色或未完全炭化
- ✗ 多灰分
- ✗ 易碎

## 四、温度控制

### 4.1 判断温度

**无温度计情况下:**

| 温度 | 金属颜色 | 说明 |
|------|----------|------|
| 500°C | 暗红色 | 刚可见红 |
| 700°C | 深红色 | - |
| 900°C | 樱桃红 | 锻造温度 |
| 1100°C | 橙红色 | 铜熔化 |
| 1300°C | 黄色 | 高温区 |
| 1500°C | 亮黄色 | 炼铁温度 |
| 1700°C | 白色 | 极高温 |

**辅助方法:**
- 投入小块铜: 融化表示>1085°C
- 投入铁钉: 融化表示>1538°C

### 4.2 提高温度的方法

**1. 加强鼓风**
- 持续、稳定的鼓风
- 双风箱轮流工作

**2. 优化燃料**
- 使用优质硬木炭
- 木炭块大小适中(核桃大小)

**3. 炉体保温**
- 加厚炉壁
- 炉顶部分封闭(减少热损失)

**4. 预热进料**
- 利用废气预热空气

**5. 燃料与矿石的比例**
- 炼铁: 木炭:矿石 ≈ 1:1(重量)
- 炼铜: 木炭:矿石 ≈ 1:2

## 五、矿石识别与采集

### 5.1 铁矿石

#### 磁铁矿 Fe₃O₄
- **外观:** 黑色、有金属光泽
- **特征:** 有磁性(关键!)
- **含铁:** 60-70%
- **识别:** 用磁铁吸附
- **分布:** 山区、河床

#### 赤铁矿 Fe₂O₃
- **外观:** 红褐色
- **特征:** 粉末状时呈红色
- **含铁:** 50-60%
- **识别:** 在白瓷板上划红色条痕
- **分布:** 沉积岩地区

#### 褐铁矿 FeO(OH)
- **外观:** 黄褐色至褐色
- **特征:** 土状、块状
- **含铁:** 40-50%
- **识别:** 较轻、易碎
- **分布:** 风化带、沼泽

### 5.2 铜矿石

#### 孔雀石 Cu₂(OH)₂CO₃
- **外观:** 鲜艳的绿色
- **特征:** 条带状花纹
- **含铜:** 55-60%
- **识别:** 颜色特殊,易辨认
- **最易冶炼的铜矿!**

#### 辉铜矿 Cu₂S
- **外观:** 铅灰色、金属光泽
- **特征:** 较重、柔软
- **含铜:** 79%
- **识别:** 刀刻有铜的光泽

#### 黄铜矿 CuFeS₂
- **外观:** 黄铜色
- **特征:** 金属光泽
- **含铜:** 34%
- **识别:** 颜色似黄金但更硬

### 5.3 其他有用矿石

#### 锡石 SnO₂
- **外观:** 黑色或棕色,光泽强
- **用途:** 制作青铜(铜+锡)
- **熔点低:** 232°C,极易冶炼

#### 方铅矿 PbS
- **外观:** 铅灰色、金属光泽强
- **特征:** 很重、立方体晶体
- **用途:** 铅制品、合金
- **警告:** 铅有毒!

### 5.4 矿石富集

**原矿石通常含杂质多,需要富集:**

#### 重力选矿(淘洗法)
```
1. 粉碎矿石(拳头大小)
2. 放入水槽中
3. 摇晃淘洗
4. 轻的杂质浮起或流走
5. 重的金属矿石沉底
```

**适用:** 密度大的矿石(铁、铜、锡)

#### 磁选(仅限磁铁矿)
```
1. 粉碎矿石
2. 用强磁铁吸附
3. 分离出磁性的磁铁矿
```

#### 焙烧(去硫)
```
某些硫化矿(如黄铁矿FeS₂):
1. 露天堆放
2. 点火焙烧
3. 硫变成SO₂逸出
4. 残留物为氧化铁(可冶炼)
```

### 5.5 矿石粉碎

**为什么要粉碎?**
- 增大反应面积
- 提高冶炼效率
- 便于与燃料混合

**方法:**
1. **石臼捣碎** - 人力
2. **石锤敲碎** - 人力
3. **石磨碾碎** - 畜力
4. **水力碾磨** - 利用水流

**粒度:**
- 炼铁: 花生到核桃大小
- 炼铜: 黄豆到花生大小
- 越小越好,但粉末太细会影响透气

## 六、安全防护

### 6.1 必备防护装备

| 装备 | 用途 | 简易替代 |
|------|------|----------|
| **厚皮手套** | 防高温 | 多层湿布包手 |
| **护目镜** | 防飞溅 | 透明塑料片+绳子 |
| **口罩** | 防粉尘 | 多层湿布 |
| **厚围裙** | 防烧伤 | 湿皮革 |
| **长柄钳** | 夹持高温物 | 木棍+铁丝 |
| **长柄勺** | 舀熔融金属 | 耐火黏土做碗+长木柄 |

### 6.2 作业安全规则

#### 环境安全
- [ ] 10米内无可燃物
- [ ] 上风向无人居住
- [ ] 准备灭火沙和水
- [ ] 夜间作业有照明和守夜人

#### 操作安全
- [ ] 所有工具完全干燥(湿气会爆炸!)
- [ ] 绝不在下雨天作业
- [ ] 熔融金属倒入预热的模具
- [ ] 远离风口观察

#### 个人安全
- [ ] 穿长袖长裤,扎紧袖口
- [ ] 头发扎起或戴帽
- [ ] 不穿易燃材料(化纤)
- [ ] 保持清醒,禁止饮酒

### 6.3 中毒预防

**一氧化碳中毒(头号杀手!)**

**来源:** 不完全燃烧产生CO

**症状:**
- 轻度: 头痛、头晕、恶心
- 中度: 意识模糊、乏力
- 重度: 昏迷、死亡

**预防:**
- ✓ 必须在室外或开阔处
- ✓ 多人作业,互相监督
- ✓ 感觉不适立即离开
- ✓ 炉子附近不要躺卧

**急救:**
1. 立即转移到新鲜空气处
2. 平躺、保暖
3. 清醒者喝热水
4. 昏迷者侧卧,防窒息
5. 呼吸停止立即CPR

**重金属中毒**

**高危金属:** 铅、汞、砷

**预防:**
- 作业后彻底洗手
- 不在作业区饮食
- 怀孕妇女和儿童禁止接触

### 6.4 烧伤处理

**轻度烧伤(发红、起泡):**
1. 立即冷水冲洗15-20分钟
2. 不要弄破水泡
3. 涂烧伤膏或芦荟汁
4. 无菌纱布包扎

**重度烧伤(皮肤焦黑):**
1. 用大量清水冲洗
2. 剪开衣物(不要撕拉)
3. 清洁敷料覆盖
4. 立即就医
5. 给予大量饮水(防休克)

**熔融金属飞溅:**
- 绝不能立即用水!
- 先扑灭衣物上的火
- 用干净布料包裹
- 就医处理

## 七、常见问题

### Q1: 完全新手要多久才能成功冶炼?

**A:**
- 铜: 1-2周准备,成功率60%
- 铁: 1-2个月准备,初次成功率20-30%
- 建议从铜开始,积累经验

### Q2: 一次能产多少金属?

**A:**
- 小型炉(1次): 0.5-2公斤
- 中型炉(1次): 2-10公斤
- 大型炉(1次): 10-50公斤

### Q3: 木炭消耗量有多大?

**A:**
- 炼1公斤铁: 需要1-1.5公斤木炭
- 炼1公斤铜: 需要0.5-1公斤木炭
- 制1公斤木炭: 需要3-5公斤木材

**结论:** 需要大量木材!可持续性是关键。

### Q4: 可以用煤代替木炭吗?

**A:**
- 可以,但煤含硫,产品质量差
- 需要先炼成焦炭(类似木炭制法)
- 优点是热值更高

### Q5: 找不到矿石怎么办?

**A:**
- 回收废弃金属制品
- 旧汽车、机械(大量钢铁)
- 废弃电器(铜线)
- 旧建筑钢筋
- 重新熔化、锻造即可

### Q6: 一个人能完成冶炼吗?

**A:**
- 理论可以,但强度很大
- 建议至少2-3人:
  - 1人鼓风
  - 1人加料
  - 1人监控和安全
- 冶炼是社区性工作

## 八、检查清单

### 开工前检查

- [ ] 炉子完全干燥,无裂缝
- [ ] 燃料充足(准备量的2倍)
- [ ] 鼓风装置工作正常
- [ ] 所有工具齐备且干燥
- [ ] 防护装备穿戴完毕
- [ ] 灭火设施就位
- [ ] 风向检查(下风向无人)
- [ ] 天气晴朗,无雨

### 作业中监控

- [ ] 每15分钟检查炉温
- [ ] 鼓风稳定持续
- [ ] 观察烟色变化
- [ ] 人员状态正常(无头晕等)
- [ ] 燃料及时补充

### 结束后处理

- [ ] 熄火后至少等待12小时
- [ ] 余烬完全熄灭
- [ ] 残渣妥善处理(可能含有毒物质)
- [ ] 工具清洁归位
- [ ] 记录本次经验(温度、配比、结果)
