## 一、热处理基础

### 1.1 什么是热处理?

**热处理 = 加热 + 保温 + 冷却**

**目的:** 改变金属内部结构,获得需要的性能

**关键要素:**
- 加热温度
- 保温时间
- 冷却速度

### 1.2 钢的相变(简化理解)

```
常温: 铁素体(软) + 渗碳体(硬)
  ↓ 加热到临界温度(约727°C)
高温: 奥氏体(可塑)
  ↓
  ├→ 快速冷却(淬火)
  │   → 马氏体(极硬、脆)
  │
  ├→ 慢速冷却(退火)
  │   → 铁素体+珠光体(软、韧)
  │
  └→ 中速冷却(正火)
      → 铁素体+珠光体(细化,强度好)
```

**关键:**
- **冷却速度决定性能!**
- 快冷 → 硬
- 慢冷 → 软

### 1.3 温度判断(无温度计)

| 温度 | 颜色 | 用途 |
|------|------|------|
| 200°C | 浅黄色(稻草色) | 回火-剪刀、刀具 |
| 250°C | 金黄色 | 回火-斧头、凿子 |
| 300°C | 褐色 | 回火-弹簧 |
| 400°C | 紫色 | - |
| 500°C | 暗红(刚可见) | - |
| 700°C | 深红 | 退火、回火 |
| 800°C | 樱桃红 | **淬火温度** |
| 900°C | 鲜红 | 锻造 |
| 1100°C | 橙黄 | 过热 |

**记忆口诀:**
- 淬火: 樱桃红入水
- 回火: 稻草黄空冷

## 二、淬火技术

### 2.1 什么是淬火?

**淬火 = 加热至红色 + 快速冷却(水或油)**

**目的:** 硬化金属表面

**适用:** 钢(不适用纯铁或熟铁!)

### 2.2 淬火操作

**步骤:**

**1. 加热:**
```
加热至淬火温度:
- 碳钢: 780-850°C(樱桃红至鲜红)
- 颜色判断: 暗室中观察更准确
```

**2. 保温:**
```
保持红热状态:
- 小件: 30秒-2分钟
- 大件: 5-10分钟
- 目的: 均匀透热
```

**3. 快速冷却:**
```
方法A: 水淬(最快)
  - 直上直下插入水中
  - 快速搅动(均匀冷却)
  - 冷却至不发红

方法B: 油淬(较慢)
  - 冷却速度慢,不易开裂
  - 用于复杂形状、大件

方法C: 分级淬火
  - 先入热水/温油(200-300°C)
  - 停留片刻
  - 再入冷水
  - 减少开裂风险
```

**4. 检查:**
```
淬火成功的特征:
- 表面呈青灰色
- 硬度极高(锉刀打滑)
- 敲击声音清脆
```

### 2.3 淬火液选择

| 淬火液 | 冷却速度 | 优点 | 缺点 | 用途 |
|--------|----------|------|------|------|
| **冷水** | 极快 | 简单易得,硬度高 | 易开裂 | 简单形状,小件 |
| **盐水** | 最快 | 硬度最高 | 最易裂 | 特殊需求 |
| **温水(30-60°C)** | 快 | 比冷水裂少 | 需加热水 | 中等件 |
| **机油/植物油** | 慢 | 不易裂 | 冒烟、着火风险 | 复杂形状 |
| **动物脂肪** | 慢 | 古代常用 | 臭味 | 替代油 |

### 2.4 选择性淬火(局部淬火)

**为什么?**
- 刀刃需要硬(锋利)
- 刀背需要韧(不断)

**方法:**

**方法A: 遮蔽淬火**
```
1. 涂泥在不想淬火的部分(刀背)
2. 厚度5-10mm
3. 整体加热
4. 入水淬火
5. 泥保护的部分冷却慢,不淬硬
```

**方法B: 局部加热**
```
1. 只加热刃口部分至红热
2. 刀背保持暗色
3. 入水,刃口先入
4. 只有刃口淬硬
```

**方法C: 涂水淬火**
```
1. 整体加热
2. 用湿布快速擦拭刃口
3. 刃口快速冷却淬硬
4. 刀背缓慢冷却保持韧性
```

### 2.5 淬火常见问题

**问题1: 淬火后开裂**

**原因:**
- 冷却过快
- 温度过高
- 有内应力或裂纹

**预防:**
- 使用油淬代替水淬
- 温度准确控制
- 原材料质量好

**问题2: 淬不硬**

**原因:**
- 材料不是钢(是熟铁)
- 温度不够
- 冷却太慢

**解决:**
- 确认是钢材
- 提高温度至樱桃红
- 快速冷却

**问题3: 淬火后变形**

**原因:**
- 冷却不均匀
- 形状不对称

**解决:**
- 直上直下,不斜入
- 快速搅动
- 使用夹具固定

## 三、回火技术

### 3.1 为什么必须回火?

**淬火后的钢:**
- ✓ 硬度极高
- ✗ 脆性极大
- ✗ 一摔就碎!

**回火目的:**
- 减少脆性
- 保留适当硬度
- 获得硬韧兼备的状态

**原则: 淬火必回火!否则工具易碎!**

### 3.2 回火操作

**步骤:**

**1. 清理表面:**
- 用砂纸或磨石打磨淬火件
- 去除氧化皮
- 露出光亮金属面(重要!观察颜色)

**2. 加热:**
```
方法A: 炉火回火
  - 小火加热
  - 观察表面颜色变化

方法B: 明火回火
  - 木炭或煤气火焰
  - 不断移动,均匀加热

方法C: 金属浴回火
  - 熔融铅/盐(危险!)
  - 温度精确
```

**3. 观察颜色(关键!):**
```
淬火件表面会出现氧化色:

浅黄(220°C) → 剃刀、手术刀
金黄(240°C) → 菜刀、刨刀
褐色(260°C) → 斧头、凿子
紫色(280°C) → 剪刀
深蓝(300°C) → 锯条、弹簧
```

**4. 冷却:**
- 到达目标颜色后
- 空冷或油冷(不用水!)

### 3.3 不同工具的回火温度

| 工具 | 回火温度 | 回火颜色 | 性能要求 |
|------|----------|----------|----------|
| 锉刀、锯条 | 200-220°C | 浅黄 | 极硬,微脆 |
| 刀具、凿子 | 220-240°C | 金黄 | 硬,一定韧性 |
| 斧头、剪刀 | 240-270°C | 褐-紫 | 硬韧兼备 |
| 弹簧、钳子 | 300-350°C | 深蓝-灰 | 极韧,硬度适中 |

## 四、退火技术

### 4.1 什么是退火?

**退火 = 加热 + 极慢冷却**

**目的:**
- 软化金属
- 消除内应力
- 便于加工

### 4.2 退火操作

**步骤:**

**1. 加热:**
- 加热至700-800°C(深红至樱桃红)
- 保温30分钟-2小时

**2. 慢冷:**
```
方法A: 炉冷
  - 熄火后留在炉中
  - 随炉自然冷却
  - 12-24小时

方法B: 埋灰/沙中
  - 取出后埋入干灰/干沙
  - 保温慢冷

方法C: 石棉布包裹
  - 隔热保温
```

**3. 冷却到常温:**
- 完全冷却需24小时+

### 4.3 退火用途

**1. 锻前退火:**
- 金属太硬,难锻造
- 退火软化,便于加工

**2. 去应力退火:**
- 焊接、加工后有内应力
- 退火消除,防变形开裂

**3. 纠正淬火失误:**
- 淬火不满意
- 退火重来

## 五、正火技术

### 5.1 什么是正火?

**正火 = 加热 + 空冷**

**与退火区别:**
- 退火: 慢冷(炉冷)
- 正火: 空冷(自然冷却)

**目的:**
- 细化晶粒
- 均匀组织
- 提高强度和韧性

### 5.2 正火操作

**步骤:**

1. **加热:** 780-850°C(樱桃红)
2. **保温:** 10-30分钟
3. **出炉空冷:** 放在空气中自然冷却

**用途:**
- 锻造后正火(改善组织)
- 粗加工后正火(便于精加工)

## 六、渗碳技术

### 6.1 什么是渗碳?

**渗碳 = 让低碳铁吸收碳,表面变成钢**

**原理:**
- 高温下,碳原子渗入铁表面
- 表面含碳量提高
- 形成"软心硬壳"结构

**优势:**
- 省钢材(内部用熟铁)
- 性能优(外硬内韧)
- 古代宝剑常用!

### 6.2 固体渗碳(实用)

**材料:**
```
渗碳剂配方:
- 木炭粉: 70-80%
- 草木灰: 15-20%
- 食盐: 5-10%

混合均匀,备用
```

**步骤:**

**1. 准备:**
- 熟铁工件打磨光滑
- 准备密封铁盒或陶罐

**2. 装填:**
```
[剖面图]
┌────────────┐
│  渗碳剂(3cm)  │← 顶部
├────────────┤
│  ┌──────┐  │
│  │ 工件 │  │← 周围填满渗碳剂
│  └──────┘  │
├────────────┤
│  渗碳剂(3cm)  │← 底部
└────────────┘
用黏土密封
```

**3. 加热:**
```
温度: 900-950°C(鲜红)
时间:
  - 薄件(刀片2-3mm): 4-6小时
  - 中件(斧头5mm): 8-12小时
  - 厚件(10mm): 16-24小时

保持温度稳定!
```

**4. 冷却:**
- 随炉慢冷
- 或取出空冷

**5. 检验:**
- 表面应呈暗灰色
- 可见渗碳层

**6. 淬火:**
- 重新加热至樱桃红
- 淬火硬化渗碳层

**7. 回火:**
- 回火降低脆性

### 6.3 液体渗碳(快速)

**方法:**
```
1. 工件加热至樱桃红
2. 快速插入木炭粉
3. 表面粘附木炭
4. 再加热
5. 碳渗入表面
6. 重复5-10次
```

**优点:** 快(1-2小时)
**缺点:** 渗层浅(0.1-0.3mm)

### 6.4 渗碳深度

| 时间 | 温度 | 渗碳深度 |
|------|------|----------|
| 4小时 | 900°C | 0.3-0.5mm |
| 8小时 | 900°C | 0.5-1mm |
| 16小时 | 900°C | 1-1.5mm |
| 24小时 | 900°C | 1.5-2mm |

**够用深度:** 0.5-1mm(刀具)

## 七、其他表面处理

### 7.1 氮化(可选)

**原理:** 渗入氮,形成极硬氮化层

**简易方法:**
```
1. 工件埋入尿素或硝酸盐
2. 加热至500-550°C
3. 保温12-24小时
4. 形成氮化层

硬度: 比渗碳还高!
```

### 7.2 发蓝/发黑(防锈)

**原理:** 形成致密氧化膜

**方法:**
```
1. 工件表面除锈、除油
2. 加热至300-350°C(深蓝色)
3. 立即浸入油中
4. 表面形成蓝黑色氧化膜

或:
1. 浸入碱性氧化液(配方复杂)
2. 加热至135-155°C
3. 30-90分钟
4. 表面发黑
```

**效果:** 防锈,美观

### 7.3 镀层(高级)

**镀锡:**
- 熔融锡浸渍
- 防锈(马口铁)

**镀铜:**
- 电镀或化学镀
- 防锈,导电

## 八、质量控制

### 8.1 硬度测试

**方法A: 锉刀测试**
```
用锉刀在工件上推:
- 锉刀打滑,发出"咝咝"声 → 硬度高✓
- 能锉动 → 硬度不够✗
```

**方法B: 刮削测试**
```
用刀尖刮:
- 刮不动或微痕 → 硬✓
- 轻易刮出槽 → 软✗
```

**方法C: 火花测试(高级)**
```
在砂轮上打磨观察火花:
- 生铁: 火花短,爆裂
- 熟铁: 火花长,分叉少
- 钢: 火花中等,分叉多
- 淬火钢: 火花细密,明亮
```

### 8.2 韧性测试

**弯曲测试:**
```
刀刃向外弯曲:
- 90°不断裂 → 韧性好✓
- <45°断裂 → 脆,需回火✗
```

**跌落测试:**
```
从1米高跌落到水泥地:
- 完好 → 韧性好✓
- 崩口/断裂 → 脆✗
```

## 九、故障排除

### 问题1: 淬火后太脆,一用就崩口

**原因:** 未回火或回火不足

**解决:**
- 重新回火至250-280°C
- 观察颜色变化

### 问题2: 刀具不锋利,容易钝

**原因:**
- 淬火温度不够
- 冷却太慢

**解决:**
- 重新淬火
- 温度至樱桃红
- 快速水冷

### 问题3: 渗碳不均匀

**原因:**
- 渗碳剂填装不实
- 温度不稳定

**解决:**
- 渗碳剂压实
- 保持恒温

### 问题4: 工件变形严重

**原因:**
- 冷却不均匀
- 工件形状不对称

**解决:**
- 使用夹具
- 垂直入水
- 或改用油淬

## 十、实战案例

### 案例1: 制作一把高品质菜刀

**步骤:**
```
1. 材料: 熟铁500g
2. 锻造成刀形
3. 渗碳: 刃口部位,8小时
4. 淬火: 樱桃红,水淬(刃口先入)
5. 回火: 240°C(金黄色)
6. 磨刃
7. 安装柄

结果: 外硬内韧,锋利耐用!
```

### 案例2: 修复淬裂的工具

**问题:** 凿子淬火开裂

**修复:**
```
1. 退火(软化)
2. 加热至鲜红
3. 裂纹对齐
4. 锤打焊合
5. 重新磨光
6. 再淬火(这次用油)
7. 回火

成功率: 70%
```
