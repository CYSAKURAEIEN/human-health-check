# 简易工具锻造

> **末日最需要的工具 - 从零开始打造生存装备**

## 🔨 引言

**有了金属,还需要加工成工具才有用!**

本章节教你如何用熟铁、钢或青铜锻造最基本、最必需的工具。这些工具将成为你重建文明的基础。

**工具优先级:**
1. **锤子** - 制作其他工具的工具
2. **刀和斧** - 基本生存工具
3. **锄、镐** - 农业和采矿
4. **钉子** - 建造必需

---

## 一、锻造基础知识

### 1.1 什么是锻造?

**锻造 = 加热金属 + 锤打成型**

**原理:**
- 金属加热后变软
- 锤打可改变形状
- 冷却后保持新形状
- 反复锤打使金属更致密、更强

### 1.2 锻造温度

| 温度 | 颜色 | 状态 | 适用 |
|------|------|------|------|
| 暗红(500°C) | 暗红色勉强可见 | 太硬 | 不适合锻造 |
| 深红(700°C) | 深红色 | 较硬 | 可锻,但费力 |
| 樱桃红(900°C) | 樱桃红 | **最佳** | 锻造铁的理想温度 |
| 橙红(1100°C) | 橙红色 | 偏软 | 锻造,但易过烧 |
| 黄色(1300°C) | 黄色 | 过热 | 避免(金属损伤) |

**关键:**
- **铁的最佳锻造温度: 樱桃红(900°C)**
- 温度不够: 打不动,工具易裂
- 温度过高: 金属烧损,性能下降

### 1.3 必备工具

#### 最小工具集:

**1. 锻造炉:**
```
简易锻造炉:
- 比冶炼炉小得多
- 内径: 20cm
- 深度: 15cm
- 炉膛装木炭
- 侧面鼓风口
- 可用耐火砖临时砌
```

**2. 砧(铁砧):**
```
理想: 大块熟铁或钢(50-100公斤)
替代:
  - 大石块(花岗岩、玄武岩)
  - 铁轨段
  - 大铁块
要求: 平整、坚硬、稳固
```

**3. 锤子:**
```
初期: 石锤、铁锤头绑木柄
后期: 自己锻造专用锤
重量: 0.5-2公斤
```

**4. 钳子:**
```
用途: 夹持烫的工件
初期: 两根木棍+铁丝绑
后期: 锻造铁钳
```

**5. 凿子、冲子:**
```
用途: 开孔、切断
制作: 淬火硬化的铁棍
```

### 1.4 基本锻打技巧

**1. 平锤:**
- 垂直下锤
- 用于展平、延伸

**2. 斜锤:**
- 倾斜击打
- 用于塑形、收拢

**3. 折叠锻打:**
- 对折金属
- 反复锤打
- 增加层次,提升强度
- 日本刀的秘诀!

**4. 镦粗:**
- 垂直锤打端部
- 使金属增厚

**5. 拔长:**
- 持续锤打
- 使金属延伸、变薄

## 二、锤子锻造

**为什么先做锤子?**
- 锤子是"制作工具的工具"
- 有了好锤子,其他工具更好做

### 2.1 锤头锻造

**材料:**
- 熟铁块: 1公斤
- 或钢块: 0.8公斤(更好)

**步骤:**

**1. 加热:**
- 铁块加热至樱桃红

**2. 初步成型:**
```
圆柱形铁块
   ↓ 锤打一端
楔形(一端粗一端细)
   ↓ 锤打中部
中间收腰(安装柄的地方)
   ↓ 整形
锤头雏形
```

**3. 开眼(装柄的孔):**
```
方法A: 热冲法
  - 加热至樱桃红
  - 用尖凿从两面对冲
  - 打穿

方法B: 钻孔法
  - 锻造完成后
  - 用钻头钻孔(费时)
```

**4. 整形:**
- 锤击面: 平整光滑
- 楔面: 略弧形

**5. 淬火(如果是钢):**
- 加热至樱桃红
- 快速放入水中
- 表面硬化

**6. 回火:**
- 重新加热至200-300°C(浅黄色)
- 空冷
- 减少脆性

### 2.2 安装柄

**木柄制作:**
```
材料: 硬木(橡木、榆木、槐木)
长度: 30-40cm
粗细: 握持舒适,前细后粗

安装:
1. 木柄插入锤头孔
2. 打入木楔或铁楔(从顶部)
3. 楔子胀开,卡紧
```

## 三、刀具锻造

### 3.1 简易菜刀

**材料:**
- 熟铁: 500克(做刀身)
- 钢: 50克(做刃口,如有)

**步骤:**

**1. 拔长:**
- 铁块加热
- 反复锤打拉长成条
- 目标尺寸: 长20cm,宽8cm,厚5mm

**2. 成型刀身:**
```
锤打边缘,使其变薄
形成刀刃
   ___
  /   \  ← 刀刃(薄)
 |     | ← 刀身
 |_____| ← 刀背(厚,保持强度)
```

**3. 包钢(如果有钢):**
```
方法:
  - 刃口位置焊接钢片
  - 加热至接近熔化
  - 锤打使铁和钢结合
  - 形成"软铁背,钢刃口"
  - 日本刀、大马士革刀的原理!
```

**4. 磨刃:**
- 在磨石上研磨
- 形成锋利刃口

**5. 淬火(如果刃口是钢):**
- 刃口加热至樱桃红
- 只有刃口部分浸水淬火
- 刀背不淬(保持韧性)

**6. 回火:**
- 200-300°C回火
- 减少脆性

**7. 安装柄:**
- 木柄或包布

### 3.2 砍刀/匕首

**类似菜刀,但:**
- 更厚重(砍刀)
- 或更薄更尖(匕首)
- 刃口双刃或单刃
- 全钢最佳(如有)

## 四、斧头锻造

### 4.1 斧头结构

```
[侧视图]
       刃口
        ↓
    ___/  ← 斧头刃
   /
  |  ○  ← 安柄孔
  |
   \___  ← 斧头背(锤击用)
```

**关键:** 重心在刃口侧,砍伐有力

### 4.2 锻造步骤

**材料:**
- 熟铁: 1-1.5公斤

**步骤:**

**1. 对折锻打:**
```
长铁条
  ↓ 对折
双层
  ↓ 锤打焊合
一端开口(安柄孔)
  ↓ 整形
斧头雏形
```

**2. 开刃:**
- 锤打刃口使其变薄
- 形成斜面

**3. 开孔:**
- 对折处自然形成孔
- 或热冲、钻孔

**4. 淬火回火(如是钢):**
- 刃口淬火硬化
- 回火减脆

**5. 磨刃:**
- 磨石研磨
- 刃口角度20-30°

**6. 安装斧柄:**
- 木柄长60-80cm
- 前细后粗
- 楔子固定

## 五、农具锻造

### 5.1 锄头

**结构:**
```
[俯视图]
┌─────┐
│     │ ← 锄板(宽15cm,长20cm)
│  ║  │ ← 安柄套筒
└─────┘
```

**锻造:**
1. 铁块锤打成薄板
2. 一端卷成套筒(装柄)
3. 板面略弧形
4. 磨锋利边缘

**用途:** 翻土、除草

### 5.2 镐(十字镐)

**结构:**
```
  ╱╲  ← 尖头(破硬土、岩石)
 ╱  ╲
═════ ← 安柄孔
 ╲  ╱
  ╲╱  ← 凿形(可选)
```

**锻造:**
1. 铁块中部开孔(安柄)
2. 两端锤打拔尖
3. 淬火增硬(如是钢)

**用途:** 挖掘、采矿、破冰

### 5.3 镰刀

**结构:**
```
   ___
  /   ← 弧形刃口
 /
/
║ ← 柄
```

**锻造:**
1. 长铁条锤打成弯月形
2. 内侧磨成刃口
3. 一端锻造成安柄套筒

**用途:** 收割、割草

## 六、钉子和紧固件

### 6.1 铁钉锻造

**为什么重要?**
- 建筑、家具必需
- 用量巨大
- 简单但关键

**步骤:**

**1. 准备铁条:**
- 粗细: 铅笔粗细(钉子)到筷子粗细(大钉)
- 长度: 比成品钉长5cm

**2. 拔尖:**
- 加热一端至樱桃红
- 锤打成尖

**3. 做钉帽:**
```
方法A: 直接镦粗
  - 另一端加热
  - 垂直锤打,增厚成钉帽

方法B: 使用钉模
  - 铁板上钻小孔
  - 尖端插入孔
  - 露出的端部锤打成帽
```

**4. 切断:**
- 凿子切断

**5. 冷却:**
- 水冷

**效率:**
- 熟练后: 2-5分钟/根
- 一天可做100-200根!

### 6.2 铁环、链条

**铁环(扣环):**
1. 铁条绕圆柱(木棍)成螺旋
2. 切断成单个环
3. 每个环锻接闭合

**链条:**
- 环环相扣
- 用于吊挂、牵引

## 七、容器类

### 7.1 铁锅

**方法:** 铸造或锻造

**铸造法(简单):**
- 熔化铁(或用生铁)
- 倒入锅形模具
- 冷却成型

**锻造法(费力):**
1. 大铁板
2. 加热后用圆头锤反复击打
3. 形成凹形
4. 边缘向外卷

**用途:** 煮饭、烧水

### 7.2 铁桶

**锻造:**
1. 铁板围成圆筒
2. 边缘锤接或铆接
3. 焊接底部

**用途:** 运水、储物

## 八、质量检验

### 8.1 刀具检验

**1. 锋利度:**
```
测试: 削纸、削木
合格: 轻松削过
不合格: 重新磨刃
```

**2. 韧性:**
```
测试: 砍木头,观察刃口
合格: 不崩口、不卷刃
不合格: 重新回火或重做
```

**3. 硬度(钢刀):**
```
测试: 锉刀刻划
合格: 锉刀打滑(硬度够)
不合格: 能刻动(重新淬火)
```

### 8.2 工具检验

**锤子:**
- 锤头牢固不脱落
- 击打不崩裂
- 重心合适

**钉子:**
- 钉入木头不弯曲
- 钉帽不脱落

## 九、常见问题

### Q1: 锻造时金属开裂怎么办?

**原因:**
- 温度过低(冷锻)
- 杂质太多
- 锤击过猛

**解决:**
- 重新加热
- 裂纹处对齐,加热至将熔,锤打焊合
- 使用优质材料

### Q2: 刀具不够硬,容易变钝?

**原因:**
- 材料是熟铁(不是钢)
- 淬火温度不对
- 未淬火

**解决:**
- 使用钢材
- 或对熟铁进行渗碳(变成钢)
- 正确淬火

### Q3: 淬火后工具太脆,一砸就断?

**原因:**
- 淬火过度
- 未回火

**解决:**
- 必须回火!
- 淬火后重新加热至200-300°C
- 空冷

### Q4: 工具表面坑坑洼洼?

**正常!**
- 手工锻造品都这样
- 不影响使用
- 想要光滑: 反复锤打+打磨

### Q5: 一把刀要打多久?

**时间:**
- 新手: 6-10小时
- 熟练: 2-4小时
- 大师级: 1小时内

## 十、工具保养

### 10.1 防锈

**方法:**
1. 涂油(动物油、植物油、矿物油)
2. 干燥保存
3. 定期擦拭
4. 包油纸或布

### 10.2 刃口维护

**定期磨刀:**
- 粗磨: 砂轮石、粗磨刀石
- 精磨: 细磨刀石
- 磨刀角度保持一致

**钝了立即磨:**
- 不要等到严重钝化
- 越钝越难磨

### 10.3 柄的维护

**木柄:**
- 定期涂油防干裂
- 松动及时楔紧
- 断裂立即更换

## 十一、高级技巧

### 11.1 折叠锻打(百炼钢)

**方法:**
```
1. 铁块拔长
2. 对折
3. 加热锤打焊合
4. 再拔长、对折
5. 重复10-100次

结果:
  - 层次结构
  - 极致密
  - 强度、韧性大增
  - 花纹(大马士革钢)
```

### 11.2 嵌钢刃

**方法:**
- 软铁做身
- 钢做刃
- 锻接结合
- 优点: 省钢,性能好

### 11.3 表面硬化

**渗碳硬化:**
- 已在炼钢章节介绍
- 表面硬,内部韧
