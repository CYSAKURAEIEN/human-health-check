# 冶金技术

> **末日情况下从零开始的金属冶炼和工具制作**

## ⚠️ 重要警告

**冶金作业具有极高危险性!**

- 🔥 高温作业,易导致严重烧伤
- ☠️ 金属蒸汽有毒,需良好通风
- 💥 熔融金属遇水会爆炸
- ⚡ 需要专业防护装备
- 🏭 环境污染严重,需远离水源和居住区

**在非紧急情况下,请勿尝试!本指南仅供文明崩溃、工业体系瓦解等极端情况使用。**

### [01-铁的冶炼](./01-铁的冶炼.md)

从铁矿石到可用铁器的完整流程。

**核心内容:**
- 🪨 铁矿石的识别(磁铁矿、赤铁矿、褐铁矿)
- 🔥 炼铁基本原理(还原反应)
- ⚒️ 从矿石到生铁
- 🛠️ 从生铁到熟铁/锻铁
- ⚡ 炼钢技术(渗碳法)

**关键要点:**
- 铁矿石 + 木炭 → 生铁(1538°C)
- 生铁含碳高(脆),需精炼成熟铁
- 熟铁渗碳 → 钢

**时间成本:**
- 建炉: 3-7天
- 一次炼铁: 8-12小时
- 产量: 1-5公斤铁/次

铜和青铜的冶炼方法。

**核心内容:**
- 🟢 铜矿石识别(孔雀石、辉铜矿)
- 🔥 铜的冶炼(熔点1085°C,比铁容易)
- 🛡️ 青铜的制作(铜+锡)
- 🔨 黄铜的制作(铜+锌)
- 📿 简易铸造技术

**关键要点:**
- 铜是最易炼的金属
- 青铜(90%铜+10%锡)硬度高
- 可用于制作工具、武器、容器

**优势:**
- ✓ 熔点低,普通炉子即可
- ✓ 易于铸造成型
- ✓ 耐腐蚀

末日最需要的基础工具制作。

### [04-金属加工技术](./04-金属加工技术.md)

提高金属性能的处理方法。

**核心内容:**
- 🔨 锻造技术(加热锤打)
- 🏺 铸造技术(熔化浇铸)
- 🌡️ 热处理技术
  - 淬火(快速冷却增硬)
  - 回火(减少脆性)
  - 退火(软化便于加工)
  - 正火(细化晶粒)
- 🛡️ 表面处理(防锈、渗碳)

**关键要点:**
- 淬火: 加热至红色后急速水冷
- 回火: 淬火后再加热至200-300°C
- 正确的热处理可使刀具硬度提升3-5倍

## ⚠️ 关键安全提示

### 🔥 火灾预防
- 远离易燃物至少10米
- 准备灭火沙和水
- 风向下游不可有建筑
- 夜间作业需要守夜人

### 💧 防爆炸
- **绝对禁止熔融金属接触水!**
- 工具必须完全干燥
- 下雨天停止作业
- 铸模需预热去湿

### ☠️ 防中毒
- 必须在室外或通风良好处
- 避免吸入金属蒸汽
- 铅、汞等重金属极毒,慎用
- 作业后清洗手部

### 🩹 烧伤处理
- 立即冷水冲洗15分钟
- 不要挑破水泡
- 涂抹烧伤膏
- 严重烧伤需包扎防感染

## 💡 新手建议

### ✅ 推荐起步
1. **先从铜开始**: 熔点低、易操作、成功率高
2. **收集废铁**: 比炼矿容易,可练习锻造
3. **从简单工具开始**: 钉子、刀片等
4. **建立团队**: 冶金是高强度劳动,需要分工

### ❌ 常见错误
- ✗ 第一次就尝试炼铁(太难,容易放弃)
- ✗ 炉子通风不足(温度上不去)
- ✗ 淬火时工件过热(开裂)
- ✗ 单人作业(危险且效率低)

### 💰 资源优先级
1. **木炭** - 最重要,大量消耗
2. **黏土** - 建炉必需
3. **矿石** - 根据难度选择铜或铁
4. **水源** - 冷却和安全

## 📖 术语表

- **生铁**: 含碳2-4%,硬但脆,不能锻造
- **熟铁**: 含碳<0.2%,软但韧,易锻造
- **钢**: 含碳0.2-2%,既硬又韧
- **还原**: 氧化物去氧得到金属
- **渣**: 冶炼产生的废弃物
- **淬火**: 高温后急冷,增加硬度
- **回火**: 淬火后回温,减少脆性
- **渗碳**: 在表面增加碳含量
- **锻造**: 加热后锤打成型
- **铸造**: 熔化后倒模成型
