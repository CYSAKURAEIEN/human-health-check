# 铁的冶炼

> **文明的脊梁 - 从零开始掌握炼铁技术**

## ⚠️ 警告

**炼铁是冶金中最困难的技术!**

- 🔥 需要1500°C+的高温
- ⏱️ 单次冶炼需8-24小时持续作业
- 💪 体力消耗巨大(持续鼓风)
- ☠️ 一氧化碳浓度极高
- 💥 操作不当可能爆炸

**初次尝试失败率70%+,需要充分准备和多次尝试!**

---

## 一、铁矿石识别与采集

### 1.1 三大主要铁矿石

#### ① 磁铁矿 Fe₃O₄

**外观特征:**
- 颜色: 黑色、铁黑色
- 光泽: 金属光泽或半金属光泽
- 硬度: 5.5-6.5(小刀刻不动)
- 比重: 5.0-5.2(很重!)

**独特识别方法:**
```
最简单: 有磁性!
- 用磁铁吸附: 强烈吸引
- 或用指南针靠近: 指针偏转
```

**优点:**
- ✓ 含铁量最高(60-70%)
- ✓ 易识别(磁性)
- ✓ 冶炼效率高

**分布:**
- 火成岩地区
- 接触变质带
- 河流黑砂中

#### ② 赤铁矿 Fe₂O₃

**外观特征:**
- 颜色: 红褐色、钢灰色、黑色
- 光泽: 金属光泽(致密块状)或暗淡(土状)
- 条痕: **红色**(关键识别!)
- 硬度: 5.5-6.5

**识别方法:**
```
关键: 条痕颜色!
1. 在未上釉的白瓷板上摩擦
2. 留下樱桃红色或铁锈色条痕
3. 即使矿石本身是黑色,条痕也是红色
```

**优点:**
- ✓ 含铁量高(50-60%)
- ✓ 分布广泛
- ✓ 易于粉碎

**分布:**
- 沉积岩地区
- 风化壳
- 红土层

#### ③ 褐铁矿 FeO(OH)·nH₂O

**外观特征:**
- 颜色: 褐色、黄褐色、黑褐色
- 光泽: 暗淡、土状
- 形态: 钟乳状、土状、致密块状
- 比重: 3.6-4.0(相对较轻)

**识别方法:**
```
特征:
- 黄褐色条痕
- 较轻,用手能感觉出
- 多孔,像土块
```

**缺点:**
- ✗ 含铁量较低(40-50%)
- ✗ 含水,冶炼时需烘干
- ✗ 多杂质

**优点:**
- ✓ 分布广,表层即有
- ✓ 易采集
- ✓ 新手入门选择

**分布:**
- 沼泽地
- 湖泊底部
- 古老风化壳

### 1.2 简易鉴定流程

```
发现可疑矿石
     ↓
┌────┴────┐
│ 用磁铁测试 │
└────┬────┘
     │
  有磁性?
     │
 是 ↓          否 ↓
磁铁矿!      在白瓷板划痕
              ↓
          红色条痕?
              │
          是  ↓    否 ↓
         赤铁矿!  褐色条痕?
                     │
                 是  ↓
                褐铁矿!
```

### 1.3 采集要点

**优选矿点:**
- 矿石露出地表
- 矿层厚度>30cm
- 含铁量高(颜色深、密度大)
- 距离居住地3-10公里(太近污染,太远运输累)

**采集工具:**
- 铁镐或尖锄(挖掘)
- 铁锤(敲碎)
- 簸箕或背篓(运输)
- 布袋(装碎矿石)

**采集量(首次):**
- 目标产铁: 2公斤
- 需要矿石:
  - 磁铁矿: 3-4公斤
  - 赤铁矿: 4-5公斤
  - 褐铁矿: 5-8公斤

### 1.4 矿石处理

#### 步骤1: 粉碎
```
块状矿石 → 核桃大小
工具: 石锤、铁锤
时间: 1-2小时
```

#### 步骤2: 焙烧(去水、去硫)
```
针对: 褐铁矿、含硫矿石
方法:
1. 平铺在耐火砖/石板上
2. 堆木柴点燃加热
3. 焙烧2-4小时
4. 目的: 蒸发水分,氧化硫
```

#### 步骤3: 淘洗富集(可选)
```
1. 将碎矿石放入水槽
2. 加水摇晃
3. 轻的泥土流走
4. 重的铁矿石沉底
5. 可提升含铁量5-10%
```

## 二、炼铁炉建造

### 2.1 简易竖炉(推荐)

**规格:**
- 高度: 120-150cm
- 内径:
  - 炉底(出铁口): 20cm
  - 炉腰(最窄): 25cm
  - 炉腹(最宽): 40cm
  - 炉顶(装料口): 30cm
- 壁厚: 20-30cm

**剖面图:**
```
     [俯视顶部]
      ┌─────┐
      │ 30cm│ ← 装料口
      └─────┘

     [剖面侧视]
      ╔═══╗ ← 装料口(30cm)
      ║   ║
      ║   ║ ← 炉身(120cm)
     ╔╝   ╚╗
    ╔╝     ╚╗ ← 炉腹(最宽40cm)
    ║   ●   ║ ← 风口(25cm高度)
    ╚╗  ↑  ╔╝
     ╚╗   ╔╝ ← 炉腰(最窄25cm)
      ║   ║
      ╚═══╝ ← 炉底
       \_/
      出铁口
```

**关键尺寸解释:**
- **炉腹**: 最宽处,矿石和燃料充分反应区
- **炉腰**: 收窄,增加气体流速和压力
- **风口**: 鼓风进入位置,距炉底20-30cm

### 2.2 耐火材料配制

**普通黏土不行!必须用耐火材料!**

#### 耐火黏土配方A(推荐):
```
成分:
- 铝矾土(或高岭土): 50%
- 石英砂: 30%
- 耐火黏土: 15%
- 碎耐火砖(如有): 5%
- 水: 适量
```

**铝矾土/高岭土在哪找?**
- 瓷土矿区
- 白色/灰白色土层
- 摸起来细腻,吸水性强

#### 耐火黏土配方B(简易):
```
成分:
- 黄黏土: 40%
- 砂子: 30%
- 碎砖/陶片: 20%
- 生石灰: 5%
- 食盐: 5%(增加耐火度!)
```

**制作:**
1. 混合干料
2. 加水调至可塑
3. 反复践踏、揉捏30分钟
4. 静置24小时"腐化"(更均匀)

### 2.3 建炉步骤

**第1天: 砌筑**
1. 选址: 空旷、通风、远离建筑
2. 地基: 石块垫高50cm,防潮
3. 内模: 用木桶或草编筐作模型
4. 外墙: 一边砌黏土,一边抽出内模
5. 风口: 预留直径8-12cm的孔

**第2-7天: 干燥**
- 自然阴干5天(重要!)
- 不可阳光暴晒
- 下雨需遮盖

**第8-10天: 烘炉**
- 第1天: 小火烘烤2小时,停火,重复3次
- 第2天: 中火烘烤4小时
- 第3天: 大火烘烤6小时
- 看到炉壁微微发红即可

**第11天: 检查**
- 检查有无裂缝
- 裂缝<2mm: 不管
- 裂缝>2mm: 用耐火泥填补
- 风口畅通

## 三、炼铁实战

### 3.1 材料准备

**单次冶炼(目标: 2公斤铁):**

| 材料 | 用量 | 备注 |
|------|------|------|
| 铁矿石 | 4-6公斤 | 已粉碎 |
| 木炭 | 5-8公斤 | 优质硬木炭 |
| 石灰石 | 0.5公斤 | 助熔剂,降低熔点 |
| 备用木炭 | 5公斤 | 以防万一 |

**其他准备:**
- 水(2桶): 灭火、冷却
- 长柄钳: 夹持
- 长柄铁勺: 舀渣
- 沙土(1堆): 应急灭火
- 铁桶: 接铁水

### 3.2 操作流程

#### 阶段1: 预热炉子(1-2小时)

```
1. 炉底铺木炭(10cm厚)
2. 点火
3. 开始鼓风
4. 温度升至600-700°C(暗红色)
```

**鼓风节奏:**
- 1人: 每5秒鼓1次
- 2人轮换: 持续鼓风

#### 阶段2: 装料(分批进行)

**装料顺序(循环):**
```
第1批:
1. 加木炭(1层,3-5cm厚)
2. 加矿石(1层,3-5cm厚)
3. 加少量石灰石(1小把)
4. 继续鼓风5分钟

重复装料8-10批,装满炉腹
```

**关键要点:**
- 木炭和矿石交替
- 木炭略多于矿石
- 不要一次装满(窒息)
- 边装料边鼓风

#### 阶段3: 冶炼反应(6-10小时)

**持续鼓风!**
- 风力要稳定
- 不能停止(温度会骤降)
- 人员轮换

**温度管理:**
```
目标温度: 1500-1600°C
颜色判断: 亮黄色或黄白色

温度不足:
- 加大鼓风
- 增加木炭

温度过高(罕见):
- 减少鼓风
- 停止加料
```

**现象观察:**

| 时间 | 现象 | 说明 |
|------|------|------|
| 1-2小时 | 浓烟,刺鼻 | 矿石加热,挥发杂质 |
| 3-4小时 | 烟少,炉内明亮 | 进入还原阶段 |
| 5-6小时 | 偶尔火星飞出 | 铁开始熔化 |
| 7-8小时 | 炉底发亮 | 铁水聚集 |

**补料:**
- 炉料下降时,继续添加
- 保持炉腹充满
- 木炭消耗快,及时补充

#### 阶段4: 出铁(关键!)

**判断完成:**
- 炉底出铁口发亮
- 用铁棍探入,有液态感觉
- 时间已达8小时+

**出铁步骤:**
```
1. 准备接铁容器(干燥!)
   - 地面挖坑铺沙
   - 或用耐火砖砌槽

2. 打开出铁口
   - 用铁棍捅开堵塞的黏土
   - 铁水会流出

3. 分离铁和渣
   - 铁水(重)在底部
   - 渣(轻)浮在上面
   - 用铁板或木板挡住渣

4. 铸锭或铸块
   - 倒入沙模
   - 或直接倒在沙地上成薄饼状(便于后续加工)
```

**⚠️ 安全警告:**
- 铁水温度1500°C+,极度危险!
- 保持2米安全距离
- 绝对不能接触水!
- 准备好灭火沙

#### 阶段5: 冷却(2-4小时)

```
铁块在模具中:
- 自然冷却
- 不要浇水(会炸裂)
- 等待变成暗红色后可移动

炉子:
- 停止鼓风
- 让炉火自然熄灭
- 等待12小时后清理
```

### 3.3 首次成果

**你得到的是: 生铁**

**生铁的特性:**
- 含碳量: 2-4%
- 硬度: 很硬
- 脆性: 很脆,一砸就碎
- 颜色: 灰白色(铸铁)或白色(白口铁)
- **不能锻造!**

**能做什么?**
- ✓ 铸造成型(熔化后倒模)
- ✓ 制作铁锅、犁铧(耐磨损)
- ✗ 不能锻打成刀、斧(会碎)

**需要进一步精炼成熟铁或钢!**

## 四、从生铁到熟铁

### 4.1 原理

```
生铁(含碳2-4%)
    → 去碳
      → 熟铁(含碳<0.2%)
```

**目的:** 去除碳,增加韧性,便于锻造

### 4.2 方法A: 炒钢法(推荐)

**设备:**
- 浅盘状炉子(炒锅状)
- 鼓风装置

**步骤:**

**1. 装炉:**
- 炉底铺木炭
- 点火鼓风至高温

**2. 化生铁:**
- 将生铁块放入炉中
- 鼓风加热至熔化
- 温度: 1300-1400°C

**3. 搅拌(关键!):**
```
用长铁棍不断搅动铁水:
- 目的: 让铁水充分接触空气
- 反应: C + O₂ → CO₂ (碳被氧化烧掉)
- 时间: 持续30-60分钟
- 现象:
  * 火花四溅
  * 逐渐变稠(碳少,熔点升高)
```

**4. 观察变化:**
```
铁水 → 铁浆(半固态) → 铁坨(固态团块)
```

**5. 取出锻打:**
- 用钳子夹出铁坨(趁热!)
- 放在砧上快速锤打
- 目的:
  * 打出残渣
  * 压实铁块
  * 整形

**6. 反复加热锻打:**
- 加热至橙红色
- 锤打
- 重复3-5次

**成品: 熟铁(锻铁)**
- 含碳<0.2%
- 软,韧性好
- 可以锻造成各种形状
- 不易折断

### 4.3 方法B: 吹炼法

**原理:** 在熔融生铁中吹入空气,氧化碳

**设备:**
- 坩埚(耐火黏土制)
- 空心竹管或金属管

**步骤:**
1. 坩埚中熔化生铁
2. 插入空心管,深入铁水
3. 向管中吹气或鼓风
4. 碳被氧化,铁水翻腾
5. 持续20-40分钟
6. 倒出锻打

**缺点:** 危险,容易飞溅

### 4.4 熟铁检验

**方法:**
```
1. 锻打测试:
   - 加热至红色
   - 锤打能延展成薄片
   - 不开裂 → 熟铁✓
   - 碎裂 → 生铁,需继续精炼

2. 弯曲测试:
   - 常温下弯曲铁条
   - 能弯90°不断裂 → 熟铁✓
   - 折断 → 生铁

3. 火花测试:
   - 在砂轮上打磨
   - 熟铁: 火花长、分叉少
   - 生铁: 火花短、分叉多、爆裂
```

## 五、炼钢技术

### 5.1 什么是钢?

**钢 = 含碳0.2-2%的铁**

- 比生铁含碳少(不脆)
- 比熟铁含碳多(更硬)
- **既硬又韧,制刀最佳!**

### 5.2 方法: 渗碳法

**原理:** 让熟铁吸收碳

#### 固体渗碳法(简单)

**材料:**
- 熟铁块
- 木炭粉
- 密封容器(铁盒或陶罐)

**步骤:**

**1. 准备渗碳剂:**
```
木炭粉: 80%
草木灰: 10%
食盐: 10%
(混合均匀)
```

**2. 装填:**
```
铁盒/陶罐内:
底部铺渗碳剂(3cm)
  ↓
放入熟铁块
  ↓
周围填满渗碳剂
  ↓
顶部覆盖渗碳剂(3cm)
  ↓
密封(用黏土糊严)
```

**3. 加热:**
- 将密封容器放入炉火中
- 加热至900-1000°C(橙红色)
- 保温4-12小时
  - 薄件(刀片): 4-6小时
  - 厚件(斧头): 8-12小时

**4. 冷却:**
- 随炉慢冷
- 或取出空冷

**5. 检查:**
- 打开容器
- 铁表面应呈灰黑色
- 硬度明显增加

**成品: 钢!**
- 表面渗碳深度: 0.5-2mm
- 外硬内韧(最佳状态!)

#### 液体渗碳法(快速)

**步骤:**
1. 熟铁加热至1000°C(橙黄色)
2. 迅速插入木炭粉中
3. 表面吸收碳
4. 再加热、再插入
5. 重复5-10次
6. 淬火(快速冷却)

**优点:** 快
**缺点:** 渗碳深度浅

## 六、常见问题与故障

### Q1: 炉温上不去,一直不到1500°C?

**原因:**
- ✗ 鼓风不足
- ✗ 炉壁太薄,散热快
- ✗ 炉顶开口太大
- ✗ 木炭质量差

**解决:**
- ✓ 加强持续鼓风
- ✓ 加厚炉壁或外围再砌一层
- ✓ 炉顶部分封闭
- ✓ 更换木炭

### Q2: 冶炼8小时,没有铁水出来?

**可能原因:**
1. **温度不够**
   - 检查炉内颜色,应为黄白色
   - 增强鼓风

2. **还原不充分**
   - 木炭与矿石比例不对
   - 增加木炭用量

3. **铁成海绵状,未熔化**
   - 正常现象!古代炼铁很多是海绵铁
   - 取出炉料,挑出海绵铁块
   - 反复加热锻打,压实成块

### Q3: 出铁时爆炸飞溅?

**原因:** 水气!
- 出铁口潮湿
- 接铁容器不干燥
- 地面有积水

**预防:**
- 所有接触铁水的部分彻底烘干
- 提前用火烤20分钟

### Q4: 得到的铁质量很差,杂质多?

**原因:**
- 矿石品位低
- 炉渣混入

**改进:**
- 矿石淘洗富集
- 添加足够石灰石(助熔剂)
- 出铁时仔细分离渣

### Q5: 一次只能得到几百克铁,太少?

**正常!**
- 小型炉单次产量: 0.5-3公斤
- 中型炉: 5-10公斤
- 大型炉(需多人): 20-50公斤

**提高产量:**
- 扩大炉子
- 提高装料量
- 多建几座炉,同时作业

### Q6: 生铁精炼成熟铁时,铁块越炒越少?

**正常现象!**
- 碳被氧化烧掉
- 部分铁被氧化成铁渣
- 损失率: 10-30%
- 这就是为什么要多炼铁

## 七、时间与资源评估

### 单次炼铁完整流程:

| 阶段 | 时间 | 人力 |
|------|------|------|
| 建炉 | 7-10天 | 2-3人 |
| 采矿 | 1-2天 | 2人 |
| 制木炭 | 5-7天 | 1-2人 |
| 预处理矿石 | 1天 | 1人 |
| 冶炼 | 10-14小时 | 2-3人(轮换) |
| 冷却 | 12小时 | 无需看管 |
| **总计** | **约3周** | **2-3人团队** |

### 资源消耗(单次):

| 资源 | 用量 | 获取难度 |
|------|------|----------|
| 木材(制木炭用) | 30-50公斤 | ⭐⭐ |
| 黏土 | 100公斤 | ⭐ |
| 铁矿石 | 5-10公斤 | ⭐⭐⭐ |
| 石灰石 | 1公斤 | ⭐⭐ |

### 产出:
- 生铁: 1-3公斤
- 精炼后熟铁: 0.7-2公斤

**结论: 炼铁是资源和时间密集型工作,需要团队协作和充分准备!**

## 八、进阶技巧

### 8.1 提高产量

**多炉并行:**
- 建3-5座炉
- 轮流作业
- 持续产出

**连续作业:**
- 一炉出铁后,立即清理
- 检修2-3小时
- 开始下一炉

### 8.2 提高质量

**矿石精选:**
- 只用磁铁矿或优质赤铁矿
- 充分淘洗

**温度控制:**
- 训练固定鼓风人员
- 保持稳定节奏

**合理配料:**
- 木炭:矿石:石灰石 = 2:2:0.2

### 8.3 废铁利用

**末日情况下,废铁可能比矿石更易得:**
- 旧汽车(大量钢材)
- 建筑钢筋
- 机械零件
- 铁轨、铁丝

**处理:**
- 分类(去除铜、铝、塑料)
- 重新熔化
- 铸造或锻造
- **比从矿石炼铁容易10倍!**

## 九、检查清单

### 冶炼前:
- [ ] 炉子完全干燥,无裂缝
- [ ] 鼓风装置运行流畅
- [ ] 木炭充足(准备量×1.5)
- [ ] 矿石已粉碎、处理
- [ ] 石灰石准备好
- [ ] 出铁口、接铁容器干燥
- [ ] 灭火沙、水准备好
- [ ] 人员分工明确
- [ ] 天气晴朗

### 冶炼中:
- [ ] 鼓风持续、稳定
- [ ] 温度达到黄白色
- [ ] 及时补充木炭
- [ ] 人员轮换,防疲劳
- [ ] 监测一氧化碳中毒迹象

### 出铁时:
- [ ] 确认铁水聚集
- [ ] 接铁容器已预热
- [ ] 人员后退至安全距离
- [ ] 准备分离渣
