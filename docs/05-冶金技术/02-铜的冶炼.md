# 铜的冶炼

> **新手首选 - 最容易成功的金属冶炼**

## ✅ 为什么从铜开始?

**铜是冶金入门的最佳选择:**

| 对比项 | 铜 | 铁 |
|--------|----|----|
| 熔点 | 1085°C | 1538°C |
| 炉温要求 | 1200°C | 1600°C+ |
| 设备难度 | ⭐⭐ 简单 | ⭐⭐⭐⭐ 困难 |
| 成功率(初次) | 60-70% | 20-30% |
| 冶炼时间 | 3-5小时 | 8-24小时 |
| 识别难度 | ⭐ 非常简单(绿色) | ⭐⭐⭐ 较难 |

**优势:**
- ✓ 温度要求低,普通木炭炉+鼓风即可
- ✓ 冶炼时间短
- ✓ 铜矿易识别(颜色鲜艳)
- ✓ 可铸造成各种形状
- ✓ 不易锈蚀
- ✓ 可制成青铜(更硬、更耐用)

## 一、铜矿石识别

### 1.1 孔雀石 Cu₂(OH)₂CO₃ ⭐推荐

**最易识别和冶炼的铜矿!**

**外观:**
- 颜色: **鲜艳的绿色**(孔雀羽毛色)
- 花纹: 同心圆或条带状
- 光泽: 丝绢光泽或玻璃光泽
- 硬度: 3.5-4(小刀可刻动)

**识别要点:**
```
看到绿色石头 → 99%是孔雀石!
自然界很少有其他绿色矿石
```

**特性:**
- 含铜量: 55-60%
- 易分解(加热到300°C就开始分解释放CO₂)
- **最适合新手!**

**分布:**
- 铜矿床氧化带
- 岩石裂隙
- 常与蓝铜矿共生

### 1.2 蓝铜矿 Cu₃(OH)₂(CO₃)₂

**外观:**
- 颜色: **深蓝色、天蓝色**
- 光泽: 玻璃光泽
- 形态: 晶体或块状

**特性:**
- 含铜量: 55%
- 易碎
- 常与孔雀石共生(绿蓝相间很漂亮)

### 1.3 辉铜矿 Cu₂S

**外观:**
- 颜色: 铅灰色,新鲜面带金属光泽
- 特征: 软,小刀可刻出铜的光泽
- 比重: 很重(5.5-5.8)

**特性:**
- 含铜量: **79%**(最高!)
- 硫化物,需先焙烧去硫

### 1.4 黄铜矿 CuFeS₂

**外观:**
- 颜色: 黄铜色(似黄金但更硬)
- 光泽: 强金属光泽
- 硬度: 3.5-4

**识别:**
- 比黄金硬(黄金很软)
- 绿黑色条痕

**特性:**
- 含铜量: 34%
- 含铁,冶炼时铁成渣

### 1.5 自然铜 Cu(纯铜)

**极罕见!**
- 纯红色金属
- 可直接使用,无需冶炼
- 如果找到,你太幸运了!

## 二、简易炼铜

### 2.1 设备准备

**简易炼铜炉:**
- 比炼铁炉小、简单得多!
- 内径: 20-30cm
- 高度: 40-60cm
- 壁厚: 10-15cm
- 材料: 普通黏土即可(耐火黏土更好)

**坩埚(熔铜容器):**
```
可选方案:
1. 购买石墨坩埚(理想)
2. 耐火黏土自制
3. 厚铁罐(临时)
```

**自制黏土坩埚:**
1. 耐火黏土捏成碗状
2. 壁厚2-3cm
3. 容量500ml-1L
4. 慢慢烘干、烧硬
5. 可重复使用

### 2.2 炼铜流程A: 直接熔炼法(适合孔雀石、蓝铜矿)

**材料(首次):**
- 孔雀石: 2公斤
- 木炭: 3公斤
- 坩埚: 1个

**步骤:**

**1. 矿石预处理:**
```
粉碎 → 黄豆至花生大小
焙烧(可选):
  - 在空气中加热至红热
  - 持续1-2小时
  - 反应: Cu₂(OH)₂CO₃ → CuO + H₂O + CO₂
  - 目的: 去水、去CO₂,减少杂质
```

**2. 装炉:**
```
炉底: 铺木炭(10cm)
放入坩埚
坩埚周围填满木炭
坩埚内装矿石(装七分满)
```

**3. 点火:**
- 从底部点燃木炭
- 开始鼓风

**4. 加热(2-4小时):**
```
温度变化:
600°C → 矿石开始分解(冒气)
900°C → 氧化铜形成
1100°C → 铜开始熔化
1200°C → 完全熔化成铜水

判断:
- 坩埚发亮(黄色)
- 用铁棍探入,无固体感
```

**5. 扒渣:**
```
铜水表面会浮一层渣(氧化物、杂质)
用长勺或铁棍撇去
```

**6. 浇铸:**
```
准备模具:
  - 沙模(在沙子上刻槽)
  - 石模(在石板上刻凹槽)
  - 铁模(耐久)

倒铜:
  - 用长柄钳夹坩埚
  - 慢慢倒入模具
  - 注意不要带入渣
```

**7. 冷却:**
- 自然冷却10-20分钟
- 铜锭呈红色金属光泽

**产量:**
- 2公斤孔雀石 → 约0.8-1.2公斤纯铜

### 2.3 炼铜流程B: 还原熔炼法(适合氧化铜矿)

**原理:** 用木炭还原氧化铜
```
2CuO + C → 2Cu + CO₂↑
```

**与铁冶炼类似,但简单得多:**

1. 矿石和木炭混合(1:1)
2. 装入炉中,分层或混装
3. 鼓风加热
4. 3-5小时后,铜在炉底聚集
5. 打开出口,流出铜水
6. 浇铸成锭

**适用:** 氧化铜矿石(黑色或褐色)

### 2.4 炼铜流程C: 硫化铜矿处理

**针对:** 辉铜矿、黄铜矿等硫化物

**两步法:**

**第一步: 焙烧(去硫):**
```
1. 矿石粉碎
2. 露天堆放或浅坑
3. 点燃,让其缓慢燃烧
4. 反应:
   2Cu₂S + 3O₂ → 2Cu₂O + 2SO₂↑
   (硫变成臭味的SO₂逸出)
5. 持续6-12小时
6. 产物: 氧化铜(黑色)
```

**第二步: 还原熔炼:**
- 用流程B处理焙烧后的氧化铜

## 三、青铜制作

### 3.1 什么是青铜?

**青铜 = 铜 + 锡合金**

**配比:**
- 标准青铜: 90% Cu + 10% Sn
- 硬青铜(工具): 88% Cu + 12% Sn
- 软青铜(装饰): 92% Cu + 8% Sn

**为什么要青铜?**
- 纯铜太软,做工具易变形
- 加锡后硬度大幅提升
- 熔点降低(更易铸造)
- 流动性好(铸造复杂形状)

**硬度对比:**
| 材料 | 硬度(HB) | 说明 |
|------|----------|------|
| 纯铜 | 40-50 | 软,易弯曲 |
| 青铜(10%锡) | 100-120 | 硬度提升2-3倍! |
| 熟铁 | 80-100 | 青铜更硬! |
| 钢 | 200+ | 钢最硬 |

### 3.2 锡的获取

**锡矿石: 锡石 SnO₂**

**识别:**
- 颜色: 黑色、褐色、暗红色
- 光泽: 强金属光泽或金刚光泽
- 比重: **非常重**(6.8-7.0,比铜还重!)
- 硬度: 6-7(很硬)

**分布:**
- 花岗岩地区
- 砂矿(河流沙中)

**炼锡(非常简单!):**
```
锡石 + 木炭 → 加热至600-700°C → 锡熔化流出

熔点仅232°C,甚至不需要鼓风!
古人用普通火堆就能炼锡。
```

### 3.3 青铜熔炼

**材料:**
- 纯铜: 900克
- 锡: 100克
- 木炭: 2公斤

**步骤:**

1. **熔化铜:**
   - 坩埚中放铜
   - 加热至1100°C熔化

2. **加入锡:**
   - 铜熔化后,加入锡块
   - 锡瞬间熔化(熔点太低)
   - 用铁棍搅拌均匀

3. **扒渣:**
   - 撇去表面浮渣

4. **浇铸:**
   - 倒入模具
   - 青铜流动性极好,能铸造精细形状

5. **冷却:**
   - 自然冷却
   - 青铜略带黄色

**成品特性:**
- 颜色: 青黄色、青灰色
- 硬度: 比纯铜高得多
- 韧性: 不易折断
- 耐腐蚀: 数千年不锈(青铜文物!)

### 3.4 黄铜制作(替代方案)

**黄铜 = 铜 + 锌**
- 配比: 70% Cu + 30% Zn
- 颜色: 金黄色(像金子!)
- 硬度: 中等,不如青铜

**问题:** 锌熔点419°C,但沸点907°C
- 炼铜温度(1100°C)下,锌会汽化!

**古代方法(固态渗锌):**
1. 铜片+炉甘石(ZnCO₃)粉末混合
2. 密封加热至900°C
3. 锌蒸汽渗入铜中
4. 形成黄铜表面

**现代方法:**
- 回收黄铜制品重熔(简单!)
- 废弃的铜管、阀门、装饰品等

## 四、铸造技术

### 4.1 砂模铸造(最简单)

**材料:**
- 细沙(河沙)
- 黏土粉(少量,5-10%)
- 水(微量,调至湿润)

**制作模型:**
```
方法1: 直接刻槽
  - 在平整的湿沙上刻出形状
  - 适合: 简单形状(锭、片、环)

方法2: 压模
  - 用木头/石头雕刻原型
  - 压入沙中留下印模
  - 取出原型
  - 倒入金属

方法3: 双面模
  - 制作上下两半沙模
  - 留出浇注口
  - 合模后浇注
  - 适合: 复杂形状(斧头、刀)
```

**浇注步骤:**
1. 沙模压实(防塌陷)
2. 挖出浇注通道
3. 预热模具(避免冷裂)
4. 快速倒入铜水
5. 等待冷却5-10分钟
6. 扒开沙子取出铸件
7. 去除毛刺、打磨

### 4.2 失蜡法(精密铸造)

**适用:** 复杂形状、艺术品

**流程:**
```
1. 雕刻蜡模(蜂蜡或动物脂肪)
2. 包裹黏土形成外壳
3. 留出浇注口和排气孔
4. 烘烤:
   - 蜡熔化流出
   - 黏土烧硬成陶
5. 倒入铜水填充空腔
6. 冷却后打碎外壳
7. 得到精细铸件
```

**优点:** 可铸造极复杂的形状
**缺点:** 模具只能用一次

### 4.3 石模铸造(耐用)

**方法:**
- 在软石(皂石、砂岩)上雕刻模具
- 可重复使用数十次
- 适合: 箭头、刀片、钉子等小件

## 五、实用铜制品

### 5.1 工具类

**1. 铜斧、铜锄(青铜):**
- 虽不如铁,但胜过石器
- 不易锈蚀
- 适合农耕

**2. 铜刀(青铜):**
- 砍伐、切割
- 需要经常磨砺(不如钢)

**3. 铜箭头(青铜):**
- 锋利、穿透力强
- 铸造简单

**4. 铜针、铜钩:**
- 缝纫、捕鱼
- 不生锈

### 5.2 容器类

**1. 铜锅、铜盆:**
- 导热快
- 耐用
- 注意: 不能装强酸性食物(产生铜绿有毒)

**2. 铜桶、铜杯:**
- 储水
- 天然抑菌(铜离子有杀菌作用)

### 5.3 装饰和货币

**铜镜:**
- 抛光后可当镜子

**铜钱:**
- 可作为货币(古代钱币多为铜)

**铜铃、铜饰:**
- 装饰品

## 六、常见问题

### Q1: 找不到锡,能做青铜吗?

**A: 有替代方案**

**方案1: 铜-砷合金**
- 用含砷的矿石代替锡
- 硬度也会提升
- ⚠️ 砷有剧毒!需极度小心

**方案2: 用纯铜**
- 虽然软,但也能用
- 冷锻(反复锤打)可增加硬度

**方案3: 回收废铜**
- 电线(纯铜)
- 黄铜制品(含锌)
- 青铜制品(古董可能含锡)

### Q2: 炼出的铜很脆,是什么原因?

**A: 可能含氧过高**

**解决:**
- 熔化时在表面撒一层木炭粉(隔绝空气)
- 搅拌时用木棍(木炭化后脱氧)
- 浇铸后快速冷却

### Q3: 铜器有毒吗?

**A: 纯铜无毒,但要注意:**

✓ 安全使用:
- 铜锅煮饭、烧水
- 铜杯饮水

✗ 避免:
- 长期装酸性食物(醋、果汁)→ 产生铜绿(有毒)
- 如有绿锈,彻底清洗后再用

### Q4: 一次能炼多少铜?

**A:**
- 小坩埚(500ml): 0.5-1公斤
- 中坩埚(2L): 2-4公斤
- 大坩埚(5L): 5-10公斤

### Q5: 铜可以锻造吗?

**A: 可以,但要注意:**

**冷锻:**
- 纯铜可冷锻(常温锤打)
- 会越打越硬(加工硬化)
- 但也越来越脆

**热锻:**
- 加热至500-600°C(暗红)
- 锤打成型
- 退火(加热后慢冷)恢复韧性

## 七、安全提示

### 铜毒(相对较轻,但仍需注意):

**急性中毒:**
- 症状: 恶心、呕吐、腹痛
- 原因: 误食大量铜盐
- 处理: 大量喝牛奶(结合铜离子),催吐,就医

**预防:**
- 作业后洗手
- 不用破损、氧化的铜器装食物
- 定期清洗铜器

### 硫化铜矿焙烧:

**产生SO₂(有毒!):**
- 臭鸡蛋味,刺激性气体
- 必须上风向
- 距离居住区500米以上
