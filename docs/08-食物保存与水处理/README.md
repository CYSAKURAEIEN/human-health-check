# 食物保存与水处理

> **延长生命 - 食物储存和饮水安全技术**

## 📖 引言

**末日后的两大生存挑战:**

1. **食物腐败** - 没有冰箱，食物几天就坏
2. **水源污染** - 河水、湖水可能致病

**本章节解决:**
- 🥓 如何让食物保存数月甚至数年
- 💧 如何确保饮水安全无毒

**掌握这些技术 = 饥荒中生存 + 避免疾病！**

## 📚 章节目录

### [01-腌制技术](./01-腌制技术.md)

用盐、糖、醋保存食物。

**核心内容:**
- 🧂 盐腌：肉类、鱼类、蔬菜
- 🍯 糖腌：果脯、蜜饯
- 🫗 醋腌：泡菜、酸菜

**原理:**
- 高盐/糖环境抑制细菌
- 渗透压脱水
- 保存期: 3-12个月

**关键要点:**
- 盐浓度15-25%
- 完全浸没
- 密封储存

**最实用:** 盐是末日最宝贵资源！

---

### [02-烟熏与风干](./02-烟熏与风干.md)

通过脱水保存食物。

**核心内容:**
- 🔥 热熏：快速熏制（3-7天）
- 🌫️ 冷熏：长期保存（1-3周）
- 🌬️ 风干：自然干燥（鱼干、肉干）
- ☀️ 日晒：蔬菜、水果干

**原理:**
- 脱水（细菌无法生存）
- 烟熏杀菌
- 保存期: 3-24个月

**关键要点:**
- 水分含量<15%
- 烟熏用硬木（不用松木）
- 避免苍蝇污染

**优势:** 不需要盐！

---

### [03-发酵技术](./03-发酵技术.md)

利用有益菌保存和改善食物。

**核心内容:**
- 🥬 乳酸发酵：泡菜、酸菜、酸黄瓜
- 🍺 酒精发酵：酒、醋
- 🫘 豆类发酵：酱油、豆豉、腐乳
- 🍞 面团发酵：酸面包、馒头

**原理:**
- 有益菌产生酸/酒精
- 抑制有害菌
- 保存期: 3-12个月

**关键要点:**
- 温度控制（20-30°C）
- 厌氧环境
- 盐度2-5%

**双重好处:** 保存+增加营养！

---

### [04-罐装与密封](./04-罐装与密封.md)

高温杀菌后密封保存。

**核心内容:**
- 🫙 玻璃罐装：果酱、罐头
- 🏺 陶罐密封：粮食、干货
- 🍯 蜡封技术：蜂蜜、油脂
- 📦 地窖储存：根茎类蔬菜

**原理:**
- 高温杀菌
- 密封隔绝空气
- 保存期: 1-3年

**关键要点:**
- 必须煮沸消毒
- 趁热密封
- 检查气密性

**工业级保存:** 最长保质期！

---

### [05-水处理技术](./05-水处理技术.md)

确保饮水安全。

**核心内容:**
- ♨️ 煮沸消毒：最简单可靠
- 🏜️ 过滤技术：沙滤、炭滤、陶滤
- ☀️ 太阳消毒：SODIS法
- 💧 化学消毒：漂白粉、碘片
- 🌧️ 雨水收集：清洁水源
- 🕳️ 井的挖掘：长期水源

**原理:**
- 杀灭病原体
- 去除杂质
- 水质达标

**关键要点:**
- 煮沸是金标准（1分钟）
- 多级过滤更安全
- 储水容器需清洁

**生命第一要务:** 水比食物更重要！

## 🧂 盐的重要性

**末日最宝贵资源之一！**

### 盐的用途:

1. **食物保存** - 腌制
2. **调味** - 维持食欲
3. **医疗** - 生理盐水
4. **消毒** - 盐水杀菌
5. **生理需求** - 必需矿物质

### 盐的来源:

**海盐（沿海地区）:**
```
1. 收集海水
2. 浅池蒸发
3. 晒至结晶
4. 收集盐粒
```

**井盐/矿盐（内陆）:**
- 盐井
- 盐矿开采
- 盐湖采集

**替代品（紧急）:**
- 动物血液（含钠）
- 某些植物（灰分含盐）

### 盐的储备:

**建议储备量（每人每年）:**
- 食用: 5-10公斤
- 腌制: 20-50公斤
- **总计: 30-60公斤/人/年**

**储存:**
- 密封防潮
- 干燥处
- 永不过期

## 💧 水处理优先级

### 水源安全等级:

| 水源 | 安全等级 | 处理需求 | 说明 |
|------|----------|----------|------|
| 自来水(断水前) | ⭐⭐⭐⭐⭐ | 无 | 可直接饮用 |
| 深井水 | ⭐⭐⭐⭐ | 煮沸 | 较安全 |
| 山泉水 | ⭐⭐⭐ | 过滤+煮沸 | 可能有寄生虫 |
| 雨水 | ⭐⭐⭐ | 过滤+煮沸 | 收集容器需清洁 |
| 河水/湖水 | ⭐⭐ | 沉淀+过滤+煮沸 | 污染风险高 |
| 池塘水 | ⭐ | 多级处理 | 风险很高 |

### 最低标准:

**绝对底线:** 煮沸1分钟！
- 杀灭99.9%病原体
- 简单可靠
- 必须坚持

## ⚠️ 食物中毒预防

### 警惕信号:

**不要食用:**
- ✗ 发霉（黄曲霉素致癌）
- ✗ 异味（酸臭味）
- ✗ 变色（绿色、黑色）
- ✗ 黏稠（腐败）
- ✗ 膨胀罐头（肉毒杆菌！）

### 肉毒杆菌（致命！）:

**来源:**
- 密封不当的罐头
- 低酸性食物（肉、豆类）

**症状:**
- 视力模糊、复视
- 吞咽困难
- 肌肉麻痹
- 呼吸困难
- **死亡率高！**

**预防:**
- 罐装必须高温杀菌（121°C, 20分钟）
- 开罐后加热煮沸
- 怀疑即丢弃

### 亚硝酸盐中毒:

**来源:**
- 过度腌制的肉类
- 腐烂蔬菜

**预防:**
- 腌制肉不过量食用
- 腐烂蔬菜立即丢弃
- 腌菜腌制7天后再食用
