# 水处理技术

> **生命之源 - 确保饮水安全的完整指南**

## ⚠️ 水比食物更重要！

**人类生存极限:**
- 无水: 3-5天死亡
- 无食物: 21-30天死亡

**不洁水的危害:**
- 霍乱、伤寒、痢疾 → 脱水死亡
- 寄生虫病 → 长期虚弱
- 化学污染 → 慢性中毒

**原则: 宁可渴几小时也不喝不洁水！**

## 一、煮沸消毒

### 1.1 为什么煮沸最可靠?

**煮沸的优势:**
- ✓ 杀灭99.99%病原体
- ✓ 无需化学药剂
- ✓ 任何人都能做
- ✓ 对所有水源有效

**杀灭对象:**
- 细菌（霍乱、伤寒、大肠杆菌）
- 病毒（甲肝、诺如）
- 寄生虫卵（蛔虫、鞭虫、贾第虫囊肿）

### 1.2 正确煮沸方法

**标准流程:**

```
1. 预过滤:
   - 去除泥沙、树叶等大颗粒
   - 用布/纱布过滤

2. 煮沸:
   - 大火加热至沸腾（100°C）
   - 保持沸腾状态至少1分钟
   - 高海拔地区（>2000米）: 3分钟

3. 冷却:
   - 自然冷却至适饮温度
   - 不要加生水降温！

4. 储存:
   - 清洁容器密封保存
   - 24小时内饮用
   - 超过24小时需重新煮沸
```

**常见错误:**
- ✗ 刚冒泡就停火（未达到沸腾）
- ✗ 煮沸后敞开放置（再次污染）
- ✗ 使用污染的容器储存

### 1.3 应急煮沸

**无容器情况:**

**方法1: 烧石法**
```
1. 挖坑或利用天然石坑
2. 铺树叶或兽皮防渗
3. 倒入水
4. 烧红石头投入水中
5. 反复投入直至沸腾
```

**方法2: 竹筒法**
```
1. 选择粗竹筒
2. 一端削尖插入地面
3. 倒入水
4. 下方生火
5. 竹筒不会烧穿（有水冷却）
```

## 二、过滤技术

### 2.1 简易沙滤器

**原理:** 多层过滤去除杂质和部分病原体

**制作:**

```
[剖面示意图]

    水 ↓
  ┌──────┐
  │  布   │ ← 过滤大颗粒
  ├──────┤
  │细沙层│ ← 3-5cm，过滤细菌
  ├──────┤
  │粗沙层│ ← 5-10cm
  ├──────┤
  │碎石层│ ← 10cm，防堵塞
  ├──────┤
  │木炭层│ ← 10cm，吸附有机物、异味
  ├──────┤
  │碎石层│ ← 5cm，支撑
  └───┬──┘
    出水 ↓
```

**材料:**
- 容器: 木桶、陶罐、塑料瓶
- 细沙: 河沙洗净晒干
- 粗沙/碎石
- 木炭: 烧柴后的木炭捣碎
- 布/纱布

**制作步骤:**

```
1. 容器底部开孔（出水口）
2. 从下往上依次装填:
   - 碎石层（支撑）
   - 木炭层（吸附）
   - 碎石层（防堵）
   - 粗沙层
   - 细沙层
   - 顶部铺布

3. 使用前冲洗:
   - 倒入清水
   - 让前几升水流走
   - 冲洗掉杂质

4. 过滤:
   - 缓慢倒入待处理水
   - 收集出水
   - 仍需煮沸！
```

**效果:**
- 去除泥沙、悬浮物: 95%+
- 去除细菌: 50-80%
- 去除病毒: 20-40%
- 改善口感和颜色

**维护:**
- 定期清洗沙层（1-2周）
- 更换木炭（1-2个月）
- 清除表面污垢

### 2.2 陶瓷过滤器

**原理:** 微孔陶瓷拦截病原体

**制作陶瓷滤芯:**

```
材料:
- 黏土: 细腻、纯净
- 锯末或稻壳: 10-30%（烧掉后形成微孔）
- 胶体银溶液（可选，杀菌）

步骤:
1. 黏土与锯末混合
2. 加水揉成可塑性团
3. 制作成蜡烛形或圆筒形滤芯
4. 阴干3-7天
5. 烧制:
   - 温度800-1000°C
   - 时间8-12小时
   - 锯末燃烧形成微孔
6. 浸泡胶体银溶液（可选）
7. 干燥

使用:
- 滤芯装入容器
- 水从外侧渗透到内侧
- 流速慢（1-2升/小时）
- 需定期刷洗表面
```

**效果:**
- 去除细菌: 99%+
- 去除原生动物: 99%+
- 去除病毒: 50-80%

**商业产品:** Berkey滤芯、Katadyn陶瓷滤芯

## 三、化学消毒

### 3.1 漂白粉（次氯酸钙）

**用量:**

| 水质 | 每升水加漂白粉 | 静置时间 |
|------|----------------|----------|
| 清澈水 | 2滴（1%溶液） | 30分钟 |
| 浑浊水 | 4滴（1%溶液） | 30分钟 |

**配制1%溶液:**
- 漂白粉10克 + 水1升

**步骤:**
```
1. 将水倒入容器
2. 加入漂白粉溶液
3. 搅拌均匀
4. 盖上盖子静置30分钟
5. 有淡淡氯味即可饮用
6. 如无氯味，再加漂白粉重复
```

**注意:**
- 漂白粉受潮失效（密封保存）
- 过量会刺激胃肠
- 适当余氯可持续保护

### 3.2 碘片/碘酒

**用量:**
- 碘片: 1片/升水
- 2%碘酒: 5滴/升水

**步骤:**
```
1. 加入碘片/碘酒
2. 摇匀
3. 静置30分钟
4. 水呈微黄色可饮用
```

**注意:**
- 孕妇、甲状腺病人慎用
- 长期使用可能碘过量
- 有碘味（可加维C去碘味）

### 3.3 高锰酸钾

**用量:** 极少！

**方法:**
```
1. 水变淡粉色即可（1:10000）
2. 静置30分钟
3. 有机物多的水不适用（消耗高锰酸钾）
```

**警告:** 浓度过高有毒！

## 四、太阳消毒（SODIS）

### 4.1 原理

**紫外线杀菌:**
- UV-A射线破坏细菌DNA
- 温度升高加速杀灭

### 4.2 方法

**要求:**
- 透明PET塑料瓶（<2升）
- 或透明玻璃瓶
- 晴天

**步骤:**

```
1. 选择透明瓶:
   - 洗净
   - 无划痕（影响透光）

2. 装水:
   - 装3/4满
   - 摇晃增氧（加速反应）
   - 装满

3. 曝晒:
   - 平放在金属板/瓦片上（增温）
   - 或涂黑一半瓶身
   - 晴天: 曝晒6小时
   - 阴天: 曝晒2天

4. 饮用:
   - 冷却后饮用
   - 或继续储存
```

**效果:**
- 6小时阳光 = 煮沸效果的80-90%
- 去除细菌、病毒、原生动物

**限制:**
- 浑浊水效果差（预先过滤）
- 阴雨天不可用
- 化学污染无效

## 五、雨水收集

### 5.1 为什么收集雨水?

**优点:**
- 相对清洁（天然蒸馏）
- 免费、可再生
- 量大（一场雨数百升）

**用途:**
- 饮用（处理后）
- 洗涤
- 灌溉

### 5.2 收集系统

**简易收集:**

```
[示意图]

  ☔️ 雨
    ↓↓↓
  ╔═══════╗
  ║  屋顶  ║
  ╚═══╦═══╝
      ║ ← 雨槽
    ┌─╨─┐
    │滤网│ ← 去树叶
    └─┬─┘
      ↓
    ╔═══╗
    ║储罐║
    ╚═══╝
```

**组件:**

1. **收集面:** 屋顶（瓦、铁皮、帆布）
2. **导流:** 雨槽/竹管
3. **初滤:** 网筛去树叶、虫子
4. **弃流装置:** 前5-10分钟的雨水丢弃（冲洗屋顶）
5. **储水罐:** 陶缸、水桶、水池

**弃流装置（重要！）:**

```
[原理]
初雨冲洗屋顶脏物
    ↓
进入弃流管
    ↓
管满后，干净雨水
    ↓
进入储水罐
```

**简易版:**
- 前10分钟手动移开导流管
- 之后再接入储罐

### 5.3 雨水处理

**仍需处理！**

```
步骤:
1. 沉淀24小时
2. 过滤（沙滤）
3. 煮沸或化学消毒
```

**原因:**
- 空气污染物
- 屋顶污染（鸟粪、灰尘）
- 储存过程污染

### 5.4 储存

**容器:**
- 陶缸（最佳）
- 水泥池
- 食品级塑料桶

**要点:**
- 密封（防蚊虫、灰尘）
- 遮光（防藻类生长）
- 定期清洗（3-6个月）
- 底部开龙头（避免搅动沉淀）

**容量计算:**

```
屋顶面积(m²) × 降雨量(mm) = 收集量(升)

例: 50m²屋顶，降雨20mm
    50 × 20 = 1000升
```

**储备建议:**
- 最少: 30天用水量（60-120升/人）
- 理想: 90天用水量

## 六、挖井技术

### 6.1 井的类型

#### 浅井（<15米）

**适用:**
- 地下水位高
- 短期使用
- 应急

**特点:**
- 易挖
- 水量少
- 易污染

**挖掘:**
- 人工挖掘
- 圆形或方形
- 直径1-1.5米

#### 深井（15-50米）

**适用:**
- 永久水源
- 大量用水
- 水质好

**特点:**
- 难挖
- 需设备
- 水质好

**挖掘:**
- 需专业钻井设备
- 或人工凿井（极费力）

### 6.2 选址

**地下水指示:**

**植物指示:**
- 芦苇、香蒲: 地下水浅
- 柳树、白杨: 地下水丰富
- 绿色植物集中: 可能有水

**地形指示:**
- 山谷、洼地
- 河流古道
- 泉水下游

**避开:**
- 厕所、垃圾场下游
- 畜栏附近
- 易渗化粪池

**距离要求:**
- 距污染源>30米
- 地势高于周围

### 6.3 挖井步骤（浅井）

**准备:**
- 铁锹、铁镐
- 水桶（提土）
- 绳索
- 井圈（砖、石、混凝土圈）

**步骤:**

```
1. 挖掘:
   - 直径1.5米
   - 一边挖一边砌井壁
   - 防止塌方！

2. 遇到地下水:
   - 继续挖深1-2米
   - 水会不断渗入

3. 清淤:
   - 掏出泥沙
   - 底部铺鹅卵石（过滤）

4. 封顶:
   - 井口高出地面30cm
   - 加盖防污染
   - 周围砌台防雨水倒灌

5. 洗井:
   - 反复抽水
   - 直至水清
```

**安全:**
- 必须2人以上
- 挖掘者系安全绳
- 深度>3米需通风（点蜡烛测氧）
- 警惕塌方和有毒气体

### 6.4 井水处理

**即使井水也需处理！**

```
步骤:
1. 静置沉淀
2. 过滤
3. 煮沸或消毒
```

**井水可能的污染:**
- 地表水渗入
- 地层矿物质（氟、砷等）
- 微生物

## 七、水质检测

### 7.1 感官检测

**看:**
- 清澈透明 ✓
- 浑浊、有色 ✗

**闻:**
- 无异味 ✓
- 腐臭、化学味 ✗

**尝:**
- 无异味、微甘 ✓
- 苦、咸、酸、涩 ✗

### 7.2 简易测试

**煮沸测试:**
- 煮沸后沉淀物多 → 硬度高（多矿物质）

**肥皂测试:**
- 易起泡 → 软水
- 难起泡 → 硬水（高钙镁）

**PH试纸:**
- pH 6.5-8.5 ✓
- 过酸或过碱 ✗

### 7.3 生物指标

**水质指示生物:**

**清洁水:**
- 水生昆虫（蜻蜓幼虫）
- 鱼虾

**污染水:**
- 蚊子幼虫多
- 死鱼
- 无生物

## 八、应急取水

### 8.1 挖掘渗水坑

**原理:** 地下水渗出

**步骤:**
```
1. 在河床、低洼处挖坑
2. 深度50cm-1米
3. 等待渗水
4. 舀出沉淀后的水
5. 必须过滤+煮沸
```

### 8.2 露水收集

**方法:**
```
清晨用布擦草叶
  ↓
吸饱露水
  ↓
拧入容器
```

**产量:** 1小时约0.5-1升

### 8.3 蒸馏法（海水淡化）

**太阳蒸馏器:**

```
[示意图]

  塑料布
  ╱     ╲
 ╱   ○   ╲ ← 小石子压低中心
╱    ↓    ╲
──┬───┬──
  │杯子│   ← 接水
  └───┘
 海水/不洁水

原理:
1. 太阳加热水蒸发
2. 蒸汽凝结在塑料布上
3. 沿塑料布流到中心
4. 滴入杯中
```

**效率:**
- 晴天: 1-2升/天
- 需大面积（1-2m²）

### 8.4 植物取水

**竹子:**
- 老竹节间有水
- 摇晃听水声
- 钻孔取水

**藤蔓:**
- 割断粗藤
- 水会流出
- 立即接住

**仙人掌:**
- 捣碎取汁
- 过滤饮用

## 九、储水技巧

### 9.1 容器选择

**最佳:**
- 陶罐（透气、保鲜）
- 玻璃瓶（不变质）

**可用:**
- 食品级塑料桶
- 不锈钢桶

**避免:**
- 铁桶（生锈）
- 非食品级塑料

### 9.2 防污染

**措施:**
- 密封盖
- 专用舀水勺（不要直接用杯子舀）
- 定期清洗（1-2周）
- 遮光防藻

### 9.3 轮换使用

**原则:**
- 先进先出
- 定期更换（7天）
- 避免长期储存

## 十、每日需水量

### 10.1 基本需求

| 用途 | 每人每天用量 |
|------|--------------|
| 饮水 | 2-4升 |
| 烹饪 | 2-3升 |
| 洗漱 | 5-10升 |
| 洗衣 | 10-20升 |
| 总计（节约模式） | 20-40升 |
| 总计（正常模式） | 50-100升 |

### 10.2 节水策略

**饮用/烹饪:**
- 重复利用（洗菜水洗碗）
- 不浪费一滴

**洗漱:**
- 擦洗代替冲洗
- 接水洗，不长流水

**洗衣:**
- 集中洗（攒够一缸）
- 洗衣水再利用（拖地、浇地）
