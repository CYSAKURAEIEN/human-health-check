## 眼睛异物

### 常见异物

- 灰尘、沙子
- 睫毛
- 小虫
- 木屑、金属碎片

### 症状

- 刺痛、异物感
- 流泪
- 发红
- 眨眼频繁
- 视力模糊

### 处理方法

#### 浅表异物(灰尘、睫毛)

```
1. 洗手
2. 不要揉眼!
3. 冲洗:
   - 用清水冲洗眼睛15分钟
   - 从内眼角向外
   - 或将脸浸入水中,睁眼眨动
4. 拉动眼皮:
   - 上眼皮拉起盖住下眼皮
   - 让下睫毛刷掉上眼皮内的异物
5. 如能看见异物:
   - 用湿润的棉签轻轻粘除
   - 仅限于眼白部分
```

#### ❌ 禁止

```
- 揉眼睛
- 用干棉签或硬物接触眼球
- 试图取出嵌入的异物
- 触碰角膜(黑眼球)
```

### ⚠️ 立即就医

- 异物嵌入眼球
- 金属或木头碎片
- 化学物质溅入(先冲洗15分钟,再就医)
- 眼球受撞击
- 疼痛剧烈
- 视力明显下降
- 无法取出异物
- 取出后仍不适

### 化学品入眼(紧急!)

```
立即处理:
1. 立即冲洗! 不要延误!
2. 流动清水冲洗至少15-20分钟
3. 翻开眼睑冲洗
4. 持续冲洗,同时拨打急救电话
5. 带上化学品容器或标签
6. 就医

注意:
- 酸碱伤害都严重
- 时间就是视力!
- 越早冲洗越好
```

## 耳朵异物

### 常见异物

- 小虫
- 小珠子、玩具零件(儿童)
- 棉花、纸团
- 食物

### 症状

- 耳痛
- 听力下降
- 耳鸣
- 不适感
- 分泌物(虫子活动)

### 处理方法

#### 活虫入耳

```
1. 倾斜头部(患耳向上)
2. 滴入:
   - 温植物油(橄榄油、婴儿油)
   - 或温水
   - 数滴即可
3. 虫子会浮出或死亡
4. 倾斜头部(患耳向下)让其流出
5. 如未流出,就医取出
```

#### 可见异物

```
如果能看到且靠近外耳:
1. 倾斜头部(患耳向下)
2. 轻拍对侧头部
3. 可能掉出

或:
- 用镊子(非常小心!)
- 仅限于能完全看到的异物
- 在外耳道口部位
```

### ❌ 禁止

```
- 用棉签或硬物掏耳
- 掏得更深(可能推得更深)
- 冲洗(除非是活虫)
- 盲目使用镊子
- 自行处理深部异物
```

### ⚠️ 立即就医

- 异物在耳道深处
- 电池(会腐蚀,紧急!)
- 尖锐物体
- 耳痛剧烈
- 流血
- 听力明显下降
- 自行取不出

## 鼻子异物

### 常见(儿童多见)

- 小玩具、珠子
- 食物
- 纸团、棉花
- 小电池(危险!)

### 症状

- 单侧鼻塞
- 鼻涕(可能有臭味、血丝)
- 呼吸困难
- 鼻痛

### 处理方法

#### 可见异物

```
方法1: 擤鼻子
1. 压住健侧鼻孔
2. 轻轻从患侧擤鼻子
3. 不要太用力

方法2: 亲吻法(儿童)
1. 压住健侧鼻孔
2. 家长对孩子嘴巴快速吹气
3. 气流可能将异物冲出

方法3: 镊子(仅外部可见)
- 非常小心
- 仅限于鼻孔口
```

### ❌ 禁止

```
- 用硬物捅
- 试图钩出(可能推深)
- 让孩子用力吸鼻子
- 盲目使用镊子
```

### ⚠️ 立即就医

- 电池(立即!会腐蚀)
- 磁铁(尤其是两个)
- 尖锐物体
- 无法取出
- 呼吸困难
- 大量出血

## 喉咙异物/气道梗阻

### 严重程度判断

#### 部分梗阻(轻度)
**症状**:
- 能咳嗽
- 能说话或哭
- 能呼吸(虽困难)

**处理**:
```
✓ 鼓励用力咳嗽
✓ 不要拍背或干预
✓ 持续观察
✓ 准备急救

如转为完全梗阻,立即急救!
```

#### 完全梗阻(重度)
**症状**:
- 不能咳嗽
- 不能说话或哭
- 不能呼吸
- 双手抓喉咙(窒息信号)
- 脸色发紫

**处理**: 立即急救!

### 急救方法

#### 成人和1岁以上儿童: 海姆立克急救法

**伤者站立或坐位**:

```
1. 站在伤者背后
2. 双臂环抱腰部
3. 一手握拳,拳头拇指侧放在:
   - 肚脐上方
   - 肋骨下缘
4. 另一手握住拳头
5. 快速向内上方冲击
6. 重复直到:
   - 异物排出,或
   - 伤者昏迷

力度:
- 快速、有力
- 向内上方
- 每次独立的冲击
```

**伤者昏迷**:

```
1. 平躺
2. 开始CPR(胸外按压)
3. 每次准备人工呼吸前:
   - 检查口腔
   - 如看到异物,取出
   - 不要盲目掏
4. 持续CPR直到急救到达
```

**孕妇或肥胖者**:

```
胸部冲击:
1. 双臂环抱胸部
2. 拳头放在胸骨中部
3. 向后快速冲击
```

**自救**(独自一人):

```
方法1: 腹部冲击
1. 拳头放在腹部
2. 另一手握住
3. 向内上方快速冲击

方法2: 椅背法
1. 俯身在椅背上
2. 腹部抵住椅背顶端
3. 快速向下压
```

#### 1岁以下婴儿

**拍背法 + 胸部冲击法**:

```
1. 拍背:
   - 将婴儿面朝下放在前臂上
   - 头低于身体
   - 支撑头颈
   - 用手掌根拍背(两肩胛骨之间)
   - 5次,用力

2. 翻转婴儿:
   - 面朝上
   - 头仍低于身体

3. 胸部冲击:
   - 两指放在胸骨中部(两乳头连线下方)
   - 快速向下压(深度约4厘米)
   - 5次

4. 重复拍背和胸部冲击
   - 直到异物排出
   - 或婴儿昏迷

如昏迷:
- 开始婴儿CPR
- 检查口腔异物
- 呼叫急救
```

#### ❌ 禁止(婴儿)

```
- 不要用海姆立克法(会伤内脏)
- 不要头朝上拍背(异物可能更深)
- 不要盲目掏喉咙
```

### 预防

```
✓ 细嚼慢咽
✓ 小口进食
✓ 不要边吃边说笑
✓ 儿童:
  - 避免坚果(3岁以下)
  - 切小块
  - 远离小玩具
  - 监督进食
✓ 老人:
  - 假牙固定好
  - 食物煮软
```

## 误吞异物

### 常见误吞物

- 硬币
- 纽扣电池(危险!)
- 磁铁(尤其多个)
- 鱼刺、骨头
- 图钉、针
- 玩具零件

### 症状

- 可能无症状
- 吞咽困难或疼痛
- 流口水
- 胸痛或腹痛
- 呕吐
- 拒绝进食

### 处理

#### 一般异物(硬币、小圆物)

```
如果:
✓ 能说话
✓ 能呼吸
✓ 不痛或轻微不适

处理:
1. 保持冷静
2. 不要催吐
3. 不要喝水或进食
4. 联系医生咨询
5. 观察大便(2-3天内排出)

就医如果:
- 持续疼痛
- 吞咽困难
- 呕吐
- 腹痛
- 3天未排出(视物体大小)
```

#### 危险异物(立即就医!)

**纽扣电池**:
```
极度危险!
- 2小时内可造成严重化学烧伤
- 立即急诊!
- 不要等待
- 不要催吐
- 不要进食
- 带上电池包装或相同电池
```

**磁铁(两个或多个)**:
```
极度危险!
- 可能在肠道内互相吸引
- 导致肠穿孔
- 立即就医!
```

**尖锐物体**:
```
危险!
- 鱼刺、骨头、针、图钉
- 可能刺穿消化道
- 立即就医
- 不要吞食物(企图推下)
- 不要催吐
```

#### 鱼刺卡喉

```
❌ 错误方法(不要用!):
- 吞饭团、馒头(可能刺更深)
- 喝醋(无效)
- 用手抠

✓ 正确方法:
1. 停止进食
2. 如能看到鱼刺,用镊子小心取出
3. 如在喉咙深处:
   - 不要自行处理
   - 立即就医
4. 医生会:
   - 喉镜检查
   - 取出鱼刺

危险信号:
- 剧烈疼痛
- 呼吸困难
- 吞咽困难
- 流口水
- 呕血
```

### ⚠️ 立即就医

- 纽扣电池
- 多个磁铁
- 尖锐物体
- 较大物体
- 呼吸困难
- 剧烈疼痛
- 吞咽困难
- 呕吐(尤其是血)
- 流口水
- 发烧
- 腹痛

### 预防(儿童)

```
✓ 小物品远离儿童
✓ 检查玩具(无小零件)
✓ 纽扣电池妥善保管
✓ 硬币、磁铁收好
✓ 监督进食
✓ 食物切小块
✓ 去除鱼刺、骨头
```

## 📝 总结要点

### 异物处理原则

```
✓ 保持冷静
✓ 评估严重程度
✓ 简单、安全的方法优先
✓ 不要盲目操作
✓ 及时就医

❌ 不要:
- 盲目掏取
- 用硬物捅
- 将异物推深
- 延误就医(危险异物)
```

### 危险异物(立即就医)

- 纽扣电池
- 多个磁铁
- 尖锐物体
- 化学品入眼
- 完全气道梗阻

### 掌握的急救技能

- 海姆立克急救法
- 婴儿拍背法
- 眼睛冲洗
- 基础异物取出

### 预防最重要

- 儿童监护
- 安全存放小物品
- 细嚼慢咽
- 食物安全处理
