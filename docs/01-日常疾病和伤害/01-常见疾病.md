## 感冒

### 症状识别

**普通感冒**:
- 鼻塞、流鼻涕
- 打喷嚏
- 喉咙轻微疼痛
- 轻微咳嗽
- 一般不发烧或低烧(<38°C)

**流感**:
- 突然高烧(>38.5°C)
- 全身肌肉酸痛
- 严重乏力
- 头痛
- 干咳

### 家庭护理

#### 1. 休息和补水
```
✓ 充足睡眠(每天8小时以上)
✓ 多喝温水(每天2-3升)
✓ 温热饮品(蜂蜜柠檬水、姜茶)
✓ 避免脱水
```

#### 2. 缓解症状
- **鼻塞**:
  - 热水蒸汽熏蒸(10-15分钟)
  - 生理盐水鼻喷雾
  - 提高枕头高度睡觉

- **喉咙痛**:
  - 温盐水漱口(半茶匙盐+温水)
  - 蜂蜜(1岁以上)
  - 润喉糖

- **咳嗽**:
  - 蜂蜜(睡前一勺)
  - 保持空气湿润
  - 避免刺激性气味

#### 3. 药物治疗(非必需)
- 退烧止痛: 对乙酰氨基酚、布洛芬
- 减充血剂: 缓解鼻塞
- **注意**: 抗生素对病毒性感冒无效

### 预防措施

```
✓ 勤洗手(20秒以上)
✓ 避免触摸脸部
✓ 保持社交距离
✓ 佩戴口罩(人群密集处)
✓ 增强免疫力(均衡饮食、适度运动)
✓ 接种流感疫苗
```

### ⚠️ 就医指征

立即就医如果出现:
- 高烧持续3天以上
- 呼吸困难
- 胸痛
- 严重头痛或颈部僵硬
- 持续呕吐
- 意识混乱
- 症状突然恶化

## 发烧

### 体温标准

| 测量部位 | 正常体温 | 低烧 | 中度发烧 | 高烧 |
|----------|----------|------|----------|------|
| 口腔 | 36.3-37.2°C | 37.3-38.0°C | 38.1-39.0°C | >39°C |
| 腋下 | 36.0-37.0°C | 37.1-37.7°C | 37.8-38.7°C | >38.7°C |
| 肛门 | 36.5-37.5°C | 37.6-38.3°C | 38.4-39.3°C | >39.3°C |
| 耳温 | 35.8-38.0°C | 38.1-38.5°C | 38.6-39.5°C | >39.5°C |

### 何时需要退烧

**成人**:
- 体温≥38.5°C,且感到不适
- 有基础疾病者,体温≥38°C

**儿童**:
- 3个月以下: 任何发烧都应就医
- 3-6个月: 体温≥38.5°C
- 6个月以上: 体温≥39°C或明显不适

### 退烧方法

#### 1. 物理降温

**温水擦浴**:
```
1. 水温: 32-34°C(略低于体温)
2. 擦拭部位: 颈部、腋下、腹股沟、四肢
3. 时间: 15-20分钟
4. 禁止: 酒精擦浴(可能导致中毒)
```

**环境调节**:
- 室温保持在22-24°C
- 减少衣物和被子
- 保持通风

**补充水分**:
- 每小时喝200-300ml水
- 可以喝: 水、果汁、清汤
- 避免: 含咖啡因饮料

#### 2. 药物退烧

**对乙酰氨基酚(扑热息痛)**
- 成人剂量: 500-1000mg,每4-6小时一次
- 儿童剂量: 10-15mg/kg体重,每4-6小时一次
- 最大剂量: 成人每天4000mg

**布洛芬**
- 成人剂量: 200-400mg,每4-6小时一次
- 儿童剂量: 5-10mg/kg体重,每6-8小时一次
- 最大剂量: 成人每天1200mg(非处方)

**用药注意**:
- ❌ 不要同时使用两种退烧药
- ❌ 18岁以下不要使用阿司匹林
- ✓ 饭后服用减少胃部不适

### 发烧的护理

#### 监测记录
```
记录内容:
- 测量时间
- 体温数值
- 伴随症状
- 用药情况
- 饮水量
```

#### 饮食建议
- **宜**: 清淡易消化食物
  - 稀粥、面条
  - 新鲜水果
  - 蔬菜汤
- **忌**:
  - 油腻食物
  - 辛辣刺激
  - 生冷食物

### ⚠️ 紧急就医

以下情况立即就医:
- 婴儿(<3个月)任何发烧
- 体温≥40°C
- 发烧伴随:
  - 剧烈头痛
  - 颈部僵硬
  - 意识改变
  - 抽搐
  - 呼吸困难
  - 胸痛
  - 严重腹痛
  - 皮疹(尤其是不褪色的)
  - 持续呕吐
- 发烧超过3天无改善

## 腹泻

### 腹泻分类

**急性腹泻** (持续<2周)
- 感染性: 细菌、病毒、寄生虫
- 非感染性: 食物不耐受、药物反应

**慢性腹泻** (持续>4周)
- 需就医检查原因

### 症状评估

**轻度**:
- 每天3-5次稀便
- 无脱水症状
- 能正常饮食

**中度**:
- 每天6-10次
- 轻度脱水
- 腹痛

**重度**:
- 每天>10次
- 明显脱水
- 发烧、血便

### 脱水症状识别

**轻度脱水**:
- 口渴
- 尿量减少,颜色深
- 轻微头晕

**中度脱水**:
- 口干舌燥
- 皮肤弹性差
- 眼窝凹陷
- 尿量明显减少

**重度脱水**(立即就医):
- 极度口渴或不渴
- 无尿或极少
- 皮肤干燥
- 意识模糊
- 脉搏快而弱

### 家庭治疗

#### 1. 补液(最重要!)

**口服补液盐(ORS)**:
```
WHO推荐配方(1升水):
- 食盐: 3.5g (约3/4茶匙)
- 糖: 20g (约4茶匙)
- 可选: 小苏打 2.5g

用法:
- 少量多次饮用
- 成人: 每次腹泻后200-400ml
- 儿童: 每次50-100ml
- 室温或微温饮用
```

**其他补液**:
- 清汤
- 米汤
- 稀释的果汁
- 椰子水

#### 2. 饮食调整

**BRAT饮食** (初期)
- **B**ananas: 香蕉
- **R**ice: 白米饭/粥
- **A**pplesauce: 苹果泥
- **T**oast: 烤面包

**逐渐添加**:
- 煮熟的胡萝卜
- 蒸土豆
- 鸡肉(去皮)
- 酸奶(含益生菌)

**避免**:
- ❌ 油腻食物
- ❌ 辛辣食物
- ❌ 高纤维食物
- ❌ 乳制品(乳糖不耐受期)
- ❌ 咖啡因
- ❌ 酒精

#### 3. 药物治疗

**止泻药**(仅成人,医生建议下使用):
- 洛哌丁胺(Imodium)
- **禁用于**:
  - 血性腹泻
  - 高烧
  - 儿童(无医嘱)

**益生菌**:
- 有助于恢复肠道菌群
- 建议持续服用1-2周

**锌补充剂**(儿童):
- 可缩短腹泻持续时间

### 预防措施

```
✓ 勤洗手(尤其是饭前便后)
✓ 食物煮熟煮透
✓ 饮用安全水源
✓ 妥善保存食物
✓ 避免生冷食物
✓ 注意食物卫生
✓ 餐具清洁消毒
```

### ⚠️ 就医指征

立即就医如果:
- 严重脱水症状
- 血便或黑便
- 高烧(>39°C)
- 剧烈腹痛
- 持续呕吐无法补液
- 腹泻超过3天无改善
- 婴幼儿腹泻
- 近期旅行史(可能寄生虫感染)
- 免疫力低下者

## 呕吐

### 常见原因

- 肠胃炎(病毒或细菌)
- 食物中毒
- 晕动病
- 妊娠
- 药物副作用
- 过度饮酒

### 即时处理

#### 呕吐时
```
1. 坐起或侧卧(防止呛咳)
2. 准备容器
3. 漱口清洁
4. 深呼吸
```

#### 呕吐后
```
1. 先禁食1-2小时
2. 小口喝水测试
3. 如不再吐,逐渐增加液体
4. 休息
```

### 补液方案

**第一阶段**(呕吐后0-4小时):
```
- 每5-10分钟一小口水(5-10ml)
- 可以含冰块
- 可喝: 清水、口服补液盐
- 避免: 果汁、牛奶、饮料
```

**第二阶段**(4-8小时,如无呕吐):
```
- 逐渐增加饮水量
- 每15分钟30-60ml
- 可添加: 清汤、米汤
```

**第三阶段**(8小时后,如耐受):
```
- 开始进食
- 从流食开始: 稀粥、面汤
- 逐渐过渡到固体食物
```

### 饮食恢复

**可以吃**:
- 清淡的碳水化合物
  - 白米粥
  - 面条
  - 苏打饼干
  - 烤面包
- 香蕉
- 煮熟的土豆

**避免**(至少24-48小时):
- 油腻食物
- 辛辣食物
- 乳制品
- 酸性食物
- 咖啡因
- 酒精

### 药物治疗

**成人止吐药**(医生建议下):
- 甲氧氯普胺(胃复安)
- 多潘立酮(吗丁啉)

**儿童**:
- 不建议自行用药
- 应咨询医生

**晕车药**:
- 茶苯海明(乘晕宁)
- 出行前30分钟服用

### ⚠️ 就医指征

立即就医如果:
- 持续呕吐>24小时
- 无法保持任何液体
- 呕吐物含血或呈咖啡色
- 严重腹痛
- 脱水症状严重
- 剧烈头痛或颈部僵硬
- 意识改变
- 婴儿>8小时未排尿

## 食物中毒

### 症状

通常在进食后1-6小时出现:
- 恶心、呕吐
- 腹泻(可能带血)
- 腹痛、痉挛
- 发烧
- 头痛、乏力

### 处理步骤

#### 1. 停止进食可疑食物
```
- 保留剩余食物样本(供检测)
- 记录: 食物种类、进食时间、症状出现时间
```

#### 2. 补液
```
- 每15分钟喝少量液体
- 口服补液盐首选
- 避免咖啡因和酒精
```

#### 3. 休息
```
- 卧床休息
- 避免剧烈活动
```

#### 4. 逐渐恢复饮食
```
- 症状缓解后开始进食
- 从清淡食物开始
```

### 不同病原体特点

| 病原体 | 潜伏期 | 典型症状 | 常见来源 |
|--------|--------|----------|----------|
| 金黄色葡萄球菌 | 1-6小时 | 剧烈呕吐 | 熟食、糕点 |
| 沙门氏菌 | 6-72小时 | 腹泻、发烧 | 蛋、禽肉 |
| 大肠杆菌 | 1-8天 | 血性腹泻 | 生肉、未消毒奶 |
| 诺如病毒 | 12-48小时 | 呕吐、腹泻 | 贝类、污染水 |
| 肉毒杆菌 | 12-36小时 | 神经症状 | 罐头食品 |

### 预防措施

#### 采购
```
✓ 检查保质期
✓ 包装完整
✓ 冷藏食品温度适当
```

#### 储存
```
✓ 冰箱温度: ≤4°C
✓ 冷冻: ≤-18°C
✓ 生熟分开
✓ 密封保存
```

#### 烹饪
```
✓ 煮熟煮透
✓ 肉类中心温度≥75°C
✓ 避免交叉污染
✓ 切菜板生熟分开
```

#### 进食
```
✓ 热食热吃(>60°C)
✓ 冷食冷吃(<5°C)
✓ 食物在室温下不超过2小时
```

### ⚠️ 紧急就医

以下情况立即就医:
- 血性腹泻
- 高烧(>39°C)
- 严重脱水
- 持续呕吐>24小时
- 神经症状(视力模糊、肌肉无力、说话困难)
- 症状持续>3天
- 孕妇、儿童、老人、免疫力低下者

## 便秘

### 定义

以下任一情况:
- 每周排便<3次
- 大便干硬
- 排便困难,需用力
- 排便不尽感

### 常见原因

- 饮食纤维不足
- 饮水少
- 缺乏运动
- 忽视便意
- 药物副作用
- 生活习惯改变
- 压力

### 家庭治疗

#### 1. 增加纤维摄入

**目标**: 每天25-35克纤维

**高纤维食物**:
- 蔬菜: 西兰花、胡萝卜、菠菜
- 水果: 苹果、梨、浆果、西梅
- 全谷物: 燕麦、糙米、全麦面包
- 豆类: 扁豆、黑豆、鹰嘴豆
- 坚果和种子

**注意**: 逐渐增加纤维,避免腹胀

#### 2. 充足饮水

```
- 成人: 每天2-3升水
- 早晨空腹喝温水
- 多喝: 水、温水、果汁
```

#### 3. 规律运动

```
- 每天至少30分钟
- 散步、慢跑、游泳
- 腹部按摩(顺时针)
```

#### 4. 建立规律

```
- 固定时间如厕(如早餐后)
- 不要忽视便意
- 不要久坐马桶(不超过10分钟)
- 放松,不要过度用力
```

#### 5. 姿势优化

```
- 使用脚凳(蹲姿更自然)
- 身体前倾
- 膝盖高于臀部
```

### 药物治疗(短期使用)

**膨胀性泻药**(首选):
- 车前草、甲基纤维素
- 增加大便体积
- 需大量饮水

**渗透性泻药**:
- 乳果糖、聚乙二醇
- 软化大便

**刺激性泻药**(谨慎使用):
- 番泻叶、比沙可啶
- 仅短期、偶尔使用
- 可能产生依赖

**润滑剂**:
- 开塞露(甘油)
- 用于紧急缓解

### ⚠️ 就医指征

立即就医如果:
- 突然便秘伴剧烈腹痛
- 便血或黑便
- 不明原因的体重下降
- 便秘与腹泻交替
- 便秘持续>2周无改善
- 伴随呕吐
- 腹部明显肿胀

## 头痛

### 头痛类型

#### 1. 紧张性头痛(最常见)

**症状**:
- 双侧头部紧缩感
- 像被绷带勒紧
- 轻到中度疼痛
- 持续30分钟到数天

**诱因**:
- 压力、焦虑
- 姿势不良
- 眼睛疲劳
- 睡眠不足

#### 2. 偏头痛

**症状**:
- 单侧搏动性疼痛
- 中到重度
- 恶心、呕吐
- 畏光、畏声
- 可能有先兆(视觉异常)
- 持续4-72小时

**诱因**:
- 特定食物(奶酪、巧克力、酒精)
- 荷尔蒙变化
- 睡眠不规律
- 强光、强烈气味

#### 3. 丛集性头痛

**症状**:
- 单侧眼部及周围剧痛
- 流泪、鼻塞
- 持续15分钟到3小时
- 可能每天发作多次

### 家庭处理

#### 即时缓解

**紧张性头痛**:
```
1. 休息,闭眼
2. 热敷颈部和肩膀(20分钟)
3. 轻柔按摩太阳穴
4. 放松练习、深呼吸
```

**偏头痛**:
```
1. 安静、黑暗的房间
2. 冷敷前额(15-20分钟)
3. 平躺休息
4. 避免光线和噪音
```

#### 药物治疗

**轻到中度头痛**:
- 对乙酰氨基酚: 500-1000mg
- 布洛芬: 200-400mg
- 阿司匹林: 500-1000mg(成人)

**注意**:
- 在头痛早期服用效果更好
- 不要频繁使用(可能导致药物过量性头痛)
- 每月使用不超过10-15天

#### 非药物方法

```
✓ 保持水分充足
✓ 规律进食
✓ 充足睡眠(7-9小时)
✓ 减少屏幕时间
✓ 改善姿势
✓ 管理压力
✓ 规律运动
```

### 预防措施

#### 识别和避免诱因

**常见诱因记录表**:
```
- 日期时间
- 头痛类型和强度
- 持续时间
- 诱因(食物、活动、压力等)
- 治疗方法和效果
```

#### 生活方式调整

```
✓ 规律作息
✓ 均衡饮食
✓ 适度运动
✓ 压力管理
✓ 限制咖啡因
✓ 戒烟限酒
```

### ⚠️ 紧急就医("雷击"信号)

立即就医如果头痛伴随:
- 突发最剧烈头痛("一生中最严重")
- 发烧、颈部僵硬
- 意识改变、混乱
- 视力突然改变
- 说话困难
- 行走困难、失去平衡
- 面部、手臂或腿部麻木/无力
- 头部受伤后的头痛
- 头痛逐渐加重
- 50岁后首次出现严重头痛

## 咳嗽

### 咳嗽类型

**按持续时间**:
- 急性: <3周(通常感冒)
- 亚急性: 3-8周(感染后)
- 慢性: >8周(需查明原因)

**按性质**:
- 干咳: 无痰
- 湿咳: 有痰

### 常见原因

- 病毒感染(感冒、流感)
- 细菌感染
- 过敏
- 哮喘
- 胃食管反流
- 药物副作用
- 吸烟

### 家庭治疗

#### 1. 缓解咳嗽

**蜂蜜**(1岁以上):
```
- 睡前1-2茶匙
- 可与温水或茶混合
- 有镇咳和抗菌作用
```

**蒸汽吸入**:
```
- 热水蒸汽(小心烫伤)
- 热水澡
- 加湿器
- 保持室内湿度40-50%
```

**温热饮品**:
- 蜂蜜柠檬水
- 姜茶
- 鸡汤

**抬高头部**:
- 睡觉时多垫一个枕头
- 减少夜间咳嗽

#### 2. 化痰

**多喝水**:
- 稀释痰液
- 每天2-3升

**拍背排痰**(适用于儿童):
```
1. 俯卧或侧卧
2. 掌心成杯状
3. 从下往上轻拍背部
4. 持续3-5分钟
5. 鼓励咳出痰液
```

#### 3. 药物

**止咳药**(干咳):
- 右美沙芬
- **注意**: 不建议给6岁以下儿童

**祛痰药**(湿咳):
- 愈创甘油醚
- 氨溴索、乙酰半胱氨酸

**不建议**:
- 抗生素(病毒性咳嗽无效)
- 除非医生确诊细菌感染

#### 4. 避免刺激

```
❌ 吸烟和二手烟
❌ 强烈气味(香水、清洁剂)
❌ 冷空气(外出戴口罩)
❌ 剧烈运动(急性期)
```

### 不同痰液的意义

| 痰液颜色 | 可能原因 | 处理 |
|----------|----------|------|
| 清白色 | 病毒感染、过敏 | 观察 |
| 黄绿色 | 细菌感染 | 可能需抗生素 |
| 棕色 | 吸烟、旧血 | 戒烟、就医 |
| 粉红泡沫 | 心脏/肺问题 | 立即就医 |
| 血丝/血块 | 多种原因 | 就医检查 |

### 预防措施

```
✓ 勤洗手
✓ 避免接触感染者
✓ 戒烟
✓ 接种疫苗(流感、肺炎)
✓ 避免过敏原
✓ 保持室内清洁
```

### ⚠️ 就医指征

立即就医如果:
- 咳血
- 呼吸困难、喘息
- 胸痛
- 高烧持续
- 咳嗽>3周无改善
- 痰液颜色变化(黄绿、粉红)
- 伴随体重下降
- 夜间盗汗
- 婴儿咳嗽伴呼吸急促

## 📝 总结要点

### 何时自我护理

大多数轻微疾病可以在家处理:
- 症状轻微
- 逐渐好转
- 能正常饮食和休息
- 无危险信号

### 何时就医

出现以下情况应就医:
- 症状严重或持续
- 症状突然恶化
- 出现危险信号
- 婴幼儿、老人、孕妇
- 免疫力低下者

### 家庭常备药

**基础药物**:
- 退烧止痛药
- 止泻药
- 口服补液盐
- 抗过敏药
- 益生菌
- 外用消毒药

**定期检查**:
- 查看有效期
- 补充用完的药品
- 妥善保存
