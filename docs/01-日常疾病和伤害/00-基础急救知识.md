## 生命体征评估

### ABC原则

在任何急救情况下,首先检查:

#### A - Airway (气道)
- **检查**: 气道是否通畅
- **处理**:
  - 清除口腔异物
  - 将头部后仰,下巴抬起
  - 侧卧位防止呕吐物堵塞

#### B - Breathing (呼吸)
- **检查**: 是否有呼吸
  - 看:胸部起伏
  - 听:呼吸声音
  - 感:脸颊感受气流
- **正常**: 成人12-20次/分钟

#### C - Circulation (循环)
- **检查**: 脉搏和出血
  - 颈动脉:喉结旁2-3厘米
  - 桡动脉:手腕拇指侧
- **正常**: 成人60-100次/分钟

### 意识水平评估(AVPU)

- **A - Alert**: 清醒,能正常对话
- **V - Voice**: 对声音有反应
- **P - Pain**: 仅对疼痛刺激有反应
- **U - Unresponsive**: 完全无反应

## 心肺复苏(CPR)

### 何时进行CPR
- 无反应
- 无呼吸或仅有濒死喘息
- 无脉搏

### 成人CPR步骤

#### 1. 确认安全和呼救
```
1. 确保现场安全
2. 轻拍肩膀:"你还好吗?"
3. 如无反应,立即呼救
4. 拨打120急救电话
```

#### 2. 胸外按压
- **位置**: 胸骨中下1/3,两乳头连线中点
- **深度**: 5-6厘米
- **频率**: 100-120次/分钟
- **手法**:
  ```
  1. 跪在伤者旁边
  2. 一手掌根放在胸骨上
  3. 另一手掌根叠放在上面
  4. 手臂伸直,垂直向下按压
  5. 每次按压后完全放松,但手不离开
  ```

#### 3. 人工呼吸(如果会做)
- 每30次按压后,进行2次人工呼吸
- 每次吹气1秒,看到胸部隆起即可
- 如果不会或不愿意,只做胸外按压也可以

#### 4. 持续CPR
- 按照30:2的比例持续
- 直到:
  - 急救人员到达
  - 伤者恢复呼吸和脉搏
  - 你已精疲力竭无法继续

### 儿童和婴儿CPR区别

| 项目 | 成人 | 儿童(1-8岁) | 婴儿(<1岁) |
|------|------|-------------|------------|
| 按压深度 | 5-6cm | 5cm | 4cm |
| 按压方式 | 双手 | 单手或双手 | 双指 |
| 按压位置 | 胸骨中下1/3 | 胸骨中下1/3 | 两乳头连线下方 |
| 按压呼吸比 | 30:2 | 30:2(单人)<br>15:2(双人) | 30:2(单人)<br>15:2(双人) |

### 自动体外除颤器(AED)使用

如果现场有AED设备:

```
1. 打开AED电源
2. 按照语音提示操作
3. 撕开衣服,擦干胸部
4. 贴上电极片(按图示位置)
5. 确保无人触碰伤者
6. 按下"分析"按钮
7. 如提示需要除颤,按下"电击"按钮
8. 立即继续CPR
```

## 止血方法

### 1. 直接压迫止血(首选)

**适用**: 大部分出血

**方法**:
```
1. 用干净纱布或布料直接压在伤口上
2. 持续用力按压至少10分钟
3. 不要频繁抬起查看
4. 如血液浸透,再加一层纱布继续按压
5. 用绷带固定
```

### 2. 抬高伤肢

**配合使用**:
- 将受伤肢体抬高至高于心脏
- 有助于减少出血
- 同时继续直接压迫

### 3. 压迫止血点

**适用**: 四肢大出血且直接压迫无效

**主要止血点**:
- **上臂**: 肱动脉(上臂内侧)
- **大腿**: 股动脉(腹股沟中点)

**方法**: 用拇指或手掌压迫动脉搏动点

### 4. 止血带(最后手段)

⚠️ **仅在以下情况使用**:
- 肢体严重出血威胁生命
- 其他止血方法无效
- 肢体已部分离断

**使用方法**:
```
1. 在伤口上方5-7厘米处(近心端)
2. 使用宽布条(至少5厘米宽)
3. 紧紧缠绕2-3圈
4. 记录使用时间
5. 不要覆盖止血带
6. 不要松开,等待医疗救援
7. 最长不超过2小时
```

**禁止**:
- ❌ 使用铁丝、绳索等细线
- ❌ 用于躯干部位
- ❌ 反复松开和收紧

### 特殊部位出血

#### 鼻出血
```
1. 坐位,头略前倾
2. 捏紧鼻翼10分钟
3. 冷敷额头和鼻梁
4. 不要仰头(会导致血液流入喉咙)
```

#### 内出血征象
- 皮肤苍白、湿冷
- 脉搏快而弱
- 呼吸急促
- 腹部肿胀或僵硬
- 吐血或便血

**处理**: 立即就医,保持平卧,不要进食

## 休克处理

### 休克征象识别

- 皮肤苍白、湿冷
- 脉搏快而弱(>100次/分)
- 呼吸急促浅表
- 烦躁不安或淡漠
- 意识模糊
- 口渴

### 休克的处理

```
1. 让伤者平躺
2. 抬高下肢20-30厘米(除非有骨折)
3. 保持体温(盖毯子)
4. 不要给食物或水
5. 侧卧位防止呕吐
6. 持续监测生命体征
7. 立即呼叫急救
```

### 不同类型休克

| 类型 | 原因 | 特点 |
|------|------|------|
| 失血性休克 | 大量出血 | 先止血 |
| 感染性休克 | 严重感染 | 保暖 |
| 心源性休克 | 心脏问题 | 半坐位 |
| 过敏性休克 | 严重过敏 | 立即用肾上腺素 |
| 神经性休克 | 脊髓损伤 | 绝对平卧 |

## 搬运伤员

### 何时不能搬运

⚠️ **以下情况不要移动伤者**:
- 疑似颈椎或脊柱损伤
- 除非:
  - 现场有危险(火灾、爆炸等)
  - 必须移动才能进行急救
  - 等待专业救援

### 安全搬运原则

1. **评估**: 先评估伤情
2. **准备**: 足够的人手和器材
3. **固定**: 先固定受伤部位
4. **保护**: 保护头颈和脊柱
5. **平稳**: 动作轻柔平稳

### 搬运方法

#### 1. 单人搬运

**扶行法** (伤者能行走)
```
1. 站在伤者一侧
2. 一手扶腰,一手握住伤者手臂
3. 让伤者手臂搭在你肩上
```

**抱持法** (伤者较轻)
```
1. 一手托背部
2. 一手托腿弯处
3. 将伤者抱起
```

**背负法** (伤者无法行走但意识清醒)
```
1. 蹲在伤者前方
2. 让伤者趴在背上
3. 双手托住臀部
4. 伤者手臂绕过肩膀
```

#### 2. 双人搬运

**椅托式**
```
1. 两人面对面,握住对方手腕形成"座椅"
2. 伤者坐在"座椅"上
3. 手臂搭在救援者肩上
4. 同时起立行走
```

**双人抬举法**
```
1. 一人托头颈和背部
2. 另一人托臀部和腿部
3. 同时发令起立
4. 步调一致
```

#### 3. 多人搬运(脊柱损伤)

**平板搬运法**
```
1. 至少3-4人
2. 使用硬板担架
3. 保持头-颈-躯干在一条直线
4. 同时翻转,同时抬起
5. 固定头部和身体
```

### 简易担架制作

**材料**: 两根结实木棍 + 毯子/衣服

**方法1**: 毯子担架
```
1. 毯子铺平
2. 两根棍子平行放置,间距约50厘米
3. 将毯子两侧向内折,包住棍子
4. 伤者平躺在中间
```

**方法2**: 衣服担架
```
1. 两根棍子穿过多件扣好的外套袖子
2. 外套开口朝下
3. 形成简易担架
```

## 📝 关键要点总结

### 记住"四不要"

1. ❌ 不要随意搬动脊柱损伤者
2. ❌ 不要给昏迷者喂食喂水
3. ❌ 不要拔出插入体内的异物
4. ❌ 不要在能力之外勉强救援

### 记住"五优先"

1. ✅ 先保护自己安全
2. ✅ 先呼叫专业救援
3. ✅ 先处理危及生命的情况
4. ✅ 先止血再固定
5. ✅ 先抢救后搬运

### 生命救援顺序

```
1. 确保现场安全
2. 评估意识和呼吸
3. 呼叫急救(120)
4. 控制大出血
5. 处理休克
6. 固定骨折
7. 包扎伤口
8. 保暖和监护
```
