## 过敏反应

### 严重程度

#### 轻度过敏
- 皮肤瘙痒、红疹
- 轻微肿胀
- 流鼻涕、打喷嚏
- 眼睛发痒

#### 中度过敏
- 大面积皮疹、荨麻疹
- 明显肿胀
- 腹痛、恶心、腹泻
- 轻度呼吸困难

#### 重度过敏 - 过敏性休克(危及生命!)
⚠️ **立即拨打120!**

**症状**:
- 呼吸困难、喘息
- 喉咙肿胀,吞咽困难
- 严重气促
- 血压下降
- 脉搏快而弱
- 头晕、意识模糊
- 皮肤苍白或发青
- 全身荨麻疹

### 处理方法

#### 轻度过敏

```
1. 脱离过敏原
2. 口服抗组胺药:
   - 氯雷他定
   - 西替利嗪
3. 冷敷皮疹处
4. 观察症状变化
```

#### 过敏性休克(紧急!)

```
立即处理:

1. 拨打120

2. 使用肾上腺素自动注射器(如有):
   - EpiPen/安瓿达
   - 注射大腿外侧
   - 握住10秒
   - 可重复(5-15分钟后)

3. 让患者平躺:
   - 抬高下肢
   - 如呕吐,侧卧位
   - 如呼吸困难,半坐位

4. 检查呼吸和脉搏:
   - 如无,开始CPR

5. 保暖

6. 不要让其站立或坐起

7. 不要给食物或饮料

8. 持续监测直到急救到达
```

### 常见过敏原

- **食物**: 花生、坚果、海鲜、鸡蛋、牛奶、大豆、小麦
- **药物**: 青霉素、阿司匹林、某些疫苗
- **昆虫**: 蜜蜂、黄蜂、火蚁
- **乳胶**
- **其他**: 花粉、尘螨、动物皮屑

### 预防

```
✓ 识别自己的过敏原
✓ 随身携带:
  - 过敏卡片(说明过敏原)
  - 肾上腺素自动注射器(严重过敏史者)
  - 抗组胺药
✓ 告知他人自己的过敏情况
✓ 外出就餐询问食材
✓ 检查药物成分
```

## 中暑

### 热相关疾病分级

#### 1. 热痉挛
**症状**: 肌肉疼痛性痉挛(腿、手臂、腹部)

**处理**:
```
1. 转移到阴凉处
2. 休息
3. 补充电解质液
   - 运动饮料
   - 口服补液盐
4. 轻轻拉伸痉挛肌肉
```

#### 2. 热衰竭
**症状**:
- 大量出汗
- 皮肤湿冷
- 脉搏快而弱
- 恶心、呕吐
- 头晕、乏力
- 体温轻度升高(<40°C)

**处理**:
```
1. 转移到阴凉处
2. 平躺,抬高下肢
3. 脱去多余衣物
4. 冷敷:
   - 湿毛巾擦身
   - 冰袋放腋下、腹股沟、颈部
5. 补液:
   - 小口频饮
   - 电解质液
6. 如30分钟未改善,就医
```

#### 3. 热射病/中暑(危及生命!)
⚠️ **紧急情况!立即拨打120!**

**症状**:
- 体温≥40°C
- **皮肤干燥、灼热**(可能不出汗)
- 意识改变(混乱、躁动、昏迷)
- 癫痫发作
- 呼吸急促
- 心跳快
- 头痛剧烈

**处理**:

```
紧急处理:

1. 立即拨打120

2. 快速降温(最重要!):

   冰水浸泡法(最有效):
   - 冰水浴
   - 持续到体温降至38.5°C

   或蒸发降温法:
   - 脱去衣物
   - 全身喷洒凉水
   - 用风扇吹风

   同时:
   - 冰袋敷在:
     * 颈部两侧
     * 腋下
     * 腹股沟

3. 侧卧位(防呕吐)

4. 不要给食物或水(可能昏迷)

5. 监测体温
   - 目标: 降至38.5°C
   - 防止过度降温

6. 检查呼吸和脉搏
   - 必要时CPR

7. 持续降温直到急救到达
```

### 高风险人群

- 婴幼儿、老人
- 慢性病患者
- 过度肥胖
- 某些药物使用者
- 剧烈运动者
- 首次暴露于高温

### 预防

```
✓ 避免高温时段户外活动(10:00-16:00)
✓ 穿轻便、浅色、透气衣物
✓ 戴帽子、墨镜
✓ 充分补水:
  - 运动前、中、后
  - 每15-20分钟喝水
  - 不要等口渴
✓ 逐渐适应高温(1-2周)
✓ 注意身体信号
✓ 高温天避免剧烈运动
```

## 失温症

### 定义
核心体温<35°C

### 严重程度

#### 轻度(32-35°C)
**症状**:
- 剧烈寒战
- 手指不灵活
- 轻度意识混乱
- 呼吸和心跳稍快

#### 中度(28-32°C)
**症状**:
- 寒战减弱或停止
- 肌肉僵硬
- 反应迟钝
- 嗜睡
- 呼吸心跳变慢

#### 重度(<28°C)
**症状**:
- 无意识
- 脉搏微弱
- 呼吸浅慢
- 瞳孔放大
- 可能看似死亡

### 处理方法

#### 轻度失温

```
1. 转移到温暖干燥处

2. 脱去湿冷衣物

3. 保暖:
   - 干燥衣物、毯子
   - 睡袋
   - 保温毯(金属箔)

4. 温暖饮料:
   - 温热的甜饮料
   - 不要酒精!
   - 不要咖啡因

5. 温暖食物:
   - 高能量食物
   - 巧克力、糖果

6. 温和复温:
   - 温暖的房间
   - 多层覆盖
   - 体温传递(皮肤接触)

7. 活动:
   - 轻度活动
   - 不要剧烈运动
```

#### 中重度失温(紧急!)

```
⚠️ 立即拨打120

处理:
1. 轻柔操作(避免触发心律失常)

2. 水平移动(不要站立)

3. 保暖:
   - 干燥衣物
   - 毯子包裹
   - 保护头颈

4. 不要:
   ❌ 直接加热(热水、火、暖气)
   ❌ 按摩四肢
   ❌ 给食物或饮料
   ❌ 让其走动

5. 检查呼吸和脉搏:
   - 检查时间长达1分钟(可能很微弱)
   - 如无,开始CPR
   - CPR持续直到体温恢复

6. 等待专业救援
```

### 特殊注意

```
⚠️ 救援者失温("afterdrop"):
- 四肢复温时,冷血回流心脏
- 可能导致核心体温进一步下降
- 危险!

因此:
- 仅温暖躯干核心
- 不要按摩四肢
- 专业复温很重要
```

### 预防

```
✓ 多层穿衣(3层系统):
  1. 内层:排汗
  2. 中层:保暖
  3. 外层:防风防水
✓ 保持干燥
✓ 保护头部(30%热量从头部散失)
✓ 充足食物和水
✓ 避免酒精和吸烟
✓ 识别早期症状
✓ 携带应急装备
```

## 电击伤

### 紧急处理

#### 1. 确保安全

```
⚠️ 首要原则:保护自己!

如果伤者仍接触电源:
1. 不要直接触碰!
2. 切断电源:
   - 关闭总闸
   - 拔掉插头
3. 如无法切断:
   - 用干燥非导电物品移开电源
   - 木棍、塑料、厚报纸
   - 站在干燥绝缘物上
4. 高压电:
   - 不要靠近!
   - 等待专业人员
```

#### 2. 检查伤者

```
1. 评估意识

2. 检查呼吸和脉搏

3. 如无呼吸/脉搏:
   - 立即CPR
   - 拨打120
   - 持续CPR直到急救到达

4. 如有呼吸:
   - 侧卧位
   - 监测生命体征
```

#### 3. 处理烧伤

```
电击常导致烧伤:
1. 冷水冲洗烧伤处
2. 无菌纱布覆盖
3. 不要涂抹任何东西
```

#### 4. 就医

```
所有电击伤都应就医!

即使看似轻微:
- 可能有内部损伤
- 心律可能延迟异常
- 需心电图监测
```

### 特殊类型

#### 雷击

```
处理:
1. 可以立即触碰伤者(不会带电)
2. 呼叫急救
3. CPR(如需要)
4. 处理烧伤
5. 保暖
6. 就医

特点:
- 常导致心脏骤停
- 积极CPR至关重要
- 看似死亡可能被救活
```

### 预防

```
✓ 电器安全:
  - 插座保护盖(有儿童时)
  - 湿手不碰电器
  - 检查电线损坏
  - 漏电保护器
✓ 雷雨天:
  - 室内躲避
  - 远离窗户、金属
  - 不要打电话(有线)
  - 不要洗澡
  - 户外:蹲低,双脚并拢
✓ 专业电工处理电气问题
```

## 溺水

### 紧急处理

#### 1. 水中救援

```
⚠️ 安全第一!

优先顺序:
1. 伸:用杆、树枝伸向溺水者
2. 抛:抛救生圈、绳索
3. 划:用船靠近
4. 游:最后选择(需训练)

注意:
- 不会游泳者不要下水!
- 溺水者可能拉你下水
- 背后接近较安全
```

#### 2. 上岸后处理

```
立即评估:

1. 拨打120

2. 检查反应和呼吸:
   - 观察10秒

3. 如无反应、无呼吸或仅濒死喘息:
   ⚠️ 立即开始CPR!

   - 不要先控水!
   - 直接胸外按压
   - 30次按压:2次人工呼吸
   - 持续直到:
     * 恢复呼吸
     * 急救到达
     * 精疲力竭

4. 如有呼吸:
   - 侧卧位(恢复体位)
   - 保暖
   - 监测呼吸
   - 可能会呕吐水

5. 保暖:
   - 脱去湿衣
   - 干毛巾擦干
   - 毯子包裹
   - 防止失温
```

### ❌ 常见错误

```
- 先控水再CPR(错!)
  * 延误CPR
  * 浪费黄金时间
  * 可能导致呕吐、误吸

正确:
- 直接CPR
- 如在CPR时呕吐,侧转头部清除,继续CPR
```

### 干性溺水/二次溺水

⚠️ **即使救起,也要就医观察!**

**症状**(数小时内可能出现):
- 持续咳嗽
- 呼吸困难
- 乏力、嗜睡
- 行为异常
- 呕吐

**原因**:
- 肺部少量水
- 喉痉挛
- 肺水肿

**处理**:
- 密切观察24小时
- 如出现症状,立即就医

### 预防

```
✓ 永远不要单独游泳
✓ 儿童必须有成人监护
✓ 学会游泳
✓ 穿救生衣(船上、水上活动)
✓ 了解水域情况
✓ 不要高估自己能力
✓ 避免酒后游泳
✓ 疲劳时不要游泳
✓ 家中泳池:
  - 围栏
  - 监护
  - 移除玩具(防儿童独自下水)
```

## 📝 总结要点

### 急性症状识别

记住危及生命的症状:
- 呼吸困难
- 意识改变
- 休克征象
- 体温异常
- 心跳异常

### 处理原则

1. **安全第一** - 保护自己
2. **评估紧急程度** - ABC原则
3. **呼叫帮助** - 拨打120
4. **提供急救** - 在能力范围内
5. **持续监测** - 直到专业救援

### 必须掌握的技能

- CPR
- 过敏性休克识别和处理
- 中暑快速降温
- 失温复温原则
- 电击安全救援
- 溺水CPR

### 预防意识

- 识别风险
- 采取预防措施
- 准备应急装备
- 学习急救技能
