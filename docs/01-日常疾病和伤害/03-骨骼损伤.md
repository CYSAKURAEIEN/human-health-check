## 骨折

### 骨折类型

**闭合性骨折**: 皮肤完整,骨折未穿出
**开放性骨折**: 骨折端刺穿皮肤,严重!

### 症状识别

明确征象:
- 畸形(异常角度或形状)
- 异常活动(不该动的地方能动)
- 骨擦音或骨擦感

可能征象:
- 剧烈疼痛
- 肿胀
- 淤血
- 功能丧失(无法活动)
- 压痛

### 紧急处理原则

#### 1. 评估情况

```
首先检查:
✓ 伤者意识状态
✓ 呼吸和心跳
✓ 严重出血
✓ 其他危及生命的伤

然后处理骨折
```

#### 2. 不要移动!

```
❌ 禁止:
- 试图复位骨折
- 不必要的移动
- 让伤者站立或行走

✓ 除非:
- 现场有危险
- 必须移动才能获救
```

#### 3. 止血(开放性骨折)

```
⚠️ 开放性骨折:
1. 不要推回突出的骨头
2. 无菌纱布覆盖伤口
3. 压迫止血(小心,不要压在骨折处)
4. 立即呼叫急救
```

#### 4. 固定

**目的**:
- 减轻疼痛
- 防止进一步损伤
- 减少出血
- 防止闭合骨折变为开放骨折

**原则**:
```
1. 原位固定(保持受伤时的位置)
2. 固定上下两个关节
3. 松紧适度
4. 检查远端血液循环
```

### 各部位固定方法

#### 上臂骨折

```
1. 三角巾或宽布条制作悬臂带
2. 前臂屈曲90度
3. 悬挂于胸前
4. 另一布条绕过胸部固定上臂
```

#### 前臂或手腕骨折

```
1. 夹板: 从肘部到手掌
2. 两侧放置硬物:
   - 木板、硬纸板
   - 杂志、报纸卷
3. 布条绑扎(避开骨折处)
4. 三角巾悬吊
```

#### 大腿骨折

```
1. 长夹板: 从腋下到脚跟
2. 短夹板: 从腹股沟到脚跟
3. 两侧固定
4. 多处绑扎:
   - 胸部、腰部、大腿、膝盖、小腿、脚踝
5. 不要移动伤者
6. 等待专业救援
```

#### 小腿骨折

```
1. 夹板: 从大腿到脚跟
2. 包含膝关节和踝关节
3. 可用:
   - 木板
   - 雨伞
   - 健侧腿(绑在一起)
```

#### 锁骨骨折

```
1. 8字绷带固定双肩
2. 或三角巾悬吊患侧上肢
3. 坐位,挺胸
```

#### 肋骨骨折

```
⚠️ 注意气胸、血胸!

处理:
1. 半坐位
2. 宽布条或多头巾固定胸部
3. 呼气末固定
4. 浅呼吸
5. 立即就医

禁止:
❌ 固定过紧(影响呼吸)
```

#### 脊柱骨折(极其危险!)

```
⚠️ 症状:
- 背部剧痛
- 活动受限
- 可能麻木、无力
- 可能大小便失禁

处理:
1. 绝对不要移动!
2. 保持头-颈-背一直线
3. 等待专业救援
4. 硬板担架搬运(至少4人)

除非:
- 现场有生命危险(火灾等)
```

#### 颈椎骨折

```
最危险!可能导致瘫痪或死亡!

固定:
1. 双手固定头部两侧
2. 保持头-颈-身体在一直线
3. 不要屈曲、伸展、旋转
4. 等待专业救援
5. 专业颈托固定
```

### 检查血液循环

固定后必须检查:

```
每15分钟检查一次:

1. 皮肤颜色:
   ✓ 正常: 粉红
   ⚠️ 苍白或青紫: 可能太紧

2. 温度:
   ✓ 正常: 温暖
   ⚠️ 冰冷: 血液循环差

3. 脉搏:
   ✓ 能触及
   ⚠️ 触及不到: 太紧

4. 感觉:
   ✓ 正常
   ⚠️ 麻木或刺痛: 可能压迫神经

5. 活动:
   ✓ 能活动手指/脚趾
   ⚠️ 无法活动: 检查固定

如发现异常: 稍微松开固定
```

### 冰敷

```
✓ 减轻肿胀和疼痛
- 冰袋用布包裹
- 放在骨折部位周围
- 每次15-20分钟
- 每小时一次
❌ 不要直接接触皮肤
```

### ⚠️ 立即就医

所有骨折都应就医!

特别紧急:
- 开放性骨折
- 大量出血
- 严重变形
- 远端麻木、苍白、无脉
- 脊柱或颈椎损伤
- 头部、胸部、骨盆骨折

## 脱臼

### 定义

关节骨端脱离正常位置

### 常见部位

- 肩关节(最常见)
- 肘关节
- 手指
- 髋关节
- 膝盖骨(髌骨)
- 下颌

### 症状

- 关节变形
- 剧烈疼痛
- 肿胀
- 活动受限或无法活动
- 可能伴随麻木

### 处理原则

```
❌ 绝对禁止:
- 自行复位(除非受过专业训练)
- 强行活动

✓ 正确做法:
1. 保持原位
2. 固定关节
3. 冰敷
4. 尽快就医
```

### 固定方法

**肩关节脱臼**:
```
1. 在腋下垫软垫(如毛巾卷)
2. 前臂悬吊于胸前
3. 宽布条固定上臂于躯干
```

**肘关节脱臼**:
```
1. 保持原位(不要强行弯曲或伸直)
2. 软垫支撑
3. 三角巾悬吊或夹板固定
```

**手指脱臼**:
```
1. 夹板固定在原位
2. 冰敷
3. 就医
```

### 复位

⚠️ 仅由医生操作!

**例外**: 在野外偏远地区,无法及时就医,且:
- 远离医疗点数小时以上
- 受过专业训练
- 肩关节脱臼且有经验

### 脱臼 vs 骨折

有时难以区分,处理相同:
- 固定
- 冰敷
- 就医

## 扭伤

### 分级

**I度(轻度)**:
- 轻微疼痛和肿胀
- 韧带拉伸,无撕裂
- 能行走

**II度(中度)**:
- 明显疼痛和肿胀
- 韧带部分撕裂
- 关节不稳
- 行走困难

**III度(重度)**:
- 剧烈疼痛(后可能减轻)
- 严重肿胀和淤血
- 韧带完全断裂
- 关节明显不稳
- 无法承重

### 扭伤 vs 骨折

难以区分时,按骨折处理!

**考虑骨折如果**:
- 剧烈疼痛
- 严重变形
- 无法活动
- 骨性压痛

### RICE处理(I-II度)

#### R - Rest(休息)

```
- 避免使用受伤关节
- 前48-72小时尤其重要
- 使用拐杖(下肢扭伤)
```

#### I - Ice(冰敷)

```
前48小时:
- 每2小时一次
- 每次15-20分钟
- 冰袋用毛巾包裹

作用:
- 减轻肿胀
- 减少疼痛
- 减少内出血
```

#### C - Compression(加压包扎)

```
方法:
- 弹性绷带(8字包扎)
- 从远端向近端
- 松紧适中

检查:
- 每小时检查血液循环
- 脚趾/手指温暖、颜色正常
- 如肿胀加重,松开重新包扎
```

#### E - Elevation(抬高)

```
- 高于心脏水平
- 尽可能多的时间
- 休息和睡觉时
```

### 48-72小时后

**热敷**:
```
- 促进血液循环
- 加速愈合
- 温热毛巾或热水袋
- 每次15-20分钟
```

**轻柔活动**:
```
- 逐渐恢复活动范围
- 拉伸运动
- 不要疼痛
```

**加强锻炼**:
```
- 1-2周后(视恢复情况)
- 力量训练
- 平衡训练
- 预防再次扭伤
```

### 药物

**口服**:
- 布洛芬(抗炎止痛)
- 对乙酰氨基酚(止痛)

**外用**:
- 消肿止痛喷雾
- 跌打损伤膏

### 恢复时间

- **I度**: 1-2周
- **II度**: 3-6周
- **III度**: 数月,可能需手术

### 预防再次扭伤

```
✓ 完全康复后再恢复运动
✓ 充分热身
✓ 使用护具(护踝、护腕)
✓ 力量和平衡训练
✓ 选择合适的鞋
✓ 注意地面情况
```

### ⚠️ 就医指征

立即就医如果:
- III度扭伤
- 无法承重或行走
- 严重变形
- 麻木或刺痛
- 关节明显不稳
- 肿胀严重
- 自行处理3天无改善

## 📝 总结要点

### 骨骼损伤ABC

**A - Assessment(评估)**
- 评估伤情
- 检查生命体征
- 判断骨折、脱臼或扭伤

**B - Be still(制动)**
- 不要移动
- 原位固定
- 保护受伤部位

**C - Call for help(寻求帮助)**
- 严重伤及时就医
- 脊柱伤等待专业救援

### 记住

```
✓ 严重伤不要移动
✓ 固定上下关节
✓ 检查血液循环
✓ 冰敷减轻肿胀
✓ 及时就医

❌ 不要复位
❌ 不要推回骨头
❌ 不要活动受伤部位
```

### 何时立即就医

- 所有骨折
- 严重脱臼
- III度扭伤
- 脊柱或颈椎伤
- 变形明显
- 疼痛剧烈
- 麻木或血循环差
