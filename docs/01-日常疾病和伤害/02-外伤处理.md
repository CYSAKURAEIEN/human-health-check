## 割伤与擦伤

### 伤口分类

**轻微伤口**:
- 浅表,出血少
- 边缘整齐
- 长度<2厘米
- 可自行处理

**需就医的伤口**:
- 深度超过0.5厘米
- 长度>2厘米
- 出血严重不止
- 边缘不整齐
- 脏污严重
- 特殊部位(面部、关节、手掌)

### 处理步骤

#### 1. 止血

```
轻微出血:
1. 清水冲洗
2. 干净纱布按压5-10分钟
3. 抬高伤处(高于心脏)

严重出血:
1. 立即按压止血
2. 持续按压至少10分钟
3. 必要时使用止血点
4. 呼叫急救
```

#### 2. 清洁伤口

```
步骤:
1. 洗手(或戴医用手套)
2. 流动清水冲洗伤口3-5分钟
3. 去除明显污物
4. 温和肥皂水清洁周围皮肤
5. 再次清水冲洗

注意:
❌ 不要用酒精直接冲洗伤口(疼痛且损伤组织)
❌ 不要用棉花直接接触伤口(纤维残留)
✓ 可用碘伏消毒
```

#### 3. 消毒

```
推荐消毒剂:
- 碘伏(首选):从伤口中心向外涂抹
- 双氧水:清除污垢,有气泡是正常现象
- 75%酒精:仅用于周围皮肤,避免直接接触伤口
```

#### 4. 包扎

```
小伤口:
- 创可贴(每天更换)
- 保持干燥

大伤口:
1. 无菌纱布覆盖
2. 医用胶带固定
3. 每1-2天更换
4. 观察感染迹象
```

### 换药步骤

```
1. 洗手
2. 轻轻揭开旧敷料(先用温水浸湿)
3. 检查伤口
4. 清水轻轻冲洗
5. 重新消毒
6. 覆盖新的无菌纱布
7. 固定
```

### 擦伤特殊处理

**"脏擦伤"**(如摔倒在地):
```
1. 细致清洗非常重要!
2. 用软毛刷轻刷去除嵌入的沙土
3. 如无法清除干净,必须就医
4. 防止"外伤性文身"和感染
```

### 伤口愈合阶段

**正常愈合**:
- 第1-2天:轻微肿胀,轻微疼痛
- 第3-7天:结痂,瘙痒
- 第7-14天:痂皮脱落,新皮肤生成

**注意**:
- 不要撕痂(会留疤)
- 保持适度湿润(促进愈合)
- 避免阳光直射

### ⚠️ 感染迹象(立即就医)

- 红肿扩大
- 疼痛加剧
- 发热、发烫
- 黄色或绿色脓液
- 红线从伤口向外延伸
- 发烧

### 破伤风预防

**高风险伤口**:
- 脏污伤口
- 深刺伤
- 动物咬伤
- 泥土污染
- 超过6小时未处理

**破伤风疫苗**:
```
- 完整免疫者:5年内受伤无需加强
- 5-10年:根据伤口情况决定
- >10年或未接种:需注射破伤风免疫球蛋白
```

## 烫伤与烧伤

### 烧伤程度分类

#### 一度烧伤(浅表)
- **症状**: 皮肤发红、疼痛、无水疱
- **影响**: 仅表皮层
- **愈合**: 3-5天,不留疤
- **例子**: 轻微日晒伤

#### 二度烧伤(中度)
- **浅二度**:
  - 红肿、剧痛、大水疱
  - 基底红润湿润
  - 1-2周愈合,可能轻微疤痕
- **深二度**:
  - 基底苍白或红白相间
  - 疼痛较轻(神经受损)
  - 3-4周愈合,可能明显疤痕

#### 三度烧伤(重度)
- **症状**: 皮肤焦黑或蜡白、无痛(神经完全破坏)
- **影响**: 全层皮肤及皮下组织
- **愈合**: 需植皮,疤痕明显
- **必须就医**: 这是严重烧伤

### 烧伤面积评估

**手掌法则**:
- 伤者自己的手掌(包括手指)≈1%体表面积

**严重烧伤**(立即就医):
- 成人: 二度烧伤>10%体表面积
- 儿童: 二度烧伤>5%体表面积
- 任何三度烧伤
- 面部、手、脚、生殖器、关节部位烧伤

### 紧急处理:"冲、脱、泡、盖、送"

#### 1. 冲 - 冷水冲洗

```
时间: 至少15-20分钟
水温: 15-25°C(自来水即可)
方法: 轻柔流动冷水冲洗

作用:
✓ 降低局部温度
✓ 减轻疼痛
✓ 减少组织进一步损伤

禁忌:
❌ 不要用冰块直接敷(可能冻伤)
❌ 大面积烧伤不要长时间冷水浸泡(防失温)
❌ 不要用酱油、牙膏等涂抹
```

#### 2. 脱 - 小心脱掉衣物

```
✓ 在冷水中小心剪开衣物
✓ 轻轻脱除
✓ 如衣物粘连,不要强行撕脱
✓ 保留粘连部分,就医处理
✓ 去除戒指、手表等(防肿胀后嵌顿)
```

#### 3. 泡 - 继续冷水浸泡

```
时间: 10-30分钟
或直到疼痛明显减轻

注意:
- 面积大时不要过久(防失温)
- 保持水温
```

#### 4. 盖 - 覆盖保护

```
用料:
✓ 干净湿润的纱布或布料
✓ 保鲜膜(轻轻覆盖,不要缠绕)

禁止:
❌ 涂抹任何药膏、油脂、酱油、牙膏
❌ 刺破水疱
❌ 涂抹有色物质(影响医生判断)
```

#### 5. 送 - 尽快就医

```
需就医情况:
- 二度烧伤面积较大
- 三度烧伤
- 面部、手、脚、生殖器、关节烧伤
- 儿童或老人烧伤
- 化学或电烧伤
```

### 小面积轻度烫伤家庭处理

**条件**: 一度或小面积浅二度(直径<2厘米)

**处理**:
```
1. 充分冷水冲洗/浸泡
2. 清洁干燥后涂抹:
   - 烧伤膏(磺胺嘧啶银)
   - 芦荟胶
3. 覆盖无菌纱布(可选)
4. 每天换药1-2次
```

**水疱处理**:
```
小水疱(<5mm):
- 不要刺破
- 自然吸收

大水疱(>5mm):
- 建议就医处理
- 如必须自行处理:
  1. 消毒水疱及周围
  2. 无菌针头刺破边缘
  3. 轻压排出液体
  4. 保留水疱皮(保护作用)
  5. 消毒后覆盖
```

### 特殊烧伤

#### 化学烧伤

```
立即处理:
1. 迅速脱去污染衣物
2. 大量清水持续冲洗30分钟以上
3. 不要试图中和化学物质
4. 立即就医,带上化学品信息

注意:
- 生石灰:先刷去,再冲水
- 浓硫酸:先用干布吸,再冲水
```

#### 电烧伤

```
1. 切断电源(不要直接触碰伤者)
2. 检查心跳呼吸
3. 必要时CPR
4. 所有电烧伤都应就医(内部损伤可能严重)
```

### 预防感染

```
✓ 保持伤口清洁
✓ 定期换药
✓ 观察感染迹象
✓ 避免抓挠
✓ 充足营养(蛋白质、维生素C)
```

### ❌ 常见错误做法

- 涂抹牙膏、酱油、醋、酒精
- 涂抹油脂、香油
- 使用冰块直接敷
- 撒盐、糖等
- 涂抹有色药物

这些做法可能:
- 加重损伤
- 增加感染风险
- 影响医生判断
- 增加疤痕

## 冻伤

### 冻伤程度

**一度**: 皮肤红肿、刺痛
**二度**: 水疱形成
**三度**: 组织坏死,变黑
**四度**: 深达骨骼

### 易受冻部位

- 手指、脚趾
- 鼻子、耳朵
- 脸颊

### 复温处理

#### 轻度冻伤(一度)

```
1. 转移到温暖环境
2. 脱去湿冷衣物
3. 温水浸泡复温:
   - 水温: 37-40°C(温热,不烫手)
   - 时间: 15-30分钟
   - 直到皮肤恢复红润、柔软
4. 轻轻拍干
5. 保暖

禁止:
❌ 热水烫
❌ 火烤
❌ 摩擦搓揉
❌ 冻伤后行走(脚部冻伤)
```

#### 严重冻伤(二度及以上)

```
1. 保持冻伤部位不动
2. 包裹保暖(但不要加热)
3. 立即就医
4. 不要试图复温(可能造成更大伤害)
```

### 预防

```
✓ 多层穿衣
✓ 保持干燥
✓ 覆盖暴露部位
✓ 避免紧身衣物
✓ 定期活动肢体
✓ 避免酒精和吸烟
```

## 咬伤

### 动物咬伤

#### 即时处理

```
1. 冲洗(重要!):
   - 立即用流动清水冲洗15分钟以上
   - 肥皂水交替冲洗
   - 深度冲洗伤口
2. 消毒:
   - 碘伏或75%酒精
3. 覆盖无菌纱布
4. 尽快就医
```

#### 狂犬病预防(关键!)

**高风险动物**: 狗、猫、蝙蝠、狐狸、狼等

**所有咬伤都应考虑狂犬病风险!**

**疫苗接种**:
```
时间: 越早越好,24小时内最佳
程序: 通常5针
  - 0天(当天)
  - 3天
  - 7天
  - 14天
  - 28天

严重咬伤: 需注射狂犬病免疫球蛋白
```

**何时必须接种**:
- 任何被动物咬伤、抓伤
- 皮肤破损被动物舔
- 粘膜被动物体液污染

#### 伤口处理原则

```
❌ 不要缝合(除非特殊部位且咬伤浅表)
✓ 保持开放引流
✓ 密切观察感染
✓ 可能需要抗生素
```

### 人咬伤

**特点**: 感染风险高(口腔细菌多)

**处理**:
```
1. 彻底冲洗
2. 消毒
3. 必须就医
4. 预防性抗生素
5. 乙肝、艾滋病暴露评估
```

### 蛇咬伤

#### 识别毒蛇

**毒蛇特征**:
- 三角形头部
- 粗短身体
- 单排尾下鳞
- 咬痕: 2个深牙痕

**无毒蛇**:
- 椭圆形头部
- 细长身体
- 双排尾下鳞
- 咬痕: 多个浅齿痕

#### 紧急处理

```
✓ 做:
1. 立即远离蛇
2. 保持冷静,减少活动
3. 移除饰物(防肿胀)
4. 伤口低于心脏
5. 记住蛇的特征(拍照,但不要抓蛇)
6. 立即就医

❌ 不要:
- 切开伤口
- 吸吮毒液
- 冰敷
- 喝酒
- 奔跑
- 使用止血带(除非极端情况)
```

**固定**:
```
如果是四肢:
1. 用弹性绷带包扎(不要太紧)
2. 固定肢体(夹板)
3. 减少毒液扩散
```

### 昆虫叮咬

#### 一般处理

```
1. 移除蜇刺(蜜蜂):
   - 用卡片刮除
   - 不要捏取(会挤出更多毒液)
2. 清水清洗
3. 冰敷15-20分钟(减轻肿胀)
4. 抗组胺药膏
5. 口服抗过敏药(如瘙痒严重)
```

#### 过敏反应监测

**轻度**: 局部肿胀、瘙痒
**中度**: 大面积肿胀

**严重(过敏性休克)**:
⚠️ 立即就医:
- 呼吸困难
- 喉咙肿胀
- 全身荨麻疹
- 头晕、低血压
- 心跳加快

**处理**: 立即使用肾上腺素笔(如有)+ 呼叫急救

## 刺伤

### 浅表刺伤

**小刺(木刺、玻璃碎片)**:

```
1. 清洁:
   - 肥皂水洗手和周围皮肤
2. 去除:
   - 用镊子(先消毒)
   - 顺着刺入方向拔出
   - 如看不见,用针轻轻挑开皮肤
3. 消毒伤口
4. 观察感染迹象
```

**无法取出**:
- 就医处理
- 不要强行挖出(可能推得更深)

### 深刺伤

**特点**:
- 伤口小,深度深
- 出血少
- 感染和破伤风风险高

**⚠️ 不要拔出插入物**:
```
如果有物体刺入体内:
1. 不要拔出!
2. 固定异物
3. 包扎周围
4. 立即就医
```

**就医处理**:
```
1. 清洗消毒
2. 检查伤口深度
3. 可能需要X光(查异物残留)
4. 抗生素
5. 破伤风预防
```

### 高风险刺伤

立即就医:
- 被生锈金属刺伤
- 被脏污物品刺伤
- 动物骨刺、牙齿
- 刺入关节、眼睛等特殊部位
- 无法完全取出异物

## 挫伤与淤血

### 症状

- 疼痛
- 肿胀
- 淤青(皮下出血)
- 功能受限

### RICE原则

#### R - Rest(休息)
```
- 避免使用受伤部位
- 前48小时尤其重要
```

#### I - Ice(冰敷)
```
时间: 受伤后48小时内
频率: 每2-3小时一次
时长: 每次15-20分钟
方法:
  - 冰袋用毛巾包裹
  - 不要直接接触皮肤
  - 可用冰冻蔬菜袋替代
作用:
  - 减轻肿胀
  - 减少疼痛
  - 减少内出血
```

#### C - Compression(加压包扎)
```
方法:
  - 弹性绷带包扎
  - 松紧适中(不影响血液循环)
  - 检查: 手指/脚趾是否温暖、颜色正常
作用:
  - 减少肿胀
  - 支撑保护
```

#### E - Elevation(抬高)
```
位置: 高于心脏水平
时间: 尽可能多的时间
尤其: 休息和睡觉时
作用:
  - 减少肿胀
  - 促进静脉回流
```

### 48小时后护理

**热敷**:
```
时间: 受伤48小时后
方法:
  - 温热湿毛巾
  - 热水袋(用毛巾包裹)
  - 温水浸泡
时长: 每次15-20分钟
作用:
  - 促进血液循环
  - 加速淤血吸收
  - 放松肌肉
```

**轻柔活动**:
```
- 轻微拉伸
- 逐渐恢复活动
- 不要过度
```

### 药物治疗

**外用**:
- 跌打损伤药膏
- 消肿止痛凝胶
- 活血化瘀药物

**口服**:
- 布洛芬(抗炎止痛)
- 对乙酰氨基酚(止痛)

### 淤血消散

**正常过程**:
- 初期: 红色
- 1-2天: 青紫色
- 3-5天: 绿色
- 5-7天: 黄褐色
- 1-2周: 完全消退

**加速消散**:
- 热敷(48小时后)
- 按摩(轻柔)
- 活血化瘀药
- 维生素K

### ⚠️ 就医指征

立即就医如果:
- 疼痛剧烈
- 肿胀严重
- 无法活动或负重
- 变形
- 麻木或刺痛
- 淤血异常扩大
- 伴随开放性伤口

---

## 📝 总结要点

### 伤口处理"五步法"

1. **止血**: 压迫止血
2. **清洗**: 流动清水冲洗
3. **消毒**: 碘伏或酒精
4. **包扎**: 无菌纱布覆盖
5. **观察**: 注意感染迹象

### 何时必须就医

- 伤口深、大、脏
- 出血不止
- 特殊部位受伤
- 动物咬伤
- 感染迹象
- 异物无法取出
- 疼痛剧烈
- 功能受限

### 家庭急救包

**外伤用品**:
- 各种尺寸创可贴
- 无菌纱布
- 医用胶带
- 弹性绷带
- 三角巾
- 医用手套
- 剪刀、镊子
- 碘伏、酒精
- 烧伤膏
- 抗菌软膏
